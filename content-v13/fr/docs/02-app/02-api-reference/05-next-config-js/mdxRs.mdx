---
title: mdxRs
description: Utilisez le nouveau compilateur Rust pour compiler les fichiers MDX dans le routeur d'application (App Router).
---

Pour une utilisation avec `@next/mdx`. Compilez les fichiers MDX en utilisant le nouveau compilateur Rust.

```js filename="next.config.js"
const withMDX = require('@next/mdx')()

/** @type {import('next').NextConfig} */
const nextConfig = {
  pageExtensions: ['ts', 'tsx', 'mdx'],
  experimental: {
    mdxRs: true,
  },
}

module.exports = withMDX(nextConfig)
```
