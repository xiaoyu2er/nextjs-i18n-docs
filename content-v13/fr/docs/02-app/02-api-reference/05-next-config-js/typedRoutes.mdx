---

Support expérimental pour les [liens typés statiquement](/docs/app/building-your-application/configuring/typescript#statically-typed-links). Cette fonctionnalité nécessite l'utilisation du routeur App ainsi que TypeScript dans votre projet.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
