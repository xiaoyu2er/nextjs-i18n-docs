---
title: Version 10
description: Mettez à jour votre application Next.js de la version 9 à la version 10.
---

Il n'y a eu aucun changement cassant entre les versions 9 et 10.

Pour effectuer la mise à jour vers la version 10, exécutez la commande suivante :

```bash filename="Terminal"
npm install next@10

yarn add next@10
```
