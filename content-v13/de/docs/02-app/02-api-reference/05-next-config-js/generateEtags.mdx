Next.js generiert standardmäßig [ETags](https://en.wikipedia.org/wiki/HTTP_ETag) für jede Seite. Je nach Cache-Strategie kann es sinnvoll sein, die ETag-Generierung für HTML-Seiten zu deaktivieren.

Öffnen Sie `next.config.js` und deaktivieren Sie die Option `generateEtags`:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
