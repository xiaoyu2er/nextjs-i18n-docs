Next.js по умолчанию генерирует [etags](https://en.wikipedia.org/wiki/HTTP_ETag) для каждой страницы. В зависимости от вашей стратегии кэширования вы можете отключить генерацию etags для HTML-страниц.

Откройте `next.config.js` и отключите опцию `generateEtags`:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
