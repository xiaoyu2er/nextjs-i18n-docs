До версии Next.js 13 роутер страниц (Pages Router) был основным способом создания маршрутов в Next.js. Он использовал интуитивно понятную файловую систему маршрутизации, где каждый файл соответствовал определённому маршруту. Роутер страниц по-прежнему поддерживается в новых версиях Next.js, но мы рекомендуем перейти на новый [Роутер приложений (App Router)](/docs/app), чтобы использовать последние возможности React.

Используйте этот раздел документации для существующих приложений, использующих роутер страниц (Pages Router).
