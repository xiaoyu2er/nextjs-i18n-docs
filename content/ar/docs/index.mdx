## ما هي Next.js؟

Next.js هو إطار عمل React لتطبيقات الويب الشاملة. تستخدم مكونات React لبناء واجهات المستخدم، و Next.js للميزات الإضافية والتحسينات.

كما أنه يضبط الأدوات ذات المستوى الأدنى تلقائياً مثل أدوات التجميع والمترجمات. يمكنك بدلاً من ذلك التركيز على بناء منتجك وإطلاقه بسرعة.

سواء كنت مطوراً منفرداً أو عضواً في فريق أكبر، يمكن لـ Next.js مساعدتك في بناء تطبيقات React تفاعلية وديناميكية وسريعة.

## كيفية استخدام المستندات

هذه المستندات منظمة في 3 أقسام:

- [بدء](/docs/app/getting-started): دروس تعليمية خطوة بخطوة لمساعدتك على إنشاء تطبيق جديد وتعلم الميزات الأساسية لـ Next.js.
- [أدلة](/docs/app/guides): دروس تعليمية حول حالات استخدام محددة، اختر ما يناسبك.
- [مرجع واجهة برمجة التطبيقات (API Reference)](/docs/app/api-reference): مرجع تقني مفصل لكل ميزة.

استخدم الشريط الجانبي للتنقل عبر الأقسام، أو ابحث (`Ctrl+K` أو `Cmd+K`) لإيجاد صفحة بسرعة.

## موجه التطبيق (App Router) وموجه الصفحات (Pages Router)

يحتوي Next.js على موجهين مختلفين:

- **موجه التطبيق (App Router)**: هو الموجه الأحدث الذي يدعم ميزات React الجديدة مثل المكونات الخادم (Server Components).
- **موجه الصفحات (Pages Router)**: الموجه الأصلي، وهو لا يزال مدعوماً ويتم تحسينه.

في أعلى الشريط الجانبي، ستلاحظ قائمة منسدلة تتيح لك التبديل بين مستندات [موجه التطبيق (App Router)](/docs/app) و [موجه الصفحات (Pages Router)](/docs/pages).

### معالجة إصدارات React

يتم التعامل مع إصدارات React بشكل مختلف في موجه التطبيق (App Router) وموجه الصفحات (Pages Router):

- **موجه التطبيق (App Router)**: يستخدم إصدارات Canary لـ React (React canary releases) مدمجة، والتي تتضمن جميع تغييرات React 19 المستقرة، بالإضافة إلى الميزات الأحدث التي يتم التحقق منها في الأطر، قبل إصدار React الجديد.

- **موجه الصفحات (Pages Router)**: يستخدم إصدار React المثبت في `package.json` الخاص بمشروعك.

يضمن هذا النهج أن تعمل ميزات React الجديدة بشكل موثوق في موجه التطبيق (App Router) مع الحفاظ على التوافق العكسي لتطبيقات موجه الصفحات (Pages Router) الحالية.

## المعرفة المسبقة

تفترض وثائقنا بعض الإلمام بتطوير الويب. قبل البدء، سيكون من المفيد أن تكون على دراية بـ:

- HTML
- CSS
- JavaScript
- React

إذا كنت جديدًا على React أو تحتاج إلى تحديث معلوماتك، نوصي بالبدء مع [دورة أساسيات React](/learn/react-foundations)، ودورة [أساسيات Next.js](/learn/dashboard-app) التي تمكنك من بناء تطبيق أثناء التعلم.

## إمكانية الوصول

للحصول على أفضل تجربة عند استخدام قارئ الشاشة، نوصي باستخدام Firefox و NVDA، أو Safari و VoiceOver.
