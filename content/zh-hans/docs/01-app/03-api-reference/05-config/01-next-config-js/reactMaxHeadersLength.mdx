---
title: reactMaxHeadersLength
description: React 发出并添加到响应中的标头的最大长度。
---

在预渲染 (pre-rendering) 期间，React 可以生成响应头 (headers)，并将其添加到响应 (response) 中。这些头信息可用于提高性能，使浏览器能够预加载 (preload) 资源，例如字体 (fonts)、脚本 (scripts) 和样式表 (stylesheets)。默认值为 `6000`，但你可以通过在 `next.config.js` 中配置 `reactMaxHeadersLength` 选项来覆盖此值：

```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **要知道**：此选项仅在 App Router 中可用。


根据浏览器和服务器之间的代理类型，标头可能会被截断。例如，如果您使用的是不支持长标头的反向代理，则应该设置较低的值以确保标头不会被截断。
