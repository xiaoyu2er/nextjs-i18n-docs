`viewTransition` 是一个实验性标志，可在 React 中启用新的 [视图转换 API (View Transitions API)](https://developer.mozilla.org/en-US/docs/Web/API/View_Transition_API)。此 API 允许您利用原生的浏览器视图转换 API，在 UI 状态之间创建无缝过渡。

要启用此功能，你需要在 `next.config.js` 文件中将 `viewTransition` 属性设置为 `true`。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    viewTransition: true,
  },
}

module.exports = nextConfig
```

> 重要提示：`<ViewTransition>` 组件已可用在 React 的 Canary 发布渠道中。
> 仅开启 `experimental.viewTransition` 即可启用与 Next.js 功能的深度集成，例如自动
> [添加过渡类型](https://react.dev/reference/react/addTransitionType) 以用于导航。尚未实现 Next.js 特定的过渡类型。

## 使用方法

您可以在应用中从 React 导入 [`<ViewTransition>` 组件](https://react.dev/reference/react/ViewTransition)：

```jsx
import { ViewTransition } from 'react'
```
