---
title: typedRoutes
description: 启用对静态类型链接的支持。
---

> **注意**：此选项已标记为稳定，因此请使用 `typedRoutes` 代替 `experimental.typedRoutes`。

支持 [静态类型链接](/docs/app/api-reference/config/typescript#statically-typed-links)。此功能要求您的项目使用 TypeScript。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  typedRoutes: true,
}

module.exports = nextConfig
```

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  typedRoutes: true,
}

module.exports = nextConfig
```
