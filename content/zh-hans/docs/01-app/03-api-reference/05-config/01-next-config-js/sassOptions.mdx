---
title: sassOptions
description: 配置 Sass 选项。
---

`sassOptions` 允许您配置 Sass 编译器。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const sassOptions = {
  additionalData: `
    $var: red;
  `,
}

const nextConfig: NextConfig = {
  sassOptions: {
    ...sassOptions,
    implementation: 'sass-embedded',
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */

const sassOptions = {
  additionalData: `
    $var: red;
  `,
}

const nextConfig = {
  sassOptions: {
    ...sassOptions,
    implementation: 'sass-embedded',
  },
}

module.exports = nextConfig
```

> **小知识：**

> - `sassOptions` 在 `implementation` 之外未定义类型，因为 Next.js 不维护其他可能的属性。
> - 用于定义自定义 Sass 函数的 `functions` 属性仅支持 webpack。使用 Turbopack 时，无法使用自定义 Sass 函数，因为 Turbopack 基于 Rust 的架构无法直接执行通过此选项传入的 JavaScript 函数。
