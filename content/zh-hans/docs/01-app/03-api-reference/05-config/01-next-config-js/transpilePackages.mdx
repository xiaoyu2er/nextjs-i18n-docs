Next.js 可以自动转换和打包来自本地包（如 monorepos）或外部依赖（`node_modules`）的依赖项。此功能取代了 `next-transpile-modules` 包。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['package-name'],
}

module.exports = nextConfig
```
