某些包可以导出数百或数千个模块，这可能会导致开发和生产环境中的性能问题。

将包添加到 `experimental.optimizePackageImports` 中，将只加载您实际使用的模块，同时仍为您提供编写包含多个具名导出的导入语句的便利。

```js filename="next.config.js"
module.exports = {
  experimental: {
    optimizePackageImports: ['package-name'],
  },
}
```

```js filename="next.config.js"
module.exports = {
  experimental: {
    optimizePackageImports: ['package-name'],
  },
}
```
