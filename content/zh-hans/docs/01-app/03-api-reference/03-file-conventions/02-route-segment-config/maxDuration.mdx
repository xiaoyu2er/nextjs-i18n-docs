使用 `maxDuration` 选项，你可以为路由段中的服务端逻辑设置最大执行时间（以秒为单位）。部署平台可以使用 Next.js 构建输出中的 `maxDuration` 来添加特定的执行限制。

```tsx filename="layout.tsx | page.tsx | route.ts" switcher
export const maxDuration = 5
```

```js filename="layout.js | page.js | route.js" switcher
export const maxDuration = 5
```

## Server Actions

如果使用 [服务端动作 (Server Actions)](/docs/app/getting-started/mutating-data)，请在页面级别设置 `maxDuration`，以更改页面上使用的所有服务端动作 (Server Actions) 的默认超时。
