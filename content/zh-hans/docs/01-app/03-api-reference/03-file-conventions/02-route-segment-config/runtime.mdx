---
title: 运行时 (runtime)
description: 运行时路由段配置选项的 API 参考。
---

该 `runtime` 选项允许您选择用于渲染路由的 JavaScript 运行时。

```tsx filename="layout.tsx | page.tsx | route.ts" switcher
export const runtime = 'nodejs'
// 'nodejs' | 'edge'
```

```js filename="layout.js | page.js | route.js" switcher
export const runtime = 'nodejs'
// 'nodejs' | 'edge'
```

- **`'nodejs'`**（默认）
- **`'edge'`**

> **值得了解**:
>
> - 使用 `runtime: 'edge'` 在 Cache 组件 (Cache Components) 中 **不受支持**。
> - 此选项无法在 [Proxy](/docs/app/api-reference/file-conventions/proxy) 中使用。
