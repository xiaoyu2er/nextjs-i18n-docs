---
title: dynamicParams
description: dynamicParams 路由段配置选项的 API 参考。
---

`dynamicParams` 选项允许您控制在访问未通过 [生成静态参数 (generateStaticParams)](/docs/app/api-reference/functions/generate-static-params) 生成的动态段时发生什么情况。

```tsx filename="layout.tsx | page.tsx" switcher
export const dynamicParams = true // true | false
```

```js filename="layout.js | page.js | route.js" switcher
export const dynamicParams = true // true | false
```

- **`true`** (默认): 未包含在 `generateStaticParams` 中的动态路由段将在请求时生成。
- **`false`**: 未包含在 `generateStaticParams` 中的动态路由段将返回 404。

> **小提示**:
>
> - 此选项替换了 `pages` 目录中 `getStaticPaths` 的 `fallback: true | false | blocking` 选项。
> - `dynamicParams` 在启用 [缓存组件 (Cache Components)](/docs/app/api-reference/config/next-config-js/cacheComponents) 时不可用。
