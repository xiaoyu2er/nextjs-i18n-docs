`refresh` 允许你在 [服务器操作 (Server Action)](/docs/app/getting-started/mutating-data) 内部刷新客户端路由器。

## 使用方法

`refresh` **仅**能调用自服务器操作 (Server Actions) 内部。它不能用于路由处理器 (Route Handlers)、客户端组件 (Client Components) 或其他任何上下文。

## 参数

```tsx
refresh(): void;
```

## 返回

`refresh` 不返回值。

## 示例

```ts filename="app/actions.ts" switcher
'use server'

import { refresh } from 'next/cache'

export async function createPost(formData: FormData) {
  const title = formData.get('title')
  const content = formData.get('content')

  // Create the post in your database
  const post = await db.post.create({
    data: { title, content },
  })

  refresh()
}
```

### 在服务器操作 (Server Actions) 外部使用时报错

```ts filename="app/api/posts/route.ts" switcher
import { refresh } from 'next/cache'

export async function POST() {
  // This will throw an error
  refresh()
}
```
