## 最新版本

要更新到最新版本的 Next.js，您可以使用 `upgrade` 命令：

```bash package="pnpm"
pnpm next upgrade
```

```bash package="npm"
npx next upgrade
```

```bash package="yarn"
yarn next upgrade
```

```bash package="bun"
bunx next upgrade
```

Next.js 16.1.0 之前的版本不支持 `upgrade` 命令，需要使用单独的包：

```bash filename="Terminal"
npx @next/codemod@canary upgrade latest
```

如果您更喜欢手动升级，请安装最新的 Next.js 和 React 版本：

```bash package="pnpm"
pnpm i next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash package="npm"
npm i next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash package="yarn"
yarn add next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash package="bun"
bun add next@latest react@latest react-dom@latest eslint-config-next@latest
```

## Canary 版本

要更新到最新的 canary 版本，请确保您使用的是最新版本的 Next.js，并且一切运行正常。然后，运行以下命令：

```bash package="pnpm"
pnpm add next@canary
```

```bash package="npm"
npm i next@canary
```

```bash package="yarn"
yarn add next@canary
```

```bash package="bun"
bun add next@canary
```

### Canary 版本中的功能

以下功能当前在 canary 版本中可用：

**身份验证**：

- [`forbidden`](/docs/app/api-reference/functions/forbidden)

- [`unauthorized.js`](/docs/app/api-reference/file-conventions/unauthorized)
- [`authInterrupts`](/docs/app/api-reference/config/next-config-js/authInterrupts)
