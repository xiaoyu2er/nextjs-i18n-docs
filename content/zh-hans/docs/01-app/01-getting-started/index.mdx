欢迎来到 Next.js 文档！

这个 **开始使用** 部分将帮助您创建您的第一个 Next.js 应用程序，并学习您将在每个项目中使用的核心功能。

## 前提知识

我们的文档假定您对 Web 开发有一定了解。在开始之前，如果您对以下内容感到熟悉会有所帮助：

- HTML
- CSS
- JavaScript
- React

如果您是 React 新手或需要复习，我们建议从我们的 [React 基础课程](/learn/react-foundations) 开始，以及 [Next.js 基础课程](/learn/dashboard-app)，后者会让您在学习的同时构建一个应用程序。
