title: 如何升级至版本 10
nav_title: 版本 10
description: 将您的 Next.js 应用从版本 9 升级至版本 10。
---

在版本 9 和版本 10 之间没有破坏性更改 (breaking changes)。

要升级到版本 10，请运行以下命令：

```bash filename="Terminal"
npm i next@10
```

```bash filename="Terminal"
yarn add next@10
```

```bash filename="Terminal"
pnpm up next@10
```

```bash filename="Terminal"
bun add next@10
```

> **须知：** 如果您使用 TypeScript，请确保同时升级 `@types/react` 和 `@types/react-dom` 至对应版本。
