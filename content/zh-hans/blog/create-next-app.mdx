我们很高兴今天推出新的 Create Next App。

Create Next App 一条命令即可设置由 Next.js 驱动的现代化 React 应用程序。

要开始，您只需运行：

终端

```
npx create-next-app
```

Create Next App 已完全重新构建，以提供尽可能最佳的开发体验：

*   **交互体验**: 运行 `npx create-next-app`（无参数）现在将启动交互式体验，引导您完成项目设置。
*   **零依赖**: 初始化项目现在只需**一秒钟**。Create Next App 没有依赖项，安装仅需 **604 kB**。在我们优化之前，上一个版本是 **5.38 MB**。减少了超过 **4.7 MB**!
*   **离线支持**:  Create Next App 将自动检测到您处于离线状态，并使用本地包缓存引导您的项目。
*   **新默认项目模板**:  Create Next App 使用新的项目模板，专为现代化的 Next.js 应用程序设计。由于 Create Next App 现在与 Next.js 本身一起维护，此模板将始终与最新的 Next.js 版本同步！
*   **示例支持**:  Create Next App 可以使用 [Next.js 示例集合](https://github.com/vercel/next.js/tree/canary/examples) 中的示例来引导您的应用程序（例如 `npx create-next-app --example api-routes`）。
*   **已测试**: 该包是 Next.js monorepo 的一部分，并使用与 Next.js 本身相同的集成测试套件进行测试，确保每个版本都按预期工作。

Create Next App 以前是一个 [社区维护](https://open.segment.com/create-next-app/) 项目，但我们觉得整理 Next.js 的第一印象很重要。特别是我们推荐该 [Next.js 示例集合](https://github.com/vercel/next.js/tree/canary/examples)。
