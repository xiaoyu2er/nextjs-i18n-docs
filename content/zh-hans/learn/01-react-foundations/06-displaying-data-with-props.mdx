到目前为止，如果您重复使用 <Header /> 组件，两次显示的将是相同的内容。

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
function HomePage() {
  return (
    <div>
      <Header />
      <Header />
    </div>
  );
}
```

但是，如果你想传入不同的文本，或者因为从外部源（external source）获取数据而提前不知道信息怎么办？

普通 HTML 元素具有属性，你可以用来传递改变元素行为的信息。例如，更改 <img> 元素的 `src` 属性会改变显示的图片。更改 <a> 标签的 `href` 属性会改变链接的目标。

同样，你可以将信息作为属性 (props) 传递给 React 组件。这些被称为属性 (props)。以按钮的可能变体为例：

![展示按钮组件 3 种变体的图表：主要 (Primary)、次要 (Secondary) 和禁用 (Disabled)](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/light/learn-props.png)![展示按钮组件 3 种变体的图表：主要 (Primary)、次要 (Secondary) 和禁用 (Disabled)](/_next/image?url=https%3A%2F%2Fh8DxKfmAPhn8O0p3.public.blob.vercel-storage.com%2Flearn%2Fdark%2Flearn-props.png&w=3840&q=75)

类似于 JavaScript 函数，你可以设计接受自定义参数（或属性 (props)）的组件，以改变组件的行为或在屏幕上渲染时的可见内容。然后，你可以将这些属性 (props) 从父组件传递给子组件。

> **注意：** 在 React 中，数据沿组件树向下流动。这被称为_单向数据流 (one-way data flow)_。将在下一章讨论的 State 可以当作 props 从父组件传递给子组件。

使用属性 (props)[](#using-props)
---------------------------

在你的 HomePage 组件中，你可以向 Header 组件传递自定义 title prop，就像传递 HTML 属性一样：

index.html

```
function HomePage() {
  return (
    <div>
      <Header title="React" />
    </div>
  );
}
```

而 Header 作为子组件，可以接受这些 props 作为其第一个 **函数参数**：

index.html

```
function Header(props) {
  return <h1>Develop. Preview. Ship.</h1>;
}
```

如果你 `console.log()` props，可以看到它是一个包含 `title` 属性的对象。

index.html

```
function Header(props) {
  console.log(props); // { title: "React" }
  return <h1>Develop. Preview. Ship.</h1>;
}
```

由于 props 是一个对象，你可以使用 [对象解构](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Operators/Destructuring_assignment) 在函数参数中显式命名 props 的值：

index.html

```
function Header({ title }) {
  console.log(title); // "React"
  return <h1>Develop. Preview. Ship.</h1>;
}
```

然后你可以用 `title` 变量替换 `h1` 标签的内容。

index.html

```
function Header({ title }) {
  console.log(title);
  return <h1>title</h1>;
}
```

如果你在浏览器中打开文件，会看到它显示的是实际的单词"title"。这是因为 React 认为你打算将纯文本字符串渲染到 DOM。

你需要一种方式告诉 React，这是一个 JavaScript 变量。

在 JSX 中使用变量[](#using-variables-in-jsx)
-------------------------------------------------

为了使用 title prop，请添加花括号 `{}`。这是一种特殊的 JSX 语法，允许你在 JSX 标记内直接编写常规 JavaScript。

index.html

```
function Header({ title }) {
  console.log(title);
  return <h1>{title}</h1>;
}
```

你可以将花括号视为当你处于"JSX 领域"时进入"JavaScript 领域"的方式。你可以在花括号中添加任何 JavaScript 表达式（即计算为单个值的内容）。例如：

1.  一个带有**点号表示法**的对象属性：

example.js

```
function Header(props) {
  return <h1>{props.title}</h1>;
}
```

2.  一个 **模板字符串**：

example.js

```
function Header({ title }) {
  return <h1>{`Cool ${title}`}</h1>;
}
```

3.  函数的 **返回值**：

example.js

```
function createTitle(title) {
  if (title) {
    return title;
  } else {
    return 'Default title';
  }
}
 
function Header({ title }) {
  return <h1>{createTitle(title)}</h1>;
}
```

4.  或者 **三元运算符**：

example.js

```
function Header({ title }) {
  return <h1>{title ? title : 'Default Title'}</h1>;
}
```

现在你可以向 `title` prop 传递任何字符串，或者，如果你使用了三元运算符，你甚至可以完全不传递 `title` prop，因为你已经在组件中处理了默认情况：

example.js

```
function Header({ title }) {
  return <h1>{title ? title : 'Default title'}</h1>;
}
 
function HomePage() {
  return (
    <div>
      <Header />
    </div>
  );
}
```

你的组件现在接受一个通用的 title prop，你可以在应用的不同部分重用它。你需要做的就是更改 title 字符串：

index.html

```
function HomePage() {
  return (
    <div>
      <Header title="React" />
      <Header title="A new title" />
    </div>
  );
}
```

遍历列表[](#iterating-through-lists)
---------------------------------------------------

通常需要将数据显示为列表。你可以使用数组方法来操作数据，并生成样式相同但包含不同信息片段的 UI 元素。

将以下名称数组添加到你的 HomePage 组件中：

index.html

```
function HomePage() {
  const names = ['Ada Lovelace', 'Grace Hopper', 'Margaret Hamilton'];
 
  return (
    <div>
      <Header title="Develop. Preview. Ship." />
      <ul>
        {names.map((name) => (
          <li>{name}</li>
        ))}
      </ul>
    </div>
  );
}
```

然后你可以使用 `array.map()` 方法来遍历数组，并使用箭头函数将名称映射到列表项：

index.html

```
function HomePage() {
  const names = ['Ada Lovelace', 'Grace Hopper', 'Margaret Hamilton'];
 
  return (
    <div>
      <Header title="Develop. Preview. Ship." />
      <ul>
        {names.map((name) => (
          <li>{name}</li>
        ))}
      </ul>
    </div>
  );
}
```

注意你如何使用花括号在"JavaScript"和"JSX"领域中交织穿梭。

如果你运行此代码，React 会发出关于缺少 key prop 的警告。这是因为 React 需要一些东西来唯一识别数组中的项，以便它知道要更新 DOM 中的哪些元素。

现在你可以使用这些名称，因为它们目前是唯一的，但建议使用保证唯一的东西，比如项 ID。

index.html

```
function HomePage() {
  const names = ['Ada Lovelace', 'Grace Hopper', 'Margaret Hamilton'];
 
  return (
    <div>
      <Header title="Develop. Preview. Ship." />
      <ul>
        {names.map((name) => (
          <li key={name}>{name}</li>
        ))}
      </ul>
    </div>
  );
}
```
