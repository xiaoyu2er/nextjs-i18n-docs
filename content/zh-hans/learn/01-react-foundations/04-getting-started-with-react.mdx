要在你的新项目中开始使用 React，请从称为 [unpkg.com](https://unpkg.com/) 的外部网站加载两个 React 脚本：

*   **react** 是核心 React 库。
*   **react-dom** 提供 DOM 特定方法，使你可以使用 React 操作 DOM。

index.html

```
<html>
  <body>
    <div id="app"></div>
    <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
    <script type="text/javascript">
      const app = document.getElementById('app');
      const header = document.createElement('h1');
      const text = 'Develop. Preview. Ship.';
      const headerContent = document.createTextNode(text);
      header.appendChild(headerContent);
      app.appendChild(header);
    </script>
  </body>
</html>
```

不要直接使用纯 JavaScript 操作 DOM，而是移除之前添加的 DOM 方法，并添加 [`ReactDOM.createRoot()`](https://react.dev/reference/react-dom/client/createRoot) 方法来定位特定的 DOM 元素并创建根，以供显示你的 React 组件。然后，添加 [`root.render()`](https://react.dev/reference/react-dom/client/hydrateRoot#root-render) 方法将你的 React 代码渲染到 DOM。

这将告诉 React 在我们的 `#app` 元素内渲染我们的 `<h1>` 标题。

index.html

```
<html>
  <body>
    <div id="app"></div>
    <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
    <script>
      const app = document.getElementById('app');
      const root = ReactDOM.createRoot(app);
      root.render(<h1>Develop. Preview. Ship.</h1>);
    </script>
  </body>
</html>
```

如果你在浏览器中尝试运行这段代码，你会遇到语法错误：

终端

```
Uncaught SyntaxError: expected expression, got '<'
```

这是因为 `<h1>...</h1>` 不是有效的 JavaScript。这段代码是 **JSX**。

什么是 JSX?[](#what-is-jsx)
----------------------------

JSX 是 JavaScript 的语法扩展，允许你使用熟悉的 _HTML 风格_ 语法描述你的 UI。JSX 的好处在于，除了遵循 [三条 JSX 规则](https://react.dev/learn/writing-markup-with-jsx#the-rules-of-jsx) 之外，你不需要在 HTML 和 JavaScript 之外学习任何新符号或语法。

但是浏览器原生无法理解 JSX，因此你需要一个 JavaScript 编译器，例如 [Babel](https://babeljs.io/)，将你的 JSX 代码转换为普通 JavaScript。

向你的项目中添加 Babel[](#adding-babel-to-your-project)
-------------------------------------------------------------

要将 Babel 添加到你的项目中，请将以下脚本复制并粘贴到你的 `index.html` 文件中：

index.html

```
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
```

此外，你需要通过将脚本类型更改为 `type=text/jsx` 来告知 Babel 要转换的代码。

index.html

```
<html>
  <body>
    <div id="app"></div>
    <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
    <!-- Babel Script -->
    <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
    <script type="text/jsx">
      const domNode = document.getElementById('app');
      const root = ReactDOM.createRoot(domNode);
      root.render(<h1>Develop. Preview. Ship.</h1>);
    </script>
  </body>
</html>
```

为了确认它工作正常，请在浏览器中打开你的 HTML 文件。

比较你刚刚写的 **声明式** React 代码：

index.html

```
<script type="text/jsx">
  const domNode = document.getElementById('app');
  const root = ReactDOM.createRoot(domNode);
  root.render(<h1>Develop. Preview. Ship.</h1>);
</script>
```

与你在前部分编写的 **命令式** JavaScript 代码相比：

index.html

```
<script type="text/javascript">
  const app = document.getElementById('app');
  const header = document.createElement('h1');
  const text = 'Develop. Preview. Ship.';
  const headerContent = document.createTextNode(text);
  header.appendChild(headerContent);
  app.appendChild(header);
</script>
```

你可以开始看到使用 React 如何帮助你大幅减少重复代码。

这正是 React 所做的，它是一个包含可重用代码片段的库，用于代表你执行任务——在这个情况下，更新 UI。

用于 React 的必备 JavaScript[](#essential-javascript-for-react)
-----------------------------------------------------------------

虽然你可以同时学习 JavaScript 和 React，但熟悉 JavaScript 可以使学习 React 的过程更简单。

在接下来的章节中，你将从 JavaScript 的角度介绍一些 React 的核心概念。以下是将提到的 JavaScript 主题摘要：

*   [函数](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Functions) 和 [箭头函数](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Functions/Arrow_functions)
*   [对象](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Reference/Global_Objects/Object)
*   [数组和数组方法](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Reference/Global_Objects/Array)
*   [解构](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Reference/Operators/Destructuring_assignment)
*   [模板字符串](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Template_literals)
*   [三元运算符](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Reference/Operators/Conditional_Operator)
*   [ES 模块和导入/导出语法](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Modules)

虽然本课程没有深入讲解 JavaScript，但随时了解 JavaScript 的最新版本是很好的做法。但如果你目前还不精通 JavaScript，不要因此阻碍你开始用 React 进行构建！
