让我们探索 React 是如何通过 **状态 (state)** 和 **事件处理器 (event handlers)** 来帮助我们添加交互的。

例如，让我们在 `HomePage` 组件内部创建一个“赞”按钮。首先，在 `return()` 语句中添加一个按钮元素：

index.html

```
function HomePage() {
  const names = ['Ada Lovelace', 'Grace Hopper', 'Margaret Hamilton'];
 
  return (
    <div>
      <Header title="Develop. Preview. Ship." />
      <ul>
        {names.map((name) => (
          <li key={name}>{name}</li>
        ))}
      </ul>
      <button>Like</button>
    </div>
  );
}
```

监听事件 [](#listening-to-events)
-------------------------------------------

要让按钮在点击时执行某些操作，您可以使用 `onClick` 事件：

index.html

```
function HomePage() {
  // ...
  return (
    <div>
      {/* ... */}
      <button onClick={}>Like</button>
    </div>
  );
}
```

在 React 中，事件名称是驼峰命名的。`onClick` 事件是可用于响应交互的众多事件之一。例如，您可以对输入字段使用 `onChange`，或者对表单使用 `onSubmit`。

处理事件 [](#handling-events)
-----------------------------------

您可以定义一个函数来在触发事件时“处理”它们。在 `return` 语句之前创建一个名为 `handleClick()` 的函数：

index.html

```
function HomePage() {
  // ...
 
  function handleClick() {
    console.log("increment like count")
  }
 
  return (
    <div>
      {/* ... */}
	  <button onClick={}>Like</button>
    </div>
     )
   }
```

然后，当 `onClick` 事件触发时，您可以调用 `handleClick` 函数：

index.html

```
function HomePage() {
  // 	...
  function handleClick() {
    console.log('increment like count');
  }
 
  return (
    <div>
      {/* ... */}
      <button onClick={handleClick}>Like</button>
    </div>
  );
}
```

尝试在浏览器中运行此代码。在开发者工具中注意日志输出是如何增加的。

状态 和 Hooks [](#state-and-hooks)
-----------------------------------

React 有一组函数称为 [钩子 (hooks)](https://react.dev/learn)。Hooks 允许您向组件添加额外逻辑，例如 **状态 (state)**。您可以将 state 视为 UI 中随时间变化的任何信息，通常由用户交互触发。

![两种不同的 state 示例：1. 一个可以选择或取消选择的切换按钮。2. 一个可以被多次点击的赞按钮。](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/light/learn-state.png)![两种不同的 state 示例：1. 一个可以选择或取消选择的切换按钮。2. 一个可以被多次点击的赞按钮。](/_next/image?url=https%3A%2F%2Fh8DxKfmAPhn8O0p3.public.blob.vercel-storage.com%2Flearn%2Fdark%2Flearn-state.png&w=3840&q=75)

您可以使用 _状态 (state)_ 来存储并递增用户点击“赞”按钮的次数。事实上，用于管理 state 的 React hook (钩子) 名为：`useState()`。

将 `useState()` 添加到您的项目中。它返回一个数组，您可以使用 **数组解构** 在组件中访问和使用这些数组值：

index.html

```
function HomePage() {
  // ...
  const [] = React.useState();
 
  // ...
}
```

数组中的第一项是 state `value`，您可以将其命名为任何名称。建议将其命名为具有描述性的名称：

index.html

```
function HomePage() {
  // ...
  const [likes] = React.useState();
 
  // ...
}
```

数组中的第二项是一个用于 `update` 的函数。您可以将更新函数命名为任何内容，但通常以其前缀为 `set` 后跟您要更新的 state 变量名称：

index.html

```
function HomePage() {
  // ...
  const [likes, setLikes] = React.useState();
 
  // ...
}
```

您还可以利用这个机会将您的 `likes` state 的初始值设置为 `0`：

index.html

```
function HomePage() {
  // ...
  const [likes, setLikes] = React.useState(0);
}
```

然后，您可以在组件内部使用 state 变量来检查初始 state 是否正常工作。

index.html

```
function HomePage() {
  // ...
  const [likes, setLikes] = React.useState(0);
  // ...
 
  return (
    // ...
    <button onClick={handleClick}>Like({likes})</button>
  );
}
```

最后，您可以在 `HomePage` 组件中调用您的 state 更新函数 `setLikes`，让我们将其添加到您之前定义的 `handleClick()` 函数中：

index.html

```
function HomePage() {
  // ...
  const [likes, setLikes] = React.useState(0);
 
  function handleClick() {
    setLikes(likes + 1);
  }
 
  return (
    <div>
      {/* ... */}
      <button onClick={handleClick}>Likes ({likes})</button>
    </div>
  );
}
```

点击按钮现在将调用 `handleClick` 函数，该函数以当前点赞数 + 1 为参数调用 `setLikes` state 更新函数。

> **注意**：与作为第一个函数参数传递给组件的 props 不同，state 是在组件内初始化和存储的。您可以将 state 信息作为 props 传递给子组件，但更新 state 的逻辑应保留在最初创建 state 的组件中。
