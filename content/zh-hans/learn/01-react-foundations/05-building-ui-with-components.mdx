React 核心概念 [](#react-core-concepts)
-------------------------------------------

开始构建 React 应用程序需要熟悉 React 的三个核心概念。它们是：

-   组件
-   属性
-   状态

在接下来的章节中，我们将详细介绍这些概念，并提供资源供您继续学习。熟悉这些概念后，我们将向您展示如何安装 Next.js，以及如何使用 React 的新功能，例如 Server (服务端) 和 Client (客户端) 组件。

组件 [](#components)
-------------------------

用户界面可以分解为称为 **组件** 的较小构建块。

组件允许您构建自包含、可重用的代码片段。如果您将组件视为 **乐高积木**，您就可以取这些单个积木并将它们组合在一起形成更大的结构。如果您需要更新 UI 的一部分，您可以更新特定的组件或积木。

![由 3 个较小组件组成的媒体组件示例：图片、文本和按钮](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/light/learn-components.png)![由 3 个较小组件组成的媒体组件示例：图片、文本和按钮](/_next/image?url=https%3A%2F%2Fh8DxKfmAPhn8O0p3.public.blob.vercel-storage.com%2Flearn%2Fdark%2Flearn-components.png&w=3840&q=75)

这种模块化使得您的代码在增长时更具可维护性，因为您可以添加、更新和删除组件，而无需触碰应用程序的其余部分。

React 组件的妙处在于它们只是 JavaScript。让我们从 JavaScript 的角度来看看如何编写 React 组件：

### 创建组件 [](#creating-components)

在 React 中，组件是 **函数**。
在您的 `script` 标签中，创建一个名为 `header` 的新函数：

index.html

```
<script type="text/jsx">
  const app = document.getElementById('app');
 
  function header() {}
 
  const root = ReactDOM.createRoot(app);
  root.render(<h1>Develop. Preview. Ship.</h1>);
</script>
```

组件是一个 **返回 UI 元素** 的函数。在函数的 return 语句中，您可以编写 JSX：

index.html

```
<script type="text/jsx">
  const app = document.getElementById('app');
 
  function header() {
    return <h1>Develop. Preview. Ship.</h1>;
  }
 
  const root = ReactDOM.createRoot(app);
  root.render(<h1>Develop. Preview. Ship.</h1>);
</script>
```

要将此组件渲染到 DOM，请将其作为 `root.render()` 方法中的第一个参数传递：

index.html

```
<script type="text/jsx">
  const app = document.getElementById('app');
 
  function header() {
    return <h1>Develop. Preview. Ship.</h1>;
  }
 
  const root = ReactDOM.createRoot(app);
  root.render(header);
</script>
```

但是，等一下。如果您在浏览器中尝试运行上述代码，您会得到一个错误。为了使此代码正常运行，您必须做两件事：

首先，React 组件应该大写，以区分普通 HTML 和 JavaScript：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
const root = ReactDOM.createRoot(app);
// Capitalize the React Component
root.render(Header);
```

其次，您使用 React 组件的方式与使用常规 HTML 标签相同，使用尖括号 `<>`：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
const root = ReactDOM.createRoot(app);
root.render(<Header />);
```

如果您再次在浏览器中运行代码，您将看到您的更改。

### 嵌套组件 [](#nesting-components)

应用程序通常包含的内容比单个组件要多。您可以像对待普通 HTML 元素一样，将 React 组件相互 **嵌套**。

在您的示例中，创建一个名为 `HomePage` 的新组件：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
function HomePage() {
  return <div></div>;
}
 
const root = ReactDOM.createRoot(app);
root.render(<Header />);
```

然后将 `<Header>` 组件嵌套在新的 `<HomePage>` 组件中：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
function HomePage() {
  return (
    <div>
      {/* Nesting the Header component */}
      <Header />
    </div>
  );
}
 
const root = ReactDOM.createRoot(app);
root.render(<Header />);
```

### 组件树 [](#component-trees)

您可以通过这种方式不断嵌套 React 组件以形成组件树。

![显示组件如何相互嵌套的组件树](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/light/learn-component-tree.png)![显示组件如何相互嵌套的组件树](/_next/image?url=https%3A%2F%2Fh8DxKfmAPhn8O0p3.public.blob.vercel-storage.com%2Flearn%2Fdark%2Flearn-component-tree.png&w=3840&q=75)

例如，您的顶层 HomePage 组件可以包含 Header、Article 和 Footer 组件。此外，这些组件也可以分别拥有自己的子组件，依此类推。例如，Header 组件可以包含 Logo、Title 和 Navigation 组件。

这种模块化格式允许您在应用程序的不同地方重用组件。

在你的项目中，由于 `<HomePage>` 现已成为你的顶级组件，你可以将它传递给 `root.render()` 方法：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
function HomePage() {
  return (
    <div>
      <Header />
    </div>
  );
}
 
const root = ReactDOM.createRoot(app);
root.render(<HomePage />);
```
