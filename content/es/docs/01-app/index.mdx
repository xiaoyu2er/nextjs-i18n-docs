El **Enrutador de Aplicación (App Router)** es un sistema de enrutamiento basado en archivos que utiliza las últimas características de React como los [Componentes de Servidor](https://react.dev/reference/rsc/server-components), [Suspense](https://react.dev/reference/react/Suspense) y [Funciones de Servidor](https://react.dev/reference/rsc/server-functions).
