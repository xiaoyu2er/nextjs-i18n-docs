Le **Routeur d'application (App Router)** est un système de routage basé sur les fichiers qui utilise les dernières fonctionnalités de React comme les [composants serveur (Server Components)](https://react.dev/reference/rsc/server-components), [Suspense](https://react.dev/reference/react/Suspense) et les [fonctions serveur (Server Functions)](https://react.dev/reference/rsc/server-functions).
