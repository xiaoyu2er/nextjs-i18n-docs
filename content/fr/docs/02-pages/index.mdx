> Vous consultez la documentation du Routeur Pages. Consultez la documentation du [Routeur App (App Router)](/docs/app) pour les dernières fonctionnalités.

Le **Routeur Pages** utilise un système de routage basé sur les fichiers pour associer chaque fichier à une route.

Avant Next.js 13, le Routeur Pages était la méthode principale pour créer des routes dans Next.js. Il est toujours pris en charge dans les versions plus récentes de Next.js, mais nous recommandons de migrer vers le nouveau [Routeur App (App Router)](/docs/app) pour bénéficier des dernières fonctionnalités de React.
