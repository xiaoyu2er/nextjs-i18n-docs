我們很高興今天要介紹新的 Create Next App。

Create Next App 透過單一命令即可設定使用 Next.js 驅動的現代 React 應用程式。

要開始操作，您只需要執行：

終端機

```
npx create-next-app
```

Create Next App 已從頭重構，以提供最佳開發者體驗：

*   **互動體驗**：執行 `npx create-next-app`（無參數）現在會啟動互動式體驗，指導您設定專案。
*   **零依賴**：初始化專案現在只需 **一秒** 鐘。Create Next App 零依賴，安裝僅需 **604 kB**。經優化前，上一版為 **5.38 MB**。這意味著減少了 **4.7 MB** 以上！
*   **離線支援**：Create Next App 會自動檢測您是否離線，並使用本地套件快取啟動您的專案。
*   **新的預設專案範本**：Create Next App 使用為現代 Next.js 應用設計的新的專案範本。由於 Create Next App 現在與 Next.js 本身並行維護，此範本將始終與最新的 Next.js 版本保持同步！
*   **範例支援**：Create Next App 可以使用從 [Next.js 範例集合](https://github.com/vercel/next.js/tree/canary/examples) 中的範例啟動您的應用程式（例如 `npx create-next-app --example api-routes`）。
*   **已測試**：此套件是 Next.js monorepo 的一部分，並使用與 Next.js 本身的同一整合測試套件進行測試，確保其在每次發布時按預期運作。

Create Next App 過去是一個 [社群維護](https://open.segment.com/create-next-app/) 專案，然而，我們覺得策展 Next.js 的第一印象很重要。特別是因為我們在 [Next.js 範例集合](https://github.com/vercel/next.js/tree/canary/examples) 中推薦它。

我們與 [Segment](https://segment.com/) 合作轉移了套件的擁有權，並非常感謝他們之前的管理，特別是 [Fouad Matin](https://twitter.com/fouadmatin)。
