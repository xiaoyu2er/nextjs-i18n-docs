使用 [Lightning CSS](https://lightningcss.dev) 與 webpack 的實驗性支援。Lightning CSS 是由 Rust 撰寫的快速 CSS 轉換器與壓縮器。

若未設定此選項，webpack 上的 Next.js 預設會使用 [PostCSS](https://postcss.org/) 搭配 [`postcss-preset-env`](https://www.npmjs.com/package/postcss-preset-env)。

自 Next 14.2 起，Turbopack 預設使用 Lightning CSS。此設定選項對 Turbopack 無效。Turbopack 始終使用 Lightning CSS。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useLightningcss: false, // default, ignored on Turbopack
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    useLightningcss: true, // disables PostCSS on webpack
  },
}

module.exports = nextConfig
```

## lightningCssFeatures

預設情況下，Lightning CSS 會根據您的 [browserslist](https://browsersl.ist/) 目標決定要轉譯哪些 CSS 功能。
`lightningCssFeatures` 選項允許您覆寫此行為，強制特定功能始終轉譯 (`include`) 或永不轉譯 (`exclude`)，而不考慮瀏覽器支援情況。

此規則同時適用於 webpack（啟用 `useLightningcss` 時）和 Turbopack。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useLightningcss: true,
    lightningCssFeatures: {
      // Always transpile these features, even if targets support them
      include: ['light-dark', 'oklab-colors'],
      // Never transpile these features, even if targets don't support them
      exclude: ['nesting'],
    },
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    useLightningcss: true,
    lightningCssFeatures: {
      // Always transpile these features, even if targets support them
      include: ['light-dark', 'oklab-colors'],
      // Never transpile these features, even if targets don't support them
      exclude: ['nesting'],
    },
  },
}

module.exports = nextConfig
```

### 選項

| 選項    | 類型       | 描述                                                                |
| --------- | ---------- | -------------------------------------------------------------------------- |
| `include` | `string[]` | 無論瀏覽器目標為何，一律轉譯的功能。               |
| `exclude` | `string[]` | 即使瀏覽器目標需要，也絕不轉譯的功能。 |

### 可用功能

個別功能：

| 功能名稱                         | 描述                                          |
| ----------------------------------- | ---------------------------------------------------- |
| `nesting`                           | [CSS Nesting](https://drafts.csswg.org/css-nesting/) |
| `not-selector-list`                 | `:not` 搭配多個選擇器                       |
| `dir-selector`                      | `:dir()` 選擇器                                    |
| `lang-selector-list`                | `:lang()` 搭配多個語言                    |
| `is-selector`                       | `:is()` 選擇器                                     |
| `text-decoration-thickness-percent` | `text-decoration-thickness` 的百分比值           |
| `media-interval-syntax`             | 媒體查詢區間語法                    |
| `media-range-syntax`                | 媒體查詢範圍語法 (`width >= 600px`)          |
| `custom-media-queries`              | `@custom-media` 規則                                |
| `clamp-function`                    | `clamp()` 函數                                   |
| `color-function`                    | `color()` 函數                                   |
| `oklab-colors`                      | `oklab()` 與 `oklch()` 顏色                     |
| `lab-colors`                        | `lab()` 與 `lch()` 顏色                           |
| `p3-colors`                         | Display P3 顏色                                    |
| `hex-alpha-colors`                  | 4 位和 8 位十六進制顏色含透明度                |
| `space-separated-color-notation`    | 空格分隔的色彩表示法 (`rgb(0 0 0)`)        |
| `font-family-system-ui`             | `system-ui` 字體                              |
| `double-position-gradients`         | 雙位置漸層停止符                       |
| `vendor-prefixes`                   | 廠商前綴屬性與值                |
| `logical-properties`                | 邏輯屬性與值                        |
| `light-dark`                        | `light-dark()` 顏色函數                          |

組合群組（同時啟用多個功能的簡寫）：

| 群組名稱      | 包含內容                                                                                                                        |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `selectors`     | `nesting`, `not-selector-list`, `dir-selector`, `lang-selector-list`, `is-selector`                                             |
| `media-queries` | `media-interval-syntax`, `media-range-syntax`, `custom-media-queries`                                                           |
| `colors`        | `color-function`, `oklab-colors`, `lab-colors`, `p3-colors`, `hex-alpha-colors`, `space-separated-color-notation`, `light-dark` |

## 版本歷史

| 版本  | 變更                                                                                                                                                                                      |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 16.2.0    | 新增 `lightningCssFeatures`                                                                                                                                                                |
| 15.1.0    | 移除 Turbopack 對 `useSwcCss` 的支援                                                                                                                                                                                                          |
| 14.2.0    | Turbopack 的預設 CSS 處理器從 `@swc/css` 改為 Lightning CSS。`useLightningcss` 在 Turbopack 上被忽略，並新增了傳統 `experimental.turbo.useSwcCss` 選項 |
