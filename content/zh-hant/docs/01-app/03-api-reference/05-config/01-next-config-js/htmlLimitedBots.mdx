---
title: htmlLimitedBots
description: 指定會接收封鎖元數據 (blocking metadata) 的使用者代理 (User Agents) 列表。
---

The `htmlLimitedBots` 設定允許您指定一組使用者代理程式，這些程式會收到阻擋元數據，而非 [流式元數據](/docs/app/api-reference/functions/generate-metadata#streaming-metadata)。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const config: NextConfig = {
  htmlLimitedBots: /MySpecialBot|MyAnotherSpecialBot|SimpleCrawler/,
}

export default config
```

```js filename="next.config.js" switcher
module.exports = {
  htmlLimitedBots: /MySpecialBot|MyAnotherSpecialBot|SimpleCrawler/,
}
```

## 預設清單

Next.js 包含一組預設的 HTML 受限機器人清單，包括：

- Google 爬蟲 (例如: Mediapartners-Google, AdsBot-Google, Google-PageRenderer)
- Bingbot
- Twitterbot
- Slackbot

請參閱 [完整清單](https://github.com/vercel/next.js/blob/canary/packages/next/src/shared/lib/router/utils/html-bots.ts)。

指定 `htmlLimitedBots` 設定將覆蓋 Next.js 的預設清單。不過，這屬於進階功能，一般情況下預設值應已足夠。

```ts filename="next.config.ts" switcher
const config: NextConfig = {
  htmlLimitedBots: /MySpecialBot|MyAnotherSpecialBot|SimpleCrawler/,
}

export default config
```

```js filename="next.config.js" switcher

}
```

## 停用
若要完全停用流式元數據：

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const config: NextConfig = {
  htmlLimitedBots: /.*/,
}

export default config
```

```js filename="next.config.js" switcher
module.exports = {
  htmlLimitedBots: /.*/,
}
```
