> **溫馨提示**: 自 Next.js 13.4 起，此選項 **不再** 需要。App Router 已穩定。

App Router ([`app` 目錄](/docs/app)) 支援 [佈局](/docs/app/api-reference/file-conventions/layout)、[伺服器組件 (Server Components)](/docs/app/getting-started/server-and-client-components)、[流式傳輸](/docs/app/api-reference/file-conventions/loading) 以及 [並置資料獲取 (colocated data fetching)](/docs/app/getting-started/fetching-data)。

使用 `app` 目錄將自動啟用 [React Strict Mode](https://react.dev/reference/react/StrictMode)。學習如何 [逐步採用 `app`](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app)。
