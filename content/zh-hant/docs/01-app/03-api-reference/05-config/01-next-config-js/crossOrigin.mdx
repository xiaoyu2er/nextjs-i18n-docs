使用 [`crossOrigin`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) 選項可在由 <AppOnly>[`next/script`](/docs/app/guides/scripts) 元件</AppOnly> <PagesOnly>[`next/script`](/docs/pages/guides/scripts) 和 [`next/head`](/docs/pages/api-reference/components/head) 元件</PagesOnly> 產生的所有 [`<script>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/script) 標記中加入 [`crossOrigin` 屬性](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin)，並定義跨來源請求應如何處理。

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## 選項

## 選項

- `anonymous`: 加入 [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous) 屬性。
- `use-credentials`: 加入 [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials) 屬性。
