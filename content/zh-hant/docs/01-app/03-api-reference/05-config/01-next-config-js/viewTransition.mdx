`viewTransition` 是一個實驗性標誌，可在 React 中啟用新的 [View Transitions API (視圖轉換 API)](https://developer.mozilla.org/en-US/docs/Web/API/View_Transition_API)。此 API 允許您利用原生的 View Transitions (視圖轉換) 瀏覽器 API 來在不同的 UI 狀態之間建立無縫過渡。

要啟用此功能，您需要在 `next.config.js` 文件中將 `viewTransition` 屬性設置為 `true`。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    viewTransition: true,
  },
}

module.exports = nextConfig
```

> 重要通知：React 的 Canary 發布渠道中已有 <ViewTransition> 組件。
> `experimental.viewTransition` 僅用於與 Next.js 功能進行更深度的整合，例如為導航自動 [添加過渡類型](https://react.dev/reference/react/addTransitionType)。Next.js 特定的過渡類型尚未實現。

## 使用方法

您可以在應用程式中從 React 匯入 [`<ViewTransition>` 組件](https://react.dev/reference/react/ViewTransition):

```jsx
import { ViewTransition } from 'react'
```
