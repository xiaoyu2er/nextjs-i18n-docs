`devIndicators`  允許您配置螢幕指示器，以提供關於您在開發期間查看之當前路徑的上下文資訊。

```ts filename="Types"
  devIndicators: false | {
    position?: 'bottom-right'
    | 'bottom-left'
    | 'top-right'
    | 'top-left', // defaults to 'bottom-left',
  },
```

將 `devIndicators`  設定為 `false`  會隱藏指示器，不過 Next.js  仍會繼續顯示任何遇到的建置或執行階段錯誤。

## 疑難排解

### 路徑未被標記為靜態

如果您預期路由為靜態，而指示器將其標記為動態，該路由很可能已選擇退出預先渲染 (Prerendering)。

您可以透過使用 `next build --debug` 建置應用程式並檢查終端機輸出來确认路由是否為 [已預先渲染](/docs/app/glossary#prerendering) 或 [已動態渲染](/docs/app/glossary#dynamic-rendering)。靜態 (或預先渲染) 路由將顯示 `○` 符號，而動態路由則顯示 `ƒ` 符號。例如：

```bash filename="Build Output"
Route (app)
┌ ○ /_not-found
└ ƒ /products/[id]

○  (Static)   prerendered as static content
ƒ  (Dynamic)  server-rendered on demand
```

<AppOnly>

路由選擇退出預先渲染的原因有兩個：

- 存在依賴請求資訊的 [請求時 API](/docs/app/glossary#request-time-apis)。
- 一個 [未快取的資料請求](/docs/app/getting-started/fetching-data)，例如對 ORM 或資料庫驅動程式的呼叫。

請檢查您的路徑是否存在任何這些條件，若您無法以靜態方式渲染該路徑，則請考慮使用 [`loading.js`](/docs/app/api-reference/file-conventions/loading)  或 [`<Suspense />`](https://react.dev/reference/react/Suspense)  來利用 [串流](/docs/app/getting-started/linking-and-navigating#streaming)。

</AppOnly>

<PagesOnly>

當從頁面匯出 [`getServerSideProps`](/docs/pages/building-your-application/data-fetching/get-server-side-props)  或 [`getInitialProps`](/docs/pages/api-reference/functions/get-initial-props)  時，它將被標記為動態。

</PagesOnly>

## 版本歷史

| 版本   | 變更                                                                                                                                             |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `v16.0.0` | `appIsrStatus`、`buildActivity` 和 `buildActivityPosition` 選項已被移除。                                                             |
| `v15.2.0` | 使用新的 `position` 選項改進螢幕指示器。 `appIsrStatus`、`buildActivity` 和 `buildActivityPosition` 選項已被棄用。 |
| `v15.0.0` | 使用 `appIsrStatus` 選項新增靜態螢幕指示器。                                                                                        |
