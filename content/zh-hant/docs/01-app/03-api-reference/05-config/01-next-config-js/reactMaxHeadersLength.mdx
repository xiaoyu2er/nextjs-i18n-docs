---
title: reactMaxHeadersLength
description: React 所發出並加入回應中的 HTTP 標頭的最大長度。
---

在預渲染期間，React 可以發出標頭，這些標頭可以添加到響應中。這些標頭可用於提升效能，透過讓瀏覽器預先載入資源，例如字體、腳本和樣式表。預設值是 `6000`，但您可以透過在 `next.config.js` 中配置 `reactMaxHeadersLength` 選項來覆寫此值：

```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **小提示**：此選項僅適用於 App Router。

根據瀏覽器與伺服器之間代理伺服器的類型，HTTP 標頭可能會被截斷。例如，如果您使用的是不支援長 HTTP 標頭的反向代理 (reverse proxy)，您應該設定一個較低的值，以確保標頭不會被截斷。
