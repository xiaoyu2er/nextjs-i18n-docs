cacheTag 函數允許你為快取資料標上標記，以便進行按需無效化。透過將標記與快取項目關聯，你可以選擇性地清除或重新驗證特定的快取項目，而不影響其他快取資料。

## 使用方法

若要使用 cacheTag，請在您的 next.config.js 檔案中啟用 cacheComponents 旗標 (flag)：

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  cacheComponents: true,
}

export default nextConfig
```

```js filename="next.config.js" switcher
const nextConfig = {
  cacheComponents: true,
}

export default nextConfig
```

cacheTag 函數會接受一個或多個字串值。

```tsx filename="app/data.ts" switcher
import { cacheTag } from 'next/cache'

export async function getData() {
  'use cache'
  cacheTag('my-data')
  const data = await fetch('/api/data')
  return data
}
```

```jsx filename="app/data.js" switcher
import { cacheTag } from 'next/cache'

export async function getData() {
  'use cache'
  cacheTag('my-data')
  const data = await fetch('/api/data')
  return data
}
```

接著，你可以在另一個函式中，例如 [路由處理程式](/docs/app/api-reference/file-conventions/route) 或 [伺服器動作 (Server Action)](/docs/app/getting-started/mutating-data)，使用 [`revalidateTag`](/docs/app/api-reference/functions/revalidateTag) API 按需清除快取：

```tsx filename="app/action.ts" switcher
'use server'

import { revalidateTag } from 'next/cache'

export default async function submit() {
  await addPost()
  revalidateTag('my-data')
}
```

```jsx filename="app/action.js" switcher
'use server'

import { revalidateTag } from 'next/cache'

export default async function submit() {
  await addPost()
  revalidateTag('my-data')
}
```

## 小知識

- **冪等標記 (Idempotent Tags)**: 多次套用相同的標記不會有額外效果。
- **多個標記 (Multiple Tags)**: 你可以透過向 cacheTag 傳遞多個字串值，將多個標記指派給單一快取項目。

```tsx
cacheTag('tag-one', 'tag-two')
```

- **限制**：自訂標籤的最大長度為 256 個字元，且最大標籤項目數目為 128。

## 範例

### 標記元件或函數

在快取函數或元件中呼叫 cacheTag 來標記你的快取資料：

```tsx filename="app/components/bookings.tsx" switcher
import { cacheTag } from 'next/cache'

interface BookingsProps {
  type: string
}

export async function Bookings({ type = 'haircut' }: BookingsProps) {
  'use cache'
  cacheTag('bookings-data')

  async function getBookingsData() {
    const data = await fetch(`/api/bookings?type=${encodeURIComponent(type)}`)
    return data
  }

  return //...
}
```

```jsx filename="app/components/bookings.js" switcher
import { cacheTag } from 'next/cache'

export async function Bookings({ type = 'haircut' }) {
  'use cache'
  cacheTag('bookings-data')

  async function getBookingsData() {
    const data = await fetch(`/api/bookings?type=${encodeURIComponent(type)}`)
    return data
  }

  return //...
}
```

### 從外部資料建立標記

你可以使用從非同步函數返回的資料來標記快取項目。

```tsx filename="app/components/bookings.tsx" switcher
import { cacheTag } from 'next/cache'

interface BookingsProps {
  type: string
}

export async function Bookings({ type = 'haircut' }: BookingsProps) {
  async function getBookingsData() {
    'use cache'
    const data = await fetch(`/api/bookings?type=${encodeURIComponent(type)}`)
    cacheTag('bookings-data', data.id)
    return data
  }
  return //...
}
```

```jsx filename="app/components/bookings.js" switcher
import { cacheTag } from 'next/cache'

export async function Bookings({ type = 'haircut' }) {
  async function getBookingsData() {
    'use cache'
    const data = await fetch(`/api/bookings?type=${encodeURIComponent(type)}`)
    cacheTag('bookings-data', data.id)
    return data
  }
  return //...
}
```

### 無效化已標記的快取

使用 revalidateTag，你可以在需要時為特定標記無效化快取：

```tsx filename="app/actions.ts" switcher
'use server'

import { revalidateTag } from 'next/cache'

export async function updateBookings() {
  await updateBookingData()
  revalidateTag('bookings-data')
}
```

```jsx filename="app/actions.js" switcher
'use server'

import { revalidateTag } from 'next/cache'

export async function updateBookings() {
  await updateBookingData()
  revalidateTag('bookings-data')
}
```
