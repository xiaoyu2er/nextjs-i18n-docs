---
title: 重新整理 (refresh)
description: refresh 函式的 API 參考
---

`refresh` 允許您在 [Server Action](/docs/app/getting-started/mutating-data) 內刷新客戶端路由。

## 使用方法

`refresh` **僅能** 從 Server Actions 內呼叫。它不能用於 Route Handlers, Client Components 或任何其他情境。

## 參數

```tsx
refresh(): void;
```

## 返回值

`refresh` 不返回值。

## 範例

```ts filename="app/actions.ts" switcher
'use server'

import { refresh } from 'next/cache'

export async function createPost(formData: FormData) {
  const title = formData.get('title')
  const content = formData.get('content')

  // Create the post in your database
  const post = await db.post.create({
    data: { title, content },
  })

  refresh()
}
```

```js filename="app/actions.js" switcher
'use server'

  const content = formData.get('content')

```

### 在 Server Actions 外使用時出現錯誤

```ts filename="app/api/posts/route.ts" switcher
import { refresh } from 'next/cache'

export async function POST() {
  // This will throw an error
  refresh()
}
```
