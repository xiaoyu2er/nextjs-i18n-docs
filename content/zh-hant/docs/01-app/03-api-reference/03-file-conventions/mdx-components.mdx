`mdx-components.js|tsx` 檔案是**必須**的，才能使用 [`@next/mdx` 搭配 App Router](/docs/app/guides/mdx)，沒有它無法運作。此外，您還可以用它來 [自訂樣式](/docs/app/guides/mdx#using-custom-styles-and-components)。

使用您專案根目錄中的 `mdx-components.tsx`（或 `.js`）檔案來定義 MDX 元件。例如，與 `pages` 或 `app` 同一層級，或放在 `src` 內部（如適用）。

```tsx filename="mdx-components.tsx" switcher
import type { MDXComponents } from 'mdx/types'

const components: MDXComponents = {}

export function useMDXComponents(): MDXComponents {
  return components
}
```

```js filename="mdx-components.js" switcher
const components = {}

export function useMDXComponents() {
  return components
}
```

## 匯出

### `useMDXComponents` 函式

此檔案必須匯出一個名為 `useMDXComponents` 的單一函式。此函式不接受任何參數。

```tsx filename="mdx-components.tsx" switcher
import type { MDXComponents } from 'mdx/types'

const components: MDXComponents = {}

export function useMDXComponents(): MDXComponents {
  return components
}
```

```js filename="mdx-components.js" switcher
const components = {}

export function useMDXComponents() {
  return components
}
```
