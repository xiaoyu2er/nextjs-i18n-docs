`sitemap.(xml|js|ts)` 是一個特殊檔案，用於匹配 [Sitemaps XML 格式 (Sitemaps xml format)](https://www.sitemaps.org/protocol.html)，以協助搜尋引擎爬蟲更有效地編製您網站的索引。

### 網站地圖檔案 (.xml)

對於較小的應用程式，您可以建立 `sitemap.xml` 檔案，並將其放置在 `app` 目錄的根目錄下。

```xml filename="app/sitemap.xml"
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://acme.com</loc>
    <lastmod>2023-04-06T15:02:24.021Z</lastmod>
    <changefreq>yearly</changefreq>
    <priority>1</priority>
  </url>
  <url>
    <loc>https://acme.com/about</loc>
    <lastmod>2023-04-06T15:02:24.021Z</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  <url>
    <loc>https://acme.com/blog</loc>
    <lastmod>2023-04-06T15:02:24.021Z</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.5</priority>
  </url>
</urlset>
```

### 使用程式碼產生 網站地圖 (.js, .ts)

您可以使用 `sitemap.(js|ts)` 檔案約定，經程式化 **產生** 網站地圖，方法為匯出一個回傳 URL 陣列的預設函式。如果使用的是 TypeScript，則可使用 [`Sitemap`](#returns) 型別。

> **值得了解**: `sitemap.js` 是一個預設會快取的獨特路由處理器 (Route Handler)，除非它使用了 [Request-time API (請求時間 API)](/docs/app/glossary#request-time-apis) 或 [動態配置 (dynamic config)](/docs/app/guides/caching-without-cache-components#dynamic) 選項。

```ts filename="app/sitemap.ts" switcher
import type { MetadataRoute } from 'next'

export default function sitemap(): MetadataRoute.Sitemap {
  return [
    {
      url: 'https://acme.com',
      lastModified: new Date(),
      changeFrequency: 'yearly',
      priority: 1,
    },
    {
      url: 'https://acme.com/about',
      lastModified: new Date(),
      changeFrequency: 'monthly',
      priority: 0.8,
    },
    {
      url: 'https://acme.com/blog',
      lastModified: new Date(),
      changeFrequency: 'weekly',
      priority: 0.5,
    },
  ]
}
```


### 圖片 網站地圖

您可以使用 `images` 屬性來建立圖片網站地圖。請參閱 [Google 開發人員文件](https://developers.google.com/search/docs/crawling-indexing/sitemaps/image-sitemaps) 以了解更多詳細資訊。

```ts filename="app/sitemap.ts" switcher
import type { MetadataRoute } from 'next'

export default function sitemap(): MetadataRoute.Sitemap {
  return [
    {
      url: 'https://example.com',
      lastModified: '2021-01-01',
      changeFrequency: 'weekly',
      priority: 0.5,
      images: ['https://example.com/image.jpg'],
    },
  ]
}
```

輸出：

```xml filename="acme.com/sitemap.xml"
<?xml version="1.0" encoding="UTF-8"?>
<urlset
  xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
  xmlns:image="http://www.google.com/schemas/sitemap-image/1.1"
>
  <url>
    <loc>https://example.com</loc>
    <image:image>
      <image:loc>https://example.com/image.jpg</image:loc>
    </image:image>
    <lastmod>2021-01-01</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.5</priority>
  </url>
</urlset>
```

### 影片 網站地圖

您可以使用 `videos` 屬性來建立影片網站地圖。請參閱 [Google 開發人員文件](https://developers.google.com/search/docs/crawling-indexing/sitemaps/video-sitemaps) 以了解更多詳細資訊。

```ts filename="app/sitemap.ts" switcher
import type { MetadataRoute } from 'next'

export default function sitemap(): MetadataRoute.Sitemap {
  return [
    {
      url: 'https://example.com',
      lastModified: '2021-01-01',
      changeFrequency: 'weekly',
      priority: 0.5,
      videos: [
        {
          title: 'example',
          thumbnail_loc: 'https://example.com/image.jpg',
          description: 'this is the description',
        },
      ],
    },
  ]
}
```

### 產生多個 網站地圖

雖然單一 網站地圖對大多數應用程式即可。對於大型網頁應用程式，您可能需要將 網站地圖分割為多個檔案。

建立多個 網站地圖有兩種方法：

- 將 `sitemap.(xml|js|ts)` 嵌套在多個路徑區段內，例如 `app/sitemap.xml` 和 `app/products/sitemap.xml`。
- 使用 [`generateSitemaps`](/docs/app/api-reference/functions/generate-sitemaps) 函式。

以 `generateSitemaps` 分割 XML 地圖為例，請返回包含 sitemap id 的物件陣列。接著，使用 `id` 來建立獨立的 XML 地圖。

```ts filename="app/product/sitemap.ts" switcher
import type { MetadataRoute } from 'next'
import { BASE_URL } from '@/app/lib/constants'

export async function generateSitemaps() {
  // Fetch the total number of products and calculate the number of sitemaps needed
  return [{ id: 0 }, { id: 1 }, { id: 2 }, { id: 3 }]
}

export default async function sitemap(props: {
  id: Promise<string>
}): Promise<MetadataRoute.Sitemap> {
  const id = await props.id
  // Google's limit is 50,000 URLs per sitemap
  const start = id * 50000
  const end = start + 50000
  const products = await getProducts(
    `SELECT id, date FROM products WHERE id BETWEEN ${start} AND ${end}`
  )
  return products.map((product) => ({
    url: `${BASE_URL}/product/${product.id}`,
    lastModified: product.date,
  }))
}
```

```js filename="app/product/sitemap.js" switcher
import { BASE_URL } from '@/app/lib/constants'

export async function generateSitemaps() {
  // Fetch the total number of products and calculate the number of sitemaps needed
  return [{ id: 0 }, { id: 1 }, { id: 2 }, { id: 3 }]
}

export default async function sitemap(props) {
  const id = await props.id
  // Google's limit is 50,000 URLs per sitemap
  const start = id * 50000
  const end = start + 50000
  const products = await getProducts(
    `SELECT id, date FROM products WHERE id BETWEEN ${start} AND ${end}`
  )
  return products.map((product) => ({
    url: `${BASE_URL}/product/${product.id}`,
    lastModified: product.date,
  }))
}
```

您產生的 網站地圖將在 `/.../sitemap/[id]` 處可用。例如，`/product/sitemap/1.xml`。

請參閱 [`generateSitemaps` API 參考文件](/docs/app/api-reference/functions/generate-sitemaps) 以取得更多資訊。

## 返回值

從 `sitemap.(xml|ts|js)` 匯出的預設函式應回傳包含以下属性的物件陣列：

```tsx
type Sitemap = Array<{
  url: string
  lastModified?: string | Date
  changeFrequency?:
    | 'always'
    | 'hourly'
    | 'daily'
    | 'weekly'
    | 'monthly'
    | 'yearly'
    | 'never'
  priority?: number
  alternates?: {
    languages?: Languages<string>
  }
}>
```
