當 [`forbidden`](/docs/app/api-reference/functions/forbidden) 函數在身份驗證期間被調用時，**forbidden** 文件可用於渲染 UI。除了允許您自定義 UI 外，Next.js 還會返回 `403` 狀態碼。

```tsx filename="app/forbidden.tsx" switcher
import Link from 'next/link'

export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>
      <p>You are not authorized to access this resource.</p>
      <Link href="/">Return Home</Link>
    </div>
  )
}
```

```jsx filename="app/forbidden.jsx" switcher
import Link from 'next/link'

export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>
      <p>You are not authorized to access this resource.</p>
      <Link href="/">Return Home</Link>
    </div>
  )
}
```

## 參考

### 屬性 (Props)

`forbidden.js` 組件不接受任何屬性。
