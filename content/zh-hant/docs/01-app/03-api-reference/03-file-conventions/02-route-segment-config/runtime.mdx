---
title: 運行時 (runtime)
description: 關於運行時 (runtime) 路由區塊配置選項的 API 參考。
---

`runtime` 選項允許您選擇用於渲染路由的 JavaScript 運行時。

```tsx filename="layout.tsx | page.tsx | route.ts" switcher
export const runtime = 'nodejs'
// 'nodejs' | 'edge'
```

```js filename="layout.js | page.js | route.js" switcher
export const runtime = 'nodejs'
// 'nodejs' | 'edge'
```

- **`nodejs`**（預設）
- **`edge`**

>
**小提示**：
>
> - 對於快取組件 (Cache Components)，使用 `runtime: 'edge'` **不支援**。
> - 此選項無法用於 [Proxy](/docs/app/api-reference/file-conventions/proxy)。
