`maxDuration` 選項允許您設定路由區段中伺服器端邏輯的最大執行時間（以秒為單位）。部署平台可以從 Next.js 的建置輸出中使用 `maxDuration` 來新增特定的執行限制。

```tsx filename="layout.tsx | page.tsx | route.ts" switcher
export const maxDuration = 5
```

```js filename="layout.js | page.js | route.js" switcher
export const maxDuration = 5
```

## 伺服器動作 (Server Actions)

如果使用 [伺服器操作 (Server Actions)](/docs/app/getting-started/mutating-data)，請在頁面層級設定 `maxDuration`，以變更頁面所使用的全部伺服器操作 (Server Actions) 的預設逾時時間。
