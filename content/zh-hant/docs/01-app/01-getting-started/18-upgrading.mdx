## 最新版本

若要更新至 Next.js 的最新版本，您可以使用 `upgrade` 指令：

```bash package="pnpm"
pnpm next upgrade
```

```bash package="npm"
npx next upgrade
```

```bash package="yarn"
yarn next upgrade
```

```bash package="bun"
bunx next upgrade
```

Next.js 16.1.0 之前的版本不支援 `upgrade` 指令，需要改用獨立的套件：

```bash filename="Terminal"
npx @next/codemod@canary upgrade latest
```

如果您偏好手動升級，請安裝最新的 Next.js 和 React 版本：

```bash package="pnpm"
pnpm i next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash package="npm"
npm i next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash package="yarn"
yarn add next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash package="bun"
bun add next@latest react@latest react-dom@latest eslint-config-next@latest
```

## Canary 版本

若要更新至最新的 canary，請確保您目前使用的是 Next.js 最新版本且一切運作正常。接著，請執行下列命令：

```bash package="pnpm"
pnpm add next@canary
```

```bash package="npm"
npm i next@canary
```

```bash package="yarn"
yarn add next@canary
```

```bash package="bun"
bun add next@canary
```

**身份驗證 (Authentication)**:

- [`unauthorized.js`](/docs/app/api-reference/file-conventions/unauthorized)
