---
title: 如何升級至第 10 版
nav_title: 第 10 版
description: 將您的 Next.js 應用程式從第 9 版升級至第 10 版。
---

第 9 版與第 10 版之間沒有重大變更。

若要升級至第 10 版，請執行以下指令：

```bash filename="Terminal"
npm i next@10
```

```bash filename="Terminal"
yarn add next@10
```

```bash filename="Terminal"
pnpm up next@10
```

```bash filename="Terminal"
bun add next@10
```

> **小提醒：** 如果您使用 TypeScript，請確保同時升級 `@types/react` 和 `@types/react-dom` 至對應版本。
