目前，您的首頁沒有任何樣式。讓我們來看看可以為您的 Next.js 應用程式設定樣式的不同方式。

全域樣式 [](#global-styles)
-------------------------------

如果您查看 `/app/ui` 資料夾內部，您會看到一個名為 `global.css` 的檔案。您可以使用此檔案來為應用程式中的**所有**路由新增 CSS 規則 - 例如 CSS 重置規則、HTML 元素（如連結）的全域樣式等。

您可以在應用程式中的任何元件匯入 `global.css`，但通常應將其新增到您的頂層元件。在 Next.js 中，這是指 [根佈局](/docs/app/api-reference/file-conventions/layout#root-layouts)（稍後再詳述）。

透過導航至 `/app/layout.tsx` 並匯入 `global.css` 檔案，將全域樣式新增到您的應用程式：

/app/layout.tsx

```
import '@/app/ui/global.css';
 
export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  );
}
```

開發伺服器仍在執行時，儲存變更並在瀏覽器中預覽。您的首頁現在應該看起來這樣：

![已設定樣式的頁面，包含 'Acme' 標誌、描述和登入連結。](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/light/home-page-with-tailwind.png)![已設定樣式的頁面，包含 'Acme' 標誌、描述和登入連結。](/_next/image?url=https%3A%2F%2Fh8DxKfmAPhn8O0p3.public.blob.vercel-storage.com%2Flearn%2Fdark%2Fhome-page-with-tailwind.png&w=1920&q=75)

但是等等，您並沒有新增任何 CSS 規則，樣式是從哪裡來的呢？

如果您查看 `global.css` 內部，您會注意到一些 `@tailwind` 指令：

/app/ui/global.css

```
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Tailwind[](#tailwind)
---------------------

[Tailwind](https://tailwindcss.com/) 是一個 CSS 框架，透過允許您在 React 程式碼中快速寫入 [效用類別](https://tailwindcss.com/docs/utility-first) 來加速開發流程。

在 Tailwind 中，您透過新增類別名稱來設定元素的樣式。例如，新增 `"text-blue-500"` 會將 `<h1>` 文字變為藍色：

```
<h1 className="text-blue-500">I'm blue!</h1>
```

雖然 CSS 樣式是全區共享的，但每個類別會單獨套用到每個元素。這意味著如果您新增或刪除元素，您就不需要擔心維護單獨的樣式表、樣式衝突，或是隨著應用程式擴大而導致 CSS 套件大小增加。

當您使用 `create-next-app` 啟動新專案時，Next.js 會詢問您是否要使用 Tailwind。如果您選擇 `yes`，Next.js 將自動安裝必要的套件並在應用程式中配置 Tailwind。

如果您查看 `/app/page.tsx`，您會看到我們在範例中使用 Tailwind 類別。

/app/page.tsx

```
import AcmeLogo from '@/app/ui/acme-logo';
import { ArrowRightIcon } from '@heroicons/react/24/outline';
import Link from 'next/link';
 
export default function Page() {
  return (
    // These are Tailwind classes:
    <main className="flex min-h-screen flex-col p-6">
      <div className="flex h-20 shrink-0 items-end rounded-lg bg-blue-500 p-4 md:h-52">
    // ...
  )
}
```

如果您第一次使用 Tailwind 不必擔心。為了節省時間，我們已經設定好您將使用的所有元件的樣式。

讓我們來玩玩 Tailwind！複製下面的程式碼，並貼到 /app/page.tsx 中的 <p> 元素上方：

/app/page.tsx

```
<div
  className="relative w-0 h-0 border-l-[15px] border-r-[15px] border-b-[26px] border-l-transparent border-r-transparent border-b-black"
/>

```

如果您偏好撰寫傳統 CSS 規則，或將樣式與 JSX 分開 - CSS Modules 是很棒的替代方案。

CSS Modules[](#css-modules)
---------------------------

[CSS Modules](/docs/app/getting-started/css#css-modules) 允許您將 CSS 範圍縮小至元件，透過自動建立唯一的類別名稱，這樣您也不會擔心樣式衝突。

在本課程中我們將繼續使用 Tailwind，但讓我們花點時間看看如何透過 CSS 模組達成上述測驗中的相同結果。

在 `/app/ui` 內部，建立一個名為 `home.module.css` 的新檔案，並新增以下 CSS 規則：

/app/ui/home.module.css

```
.shape {
  height: 0;
  width: 0;
  border-bottom: 30px solid black;
  border-left: 20px solid transparent;
  border-right: 20px solid transparent;
}
```

接著，在您的 `/app/page.tsx` 檔案中匯入樣式，並將您已新增的 <div> 中的 Tailwind 類別名稱換成 styles.shape:

/app/page.tsx

```
import AcmeLogo from '@/app/ui/acme-logo';
import { ArrowRightIcon } from '@heroicons/react/24/outline';
import Link from 'next/link';
import styles from '@/app/ui/home.module.css';
 
export default function Page() {
  return (
    <main className="flex min-h-screen flex-col p-6">
      <div className={styles.shape} />
    // ...
  )
}
```

儲存您的變更並在瀏覽器中預覽。您應該會看到與之前相同的形狀。

Tailwind 和 CSS 模組是設定 Next.js 應用程式樣式的兩種最常見方式。無論您使用哪一種取決於偏好 - 您甚至可以同時在相同的應用程式中使用兩者！

使用 `clsx` 函式庫切換類別名稱 [](#using-the-clsx-library-to-toggle-class-names)
-----------------------------------------------------------------------------------------------

可能在某些情況下，您需要根據狀態或其他條件條件式地設定元素樣式。

[`clsx`](https://www.npmjs.com/package/clsx) 是一個讓您輕鬆切換類別名稱的函式庫。我們建議查看 [文件](https://github.com/lukeed/clsx) 以了解更多內容，以下是基本用法：

*   假設您想要建立一個 `InvoiceStatus` 元件，其接受 `status`。狀態可以是 `'pending'` 或 `'paid'`。
*   如果是 `'paid'`，您希望顏色為綠色。如果是 `'pending'`，您希望顏色為灰色。

您可以使用 `clsx` 條件式套用到類別，如下所示：

/app/ui/invoices/status.tsx

```
import clsx from 'clsx';
 
export default function InvoiceStatus({ status }: { status: string }) {
  return (
    <span
      className={clsx(
        'inline-flex items-center rounded-full px-2 py-1 text-sm',
        {
          'bg-gray-100 text-gray-500': status === 'pending',
          'bg-green-500 text-white': status === 'paid',
        },
      )}
    >
    // ...
)}
```

其他樣式解決方案 [](#other-styling-solutions)
---------------------------------------------------

除了我們討論過的方法外，您也可以使用以下方式設定您的 Next.js 應用程式樣式：

*   Sass，它允許您匯入 `.css` 和 `.scss` 檔案。
*   CSS-in-JS 函式庫，例如 [styled-jsx](https://github.com/vercel/styled-jsx)、[styled-components](https://github.com/vercel/next.js/tree/canary/examples/with-styled-components) 和 [emotion](https://github.com/vercel/next.js/tree/canary/examples/with-emotion)。

請查看 [CSS 文件](/docs/app/building-your-application/styling) 以取得更多資訊。
