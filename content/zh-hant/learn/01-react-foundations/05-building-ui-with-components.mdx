React 核心概念 [](#react-core-concepts)
-------------------------------------------

有三個 React 的核心概念，你需要熟悉它們才能開始建構 React 應用程式。這些是：

*   組件 (Components)
*   屬性 (Props)
*   狀態 (State)

在接下來的章節中，我們將講解這些概念，並提供資源供你繼續學習。在你熟悉這些概念後，我們將向你展示如何安裝 Next.js 以及使用更新的 React 功能，例如伺服器組件 (Server Components) 和使用者端組件 (Client Components)。

組件 (Components)[](#components)
-------------------------

使用者介面可以分解成稱為**組件**的更小建構塊。

組件允許你建立封閉、可重複使用的程式碼片段。如果你將組件視為**LEGO 積木**，你可以取這些單個積木並將其組合在一起以形成更大的結構。如果你需要更新用戶介面的一小部分，你可以更新特定的組件或積木。

![媒體組件 (Media Component) 示例，由 3 個較小組件組成：圖片、文字和按鈕](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/light/learn-components.png)![媒體組件 (Media Component) 示例，由 3 個較小組件組成：圖片、文字和按鈕](/_next/image?url=https%3A%2F%2Fh8DxKfmAPhn8O0p3.public.blob.vercel-storage.com%2Flearn%2Fdark%2Flearn-components.png&w=3840&q=75)

這種模塊化允許你的程式碼在成長時更具可維護性，因為你可以增加、更新和刪除組件，而不必觸及應用程式的其餘部分。

React 組件的好處在於它們只是 JavaScript。讓我們來看看如何從 JavaScript 的角度編寫 React 組件：

### 組件建立 [](#creating-components)

在 React 中，組件是**函式**。在你的 `script` 標籤內，建立一個名為 `header` 的新函式：

index.html

```
<script type="text/jsx">
  const app = document.getElementById('app');
 
  function header() {}
 
  const root = ReactDOM.createRoot(app);
  root.render(<h1>Develop. Preview. Ship.</h1>);
</script>
```

組件是一個**回傳 UI 元素**的函式。在函式的 return 語句中，你可以編寫 JSX：

index.html

```
<script type="text/jsx">
  const app = document.getElementById('app');
 
  function header() {
    return <h1>Develop. Preview. Ship.</h1>;
  }
 
  const root = ReactDOM.createRoot(app);
  root.render(<h1>Develop. Preview. Ship.</h1>);
</script>
```

要將此組件渲染到 DOM，請將它作為 `root.render()` 方法的第一個參數傳遞：

index.html

```
<script type="text/jsx">
  const app = document.getElementById('app');
 
  function header() {
    return <h1>Develop. Preview. Ship.</h1>;
  }
 
  const root = ReactDOM.createRoot(app);
  root.render(header);
</script>
```

但等等。如果你嘗試在瀏覽器中運行上面的程式碼，你會收到錯誤訊息。為了讓這能運作，你有兩件事必須要做：

首先，React 組件應該大寫以區分它們與純 HTML 和 JavaScript：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
const root = ReactDOM.createRoot(app);
// Capitalize the React Component
root.render(Header);
```

其次，你使用 React 組件的方式與使用普通 HTML 標籤相同，使用尖括號 `<>`：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
const root = ReactDOM.createRoot(app);
root.render(<Header />);
```

如果你再次嘗試運行瀏覽器中的程式碼，你將看到你的變更。

### 組件巢狀 [](#nesting-components)

應用程式通常包含比單一組件更多的內容。你可以將 React 組件**巢狀**在彼此之中，就像處理普通 HTML 元素一樣。

在你的範例中，建立一個名為 `HomePage` 的新組件：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
function HomePage() {
  return <div></div>;
}
 
const root = ReactDOM.createRoot(app);
root.render(<Header />);
```

然後將 `<Header>` 組件巢狀在新建的 `<HomePage>` 組件內：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
function HomePage() {
  return (
    <div>
      {/* Nesting the Header component */}
      <Header />
    </div>
  );
}
 
const root = ReactDOM.createRoot(app);
root.render(<Header />);
```

### 組件樹 [](#component-trees)

你可以繼續以這種方式巢狀 React 組件，以形成組件樹。

![組件樹，顯示如何將組件相互嵌套](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/light/learn-component-tree.png)![組件樹，顯示如何將組件相互嵌套](/_next/image?url=https%3A%2F%2Fh8DxKfmAPhn8O0p3.public.blob.vercel-storage.com%2Flearn%2Fdark%2Flearn-component-tree.png&w=3840&q=75)

例如，你的上層 `HomePage` 組件可以包含 `Header`、`Article` 和 `Footer` 組件。而那每個組件又各自擁有自己的子組件，依此類推。例如，`Header` 組件可以包含 `Logo`、`Title` 和 `Navigation` 組件。

這種模塊化格式允許你在應用程式內部的不同位置重複使用組件。

在你的專案中，由於 `<HomePage>` 現在是你的上層組件，你可以將其傳遞給 `root.render()` 方法：

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
function HomePage() {
  return (
    <div>
      <Header />
    </div>
  );
}
 
const root = ReactDOM.createRoot(app);
root.render(<HomePage />);
```
