到目前為止，如果你重複使用你的 `<Header />` 元件 (component)，兩次都會顯示相同的內容。

index.html

```
function Header() {
  return <h1>Develop. Preview. Ship.</h1>;
}
 
function HomePage() {
  return (
    <div>
      <Header />
      <Header />
    </div>
  );
}
```

但是如果想要傳遞不同的文字，或者因為你正在從外部來源取得資料而事先不知道資訊，該怎麼辦？

一般的 HTML 元素有屬性，你可以用這些屬性來傳遞資訊，改變該元素的行為。例如，改變 <img> 元素的 `src` 屬性會改變顯示的圖片。改變 <a> 標籤的 `href` 屬性會改變連結的目標。

同樣地，你可以作為屬性將資訊傳遞給 React 元件。這些稱為 `props`。舉例來說，按鈕可能的變體有：

![顯示按鈕元件的 3 種變體圖示：主要、次要和停用](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/light/learn-props.png)![顯示按鈕元件的 3 種變體圖示：主要、次要和停用](/_next/image?url=https%3A%2F%2Fh8DxKfmAPhn8O0p3.public.blob.vercel-storage.com%2Flearn%2Fdark%2Flearn-props.png&w=3840&q=75)

類似 JavaScript 函式，你可以設計接受自訂參數（或 props）的元件，這些參數會改變元件的行為或它在螢幕上渲染時顯示的內容。接著，你可以將這些 props 從父元件傳遞到子元件。

> **注意：** 在 React 中，資料向下流動至元件樹。這稱為 `_單向資料流 (one-way data flow)_`。狀態（將在下一章討論）可以作為 props 從父元件傳遞至子元件。

使用 props[](#using-props)
---------------------------

在你的 `HomePage` 元件中，你可以像傳遞 HTML 屬性一樣，將自訂的 `title` prop 傳遞給 `Header` 元件：

index.html

```
function HomePage() {
  return (
    <div>
      <Header title="React" />
    </div>
  );
}
```

而 `Header`（子元件）可以將這些 props 作為其第一個 **函式參數**：

index.html

```
function Header(props) {
  return <h1>Develop. Preview. Ship.</h1>;
}
```

如果你 `console.log()` props，你可以看到它是一個具有 title 屬性的 **物件**：

index.html

```
function Header(props) {
  console.log(props); // { title: "React" }
  return <h1>Develop. Preview. Ship.</h1>;
}
```

因為 props 是一個物件，你可以使用 [**物件解構**](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Operators/Destructuring_assignment) 在函式參數內明確命名 props 的值：

index.html

```
function Header({ title }) {
  console.log(title); // "React"
  return <h1>Develop. Preview. Ship.</h1>;
}
```

接著你就可以將你的 `title` 變數取代 `<h1>` 標籤的內容。

index.html

```
function Header({ title }) {
  console.log(title);
  return <h1>title</h1>;
}
```

如果你在瀏覽器中開啟檔案，你會看到它顯示實際的單字 "title"。這是因為 React 認為你打算將純文字字串渲染至 DOM。

你需要一種方法告訴 React 這是一個 JavaScript 變數。

在 JSX 中使用變數 [](#using-variables-in-jsx)
-------------------------------------------------

要使用 `title` prop，請加入 **大括號** `{}`。這些是一種特殊的 JSX 語法，允許你在 JSX 標記中直接編寫一般的 JavaScript。

index.html

```
function Header({ title }) {
  console.log(title);
  return <h1>{title}</h1>;
}
```

你可以將大括號視為當你在「JSX 領域」時進入「JavaScript 領域」的方式。你可以將任何 **JavaScript 表達式**（評估為單一值的事物）加入大括號中。例如：

1.  使用點記法的 **物件屬性**：

example.js

```
function Header(props) {
  return <h1>{props.title}</h1>;
}
```

2.  **模板字串**：

example.js

```
function Header({ title }) {
  return <h1>{`Cool ${title}`}</h1>;
}
```

3.  **函式的傳回值**：

example.js

```
function createTitle(title) {
  if (title) {
    return title;
  } else {
    return 'Default title';
  }
}
 
function Header({ title }) {
  return <h1>{createTitle(title)}</h1>;
}
```

4.  或 **三元運算子**：

example.js

```
function Header({ title }) {
  return <h1>{title ? title : 'Default Title'}</h1>;
}
```

你現在可以將任何字串傳遞給你的 title prop，或者，如果你用了三元運算子，你甚至可以完全不傳遞 title prop，因為你在元件中已處理了預設情況：

example.js

```
function Header({ title }) {
  return <h1>{title ? title : 'Default title'}</h1>;
}
 
function HomePage() {
  return (
    <div>
      <Header />
    </div>
  );
}
```

你的元件現在接受通用的 title prop，你可以在應用程式的不同部分重複使用它。你需要做的只是改變 title 字串：

index.html

```
function HomePage() {
  return (
    <div>
      <Header title="React" />
      <Header title="A new title" />
    </div>
  );
}
```

遍歷列表 [](#iterating-through-lists)
---------------------------------------------------

將資料顯示為列表是很常見的。你可以使用陣列方法來操作資料，並生成樣式相同但包含不同資訊的 UI 元件。

將以下名稱陣列加入你的 `HomePage` 元件：

index.html

```
function HomePage() {
  const names = ['Ada Lovelace', 'Grace Hopper', 'Margaret Hamilton'];
 
  return (
    <div>
      <Header title="Develop. Preview. Ship." />
      <ul>
        {names.map((name) => (
          <li>{name}</li>
        ))}
      </ul>
    </div>
  );
}
```

接著你可以使用 `array.map()` 方法來遍歷陣列，並使用 **箭頭函式** 將名稱對應到列表項目：

index.html

```
function HomePage() {
  const names = ['Ada Lovelace', 'Grace Hopper', 'Margaret Hamilton'];
 
  return (
    <div>
      <Header title="Develop. Preview. Ship." />
      <ul>
        {names.map((name) => (
          <li>{name}</li>
        ))}
      </ul>
    </div>
  );
}
```

注意你是如何使用大括號在「JavaScript」和「JSX」領域之間進出的。

如果你執行這段程式碼，React 會對遺失的 `key` prop 給我們警告。這是因為 React 需要一些東西來唯一識別陣列中的項目，以便它知道要更新 DOM 中的哪些元素。

你現在可以暫時使用名稱，因為它們目前具有唯一性，但建議使用確保唯一的內容，例如項目 ID。

index.html

```
function HomePage() {
  const names = ['Ada Lovelace', 'Grace Hopper', 'Margaret Hamilton'];
 
  return (
    <div>
      <Header title="Develop. Preview. Ship." />
      <ul>
        {names.map((name) => (
          <li key={name}>{name}</li>
        ))}
      </ul>
    </div>
  );
}
```
