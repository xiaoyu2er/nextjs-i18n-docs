要在您新建立的專案中使用 React，請從名為 [unpkg.com](https://unpkg.com/) 的外部網站載入兩個 React 腳本：

*   **react** 是 React 的核心函式庫。
*   **react-dom** 提供了 DOM 特定方法，讓您能在 DOM 中使用 React。

index.html

```
<html>
  <body>
    <div id="app"></div>
    <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
    <script type="text/javascript">
      const app = document.getElementById('app');
      const header = document.createElement('h1');
      const text = 'Develop. Preview. Ship.';
      const headerContent = document.createTextNode(text);
      header.appendChild(headerContent);
      app.appendChild(header);
    </script>
  </body>
</html>
```

不要直接使用純 JavaScript 操作 DOM，請移除之前添加的 DOM 方法，並添加 [`ReactDOM.createRoot()`](https://react.dev/reference/react-dom/client/createRoot) 方法來針對特定的 DOM 元素並建立一個 root 來顯示您的 React 元件。接著，添加 [`root.render()`](https://react.dev/reference/react-dom/client/hydrateRoot#root-render) 方法來將您的 React 程式碼渲染至 DOM。

這將指示 React 在 #app 元素內渲染我們的 <h1> 標題。

index.html

```
<html>
  <body>
    <div id="app"></div>
    <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
    <script>
      const app = document.getElementById('app');
      const root = ReactDOM.createRoot(app);
      root.render(<h1>Develop. Preview. Ship.</h1>);
    </script>
  </body>
</html>
```

如果您嘗試在瀏覽器中執行這段程式碼，您將收到語法錯誤：

終端機

```
Uncaught SyntaxError: expected expression, got '<'
```

這是因為 <h1>...</h1> 不是有效的 JavaScript。這段程式碼是 **JSX**。

什麼是 JSX？[](#what-is-jsx)
----------------------------

JSX 是 JavaScript 的語法擴展，讓您能夠使用熟悉的 _HTML-like_ 語法來描述您的 UI。JSX 的好處在於，除了遵循 [三個 JSX 規則](https://react.dev/learn/writing-markup-with-jsx#the-rules-of-jsx) 之外，您不需要學習 HTML 和 JavaScript 之外的任何新符號或語法。

但瀏覽器無法原生理解 JSX，因此您需要一個 JavaScript 編譯器，例如 [Babel](https://babeljs.io/)，來將您的 JSX 程式碼轉換為通用的 JavaScript。

將 Babel 加入您的專案 [](#adding-babel-to-your-project)
-------------------------------------------------------------

要將 Babel 加入您的專案，請將以下腳本複製並貼到您的 `index.html` 檔案中：

index.html

```
<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
```

此外，您還需要通過將 script type 改為 `type=text/jsx` 來通知 Babel 要轉換哪些程式碼。

index.html

```
<html>
  <body>
    <div id="app"></div>
    <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
    <!-- Babel Script -->
    <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
    <script type="text/jsx">
      const domNode = document.getElementById('app');
      const root = ReactDOM.createRoot(domNode);
      root.render(<h1>Develop. Preview. Ship.</h1>);
    </script>
  </body>
</html>
```

為了確認運作是否正常，請在瀏覽器中開啟您的 HTML 檔案。

比較剛才撰寫的 **宣告式** React 程式碼：

index.html

```
<script type="text/jsx">
  const domNode = document.getElementById('app');
  const root = ReactDOM.createRoot(domNode);
  root.render(<h1>Develop. Preview. Ship.</h1>);
</script>
```

與您在上一節撰寫的 **命令式** JavaScript 程式碼：

index.html

```
<script type="text/javascript">
  const app = document.getElementById('app');
  const header = document.createElement('h1');
  const text = 'Develop. Preview. Ship.';
  const headerContent = document.createTextNode(text);
  header.appendChild(headerContent);
  app.appendChild(header);
</script>
```

您現在可以看出使用 React 如何能節省許多重複的程式碼。

而這正是 React 所做的，它是一個函式庫，包含可重複使用的程式碼片段，代表您執行任務——在這種情況下，更新 UI。

React 所需的基本 JavaScript [](#essential-javascript-for-react)
-----------------------------------------------------------------

雖然你可以同時學習 JavaScript 和 React，但熟悉 JavaScript 會讓學習 React 的過程更容易。

在接下來的章節中，你將從 JavaScript 的角度認識 React 的一些核心概念。以下是將提到的 JavaScript 主題摘要：

*   [函式 (Functions)](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Functions) 和 [箭頭函式 (Arrow Functions)](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Functions/Arrow_functions)
*   [物件 (Objects)](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Object)
*   [陣列與陣列方法 (Arrays and array methods)](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Array)
*   [解構 (Destructuring)](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Operators/Destructuring_assignment)
*   [模板字串 (Template literals)](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Template_literals)
*   [三元運算子 (Ternary Operators)](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Operators/Conditional_Operator)
*   [ES Modules 與匯入／匯出語法 (ES Modules and Import / Export Syntax)](https://developer.mozilla.org/docs/Web/JavaScript/Guide/Modules)

雖然此課程不深入探討 JavaScript，但保持了解最新的 JavaScript 版本是良好的實踐。但如果您尚未覺得熟悉 JavaScript，請不要因此阻礙您開始使用 React 進行建構！
