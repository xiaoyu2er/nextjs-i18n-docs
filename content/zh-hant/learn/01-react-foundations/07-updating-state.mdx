讓我們來看看 React 如何幫助我們使用 **狀態 (state)** 和 **事件處理器 (event handlers)** 來增加互動性。

作為範例，讓我們在你的 `HomePage` 元件中建立一個「Like」按鈕。首先，在 `return()` 語句中新增一個 button 元素：

index.html

```
function HomePage() {
  const names = ['Ada Lovelace', 'Grace Hopper', 'Margaret Hamilton'];
 
  return (
    <div>
      <Header title="Develop. Preview. Ship." />
      <ul>
        {names.map((name) => (
          <li key={name}>{name}</li>
        ))}
      </ul>
      <button>Like</button>
    </div>
  );
}
```

監聽事件 [](#listening-to-events)
-------------------------------------------

要讓按鈕在點擊時執行動作，你可以使用 `onClick` 事件：

index.html

```
function HomePage() {
  // ...
  return (
    <div>
      {/* ... */}
      <button onClick={}>Like</button>
    </div>
  );
}
```

在 React 中，事件名稱是大駝峰命名法 (camelCased)。 `onClick` 事件是可以用來回應使用者互動的許多可行事件之一。例如，你可以對輸入欄位使用 `onChange`，或對表單使用 `onSubmit`。

處理事件 [](#handling-events)
-----------------------------------

你可以定義一個函式，在事件被觸發時「處理」它們。在 `return` 語句之前建立一個名為 `handleClick()` 的函式：

index.html

```
function HomePage() {
  // ...
 
  function handleClick() {
    console.log("increment like count")
  }
 
  return (
    <div>
      {/* ... */}
	  <button onClick={}>Like</button>
    </div>
     )
   }
```

接著，當 `onClick` 事件被觸發時，你可以呼叫 `handleClick` 函式：

index.html

```
function HomePage() {
  // 	...
  function handleClick() {
    console.log('increment like count');
  }
 
  return (
    <div>
      {/* ... */}
      <button onClick={handleClick}>Like</button>
    </div>
  );
}
```

試著在瀏覽器中執行這段程式碼。注意你的開發者工具中，日誌輸出如何增加。

狀態與鉤子 [](#state-and-hooks)
-----------------------------------

React 有一個稱為 [鉤子 (hooks)](https://react.dev/learn) 的函式集合。鉤子讓你可以在元件中加入額外的邏輯，例如 **狀態 (state)**。你可以將狀態視為 UI 中會隨時間改變的任何資訊，通常是透過使用者互動觸發的。

你可以使用 _狀態 (state)_ 來儲存並增加使用者點擊「Like」按鈕的次數。事實上，用來管理狀態的 React 鉤子稱為：`useState()`

將 `useState()` 新增到你的專案中。它會回傳一個陣列，你可以使用 **陣列解構 (array destructuring)** 在元件中存取和使用這些陣列值：

index.html

```
function HomePage() {
  // ...
  const [] = React.useState();
 
  // ...
}
```

陣列中的第一個項目是狀態 `value`，你可以任命名的名稱為任何東西。建議命名為具有描述性的名稱：

index.html

```
function HomePage() {
  // ...
  const [likes] = React.useState();
 
  // ...
}
```

陣列中的第二個項目是一個用來 `update` 值的函式。你可以任命名為更新函式，但常見的做法是以 `set` 開頭，後接你正在更新的狀態變數名稱：

index.html

```
function HomePage() {
  // ...
  const [likes, setLikes] = React.useState();
 
  // ...
}
```

你也可以趁機將你的 `likes` 狀態的初始值設定為 `0`：

index.html

```
function HomePage() {
  // ...
  const [likes, setLikes] = React.useState(0);
}
```

接著，你可以在元件中使用狀態變數來檢查初始狀態是否運作正常。

index.html

```
function HomePage() {
  // ...
  const [likes, setLikes] = React.useState(0);
  // ...
 
  return (
    // ...
    <button onClick={handleClick}>Like({likes})</button>
  );
}
```

最後，你可以在 `HomePage` 元件中呼叫的狀態更新函式 `setLikes`，讓我們把它新增到你之前定義的 `handleClick()` 函式中：

index.html

```
function HomePage() {
  // ...
  const [likes, setLikes] = React.useState(0);
 
  function handleClick() {
    setLikes(likes + 1);
  }
 
  return (
    <div>
      {/* ... */}
      <button onClick={handleClick}>Likes ({likes})</button>
    </div>
  );
}
```

點擊按鈕現在將呼叫 `handleClick` 函式，該函式會呼叫 `setLikes` 狀態更新函式，傳入單一參數為當前讚數 + 1。

> **注意**：與作為第一個函式參數傳遞給元件的屬性 (props) 不同，狀態 (state) 是在元件內部啟動和儲存的。你可以將狀態資訊作為屬性 (props) 傳遞給子元件，但更新狀態的邏輯應保留在初始建立狀態的元件中。
