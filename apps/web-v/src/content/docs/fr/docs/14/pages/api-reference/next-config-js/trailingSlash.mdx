Par défaut, Next.js redirige les URLs avec une barre oblique finale vers leur équivalent sans barre oblique. Par exemple, `/about/` sera redirigé vers `/about`. Vous pouvez configurer ce comportement pour qu'il agisse de manière inverse, où les URLs sans barle oblique finale sont redirigées vers leurs équivalents avec une barre oblique.

Ouvrez `next.config.js` et ajoutez la configuration `trailingSlash` :

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

Avec cette option activée, les URLs comme `/about` seront redirigées vers `/about/`.

Lorsqu'elle est utilisée avec la configuration [`output: "export"`](/docs/14/app/building-your-application/deploying/static-exports), la page `/about` générera `/about/index.html` (au lieu de `/about.html` par défaut).
