Utilisez l'option `crossOrigin` pour ajouter un [attribut `crossOrigin`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) à toutes les balises `<script>` générées par le <AppOnly>[composant `next/script`](/docs/14/app/building-your-application/optimizing/scripts)</AppOnly> <PagesOnly>[composant `next/script`](/docs/14/pages/building-your-application/optimizing/scripts) et [`next/head`](/docs/14/pages/api-reference/components/head)</PagesOnly>, et définir comment les requêtes cross-origin doivent être gérées.

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## Options

- `'anonymous'` : Ajoute l'attribut [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous).
- `'use-credentials'` : Ajoute [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials).
