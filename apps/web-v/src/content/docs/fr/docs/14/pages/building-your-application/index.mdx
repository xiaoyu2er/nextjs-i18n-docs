Next.js fournit les éléments de base pour créer des applications web full-stack flexibles. Les guides dans **Construire votre application** expliquent comment utiliser ces fonctionnalités et comment personnaliser le comportement de votre application.

Les sections et pages sont organisées séquentiellement, des concepts de base aux plus avancés, afin que vous puissiez les suivre étape par étape lors de la construction de votre application Next.js. Cependant, vous pouvez les lire dans n'importe quel ordre ou sauter directement aux pages qui correspondent à votre cas d'utilisation.

<AppOnly>

Si vous débutez avec Next.js, nous recommandons de commencer par les sections [Routage](/docs/14/app/building-your-application/routing), [Rendu](/docs/14/app/building-your-application/rendering), [Récupération de données](/docs/14/app/building-your-application/data-fetching) et [Stylisation](/docs/14/app/building-your-application/styling), car elles introduisent les concepts fondamentaux de Next.js et du web pour vous aider à démarrer. Ensuite, vous pouvez approfondir les autres sections comme [Optimisation](/docs/14/app/building-your-application/optimizing) et [Configuration](/docs/14/app/building-your-application/configuring). Enfin, une fois prêt, consultez les sections [Déploiement](/docs/14/app/building-your-application/deploying) et [Mise à niveau](/docs/14/app/building-your-application/upgrading).

</AppOnly>
