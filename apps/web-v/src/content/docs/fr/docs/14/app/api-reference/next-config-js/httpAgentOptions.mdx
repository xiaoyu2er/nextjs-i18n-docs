Dans les versions de Node.js antérieures à la 18, Next.js polyfill automatiquement `fetch()` avec [undici](/docs/architecture/supported-browsers#polyfills) et active [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive) par défaut.

Pour désactiver HTTP Keep-Alive pour tous les appels `fetch()` côté serveur, ouvrez `next.config.js` et ajoutez la configuration `httpAgentOptions` :

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
