Par défaut, Next.js ajoute l'en-tête `x-powered-by`. Pour le désactiver, ouvrez `next.config.js` et désactivez la configuration `poweredByHeader` :

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
