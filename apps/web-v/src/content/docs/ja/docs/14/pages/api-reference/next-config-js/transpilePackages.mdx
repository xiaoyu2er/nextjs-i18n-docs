Next.jsは、ローカルパッケージ（モノレポなど）や外部依存関係（`node_modules`）から依存関係を自動的にトランスパイルおよびバンドルできます。これは`next-transpile-modules`パッケージの代わりになります。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['@acme/ui', 'lodash-es'],
}

module.exports = nextConfig
```
