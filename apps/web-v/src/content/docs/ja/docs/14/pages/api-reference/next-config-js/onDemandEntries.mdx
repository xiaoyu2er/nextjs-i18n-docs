Next.js は、開発環境でビルドされたページをサーバーがどのように破棄またはメモリに保持するかを制御するためのいくつかのオプションを提供しています。

デフォルト設定を変更するには、`next.config.js` を開き、`onDemandEntries` 設定を追加します：

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // サーバーがページをバッファに保持する期間（ミリ秒）
    maxInactiveAge: 25 * 1000,
    // 同時に保持されるページ数（破棄されない）
    pagesBufferLength: 2,
  },
}
```
