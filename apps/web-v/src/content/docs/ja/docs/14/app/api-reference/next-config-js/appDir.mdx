> **補足**: Next.js 13.4以降、このオプションは**不要**になりました。App Routerは現在安定版です。

App Router（[`app`ディレクトリ](/docs/14/app/building-your-application/routing)）は、[レイアウト](/docs/14/app/building-your-application/routing/pages-and-layouts)、[サーバーコンポーネント (Server Components)](/docs/14/app/building-your-application/rendering/server-components)、[ストリーミング (streaming)](/docs/14/app/building-your-application/routing/loading-ui-and-streaming)、[データ取得の共置 (colocated data fetching)](/docs/14/app/building-your-application/data-fetching)のサポートを有効にします。

`app`ディレクトリを使用すると、自動的に[React Strict Mode](https://react.dev/reference/react/StrictMode)が有効になります。[`app`への段階的な移行方法](/docs/14/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app)をご覧ください。
