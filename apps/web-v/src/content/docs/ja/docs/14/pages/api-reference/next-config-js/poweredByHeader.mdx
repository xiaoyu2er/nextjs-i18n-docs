Next.js はデフォルトで `x-powered-by` ヘッダーを追加します。このヘッダーを無効化するには、`next.config.js` を開き、`poweredByHeader` 設定を無効にします:

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
