実験的な `instrumentationHook` オプションを使用すると、Next.js アプリケーションで [`instrumentation` ファイル](/docs/14/app/api-reference/file-conventions/instrumentation) を介してインストルメンテーションを設定できます。

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
