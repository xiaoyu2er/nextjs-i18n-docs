Next.js 提供了一些选项，让您可以控制开发服务器如何处理内存中已构建页面的释放与保留。

要修改默认配置，请打开 `next.config.js` 文件并添加 `onDemandEntries` 配置：

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // 服务器将页面保留在缓冲区的时间（毫秒）
    maxInactiveAge: 25 * 1000,
    // 应同时保留而不被释放的页面数量
    pagesBufferLength: 2,
  },
}
```
