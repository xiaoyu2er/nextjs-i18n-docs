您可以指定一个自定义构建目录名称来替代 `.next`。

打开 `next.config.js` 并添加 `distDir` 配置：

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

现在当您运行 `next build` 时，Next.js 将使用 `build` 目录而非默认的 `.next` 文件夹。

> `distDir` **不应**指向项目目录之外。例如，`../build` 是**无效的**目录路径。
