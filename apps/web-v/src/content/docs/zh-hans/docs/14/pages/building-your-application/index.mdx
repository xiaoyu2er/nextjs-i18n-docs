Next.js 提供了构建灵活、全栈 Web 应用的基础模块。**构建应用**指南将解释如何使用这些功能以及如何自定义应用行为。

章节和页面按从基础到高级的顺序组织，因此您在构建 Next.js 应用时可以逐步遵循。不过，您也可以按任意顺序阅读或直接跳转到符合您用例的页面。

<AppOnly>

如果您是 Next.js 新手，我们建议从 [路由](/docs/14/app/building-your-application/routing)、[渲染](/docs/14/app/building-your-application/rendering)、[数据获取](/docs/14/app/building-your-application/data-fetching) 和 [样式](/docs/14/app/building-your-application/styling) 部分开始，这些章节介绍了 Next.js 和 Web 的基础概念以帮助您入门。然后，您可以深入探索其他部分，如 [优化](/docs/14/app/building-your-application/optimizing) 和 [配置](/docs/14/app/building-your-application/configuring)。最后，当您准备就绪时，可以查看 [部署](/docs/14/app/building-your-application/deploying) 和 [升级](/docs/14/app/building-your-application/upgrading) 部分。

</AppOnly>
