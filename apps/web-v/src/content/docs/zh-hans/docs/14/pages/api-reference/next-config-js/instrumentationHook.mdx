实验性的 `instrumentationHook` 选项允许您通过 Next.js 应用中的 [`instrumentation` 文件](/docs/14/app/api-reference/file-conventions/instrumentation) 来设置 instrumentation。

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
