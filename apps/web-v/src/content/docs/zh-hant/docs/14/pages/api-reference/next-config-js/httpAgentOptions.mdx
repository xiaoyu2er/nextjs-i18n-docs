在 Node.js 18 之前的版本中，Next.js 會自動使用 [undici](/docs/architecture/supported-browsers#polyfills) 來填充 `fetch()` 功能，並預設啟用 [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive)。

若要停用伺服器端所有 `fetch()` 呼叫的 HTTP Keep-Alive 功能，請開啟 `next.config.js` 並新增 `httpAgentOptions` 設定：

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
