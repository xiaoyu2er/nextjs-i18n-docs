您可以指定自訂建置目錄的名稱來取代 `.next`。

開啟 `next.config.js` 並新增 `distDir` 設定：

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

現在當您執行 `next build` 時，Next.js 將會使用 `build` 資料夾而非預設的 `.next` 資料夾。

> `distDir` **不應**指向專案目錄外的位置。例如 `../build` 就是一個**無效**的目錄。
