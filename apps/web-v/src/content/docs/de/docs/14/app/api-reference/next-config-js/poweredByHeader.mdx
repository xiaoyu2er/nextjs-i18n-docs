Standardmäßig fügt Next.js den `x-powered-by`-Header hinzu. Um dies zu deaktivieren, öffnen Sie die `next.config.js` und setzen die `poweredByHeader`-Konfiguration auf `false`:

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
