Die experimentelle Option `instrumentationHook` ermöglicht es Ihnen, Instrumentation über die [`instrumentation`-Datei](/docs/14/app/api-reference/file-conventions/instrumentation) in Ihrer Next.js-App einzurichten.

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
