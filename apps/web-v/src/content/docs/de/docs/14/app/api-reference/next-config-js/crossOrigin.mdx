Verwenden Sie die `crossOrigin`-Option, um ein [`crossOrigin`-Attribut](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) in allen `<script>`-Tags hinzuzufügen, die von der <AppOnly>[`next/script`](/docs/14/app/building-your-application/optimizing/scripts)-Komponente</AppOnly> <PagesOnly>[`next/script`](/docs/14/pages/building-your-application/optimizing/scripts)- und [`next/head`](/docs/14/pages/api-reference/components/head)-Komponenten</PagesOnly> generiert werden, und definieren Sie, wie Cross-Origin-Anfragen behandelt werden sollen.

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## Optionen

- `'anonymous'`: Fügt das Attribut [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous) hinzu.
- `'use-credentials'`: Fügt [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials) hinzu.
