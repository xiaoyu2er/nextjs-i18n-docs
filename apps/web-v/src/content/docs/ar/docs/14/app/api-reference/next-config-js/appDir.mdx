> **من الجيد معرفة**: هذا الخيار **لم يعد** مطلوبًا بدءًا من Next.js 13.4. حيث أصبح موجه التطبيق (App Router) مستقرًا الآن.

يُمكّن موجه التطبيق ([دليل `app`](/docs/14/app/building-your-application/routing)) من دعم [التخطيطات (layouts)](/docs/14/app/building-your-application/routing/pages-and-layouts)، [مكونات الخادم (Server Components)](/docs/14/app/building-your-application/rendering/server-components)، [البث (streaming)](/docs/14/app/building-your-application/routing/loading-ui-and-streaming)، و[جلب البيانات المترافق (colocated data fetching)](/docs/14/app/building-your-application/data-fetching).

سيؤدي استخدام دليل `app` إلى تمكين [وضع React الصارم (React Strict Mode)](https://react.dev/reference/react/StrictMode) تلقائيًا. تعلم كيفية [تبني دليل `app` تدريجيًا](/docs/14/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app).
