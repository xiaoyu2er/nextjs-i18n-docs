افتراضيًا، سيضيف Next.js رأس `x-powered-by`. لإلغاء تفعيله، افتح ملف `next.config.js` وعطل إعداد `poweredByHeader`:

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
