يفشل Next.js في عملية **بناء الإنتاج** (`next build`) عند وجود أخطاء TypeScript في مشروعك.

إذا كنت ترغب في أن يقوم Next.js بإنشاء كود إنتاج بشكل خطير حتى عند وجود أخطاء في تطبيقك، يمكنك تعطيل خطوة التحقق من الأنواع المدمجة.

إذا تم التعطيل، تأكد من أنك تقوم بإجراء فحوصات الأنواع كجزء من عملية البناء أو النشر، وإلا فقد يكون هذا خطيرًا جدًا.

افتح ملف `next.config.js` وقم بتمكين خيار `ignoreBuildErrors` في إعدادات `typescript`:

```js filename="next.config.js"
module.exports = {
  typescript: {
    // !! تحذير !!
    // السماح بشكل خطير بإكمال عمليات بناء الإنتاج بنجاح حتى لو
    // كان مشروعك يحتوي على أخطاء في الأنواع.
    // !! تحذير !!
    ignoreBuildErrors: true,
  },
}
```
