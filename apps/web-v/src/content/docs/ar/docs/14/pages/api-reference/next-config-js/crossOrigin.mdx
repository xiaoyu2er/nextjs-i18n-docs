استخدم خيار `crossOrigin` لإضافة [سمة `crossOrigin`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) في جميع وسوم `<script>` التي يتم إنشاؤها بواسطة <AppOnly>[مكون `next/script`](/docs/14/app/building-your-application/optimizing/scripts)</AppOnly> <PagesOnly>[مكون `next/script`](/docs/14/pages/building-your-application/optimizing/scripts) و[مكونات `next/head`](/docs/14/pages/api-reference/components/head)</PagesOnly>، وتحديد كيفية معالجة طلبات cross-origin.

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## الخيارات

- `'anonymous'`: يضيف سمة [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous).
- `'use-credentials'`: يضيف سمة [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials).
