> **Good to know**: This option is **no longer** needed as of Next.js 13.4. The App Router is now stable.

The App Router ([`app` directory](/docs/15/app)) enables support for [layouts](/docs/15/app/api-reference/file-conventions/layout), [Server Components](/docs/15/app/getting-started/server-and-client-components), [streaming](/docs/15/app/api-reference/file-conventions/loading), and [colocated data fetching](/docs/15/app/getting-started/fetching-data).

Using the `app` directory will automatically enable [React Strict Mode](https://react.dev/reference/react/StrictMode). Learn how to [incrementally adopt `app`](/docs/15/app/guides/migrating/app-router-migration#migrating-from-pages-to-app).
