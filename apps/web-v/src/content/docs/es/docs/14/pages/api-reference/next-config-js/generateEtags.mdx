Next.js genera [etags](https://en.wikipedia.org/wiki/HTTP_ETag) para cada página por defecto. Es posible que desee desactivar la generación de etags para páginas HTML dependiendo de su estrategia de caché.

Abra `next.config.js` y desactive la opción `generateEtags`:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
