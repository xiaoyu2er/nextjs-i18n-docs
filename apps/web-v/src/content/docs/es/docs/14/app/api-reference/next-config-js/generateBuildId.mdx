Next.js genera un ID durante `next build` para identificar qué versión de su aplicación se está sirviendo. La misma compilación debe utilizarse e iniciarse en múltiples contenedores.

Si está recompilando para cada etapa de su entorno, necesitará generar un ID de compilación consistente para usar entre contenedores. Utilice el comando `generateBuildId` en `next.config.js`:

```jsx filename="next.config.js"
module.exports = {
  generateBuildId: async () => {
    // Esto podría ser cualquier cosa, usando el último hash de git
    return process.env.GIT_HASH
  },
}
```
