> **Es bueno saberlo**: Esta opción **ya no es necesaria** a partir de Next.js 13.4. El Enrutador de Aplicación (App Router) ahora es estable.

El Enrutador de Aplicación ([directorio `app`](/docs/14/app/building-your-application/routing)) habilita soporte para [diseños (layouts)](/docs/14/app/building-your-application/routing/pages-and-layouts), [Componentes del Servidor (Server Components)](/docs/14/app/building-your-application/rendering/server-components), [transmisión en flujo (streaming)](/docs/14/app/building-your-application/routing/loading-ui-and-streaming) y [obtención de datos colocalizada (colocated data fetching)](/docs/14/app/building-your-application/data-fetching).

Usar el directorio `app` habilitará automáticamente el [Modo Estricto de React (React Strict Mode)](https://react.dev/reference/react/StrictMode). Aprende cómo [adoptar `app` incrementalmente](/docs/14/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app).
