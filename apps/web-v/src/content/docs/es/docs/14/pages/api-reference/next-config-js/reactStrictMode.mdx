> **Es bueno saberlo**: Desde Next.js 13.4, el Modo Estricto es `true` por defecto con el enrutador `app`, por lo que la configuración anterior solo es necesaria para `pages`. Aún puedes desactivar el Modo Estricto configurando `reactStrictMode: false`.

> **Recomendación**: Te sugerimos encarecidamente que actives el Modo Estricto en tu aplicación Next.js para preparar mejor tu aplicación para el futuro de React.

El [Modo Estricto (Strict Mode)](https://react.dev/reference/react/StrictMode) de React es una función solo para modo desarrollo que resalta problemas potenciales en una aplicación. Ayuda a identificar ciclos de vida inseguros, uso de APIs heredadas y otras características.

El runtime de Next.js cumple con el Modo Estricto. Para activar el Modo Estricto, configura la siguiente opción en tu `next.config.js`:

```js filename="next.config.js"
module.exports = {
  reactStrictMode: true,
}
```

Si tú o tu equipo no están listos para usar el Modo Estricto en toda la aplicación, ¡no hay problema! Puedes migrar gradualmente página por página usando `<React.StrictMode>`.
