Puedes especificar un nombre para usar un directorio de compilación personalizado en lugar de `.next`.

Abre `next.config.js` y agrega la configuración `distDir`:

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

Ahora, si ejecutas `next build`, Next.js usará `build` en lugar de la carpeta predeterminada `.next`.

> `distDir` **no debe** salir del directorio de tu proyecto. Por ejemplo, `../build` es un directorio **inválido**.
