Pour apprendre Next.js efficacement, il est utile de maîtriser JavaScript, React et les concepts associés au développement web. Mais JavaScript et React sont des sujets vastes. Comment savoir quand vous êtes prêt à utiliser Next.js ?

Bienvenue dans le cours Fondamentaux React ! Ce cours adapté aux débutants, basé sur des exemples, vous guidera à travers les connaissances prérequises pour Next.js. Vous construirez un projet simple étape par étape ; en commençant par une application JavaScript, puis en la migrant vers React et Next.js.

Chaque section s'appuie sur la précédente, vous pouvez donc choisir où commencer en fonction de ce que vous connaissez déjà.

### [Connaissances prérequises](#prerequisite-knowledge)

Ce tutoriel suppose une connaissance du HTML, CSS, JavaScript, et aucune connaissance de React. Si vous êtes déjà familier avec React, vous pouvez passer directement au chapitre [De React à Next.js](/learn/react-foundations/from-react-to-nextjs) ou suivre le cours [Application Dashboard](/learn/dashboard-app).

### [Configuration requise](#system-requirements)

Avant de commencer ce cours, assurez-vous d'avoir installé :

*   Node.js 20.12.0 ou une version ultérieure.
*   Systèmes d'exploitation : macOS, Windows (y compris WSL), ou Linux.
*   VSCode ou un autre éditeur de texte de votre choix.
