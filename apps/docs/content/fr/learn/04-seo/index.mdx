[Qu'est-ce que le SEO ?](#what-is-seo)
--------------------------------------

SEO signifie **Optimisation pour les moteurs de recherche (Search Engine Optimization)**. L'objectif du SEO est de créer une stratégie qui améliorera votre positionnement dans les résultats des moteurs de recherche. Plus le classement est élevé, plus le trafic organique vers votre site est important, ce qui se traduit finalement par plus d'activité pour vous !

### [Ce que vous apprendrez dans ce cours](#what-youll-learn-in-this-course)

Dans ce cours, nous aborderons :

*   Les systèmes de recherche et les robots d'indexation comme Googlebot.
*   L'impact qu'une stratégie SEO peut avoir sur votre site.
*   Comment intégrer les meilleures pratiques SEO.
*   L'exploration, l'indexation, le rendu et le classement avec Next.js.
*   Les sujets liés aux performances web, y compris les Core Web Vitals.
