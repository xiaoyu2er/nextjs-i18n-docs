Dans cette leçon, nous allons passer en revue les différentes métriques, l'impact que les [Core Web Vitals](https://web.dev/vitals/) peuvent avoir sur votre SEO, et leur importance pour l'expérience utilisateur.

Il existe trois valeurs différentes lors de la mesure des Core Web Vitals :

**"Bon"**, **"À améliorer"**, et **"Médiocre"**. Ces valeurs diffèrent selon le vital mesuré :

![Core Web Vitals](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/seo/vitals-light.png)

Vous pouvez aborder les Core Web Vitals de deux manières différentes :

*   **Essayer d'obtenir le score le plus élevé possible pour chaque métrique**. Viser la perfection est excellent, mais cela peut être difficile sur des sites web volumineux avec de nombreuses dépendances.
*   **Établir des benchmarks par rapport aux concurrents de votre secteur**. Vous ne rivalisez pas avec tous les sites web parfaitement optimisés dans les résultats de recherche Google, mais avec ceux qui sont classés pour vos mots-clés cibles.
