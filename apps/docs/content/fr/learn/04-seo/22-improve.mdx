Voyons comment améliorer les Core Web Vitals de notre exemple en utilisant les fonctionnalités de Next.js.

Dans cette leçon, vous apprendrez :

*   Ce qu'est Lighthouse et comment l'utiliser.
*   Comment utiliser `next/image` pour optimiser automatiquement les images.
*   Comment importer dynamiquement des bibliothèques et des composants pour réduire votre bundle JS initial.
*   Comment préconnecter des scripts tiers.
*   Comment Next.js optimise par défaut le chargement des polices web.
*   Comment optimiser le chargement de n'importe quel script tiers.
