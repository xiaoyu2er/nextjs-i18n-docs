Essayons de modifier la page de démarrage.

*   Assurez-vous que le serveur de développement Next.js est toujours en cours d'exécution.
*   Ouvrez `pages/index.js` avec votre éditeur de texte.
*   Trouvez le texte **"Welcome to"** sous la balise `<h1>` et remplacez-le par **"Learn"**.
*   Enregistrez le fichier.

Dès que vous enregistrez le fichier, le navigateur met automatiquement à jour la page avec le nouveau texte :

![Learn Next.js](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/create-nextjs-app/learn-nextjs.png)

Le serveur de développement Next.js a activé la fonctionnalité [Fast Refresh](/docs/basic-features/fast-refresh). Lorsque vous modifiez des fichiers, Next.js applique automatiquement les changements dans le navigateur presque instantanément. Aucun rafraîchissement nécessaire ! Cela vous aidera à itérer rapidement sur votre application.
