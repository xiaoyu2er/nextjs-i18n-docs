Nous souhaitons créer un blog (voici [le résultat souhaité](https://next-learn-starter.vercel.app/)), mais jusqu'à présent, nous n'avons ajouté aucun contenu de blog. Dans cette leçon, nous allons apprendre comment récupérer des données externes de blog dans notre application. Nous stockerons le contenu du blog dans le système de fichiers, mais cela fonctionnera également si le contenu est stocké ailleurs (par exemple, dans une base de données ou un [CMS Headless](https://fr.wikipedia.org/wiki/Syst%C3%A8me_de_gestion_de_contenu_headless)).

### [Ce que vous apprendrez dans cette leçon](#what-youll-learn-in-this-lesson)

Dans cette leçon, vous apprendrez :

*   La fonctionnalité de [pré-rendu](/docs/basic-features/pages#pre-rendering) de Next.js.
*   Les deux formes de pré-rendu : [Génération statique](/docs/basic-features/pages#static-generation-recommended) et [Rendu côté serveur](/docs/basic-features/pages#server-side-rendering).
*   La génération statique [avec données](/docs/basic-features/pages#static-generation-with-data) et [sans données](/docs/basic-features/pages#static-generation-without-data).
*   [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) et comment l'utiliser pour importer des données externes de blog dans la page d'accueil.
*   Quelques informations utiles sur [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation).
