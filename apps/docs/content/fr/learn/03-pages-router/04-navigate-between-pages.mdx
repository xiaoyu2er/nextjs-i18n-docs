Jusqu'à présent, l'application Next.js que nous avons créée ne comporte qu'une seule page. Les sites web et applications web ont généralement de nombreuses pages différentes.

Voyons comment ajouter plus de pages à notre application.

[Ce que vous allez apprendre dans cette leçon](#what-youll-learn-in-this-lesson)
--------------------------------------------------------------------

Dans cette leçon, vous allez :

* Créer une nouvelle page en utilisant le [routage par système de fichiers intégré](/docs/routing/introduction).
* Apprendre à utiliser le composant [`Link`](https://nextjs.org/docs/api-reference/next/link) pour activer la navigation côté client entre les pages.
* Découvrir le support intégré pour le découpage de code (code splitting) et le préchargement (prefetching).

> Si vous recherchez une documentation détaillée sur le routage dans Next.js, consultez la [documentation sur le routage](/docs/routing/introduction).
