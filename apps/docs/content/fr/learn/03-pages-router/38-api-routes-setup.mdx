### [Télécharger le code de démarrage (Optionnel)](#download-starter-code-optional)

Si vous NE continuez PAS depuis la leçon précédente, vous pouvez télécharger, installer et exécuter le code de démarrage pour cette leçon ci-dessous. Cela configure un répertoire `nextjs-blog` identique au résultat de la leçon précédente.

Encore une fois, ceci N'est PAS nécessaire si vous venez de terminer la leçon précédente.

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/api-routes-starter"
```

Suivez ensuite les instructions de la sortie de commande (`cd` dans le répertoire et démarrez le serveur de développement).

Vous devriez également mettre à jour les fichiers suivants :

*   `public/images/profile.jpg` avec votre photo (Recommandé : 400px de largeur/hauteur).
*   `const name = '[Votre Nom]'` dans `components/layout.js` avec votre nom.
*   `<p>[Votre Introduction]</p>` dans `pages/index.js` avec votre présentation.
