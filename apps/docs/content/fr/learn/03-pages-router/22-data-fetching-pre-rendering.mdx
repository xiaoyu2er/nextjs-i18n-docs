Avant d'aborder [la récupération de données](/docs/basic-features/data-fetching), parlons de l'un des concepts les plus importants dans Next.js : le [**pré-rendu**](/docs/basic-features/pages#pre-rendering).

Par défaut, Next.js effectue un pré-rendu de chaque page. Cela signifie que Next.js _génère du HTML pour chaque page à l'avance_, plutôt que de tout faire via JavaScript côté client. Le pré-rendu peut améliorer les performances et le [SEO](https://fr.wikipedia.org/wiki/Optimisation_pour_les_moteurs_de_recherche).

Chaque HTML généré est associé au code JavaScript minimal nécessaire pour cette page. Lorsqu'une page est chargée par le navigateur, son code JavaScript s'exécute et rend la page entièrement interactive. (Ce processus s'appelle **l'hydratation**.)

### [Vérifier que le pré-rendu a lieu](#check-that-pre-rendering-is-happening)

Vous pouvez vérifier que le pré-rendu est effectué en suivant ces étapes :

1.  Désactivez JavaScript dans votre navigateur. ([Voici comment faire dans Chrome](https://developer.chrome.com/docs/devtools/javascript/disable/)).
2.  [Essayez d'accéder à cette page](https://next-learn-starter.vercel.app/) (le résultat final de ce tutoriel).

Vous devriez voir que votre application s'affiche sans JavaScript. C'est parce que Next.js a pré-rendu l'application en HTML statique, vous permettant de voir l'interface sans exécuter JavaScript.

> **Remarque** : Vous pouvez aussi essayer ces étapes sur `localhost`, mais le CSS ne sera pas chargé si vous désactivez JavaScript.

Si votre application est une simple application React.js (sans Next.js), il n'y a pas de [pré-rendu](/docs/basic-features/pages#pre-rendering), donc vous ne pourrez pas voir l'application si vous désactivez JavaScript. Par exemple :

*   Activez JavaScript dans votre navigateur et [consultez cette page](https://create-react-template.vercel.app/). Il s'agit d'une simple application React.js construite avec [Create React App](https://create-react-app.dev/).
*   Maintenant, désactivez JavaScript et accédez à [la même page](https://create-react-template.vercel.app/) à nouveau.
*   Vous ne verrez plus l'application - à la place, un message indiquera "Vous devez activer JavaScript pour exécuter cette application." C'est parce que l'application n'est pas pré-rendue en HTML statique.
