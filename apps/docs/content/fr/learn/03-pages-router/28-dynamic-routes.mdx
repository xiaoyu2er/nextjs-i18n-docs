Nous avons rempli la page d'index avec les données du blog, mais nous n'avons pas encore créé de pages individuelles pour chaque article de blog (voici le [résultat souhaité](https://next-learn-starter.vercel.app/)). Nous voulons que l'URL de ces pages dépende des données du blog, ce qui signifie que nous devons utiliser des [routes dynamiques](/docs/routing/dynamic-routes).

### [Ce que vous apprendrez dans cette leçon](#what-youll-learn-in-this-lesson)

Dans cette leçon, vous apprendrez :

*   Comment générer statiquement des pages avec des [routes dynamiques](/docs/routing/dynamic-routes) en utilisant [`getStaticPaths`](/docs/basic-features/data-fetching#getstaticpaths-static-generation).
*   Comment écrire [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) pour récupérer les données de chaque article de blog.
*   Comment afficher du markdown en utilisant [`remark`](https://github.com/remarkjs/remark).
*   Comment formater joliment des dates.
*   Comment créer un lien vers une page avec des [routes dynamiques](/docs/routing/dynamic-routes).
*   Quelques informations utiles sur les [routes dynamiques](/docs/routing/dynamic-routes).
