Next.js peut être déployé sur n'importe quel fournisseur d'hébergement prenant en charge Node.js.

Si vous avez suivi les instructions jusqu'à présent, votre fichier `package.json` devrait contenir les scripts `build` et `start` suivants :

```
{
  "scripts": {
    "dev": "next",
    "build": "next build",
    "start": "next start"
  }
}
```

Sur votre propre fournisseur d'hébergement, exécutez le script `build` une fois, ce qui construit l'application de production dans le dossier `.next`.

```
npm run build
```

Après la construction, le script `start` démarre un serveur Node.js qui prend en charge les pages hybrides, servant à la fois les pages générées statiquement et rendues côté serveur, ainsi que les routes API.

```
npm run start
```

> **Astuce** : Vous pouvez personnaliser le script `start` dans `package.json` pour accepter un paramètre `PORT` en le mettant à jour comme suit : `"start": "next start -p $PORT"`.
