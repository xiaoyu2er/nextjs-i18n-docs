Tout d'abord, assurons-nous que votre environnement de développement est prêt.

*   Si vous n'avez pas **Node.js** installé, [installez-le depuis ici](https://nodejs.org/en/). Vous aurez besoin de Node.js version **18** ou supérieure.
*   Vous utiliserez votre propre éditeur de texte et terminal pour ce tutoriel.

> Si vous êtes sous Windows, nous recommandons [de télécharger Git pour Windows](https://gitforwindows.org/) et d'utiliser Git Bash qui l'accompagne, car il prend en charge les commandes spécifiques à UNIX utilisées dans ce tutoriel. [Windows Subsystem for Linux (WSL)](https://docs.microsoft.com/en-us/windows/wsl/install-win10) est une autre option.

### [Créer une application Next.js](#create-a-nextjs-app)

Pour créer une application Next.js, ouvrez votre terminal, naviguez (`cd`) dans le répertoire où vous souhaitez créer l'application, puis exécutez la commande suivante :

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/learn-starter"
```

> En arrière-plan, cela utilise l'outil appelé [`create-next-app`](/docs/api-reference/create-next-app), qui initialise une application Next.js pour vous. Il utilise [ce modèle](https://github.com/vercel/next-learn/tree/main/basics/learn-starter) via le drapeau `--example`.
> 
> Si cela ne fonctionne pas, veuillez consulter [cette page](https://github.com/vercel/next-learn/blob/main/basics/errors/install.md).

### [Lancer le serveur de développement](#run-the-development-server)

Vous avez maintenant un nouveau répertoire appelé `nextjs-blog`. Naviguez (`cd`) dedans :

```
cd nextjs-blog
```

Ensuite, exécutez la commande suivante :

```
npm run dev
```

Ceci démarre le "serveur de développement" de votre application Next.js (nous y reviendrons plus tard) sur le port **3000**.

Vérifions si cela fonctionne. Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.
