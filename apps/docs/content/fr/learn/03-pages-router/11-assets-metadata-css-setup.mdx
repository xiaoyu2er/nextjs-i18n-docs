### [Télécharger le code de départ (Optionnel)](#download-starter-code-optional)

Si vous NE continuez PAS depuis la leçon précédente, vous pouvez télécharger, installer et exécuter le code de départ pour cette leçon ci-dessous. Cela configure un répertoire `nextjs-blog` de manière identique au résultat de la leçon précédente.

Encore une fois, ceci N'est PAS nécessaire si vous venez de terminer la leçon précédente.

```
npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/assets-metadata-css-starter"
```

Puis suivez les instructions du résultat de la commande (`cd` dans le répertoire et démarrez le serveur de développement).
