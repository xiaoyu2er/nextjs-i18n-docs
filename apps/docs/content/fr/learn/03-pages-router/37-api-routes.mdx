Next.js prend en charge les [Routes API](/docs/api-routes/introduction), qui vous permettent de créer facilement un point de terminaison d'API sous forme de fonction serverless Node.js. Bien que ce ne soit pas nécessaire pour notre application de blog, nous aborderons brièvement comment l'utiliser dans cette leçon.

### [Ce que vous apprendrez dans cette leçon](#what-youll-learn-in-this-lesson)

Dans cette leçon, vous apprendrez :

*   Comment créer une Route API.
*   Quelques informations utiles sur les [Routes API](/docs/api-routes/introduction).
