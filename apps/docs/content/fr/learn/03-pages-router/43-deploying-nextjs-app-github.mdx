Avant de déployer, poussons notre application Next.js vers [GitHub](https://github.com/vercel/next.js) si vous ne l'avez pas déjà fait. Cela facilitera le déploiement.

*   Sur votre compte GitHub personnel, créez un nouveau dépôt appelé `nextjs-blog`.
*   Le dépôt peut être public ou privé. Vous n'avez **pas** besoin de l'initialiser avec un README ou d'autres fichiers.
*   Si vous avez besoin d'aide pour configurer votre dépôt, consultez [ce guide sur GitHub](https://help.github.com/en/github/getting-started-with-github/create-a-repo).

Ensuite :

*   Si vous n'avez pas initialisé le dépôt git localement pour votre application Next.js, faites-le maintenant.
*   Poussez l'application Next.js vers votre dépôt GitHub.

Pour pousser vers GitHub, vous pouvez exécuter les commandes suivantes (remplacez `<username>` par votre nom d'utilisateur GitHub) :

```
git remote add origin https://github.com/<username>/nextjs-blog.git
git push -u origin main
```

Une fois que votre dépôt GitHub est prêt, passez à la page suivante.
