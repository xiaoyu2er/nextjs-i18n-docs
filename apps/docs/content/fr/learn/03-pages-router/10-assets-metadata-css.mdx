La deuxième page que nous avons ajoutée n'a actuellement aucun style. Ajoutons un peu de CSS pour styliser la page.

Next.js prend en charge nativement [CSS](/docs/basic-features/built-in-css-support) et [Sass](/docs/basic-features/built-in-css-support#sass-support). Pour ce cours, nous utiliserons CSS.

Cette leçon abordera également la manière dont Next.js gère les ressources statiques comme les images et les métadonnées de page telles que la balise `<title>`.

### [Ce que vous apprendrez dans cette leçon](#what-youll-learn-in-this-lesson)

Dans cette leçon, vous apprendrez :

*   Comment ajouter des [fichiers statiques](/docs/basic-features/static-file-serving) (images, etc.) à Next.js.
*   Comment personnaliser le contenu de la balise `<head>` pour chaque page.
*   Comment créer un composant React réutilisable stylisé avec [CSS Modules](/docs/basic-features/built-in-css-support#adding-component-level-css).
*   Comment [ajouter du CSS global](/docs/basic-features/built-in-css-support#adding-a-global-stylesheet) dans [`pages/_app.js`](/docs/advanced-features/custom-app).
*   Quelques astuces utiles pour le style dans Next.js.

### [Prérequis](#prerequisites)

*   Connaissances de base en CSS. Ce cours expliquera comment ajouter du CSS dans une application Next.js, mais il ne couvrira pas les fondamentaux du CSS.

> Si vous recherchez une documentation détaillée sur le style dans Next.js, consultez [la documentation CSS](/docs/basic-features/built-in-css-support).
