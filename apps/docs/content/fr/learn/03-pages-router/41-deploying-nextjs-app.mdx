Dans cette dernière leçon de base, nous allons déployer notre application Next.js en production.

Nous apprendrons comment déployer Next.js sur [Vercel](https://vercel.com), la plateforme créée par les auteurs de Next.js. Nous aborderons également d'autres options de déploiement.

> **Prérequis** : Vous devez avoir un [compte GitHub](https://github.com/) pour cette leçon.

### [Ce que vous apprendrez dans cette leçon](#what-youll-learn-in-this-lesson)

Dans cette leçon, vous apprendrez :

*   Comment déployer votre application Next.js sur [Vercel](https://vercel.com).
*   Le workflow **DPS** : **D**évelopper, **P**révisualiser et **S**hipper.
*   Comment déployer votre application Next.js sur votre propre hébergeur.
