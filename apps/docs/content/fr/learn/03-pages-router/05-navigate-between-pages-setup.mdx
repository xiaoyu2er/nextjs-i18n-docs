### [Télécharger le code de démarrage (Optionnel)](#download-starter-code-optional)

Si vous NE continuez PAS depuis la leçon précédente, vous pouvez télécharger, installer et exécuter le code de démarrage pour cette leçon ci-dessous. Cela configure un répertoire `nextjs-blog` de manière identique au résultat de la leçon précédente.

Encore une fois, ceci N'est PAS nécessaire si vous venez de terminer la leçon précédente.

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/navigate-between-pages-starter"
```

Suivez ensuite les instructions du résultat de la commande (`cd` dans le répertoire et démarrez le serveur de développement).
