Avant Next.js 13, le Routeur Pages (Pages Router) était la principale méthode pour créer des routes dans Next.js. Il utilisait un système de routage basé sur le système de fichiers pour associer chaque fichier à une route. Le Routeur Pages (Pages Router) est toujours pris en charge dans les versions plus récentes de Next.js, mais nous recommandons de migrer vers le nouveau [Routeur App (App Router)](/docs/app) pour bénéficier des dernières fonctionnalités de React.

Utilisez cette section de la documentation pour les applications existantes qui utilisent le Routeur Pages (Pages Router).
