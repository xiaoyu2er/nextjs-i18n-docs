title: Récupération de données
description: Next.js permet de récupérer des données de plusieurs manières, avec pré-rendu, rendu côté serveur ou génération de site statique, et régénération statique incrémentielle. Apprenez à gérer les données de votre application dans Next.js.
---

La récupération de données dans Next.js vous permet d'afficher votre contenu de différentes manières, selon les cas d'utilisation de votre application. Cela inclut le pré-rendu avec le **rendu côté serveur (SSR)** ou la **génération statique (Static Generation)**, ainsi que la mise à jour ou la création de contenu à l'exécution avec la **régénération statique incrémentielle (ISR)**.
