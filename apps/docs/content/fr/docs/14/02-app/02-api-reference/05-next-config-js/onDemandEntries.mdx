Next.js expose certaines options qui vous permettent de contrôler comment le serveur gère en mémoire les pages construites en développement.

Pour modifier les paramètres par défaut, ouvrez `next.config.js` et ajoutez la configuration `onDemandEntries` :

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // durée (en ms) pendant laquelle le serveur conserve les pages dans le tampon
    maxInactiveAge: 25 * 1000,
    // nombre de pages qui doivent être conservées simultanément sans être libérées
    pagesBufferLength: 2,
  },
}
```
