L'option expérimentale `instrumentationHook` vous permet de configurer l'instrumentation via le [fichier `instrumentation`](/docs/app/api-reference/file-conventions/instrumentation) dans votre application Next.js.

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
