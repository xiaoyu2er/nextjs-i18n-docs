Un fichier **loading** peut créer des états de chargement instantanés basés sur [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming).

Par défaut, ce fichier est un [Composant Serveur](/docs/app/building-your-application/rendering/server-components) - mais peut également être utilisé comme Composant Client via la directive `"use client"`.

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // Ou un composant skeleton de chargement personnalisé
  return <p>Chargement...</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // Ou un composant skeleton de chargement personnalisé
  return <p>Chargement...</p>
}
```

Les composants d'interface utilisateur de chargement n'acceptent aucun paramètre.

> **Bon à savoir**
>
> - Lors de la conception d'une interface utilisateur de chargement, vous pouvez trouver utile d'utiliser les [Outils de développement React](https://react.dev/learn/react-developer-tools) pour basculer manuellement les limites Suspense.
