La fonction `draftMode` permet de détecter le [Mode Brouillon (Draft Mode)](/docs/app/building-your-application/configuring/draft-mode) à l'intérieur d'un [Composant Serveur (Server Component)](/docs/app/building-your-application/rendering/server-components).

```jsx filename="app/page.js"
import { draftMode } from 'next/headers'

export default function Page() {
  const { isEnabled } = draftMode()
  return (
    <main>
      <h1>Mon article de blog</h1>
      <p>Le Mode Brouillon est actuellement {isEnabled ? 'Activé' : 'Désactivé'}</p>
    </main>
  )
}
```
