Next.js inclut **deux** outils d'interface en ligne de commande (CLI) :

- **`create-next-app`** : Crée rapidement une nouvelle application Next.js en utilisant le modèle par défaut ou un [exemple](https://github.com/vercel/next.js/tree/canary/examples) depuis un dépôt Github public.
- **`next`** : Exécute le serveur de développement Next.js, construit votre application et plus encore.
