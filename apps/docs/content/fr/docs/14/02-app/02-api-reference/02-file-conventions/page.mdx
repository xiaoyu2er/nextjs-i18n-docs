Une **page** est une interface utilisateur unique pour une route.

```tsx filename="app/blog/[slug]/page.tsx" switcher
export default function Page({
  params,
  searchParams,
}: {
  params: { slug: string }
  searchParams: { [key: string]: string | string[] | undefined }
}) {
  return <h1>Ma Page</h1>
}
```

```jsx filename="app/blog/[slug]/page.js" switcher
export default function Page({ params, searchParams }) {
  return <h1>Ma Page</h1>
}
```

## Props

### `params` (optionnel)

Un objet contenant les [paramètres de route dynamique](/docs/app/building-your-application/routing/dynamic-routes) depuis le segment racine jusqu'à cette page. Par exemple :

| Exemple                              | URL         | `params`                       |
| ------------------------------------ | ----------- | ------------------------------ |
| `app/shop/[slug]/page.js`            | `/shop/1`   | `{ slug: '1' }`                |
| `app/shop/[category]/[item]/page.js` | `/shop/1/2` | `{ category: '1', item: '2' }` |
| `app/shop/[...slug]/page.js`         | `/shop/1/2` | `{ slug: ['1', '2'] }`         |

### `searchParams` (optionnel)

Un objet contenant les [paramètres de recherche](https://developer.mozilla.org/fr/docs/Learn/Common_questions/Web_mechanics/What_is_a_URL#param%C3%A8tres) de l'URL actuelle. Par exemple :

| URL             | `searchParams`       |
| --------------- | -------------------- |
| `/shop?a=1`     | `{ a: '1' }`         |
| `/shop?a=1&b=2` | `{ a: '1', b: '2' }` |
| `/shop?a=1&a=2` | `{ a: ['1', '2'] }`  |

> **Bon à savoir** :
>
> - `searchParams` est une **[API Dynamique](/docs/app/building-your-application/rendering/server-components#server-rendering-strategies#dynamic-functions)** dont les valeurs ne peuvent pas être connues à l'avance. Son utilisation activera le **[rendu dynamique](/docs/app/building-your-application/rendering/server-components#dynamic-rendering)** au moment de la requête.
> - `searchParams` retourne un objet JavaScript simple et non une instance de `URLSearchParams`.
