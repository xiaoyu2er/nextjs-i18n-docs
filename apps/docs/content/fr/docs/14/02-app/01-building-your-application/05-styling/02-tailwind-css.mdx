[Tailwind CSS](https://tailwindcss.com/) est un framework CSS utility-first qui fonctionne exceptionnellement bien avec Next.js.

## Installation de Tailwind

Installez les packages Tailwind CSS et exécutez la commande `init` pour générer les fichiers `tailwind.config.js` et `postcss.config.js` :

```bash filename="Terminal"
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

## Configuration de Tailwind

Dans `tailwind.config.js`, ajoutez les chemins vers les fichiers qui utiliseront les classes Tailwind CSS :

```js filename="tailwind.config.js"
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './app/**/*.{js,ts,jsx,tsx,mdx}', // Notez l'ajout du répertoire `app`.
    './pages/**/*.{js,ts,jsx,tsx,mdx}',
    './components/**/*.{js,ts,jsx,tsx,mdx}',

    // Ou si vous utilisez le répertoire `src` :
    './src/**/*.{js,ts,jsx,tsx,mdx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

Vous n'avez pas besoin de modifier `postcss.config.js`.

## Importation des styles

Ajoutez les [directives Tailwind CSS](https://tailwindcss.com/docs/functions-and-directives#directives) que Tailwind utilisera pour injecter ses styles générés dans une [feuille de style globale](/docs/app/building-your-application/styling/css-modules#global-styles) de votre application, par exemple :

```css filename="app/globals.css"
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Dans le [layout racine](/docs/app/building-your-application/routing/pages-and-layouts#root-layout-required) (`app/layout.tsx`), importez la feuille de style `globals.css` pour appliquer les styles à chaque route de votre application.

```tsx filename="app/layout.tsx" switcher
import type { Metadata } from 'next'

// Ces styles s'appliquent à chaque route de l'application
import './globals.css'

export const metadata: Metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
}

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

```jsx filename="app/layout.js" switcher
// Ces styles s'appliquent à chaque route de l'application
import './globals.css'

export const metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
}

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

## Utilisation des classes

Après avoir installé Tailwind CSS et ajouté les styles globaux, vous pouvez utiliser les classes utilitaires de Tailwind dans votre application.

```tsx filename="app/page.tsx" switcher
export default function Page() {
  return <h1 className="text-3xl font-bold underline">Hello, Next.js!</h1>
}
```

## Importation des styles

Ajoutez les [directives Tailwind CSS](https://tailwindcss.com/docs/functions-and-directives#directives) que Tailwind utilisera pour injecter ses styles générés dans une [feuille de style globale](/docs/pages/building-your-application/styling/css-modules#global-styles) de votre application, par exemple :

```css filename="styles/globals.css"
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Dans le [fichier app personnalisé](/docs/pages/building-your-application/routing/custom-app) (`pages/_app.js`), importez la feuille de style `globals.css` pour appliquer les styles à chaque route de votre application.

```tsx filename="pages/_app.tsx" switcher
// Ces styles s'appliquent à chaque route de l'application
import '@/styles/globals.css'
import type { AppProps } from 'next/app'

export default function App({ Component, pageProps }: AppProps) {
  return <Component {...pageProps} />
}
```

```jsx filename="pages/_app.js" switcher
// Ces styles s'appliquent à chaque route de l'application
import '@/styles/globals.css'

export default function App({ Component, pageProps }) {
  return <Component {...pageProps} />
}
```
