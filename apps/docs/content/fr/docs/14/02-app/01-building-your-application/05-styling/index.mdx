Next.js prend en charge différentes méthodes pour styliser votre application, notamment :

- **CSS Global** : Simple à utiliser et familier pour ceux ayant de l'expérience avec le CSS traditionnel, mais peut entraîner des fichiers CSS plus volumineux et des difficultés de gestion des styles à mesure que l'application grandit.
- **Modules CSS** : Créez des classes CSS localement scópées pour éviter les conflits de noms et améliorer la maintenabilité.
- **Tailwind CSS** : Un framework CSS utility-first qui permet des conceptions personnalisées rapides en composant des classes utilitaires.
- **Sass** : Un préprocesseur CSS populaire qui étend CSS avec des fonctionnalités comme les variables, les règles imbriquées et les mixins.
- **CSS-in-JS** : Intégrez directement le CSS dans vos composants JavaScript, permettant un style dynamique et scopé.
