Par défaut, Next.js génère des [etags](https://fr.wikipedia.org/wiki/HTTP_ETag) pour chaque page. Vous pouvez désactiver la génération d'etags pour les pages HTML selon votre stratégie de cache.

Ouvrez `next.config.js` et désactivez l'option `generateEtags` :

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
