Les options `staticGeneration*` vous permettent de configurer le processus de génération statique pour des cas d'utilisation avancés.

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
const nextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

## Options de configuration

Les options suivantes sont disponibles :

- `staticGenerationRetryCount` : Le nombre de tentatives de régénération d'une page en cas d'échec avant d'abandonner la construction.
- `staticGenerationMaxConcurrency` : Le nombre maximum de pages à traiter par worker.
- `staticGenerationMinPagesPerWorker` : Le nombre minimum de pages à traiter avant de démarrer un nouveau worker.
