L'option de configuration `authInterrupts` vous permet d'utiliser les API [`forbidden`](/docs/app/api-reference/functions/forbidden) et [`unauthorized`](/docs/app/api-reference/functions/unauthorized) dans votre application. Bien que ces fonctions soient expérimentales, vous devez activer l'option `authInterrupts` dans votre fichier `next.config.js` pour les utiliser :

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    authInterrupts: true,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
module.exports = {
  experimental: {
    authInterrupts: true,
  },
}
```
