[Tailwind CSS](https://tailwindcss.com/) est un framework CSS utility-first entièrement compatible avec Next.js. Ce guide vous explique comment installer Tailwind CSS dans votre application Next.js.

## Installation de Tailwind

Installez les paquets nécessaires de Tailwind CSS :

```bash filename="Terminal"
npm install -D tailwindcss @tailwindcss/postcss postcss
```

> **Bon à savoir** : Si vous utilisez l'interface en ligne de commande `create-next-app` pour créer votre projet, Next.js vous demandera si vous souhaitez installer Tailwind et configurera automatiquement le projet.

## Configuration de Tailwind

Créez un fichier `postcss.config.mjs` à la racine de votre projet et ajoutez le plugin `@tailwindcss/postcss` à votre configuration PostCSS :

```js filename="postcss.config.mjs" highlight={4}
/** @type {import('tailwindcss').Config} */
export default {
  plugins: {
    '@tailwindcss/postcss': {},
  },
}
```

Depuis [Tailwind v4](https://tailwindcss.com/blog/tailwindcss-v4), aucune configuration n'est requise par défaut. Si vous avez besoin de configurer Tailwind, vous pouvez suivre la [documentation officielle](https://tailwindcss.com/blog/tailwindcss-v4#css-first-configuration) pour configurer le fichier CSS global.

Il existe également un [CLI de mise à niveau](https://tailwindcss.com/docs/upgrade-guide) et un [guide](https://tailwindcss.com/docs/upgrade-guide) si vous avez un projet existant avec Tailwind v3.

## Importation des styles

Ajoutez les [directives Tailwind CSS](https://tailwindcss.com/docs/functions-and-directives#directives) que Tailwind utilisera pour injecter ses styles générés dans une [feuille de style globale](/docs/app/getting-started/css#global-css) de votre application, par exemple :

```css filename="app/globals.css"
@import 'tailwindcss';
```

Dans le [layout racine](/docs/app/api-reference/file-conventions/layout#root-layout) (`app/layout.tsx`), importez la feuille de style `globals.css` pour appliquer les styles à chaque route de votre application.

```tsx filename="app/layout.tsx" switcher
import type { Metadata } from 'next'

// Ces styles s'appliquent à chaque route de l'application
import './globals.css'

export const metadata: Metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
}

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

```jsx filename="app/layout.js" switcher
// Ces styles s'appliquent à chaque route de l'application
import './globals.css'

export const metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
}

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

## Utilisation des classes

Après avoir installé Tailwind CSS et ajouté les styles globaux, vous pouvez utiliser les classes utilitaires de Tailwind dans votre application.

```tsx filename="app/page.tsx" switcher

  return <h1 className="text-3xl font-bold underline">Hello, Next.js!</h1>
}
```

## Importation des styles

Ajoutez les [directives Tailwind CSS](https://tailwindcss.com/docs/functions-and-directives#directives) que Tailwind utilisera pour injecter ses styles générés dans une [feuille de style globale](/docs/app/getting-started/css#global-css) de votre application, par exemple :

```css filename="styles/globals.css"
@import 'tailwindcss';
```

Dans le [fichier d'application personnalisé](/docs/pages/building-your-application/routing/custom-app) (`pages/_app.js`), importez la feuille de style `globals.css` pour appliquer les styles à chaque route de votre application.

```tsx filename="pages/_app.tsx" switcher
// Ces styles s'appliquent à chaque route de l'application
import '@/styles/globals.css'
import type { AppProps } from 'next/app'

export default function App({ Component, pageProps }: AppProps) {
  return <Component {...pageProps} />
}
```

```jsx filename="pages/_app.js" switcher
// Ces styles s'appliquent à chaque route de l'application
import '@/styles/globals.css'

export default function App({ Component, pageProps }) {
  return <Component {...pageProps} />
}
```
