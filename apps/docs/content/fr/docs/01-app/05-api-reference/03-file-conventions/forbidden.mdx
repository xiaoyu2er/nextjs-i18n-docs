Le fichier **forbidden** est utilisé pour afficher l'interface utilisateur lorsque la fonction [`forbidden`](/docs/app/api-reference/functions/forbidden) est appelée pendant l'authentification. En plus de vous permettre de personnaliser l'interface, Next.js renverra un code d'état `403`.

```tsx filename="app/forbidden.tsx" switcher
import Link from 'next/link'

export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>
      <p>You are not authorized to access this resource.</p>
      <Link href="/">Return Home</Link>
    </div>
  )
}
```

```jsx filename="app/forbidden.jsx" switcher

```

## Référence

### Props

Les composants `forbidden.js` n'acceptent aucune prop.
