Utilisez l'option `crossOrigin` pour ajouter un [attribut `crossOrigin`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) à toutes les balises `<script>` générées par le <AppOnly>composant [`next/script`](/docs/app/guides/scripts)</AppOnly> <PagesOnly>composants [`next/script`](/docs/pages/guides/scripts) et [`next/head`](/docs/pages/api-reference/components/head)</PagesOnly>, et définir comment les requêtes cross-origin doivent être gérées.

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## Options

- `'anonymous'` : Ajoute l'attribut [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous).
- `'use-credentials'` : Ajoute [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials).
