Next.js peut transpiler et regrouper automatiquement les dépendances des packages locaux (comme les monorepos) ou des dépendances externes (`node_modules`). Cela remplace le package `next-transpile-modules`.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['nom-du-package'],
}

module.exports = nextConfig
```
