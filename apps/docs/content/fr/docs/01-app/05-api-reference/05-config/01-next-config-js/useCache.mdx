Le flag `useCache` est une fonctionnalité expérimentale de Next.js qui permet d'utiliser la [directive `use cache`](/docs/app/api-reference/directives/use-cache) indépendamment de [`dynamicIO`](/docs/app/api-reference/config/next-config-js/dynamicIO). Lorsqu'il est activé, vous pouvez utiliser `use cache` dans votre application même si `dynamicIO` est désactivé.

## Utilisation

Pour activer le flag `useCache`, définissez-le sur `true` dans la section `experimental` de votre fichier `next.config.ts` :

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

Lorsque `useCache` est activé, vous pouvez utiliser les fonctions et configurations de cache suivantes :

- La [directive `use cache`](/docs/app/api-reference/directives/use-cache)
- La fonction [`cacheLife`](/docs/app/api-reference/config/next-config-js/cacheLife) avec `use cache`
- La fonction [`cacheTag`](/docs/app/api-reference/functions/cacheTag)
