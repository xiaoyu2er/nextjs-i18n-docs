Support expérimental pour les [liens typés statiquement](/docs/app/api-reference/config/typescript#statically-typed-links). Cette fonctionnalité nécessite d'utiliser le routeur App ainsi que TypeScript dans votre projet.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
