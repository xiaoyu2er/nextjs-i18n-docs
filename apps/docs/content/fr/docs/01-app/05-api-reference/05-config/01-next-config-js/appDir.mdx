> **Bon à savoir** : Cette option n'est **plus nécessaire** depuis Next.js 13.4. Le routeur d'application est désormais stable.

Le routeur d'application ([répertoire `app`](/docs/app/building-your-application/routing)) permet d'utiliser les [mises en page (layouts)](/docs/app/api-reference/file-conventions/layout), les [composants serveur (Server Components)](/docs/app/getting-started/server-and-client-components), le [streaming](/docs/app/api-reference/file-conventions/loading) et la [récupération de données colocalisée (colocated data fetching)](/docs/app/building-your-application/data-fetching).

L'utilisation du répertoire `app` activera automatiquement le [mode strict React (React Strict Mode)](https://react.dev/reference/react/StrictMode). Apprenez comment [adopter `app` de manière progressive](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app).
