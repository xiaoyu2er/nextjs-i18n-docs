> **Bon à savoir** : Depuis Next.js 13.5.1, le Mode Strict est activé par défaut (`true`) avec le routeur `app`, donc la configuration ci-dessus est uniquement nécessaire pour `pages`. Vous pouvez toujours désactiver le Mode Strict en définissant `reactStrictMode: false`.

> **Recommandation** : Nous vous conseillons vivement d'activer le Mode Strict dans votre application Next.js pour mieux préparer votre application à l'avenir de React.

Le [Mode Strict](https://react.dev/reference/react/StrictMode) de React est une fonctionnalité réservée au mode développement qui met en évidence les problèmes potentiels d'une application. Il aide à identifier les cycles de vie non sécurisés, l'utilisation d'API obsolètes et plusieurs autres fonctionnalités.

Le runtime Next.js est conforme au Mode Strict. Pour l'activer, configurez l'option suivante dans votre `next.config.js` :

```js filename="next.config.js"
module.exports = {
  reactStrictMode: true,
}
```

Si vous ou votre équipe n'êtes pas prêts à utiliser le Mode Strict dans toute votre application, ce n'est pas grave ! Vous pouvez migrer progressivement page par page en utilisant `<React.StrictMode>`.
