{/* Le contenu de ce document est partagé entre l'App Router et le Pages Router. Vous pouvez utiliser le composant `<PagesOnly>Content</PagesOnly>` pour ajouter du contenu spécifique au Pages Router. Tout contenu partagé ne doit pas être encapsulé dans un composant. */}

Par défaut, Next.js ajoute l'en-tête `x-powered-by`. Pour le désactiver, ouvrez le fichier `next.config.js` et désactivez la configuration `poweredByHeader` :

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
