Les dépendances utilisées dans les [composants serveur](/docs/app/getting-started/server-and-client-components) et les [gestionnaires de route](/docs/app/building-your-application/routing/route-handlers) seront automatiquement incluses dans le bundle par Next.js.

Si une dépendance utilise des fonctionnalités spécifiques à Node.js, vous pouvez choisir d'exclure des dépendances spécifiques du bundle des composants serveur et utiliser le `require` natif de Node.js.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
