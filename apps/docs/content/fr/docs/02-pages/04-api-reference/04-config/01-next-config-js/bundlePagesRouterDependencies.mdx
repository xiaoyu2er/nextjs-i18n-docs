---

Active le regroupement automatique des dépendances côté serveur pour les applications utilisant le routeur Pages. Correspond au regroupement automatique des dépendances dans le routeur App.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  bundlePagesRouterDependencies: true,
}

module.exports = nextConfig
```

Exclure explicitement certains paquets du regroupement en utilisant l'option [`serverExternalPackages`](/docs/pages/api-reference/config/next-config-js/serverExternalPackages).
