Par défaut, Next.js redirige les URLs avec une barre oblique de fin vers leur équivalent sans barre oblique. Par exemple, `/about/` sera redirigé vers `/about`. Vous pouvez configurer ce comportement pour qu'il fonctionne de manière inverse, où les URLs sans barre oblique sont redirigées vers leur équivalent avec barre oblique.

Ouvrez `next.config.js` et ajoutez la configuration `trailingSlash` :

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

Avec cette option activée, les URLs comme `/about` seront redirigées vers `/about/`.
