Next.js fournit une compression [**gzip**](https://tools.ietf.org/html/rfc6713#section-3) pour compresser le contenu rendu et les fichiers statiques. En général, vous voudrez activer la compression sur un proxy HTTP comme [nginx](https://www.nginx.com/), afin de décharger la charge du processus `Node.js`.

Pour désactiver la **compression**, ouvrez `next.config.js` et désactivez la configuration `compress` :

```js filename="next.config.js"
module.exports = {
  compress: false,
}
```
