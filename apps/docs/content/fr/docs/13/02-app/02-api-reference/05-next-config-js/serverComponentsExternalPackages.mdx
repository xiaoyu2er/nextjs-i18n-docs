Les dépendances utilisées dans les [composants serveur (Server Components)](/docs/app/building-your-application/rendering/server-components) et les [gestionnaires de route (Route Handlers)](/docs/app/building-your-application/routing/route-handlers) sont automatiquement regroupées par Next.js.

Si une dépendance utilise des fonctionnalités spécifiques à Node.js, vous pouvez choisir d'exclure certaines dépendances du bundling des composants serveur et utiliser le `require` natif de Node.js.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    serverComponentsExternalPackages: ['@acme/ui'],
  },
}

module.exports = nextConfig
```
