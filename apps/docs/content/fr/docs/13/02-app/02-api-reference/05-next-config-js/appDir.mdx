> **Bon à savoir** : Cette option **n'est plus** nécessaire depuis Next.js 13.4. Le routeur d'application (App Router) est désormais stable.

Le routeur d'application ([répertoire `app`](/docs/app/building-your-application/routing)) permet d'utiliser les [mises en page (layouts)](/docs/app/building-your-application/routing/pages-and-layouts), les [composants serveur (Server Components)](/docs/app/building-your-application/rendering/server-components), le [streaming](/docs/app/building-your-application/routing/loading-ui-and-streaming) et la [récupération de données colocalisée (colocated data fetching)](/docs/app/building-your-application/data-fetching).

L'utilisation du répertoire `app` activera automatiquement le [mode strict React (React Strict Mode)](https://react.dev/reference/react/StrictMode). Apprenez comment [adopter progressivement `app`](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app).
