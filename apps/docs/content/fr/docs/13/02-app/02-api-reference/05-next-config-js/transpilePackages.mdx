Next.js peut transpiler et regrouper automatiquement les dépendances de packages locaux (comme les monorepos) ou de dépendances externes (`node_modules`). Cela remplace le package `next-transpile-modules`.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['@acme/ui', 'lodash-es'],
}

module.exports = nextConfig
```
