Cette page fournit une vue d'ensemble de la structure des fichiers et dossiers d'un projet Next.js. Elle couvre les fichiers et dossiers de premier niveau, les fichiers de configuration, et les conventions de routage dans les répertoires `app` et `pages`.

## Dossiers de premier niveau

|                                                                          |                                    |
| ------------------------------------------------------------------------ | ---------------------------------- |
| [`app`](/docs/app/building-your-application/routing)                     | Routeur d'application (App Router) |
| [`pages`](/docs/pages/building-your-application/routing)                 | Routeur de pages (Pages Router)    |
| [`public`](/docs/app/building-your-application/optimizing/static-assets) | Ressources statiques à servir      |
| [`src`](/docs/app/building-your-application/configuring/src-directory)   | Dossier source optionnel           |
