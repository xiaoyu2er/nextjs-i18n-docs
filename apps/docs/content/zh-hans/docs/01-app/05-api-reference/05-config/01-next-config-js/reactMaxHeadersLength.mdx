title: reactMaxHeadersLength
description: React 生成并添加到响应中的标头最大长度设置。
---

在静态渲染过程中，React 会生成可添加到响应中的标头。这些标头可用于提升性能，例如让浏览器预加载字体、脚本和样式表等资源。默认值为 `6000`，但您可以通过在 `next.config.js` 中配置 `reactMaxHeadersLength` 选项来覆盖此值：

```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **须知**：此选项仅在应用路由器 (App Router) 中可用。

根据浏览器与服务器之间代理类型的不同，标头可能会被截断。例如，如果您使用的反向代理不支持长标头，则应设置较小的值以确保标头不被截断。
