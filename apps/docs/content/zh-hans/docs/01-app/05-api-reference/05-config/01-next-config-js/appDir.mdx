> **须知**：从 Next.js 13.4 版本开始**不再**需要此选项。应用路由 (App Router) 现已稳定。

应用路由 ([`app` 目录](/docs/app/building-your-application/routing)) 支持以下特性：
- [布局 (layouts)](/docs/app/api-reference/file-conventions/layout)
- [服务端组件 (Server Components)](/docs/app/getting-started/server-and-client-components)  
- [流式传输 (streaming)](/docs/app/api-reference/file-conventions/loading)
- [同位置数据获取 (colocated data fetching)](/docs/app/building-your-application/data-fetching)

使用 `app` 目录将自动启用 [React 严格模式 (React Strict Mode)](https://react.dev/reference/react/StrictMode)。了解如何[渐进式迁移到 `app` 目录](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app)。
