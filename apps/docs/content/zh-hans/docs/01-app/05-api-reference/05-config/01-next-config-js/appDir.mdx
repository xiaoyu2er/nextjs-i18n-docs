> **须知**：从 Next.js 13.4 开始，**不再**需要此选项。App Router 现已稳定。

App Router ([`app` 目录](/docs/app/building-your-application/routing)) 提供了对[布局](/docs/app/api-reference/file-conventions/layout)、[服务端组件 (Server Components)](/docs/app/getting-started/server-and-client-components)、[流式传输 (streaming)](/docs/app/building-your-application/routing/loading-ui-and-streaming) 和[同位置数据获取 (colocated data fetching)](/docs/app/building-your-application/data-fetching) 的支持。

使用 `app` 目录将自动启用 [React 严格模式 (React Strict Mode)](https://react.dev/reference/react/StrictMode)。了解如何[逐步迁移至 `app` 目录](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app)。
