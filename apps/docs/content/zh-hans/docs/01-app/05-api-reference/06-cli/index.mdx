Next.js 提供 **两个** 命令行工具 (CLI)：

- **`create-next-app`**：使用默认模板或 [示例项目](https://github.com/vercel/next.js/tree/canary/examples) 快速创建新的 Next.js 应用（示例来自公开的 GitHub 仓库）。
- **`next`**：运行 Next.js 开发服务器、构建应用等操作。
