本**入门指南**章节将帮助您创建第一个 Next.js 应用，并学习在每个项目中都会用到的核心功能。

## 预备知识

我们的文档假设您对 Web 开发有一定了解。在开始之前，建议您熟悉以下内容：

- HTML
- CSS
- JavaScript
- React

如果您是 React 新手或需要复习，我们推荐从 [React 基础课程](/learn/react-foundations) 开始学习，随后通过 [Next.js 基础课程](/learn/dashboard-app) 边学边构建应用程序。
