**应用路由 (App Router)** 是一个基于文件系统的路由，它利用了 React 的最新特性，如 [服务端组件 (Server Components)](https://react.dev/reference/rsc/server-components)、[Suspense](https://react.dev/reference/react/Suspense) 和 [服务端函数 (Server Functions)](https://react.dev/reference/rsc/server-functions)。
