`staticGeneration*` 选项允许您为高级用例配置静态生成 (Static Generation) 过程。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
const nextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

## 配置选项

提供以下可配置选项：

- `staticGenerationRetryCount`: 在构建失败前，重试失败页面生成的次数。
- `staticGenerationMaxConcurrency`: 每个工作线程 (worker) 最多可处理的页面数量。
- `staticGenerationMinPagesPerWorker`: 启动新工作线程 (worker) 前需要处理的最小页面数量。
