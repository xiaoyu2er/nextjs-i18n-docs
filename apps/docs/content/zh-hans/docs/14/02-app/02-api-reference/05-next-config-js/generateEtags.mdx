Next.js 默认会为每个页面生成 [etags](https://en.wikipedia.org/wiki/HTTP_ETag)。根据您的缓存策略，您可能需要禁用 HTML 页面的 etag 生成功能。

打开 `next.config.js` 文件并禁用 `generateEtags` 选项：

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
