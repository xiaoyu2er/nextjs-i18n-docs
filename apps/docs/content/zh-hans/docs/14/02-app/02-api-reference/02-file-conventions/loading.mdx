**loading** 文件可以基于 [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming) 创建即时加载状态。

默认情况下，该文件是一个 [服务端组件 (Server Component)](/docs/app/building-your-application/rendering/server-components) —— 但也可以通过 `"use client"` 指令用作客户端组件 (Client Component)。

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // 或者自定义加载骨架组件
  return <p>Loading...</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // 或者自定义加载骨架组件
  return <p>Loading...</p>
}
```

加载 UI 组件不接受任何参数。

> **须知**
>
> - 在设计加载 UI 时，使用 [React 开发者工具 (React Developer Tools)](https://react.dev/learn/react-developer-tools) 手动切换 Suspense 边界可能会很有帮助。
