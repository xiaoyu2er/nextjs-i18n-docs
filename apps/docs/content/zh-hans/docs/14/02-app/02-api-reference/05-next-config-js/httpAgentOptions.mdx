在 Node.js 18 之前的版本中，Next.js 会自动使用 [undici](/docs/architecture/supported-browsers#polyfills) 为 `fetch()` 提供 polyfill，并默认启用 [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive)。

要禁用服务端所有 `fetch()` 调用的 HTTP Keep-Alive 功能，请打开 `next.config.js` 文件并添加 `httpAgentOptions` 配置：

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
