本文档章节涵盖 **元数据文件约定**。基于文件的元数据可以通过在路由段中添加特殊的元数据文件来定义。

每个文件约定既可以使用静态文件（如 `opengraph-image.jpg`）定义，也可以使用动态变体通过代码生成文件（如 `opengraph-image.js`）。

文件定义后，Next.js 会自动提供该文件（生产环境中会添加哈希值以支持缓存），并使用正确的元数据（如资源 URL、文件类型和图像尺寸）更新相关的 head 元素。
