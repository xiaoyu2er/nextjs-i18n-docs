description: Next.js 支持多种数据获取方式，包括预渲染、服务端渲染 (SSR) 或静态站点生成 (SSG)，以及增量静态再生 (ISR)。了解如何在 Next.js 中管理应用数据。
---

Next.js 的数据获取功能允许您根据应用场景以不同方式渲染内容。这些方式包括使用**服务端渲染 (SSR)** 或**静态生成 (SSG)** 进行预渲染，以及通过**增量静态再生 (ISR)** 在运行时更新或创建内容。
