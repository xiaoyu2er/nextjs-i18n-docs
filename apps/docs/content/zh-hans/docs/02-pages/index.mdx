> 您当前查看的是页面路由 (Pages Router) 的文档。如需了解最新功能，请参阅 [应用路由 (App Router)](/docs/app) 文档。

**页面路由 (Pages Router)** 采用直观的文件系统路由机制，将每个文件映射为一条路由。

在 Next.js 13 之前，页面路由 (Pages Router) 是 Next.js 中创建路由的主要方式。新版本 Next.js 仍支持该路由方式，但我们建议迁移至新的 [应用路由 (App Router)](/docs/app) 以利用 React 的最新功能。
