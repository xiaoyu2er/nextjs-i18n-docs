---

为 Pages Router 应用程序启用服务端自动依赖项打包功能。该功能与 App Router 中的自动依赖项打包行为保持一致。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  bundlePagesRouterDependencies: true,
}

module.exports = nextConfig
```

可以通过 [`serverExternalPackages`](/docs/pages/api-reference/config/next-config-js/serverExternalPackages) 选项显式排除特定包的自动打包。

## 版本历史

| 版本号     | 变更内容                                                                                                   |
| ---------- | --------------------------------------------------------------------------------------------------------- |
| `v15.0.0` | 从实验性功能转为稳定功能。配置项名称从 `bundlePagesExternals` 变更为 `bundlePagesRouterDependencies` |
