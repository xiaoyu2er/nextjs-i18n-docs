title: 如何升级至版本 10
nav_title: 版本 10
description: 将您的 Next.js 应用从版本 9 升级至版本 10。
---

版本 9 至 10 之间没有破坏性变更。

要升级至版本 10，请运行以下命令：

```bash filename="终端"
npm i next@10
```

```bash filename="终端"
yarn add next@10
```

```bash filename="终端"
pnpm up next@10
```

```bash filename="终端"
bun add next@10
```

> **须知：** 如果您使用 TypeScript，请确保同时升级 `@types/react` 和 `@types/react-dom` 至对应版本。
