Next.js 提供了构建灵活的全栈 Web 应用的基础模块。**构建应用**指南将解释如何使用这些功能以及如何自定义应用程序行为。

章节和页面按从基础到高级的顺序组织，您可以在构建 Next.js 应用时逐步遵循。当然，您也可以按任意顺序阅读或直接跳转到与您用例相关的页面。

<AppOnly>

如果您是 Next.js 新手，我们建议从 [路由](/docs/app/building-your-application/routing)、[渲染](/docs/app/building-your-application/rendering)、[数据获取](/docs/app/building-your-application/data-fetching) 和 [样式](/docs/app/building-your-application/styling) 章节开始，这些内容介绍了 Next.js 和 Web 的基础概念以帮助您入门。然后，您可以深入探索其他章节，如 [优化](/docs/app/building-your-application/optimizing) 和 [配置](/docs/app/building-your-application/configuring)。最后，当您准备就绪时，可以查看 [部署](/docs/app/building-your-application/deploying) 和 [升级](/docs/app/building-your-application/upgrading) 章节。

</AppOnly>
