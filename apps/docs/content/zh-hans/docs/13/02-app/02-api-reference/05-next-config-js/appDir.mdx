> **须知**：从 Next.js 13.4 开始**不再**需要此选项。应用路由 (App Router) 现已稳定。

应用路由 ([`app` 目录](/docs/app/building-your-application/routing)) 支持[布局 (layouts)](/docs/app/building-your-application/routing/pages-and-layouts)、[服务端组件 (Server Components)](/docs/app/building-your-application/rendering/server-components)、[流式传输 (streaming)](/docs/app/building-your-application/routing/loading-ui-and-streaming) 以及[同位置数据获取 (colocated data fetching)](/docs/app/building-your-application/data-fetching)。

使用 `app` 目录将自动启用 [React 严格模式 (React Strict Mode)](https://react.dev/reference/react/StrictMode)。了解如何[逐步采用 `app` 目录](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app)。
