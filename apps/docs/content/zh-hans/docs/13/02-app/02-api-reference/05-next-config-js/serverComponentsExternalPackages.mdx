在[服务端组件 (Server Components)](/docs/app/building-your-application/rendering/server-components) 和[路由处理器 (Route Handlers)](/docs/app/building-your-application/routing/route-handlers) 内部使用的依赖项会被 Next.js 自动打包。

如果某个依赖项使用了 Node.js 特有的功能，您可以选择将其从服务端组件打包中排除，转而使用原生 Node.js `require`。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    serverComponentsExternalPackages: ['@acme/ui'],
  },
}

module.exports = nextConfig
```
