Next.js 提供 [**gzip**](https://tools.ietf.org/html/rfc6713#section-3) 压缩功能用于压缩渲染内容和静态文件。通常建议在类似 [nginx](https://www.nginx.com/) 的 HTTP 代理上启用压缩，以减轻 `Node.js` 进程的负载。

要禁用**压缩**功能，请打开 `next.config.js` 并关闭 `compress` 配置：

```js filename="next.config.js"
module.exports = {
  compress: false,
}
```
