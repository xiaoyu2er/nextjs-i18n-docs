默认情况下，Next.js 会将带有尾部斜杠的 URL 重定向至不带斜杠的对应地址。例如 `/about/` 会被重定向到 `/about`。您可以通过配置将此行为改为相反方式，即将不带尾部斜杠的 URL 重定向至带斜杠的对应地址。

打开 `next.config.js` 文件并添加 `trailingSlash` 配置：

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

启用此选项后，类似 `/about` 的 URL 将会被重定向至 `/about/`。
