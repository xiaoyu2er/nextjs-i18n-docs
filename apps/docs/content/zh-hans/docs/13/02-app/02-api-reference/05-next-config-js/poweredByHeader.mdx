Next.js 默认会添加 `x-powered-by` 响应头。如需禁用此功能，请打开 `next.config.js` 文件并关闭 `poweredByHeader` 配置项：

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
