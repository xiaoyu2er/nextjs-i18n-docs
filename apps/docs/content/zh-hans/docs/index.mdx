## 什么是 Next.js？

Next.js 是一个用于构建全栈 Web 应用程序的 React 框架。您可以使用 React 组件构建用户界面，并通过 Next.js 获得额外的功能与优化。

它还会自动配置底层工具如打包器和编译器，让您能够专注于产品开发和快速交付。

无论您是独立开发者还是团队一员，Next.js 都能帮助您构建交互式、动态且快速的 React 应用程序。

## App 路由与 Pages 路由

Next.js 提供两种路由系统：

- **App 路由 (App Router)**：支持服务端组件等 React 新特性的新一代路由
- **Pages 路由 (Pages Router)**：原始路由系统，持续维护并保持兼容性

侧边栏顶部提供下拉菜单，可在 [App 路由](/docs/app) 和 [Pages 路由](/docs/pages) 文档间切换。

### React 版本处理机制

两种路由处理 React 版本的方式不同：

- **App 路由 (App Router)**：内置使用 [React canary 版本](https://react.dev/blog/2023/05/03/react-canaries)，包含所有稳定的 React 19 变更，以及框架验证中的新特性

- **Pages 路由 (Pages Router)**：使用项目 `package.json` 中安装的 React 版本

这种机制确保新 React 特性在 App 路由中可靠运行，同时保持 Pages 路由应用的向后兼容性。
