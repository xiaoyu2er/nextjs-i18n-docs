要高效学习 Next.js，熟悉 JavaScript、React 及相关 Web 开发概念会大有裨益。但 JavaScript 和 React 知识体系庞大，如何判断自己已经准备好使用 Next.js 了呢？

欢迎来到 React 基础课程！这个面向初学者的示例驱动课程将引导你掌握 Next.js 的预备知识。你将通过分步构建一个简单项目来学习：从 JavaScript 应用开始，然后迁移到 React，最终升级为 Next.js 应用。

每个章节都建立在前文基础上，你可以根据已有知识选择适合的起点。

### [预备知识](#prerequisite-knowledge)

本教程假设你已掌握 HTML、CSS 和 JavaScript，但不要求 React 基础。若已熟悉 React，可直接跳转至 [从 React 到 Next.js](/learn/react-foundations/from-react-to-nextjs) 章节，或学习 [仪表盘应用](/learn/dashboard-app) 课程。

### [系统要求](#system-requirements)

开始学习前请确保已安装：

*   Node.js 20.12.0 或更高版本
*   操作系统：macOS、Windows（含 WSL）或 Linux
*   VSCode 或其他你选择的代码编辑器
