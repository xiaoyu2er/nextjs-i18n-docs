要理解 React 的工作原理，我们首先需要了解浏览器如何解析代码来创建（或渲染）用户界面 (UI)。

当用户访问网页时，服务器会向浏览器返回一个 HTML 文件，内容可能如下所示：

浏览器随后会读取 HTML 并构建文档对象模型 (DOM)。

### [什么是 DOM？](#what-is-the-dom)

DOM 是 HTML 元素的对象表示形式。它充当代码与用户界面之间的桥梁，具有父子关系的树状结构。

<Image
  alt="左右并列的两个图表，左侧显示 DOM 树结构，右侧显示渲染后的用户界面。"

  srcDark="https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/dark/learn-dom-and-ui.png"
  width="1600"
  height="759"
/>

您可以使用 DOM 方法和 JavaScript 来监听用户事件，并通过选择、添加、更新和删除用户界面中的特定元素来[操作 DOM](https://developer.mozilla.org/docs/Learn/JavaScript/Client-side_web_APIs/Manipulating_documents)。DOM 操作不仅允许您定位特定元素，还能改变它们的样式和内容。

在下一节中，您将学习如何使用 JavaScript 和 DOM 方法。
