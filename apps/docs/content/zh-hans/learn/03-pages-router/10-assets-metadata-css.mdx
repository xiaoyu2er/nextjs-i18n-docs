我们添加的第二个页面目前没有样式。让我们添加一些 CSS 来美化这个页面。

Next.js 内置支持 [CSS](/docs/basic-features/built-in-css-support) 和 [Sass](/docs/basic-features/built-in-css-support#sass-support)。本课程中我们将使用 CSS。

本节课程还将介绍 Next.js 如何处理静态资源（如图片）和页面元数据（如 `<title>` 标签）。

### [本课程将学习的内容](#what-youll-learn-in-this-lesson)

在本课程中，您将学习：

* 如何向 Next.js 添加[静态文件](/docs/basic-features/static-file-serving)（图片等）。
* 如何自定义每个页面的 `<head>` 内容。
* 如何使用 [CSS Modules](/docs/basic-features/built-in-css-support#adding-component-level-css) 创建可复用的 React 组件样式。
* 如何在 [`pages/_app.js`](/docs/advanced-features/custom-app) 中[添加全局 CSS](/docs/basic-features/built-in-css-support#adding-a-global-stylesheet)。
* Next.js 样式设计的一些实用技巧。

### [先决条件](#prerequisites)

* 基础的 CSS 知识。本课程将介绍如何在 Next.js 应用中添加 CSS，但不会涉及 CSS 基础知识。

> 如果您需要 Next.js 样式设计的详细文档，请查阅 [CSS 文档](/docs/basic-features/built-in-css-support)。
