目前我们创建的 Next.js 应用仅包含一个页面。网站和 Web 应用通常需要多个不同页面。

让我们探索如何为应用添加更多页面。

[本课程将学习的内容](#what-youll-learn-in-this-lesson)
--------------------------------------------------------------------

在本课程中，您将：

*   使用内置的[文件系统路由 (file system routing)](/docs/routing/introduction) 创建新页面。
*   学习如何使用 [`Link`](https://nextjs.org/docs/api-reference/next/link) 组件实现客户端页面导航。
*   了解内置的代码分割 (code splitting) 和预取 (prefetching) 功能支持。

> 如需获取 Next.js 路由的详细文档，请参阅[路由文档](/docs/routing/introduction)。
