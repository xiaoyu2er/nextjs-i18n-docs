首先，请确保您的开发环境已准备就绪。

*   如果尚未安装 **Node.js**，请[从此处安装](https://nodejs.org/en/)。您需要 Node.js **18** 或更高版本。
*   本教程将使用您自己的文本编辑器和终端应用程序。

> 如果您使用的是 Windows 系统，建议[下载 Git for Windows](https://gitforwindows.org/) 并使用其附带的 Git Bash，它支持本教程中特定的 UNIX 命令。[适用于 Linux 的 Windows 子系统 (WSL)](https://docs.microsoft.com/en-us/windows/wsl/install-win10) 是另一个可选方案。

### [创建 Next.js 应用](#create-a-nextjs-app)

要创建 Next.js 应用，请打开终端，进入目标目录后执行以下命令：

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/learn-starter"
```

> 底层原理是使用了 [`create-next-app`](/docs/api-reference/create-next-app) 工具，它会为您初始化 Next.js 应用。通过 `--example` 参数指定了[此模板](https://github.com/vercel/next-learn/tree/main/basics/learn-starter)。
> 
> 如果执行失败，请参考[此页面](https://github.com/vercel/next-learn/blob/main/basics/errors/install.md)进行排查。

### [运行开发服务器](#run-the-development-server)

现在您会看到一个名为 `nextjs-blog` 的新目录。进入该目录：

```
cd nextjs-blog
```

然后运行以下命令：

```
npm run dev
```

这将在 **3000** 端口启动 Next.js 应用的"开发服务器"（后续会详细说明）。

打开浏览器访问 [http://localhost:3000](http://localhost:3000)，检查应用是否正常运行。
