**如果您是从上一课继续学习，**可以跳过本页内容。

### [下载初始代码（可选）](#下载初始代码-optional)

如果您不是从上一课继续学习，可以下载、安装并运行本课的初始代码。这将创建一个 `nextjs-blog` 目录，其内容与上一课的完成结果完全一致。

再次说明，如果您刚完成上一课的学习，则无需执行此操作。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/api-routes-starter"
```

然后按照命令行输出的指示操作（进入目录并启动开发服务器）。

您还需要更新以下文件：

*   将 `public/images/profile.jpg` 替换为您的照片（推荐尺寸：400px 宽/高）
*   在 `components/layout.js` 中将 `const name = '[您的姓名]'` 修改为您的姓名
*   在 `pages/index.js` 中将 `<p>[自我介绍内容]</p>` 替换为您的自我介绍
