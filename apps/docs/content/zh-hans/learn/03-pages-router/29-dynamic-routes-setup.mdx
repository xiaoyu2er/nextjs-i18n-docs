### [下载初始代码（可选）](#下载初始代码-optional)

如果未从上一课继续，可以下载、安装并运行本课的初始代码。这将创建一个 `nextjs-blog` 目录，其内容与上一课的最终结果完全一致。

再次说明，如果刚完成上一课的学习，则无需此步骤。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/dynamic-routes-starter"
```

然后按照命令行输出的指示操作（进入目录并启动开发服务器）。

还需更新以下文件：

*   将 `public/images/profile.jpg` 替换为你的照片（建议尺寸：400px 宽/高）。
*   将 `components/layout.js` 中的 `const name = '[你的名字]'` 替换为你的姓名。
*   将 `pages/index.js` 中的 `<p>[自我介绍内容]</p>` 替换为你的自我介绍。
