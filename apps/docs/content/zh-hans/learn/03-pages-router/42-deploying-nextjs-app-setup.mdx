### [下载初始代码（可选）](#下载初始代码-optional)

如果您没有学习上一课，可以下载、安装并运行本课的初始代码。这将创建一个与上一课结果完全相同的 `nextjs-blog` 目录。

再次说明，如果您刚完成上一课的学习，则无需进行此操作。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/basics-final"
```

然后按照命令行输出的指示操作（进入目录并启动开发服务器）。

您还需要更新以下文件：

*   将 `public/images/profile.jpg` 替换为您的照片（推荐尺寸：400px 宽/高）。
*   将 `components/layout.js` 中的 `const name = '[您的姓名]'` 替换为您的姓名。
*   将 `pages/index.js` 中的 `<p>[自我介绍内容]</p>` 替换为您的自我介绍。
