我们已经在首页填充了博客数据，但尚未创建独立的博客页面（这是[预期效果](https://next-learn-starter.vercel.app/)）。我们希望这些页面的 URL 能基于博客数据动态生成，这意味着需要使用[动态路由](/docs/routing/dynamic-routes)。

### [本课将学习的内容](#what-youll-learn-in-this-lesson)

在本课中，您将学习：

* 如何使用 [`getStaticPaths`](/docs/basic-features/data-fetching#getstaticpaths-static-generation) 结合[动态路由](/docs/routing/dynamic-routes)静态生成页面。
* 如何编写 [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) 来获取每篇博客文章的数据。
* 如何使用 [`remark`](https://github.com/remarkjs/remark) 渲染 markdown 内容。
* 如何格式化日期字符串。
* 如何通过[动态路由](/docs/routing/dynamic-routes)链接到页面。
* 关于[动态路由](/docs/routing/dynamic-routes)的一些实用技巧。
