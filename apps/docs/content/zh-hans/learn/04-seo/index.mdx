[什么是 SEO？](#what-is-seo)
----------------------------

SEO 是 **Search Engine Optimization（搜索引擎优化）** 的缩写。SEO 的目标是通过制定策略提升网站在搜索引擎结果中的排名。排名越高，网站获得的自然流量就越多，最终将为您带来更多业务！

### [本课程将学习的内容](#what-youll-learn-in-this-course)

在本课程中，我们将探讨：

*   搜索系统及搜索引擎机器人（如 Googlebot）
*   SEO 策略对网站的影响
*   如何实施 SEO 最佳实践
*   Next.js 中的爬取、索引、渲染和排名机制
*   包含核心 Web 指标在内的网页性能话题
