在本课程中，我们将探讨各项指标、[核心 Web 指标 (Core Web Vitals)](https://web.dev/vitals/) 对 SEO 的影响，以及它们对用户体验的重要性。

测量核心 Web 指标时存在三个不同等级：

**"良好"**、**"待改进"** 和 **"较差"**。这些等级根据所测量的指标而有所不同：

![核心 Web 指标](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/seo/vitals-light.png)

您可以通过两种方式处理核心 Web 指标：

*   **力争在每个指标上获得最高分**。追求完美固然很好，但对于依赖项众多的大型网站来说可能颇具挑战性。
*   **与同行业竞争对手进行基准对比**。您在 Google 搜索中并非与所有优化完美的网站竞争，而是与其他针对目标关键词排名的网站竞争。
