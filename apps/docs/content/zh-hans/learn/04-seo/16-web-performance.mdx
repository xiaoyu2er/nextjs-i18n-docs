[Web 指标 (Web Vitals)](https://web.dev/vitals/) 是由 Google 发起的一项计划，旨在提供统一的指导标准和指标来衡量终端用户在网页上的体验。

[核心 Web 指标 (Core Web Vitals)](https://developers.google.com/search/blog/2020/11/timing-for-page-experience) 是 Web 指标的子集，目前包含三个衡量加载速度、交互性和视觉稳定性的指标。这些指标分别是：[最大内容绘制 (Largest Contentful Paint, LCP)](/learn/seo/lcp)、[首次输入延迟 (First Input Delay, FID)](/learn/seo/fid) 和 [累积布局偏移 (Cumulative Layout Shift, CLS)](/learn/seo/cls)。

在这三项指标中获得高分将为您的用户创造更流畅的网站体验。

如果网站在核心 Web 指标的各个指标中得分不佳，将会影响其在搜索引擎中的排名，因为 Google 已开始将[核心 Web 指标作为搜索算法的排名因素](https://developers.google.com/search/blog/2020/11/timing-for-page-experience)。不佳的 Web 指标可能会对您的网站流量和业务产生影响。

在本课程中，您将学习：

*   核心 Web 指标的背景简介
*   核心 Web 指标对 SEO 和用户体验的影响，及其对网站的影响
*   为什么在开发过程中需要关注核心 Web 指标以及如何测量它们
*   如何使用 Next.js 优化您的核心 Web 指标并监控变化
