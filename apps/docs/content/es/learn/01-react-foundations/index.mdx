Para aprender Next.js de manera efectiva, es útil estar familiarizado con JavaScript, React y conceptos relacionados con el desarrollo web. Pero JavaScript y React son temas muy amplios. ¿Cómo saber cuándo estás listo para empezar a usar Next.js?

¡Bienvenido al curso de Fundamentos de React! Este curso amigable para principiantes, basado en ejemplos, te guiará a través del conocimiento previo necesario para Next.js. Construirás un proyecto simple paso a paso; comenzando con una aplicación JavaScript, luego migrándola a React y finalmente a Next.js.

Cada sección se basa en la anterior, por lo que puedes elegir dónde comenzar dependiendo de lo que ya conozcas.

### [Conocimientos previos](#prerequisite-knowledge)

Este tutorial asume conocimientos de HTML, CSS, JavaScript, pero no requiere conocimiento previo de React. Si ya estás familiarizado con React, puedes saltar al capítulo [De React a Next.js](/learn/react-foundations/from-react-to-nextjs) o tomar el curso [Aplicación de Panel](/learn/dashboard-app).

### [Requisitos del sistema](#system-requirements)

Antes de comenzar este curso, asegúrate de tener instalado lo siguiente:

*   Node.js 20.12.0 o versión posterior.
*   Sistemas operativos: macOS, Windows (incluyendo WSL) o Linux.
*   VSCode u otro editor de texto de tu preferencia.
