En esta lección, revisaremos las diferentes métricas, el impacto que los [Core Web Vitals (Métricas Web Esenciales)](https://web.dev/vitals/) pueden tener en tu SEO y la importancia que tienen para la experiencia de usuario.

Existen tres valores diferentes al medir los Core Web Vitals:

**"Bueno"**, **"Necesita Mejora"** y **"Deficiente"**. Estos valores varían según la métrica que se esté midiendo:

![Core Web Vitals](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/seo/vitals-light.png)

Puedes abordar los Core Web Vitals de dos maneras diferentes:

*   **Intentar alcanzar la puntuación más alta posible en cada métrica**. Aspirar a la perfección es excelente, pero puede ser complicado en sitios web grandes con muchas dependencias.
*   **Compararte con competidores de tu industria**. No estás compitiendo con todos los sitios web perfectamente optimizados en las búsquedas de Google, sino con otros que rankean para tus palabras clave objetivo.
