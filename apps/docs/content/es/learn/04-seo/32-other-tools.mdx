Puedes encontrar recopilación de datos de campo en las siguientes herramientas:

*   [PageSpeed Insights](https://developers.google.com/speed/pagespeed/insights/): Herramienta de medición de velocidad de páginas de Google.
*   [Chrome User Experience Report](https://developers.google.com/web/tools/chrome-user-experience-report): Conjunto de datos de campo de código abierto.
*   [Search Console](https://support.google.com/webmasters/answer/9205520): Informe de Core Web Vitals en Google Search.

Si buscas datos de laboratorio, Google también ofrece varias herramientas de medición:

*   [Lighthouse](https://developers.google.com/web/tools/lighthouse?hl=en): Herramienta automatizada de código abierto de Google para mejorar la calidad de las páginas web.
*   [Chrome DevTools](https://developer.chrome.com/docs/devtools/): Conjunto de herramientas para desarrolladores integradas directamente en el navegador Google Chrome.

Ten en cuenta que para ambas herramientas necesitarás usar el [Tiempo de Bloqueo Total (TBT)](https://web.dev/tbt/) como alternativa al [Retraso de la Primera Entrada (FID)](https://web.dev/fid/), ya que el FID solo puede medirse a través de datos de campo.
