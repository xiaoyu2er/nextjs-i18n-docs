### [Descargar Código de Inicio (Opcional)](#download-starter-code-optional)

Si NO continúas desde la lección anterior, puedes descargar, instalar y ejecutar el código de inicio para esta lección a continuación. Esto configura un directorio `nextjs-blog` para que sea idéntico al resultado de la lección anterior.

De nuevo, esto NO es necesario si acabas de terminar la lección anterior.

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/api-routes-starter"
```

Luego sigue las instrucciones de la salida del comando (`cd` para entrar al directorio e iniciar el servidor de desarrollo).

También deberías actualizar los siguientes archivos:

*   `public/images/profile.jpg` con tu foto (Recomendado: 400px de ancho/alto).
*   `const name = '[Tu Nombre]'` en `components/layout.js` con tu nombre.
*   `<p>[Tu Presentación Personal]</p>` en `pages/index.js` con tu presentación personal.
