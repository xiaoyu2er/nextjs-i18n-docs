Hemos poblado la página de índice con los datos del blog, pero aún no hemos creado las páginas individuales de cada entrada (aquí está el [resultado deseado](https://next-learn-starter.vercel.app/)). Queremos que la URL de estas páginas dependa de los datos del blog, lo que significa que necesitamos usar [rutas dinámicas](/docs/routing/dynamic-routes).

### [Qué Aprenderás en Esta Lección](#what-youll-learn-in-this-lesson)

En esta lección, aprenderás:

* Cómo generar páginas estáticamente con [rutas dinámicas](/docs/routing/dynamic-routes) usando [`getStaticPaths`](/docs/basic-features/data-fetching#getstaticpaths-static-generation).
* Cómo escribir [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) para obtener los datos de cada entrada del blog.
* Cómo renderizar markdown usando [`remark`](https://github.com/remarkjs/remark).
* Cómo formatear correctamente cadenas de fecha.
* Cómo enlazar a una página con [rutas dinámicas](/docs/routing/dynamic-routes).
* Información útil sobre [rutas dinámicas](/docs/routing/dynamic-routes).
