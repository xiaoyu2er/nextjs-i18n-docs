La segunda página que agregamos actualmente no tiene estilos. Vamos a añadir algo de CSS para darle estilo a la página.

Next.js tiene soporte integrado para [CSS](/docs/basic-features/built-in-css-support) y [Sass](/docs/basic-features/built-in-css-support#sass-support). Para este curso, usaremos CSS.

Esta lección también hablará sobre cómo Next.js maneja recursos estáticos como imágenes y metadatos de página como la etiqueta `<title>`.

### [Qué Aprenderás en Esta Lección](#what-youll-learn-in-this-lesson)

En esta lección, aprenderás:

*   Cómo añadir [archivos estáticos](/docs/basic-features/static-file-serving) (imágenes, etc.) a Next.js.
*   Cómo personalizar lo que va dentro del `<head>` para cada página.
*   Cómo crear un componente reutilizable de React que tenga estilos usando [Módulos CSS](/docs/basic-features/built-in-css-support#adding-component-level-css).
*   Cómo [añadir CSS global](/docs/basic-features/built-in-css-support#adding-a-global-stylesheet) en [`pages/_app.js`](/docs/advanced-features/custom-app).
*   Algunos consejos útiles para estilizar en Next.js.

### [Requisitos Previos](#prerequisites)

*   Conocimientos básicos de CSS. Este curso cubrirá cómo añadir CSS en una aplicación Next.js, pero no abordará fundamentos de CSS.

> Si buscas documentación detallada sobre estilos en Next.js, consulta [la documentación de CSS](/docs/basic-features/built-in-css-support).
