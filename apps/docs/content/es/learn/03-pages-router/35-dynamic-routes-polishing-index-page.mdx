A continuación, actualicemos nuestra página de inicio (`pages/index.js`). Necesitamos agregar enlaces a cada página de publicación usando el componente [`Link`](/docs/api-reference/next/link).

Abre `pages/index.js` y agrega las siguientes importaciones en la parte superior del archivo para [`Link`](/docs/api-reference/next/link) y `Date`:

```
import Link from 'next/link';
import Date from '../components/date';
```

Luego, cerca del final del componente `Home` en el mismo archivo, reemplaza la etiqueta `li` con lo siguiente:

```
<li className={utilStyles.listItem} key={id}>
  <Link href={`/posts/${id}`}>{title}</Link>
  <br />
  <small className={utilStyles.lightText}>
    <Date dateString={date} />
  </small>
</li>
```

Si vas a [http://localhost:3000](http://localhost:3000), la página ahora tendrá enlaces a cada artículo:

![Enlaces](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/dynamic-routes/links.png)

> Si algo no funciona, asegúrate de que tu código [se vea así](https://github.com/vercel/next-learn/blob/main/basics/api-routes-starter/pages/posts/%5Bid%5D.js).
