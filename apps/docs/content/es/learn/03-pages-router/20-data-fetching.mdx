Nos gustaría crear un blog (aquí está [el resultado deseado](https://next-learn-starter.vercel.app/)), pero hasta ahora no hemos añadido contenido alguno. En esta lección, aprenderemos cómo obtener datos externos de blog para nuestra aplicación. Almacenaremos el contenido del blog en el sistema de archivos, pero también funcionará si el contenido se guarda en otro lugar (por ejemplo, una base de datos o un [CMS Headless](https://en.wikipedia.org/wiki/Headless_content_management_system)).

### [Qué Aprenderás en Esta Lección](#what-youll-learn-in-this-lesson)

En esta lección, aprenderás sobre:

*   La función de [pre-renderizado](/docs/basic-features/pages#pre-rendering) de Next.js.
*   Las dos formas de pre-renderizado: [Generación Estática](/docs/basic-features/pages#static-generation-recommended) y [Renderizado del Lado del Servidor](/docs/basic-features/pages#server-side-rendering).
*   Generación Estática [con datos](/docs/basic-features/pages#static-generation-with-data) y [sin datos](/docs/basic-features/pages#static-generation-without-data).
*   [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) y cómo usarlo para importar datos externos de blog a la página de inicio.
*   Información útil sobre [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation).
