Hasta ahora, la aplicación Next.js que hemos creado solo tiene una página. Los sitios web y aplicaciones generalmente tienen muchas páginas diferentes.

Exploremos cómo agregar más páginas a nuestra aplicación.

[Lo que aprenderás en esta lección](#lo-que-aprenderas-en-esta-leccion)
--------------------------------------------------------------------

En esta lección, aprenderás:

*   Crear una nueva página usando el [enrutamiento basado en sistema de archivos](/docs/routing/introduction) integrado.
*   Aprender a usar el componente [`Link`](https://nextjs.org/docs/api-reference/next/link) para habilitar la navegación entre páginas del lado del cliente (client-side navigation).
*   Conocer el soporte integrado para división de código (code splitting) y precarga (prefetching).

> Si buscas documentación detallada sobre el enrutamiento en Next.js, consulta la [documentación de enrutamiento](/docs/routing/introduction).
