Primero, asegurémonos de que tu entorno de desarrollo esté listo.

*   Si no tienes **Node.js** instalado, [instálalo desde aquí](https://nodejs.org/en/). Necesitarás la versión **18** o superior de Node.js.
*   Utilizarás tu propio editor de texto y aplicación de terminal para este tutorial.

> Si estás en Windows, recomendamos [descargar Git para Windows](https://gitforwindows.org/) y usar Git Bash que viene incluido, el cual soporta los comandos específicos de UNIX en este tutorial. [Subsistema de Windows para Linux (WSL)](https://docs.microsoft.com/en-us/windows/wsl/install-win10) es otra opción.

### [Crear una aplicación Next.js](#create-a-nextjs-app)

Para crear una aplicación Next.js, abre tu terminal, navega con `cd` al directorio donde quieras crear la aplicación y ejecuta el siguiente comando:

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/learn-starter"
```

> Internamente, esto utiliza la herramienta llamada [`create-next-app`](/docs/api-reference/create-next-app), que prepara una aplicación Next.js para ti. Usa [esta plantilla](https://github.com/vercel/next-learn/tree/main/basics/learn-starter) a través de la bandera `--example`.
> 
> Si no funciona, por favor revisa [esta página](https://github.com/vercel/next-learn/blob/main/basics/errors/install.md).

### [Ejecutar el servidor de desarrollo](#run-the-development-server)

Ahora tienes un nuevo directorio llamado `nextjs-blog`. Navega a él con:

```
cd nextjs-blog
```

Luego, ejecuta el siguiente comando:

```
npm run dev
```

Esto inicia el "servidor de desarrollo" de tu aplicación Next.js (más sobre esto después) en el puerto **3000**.

Verifiquemos que esté funcionando. Abre [http://localhost:3000](http://localhost:3000) desde tu navegador.
