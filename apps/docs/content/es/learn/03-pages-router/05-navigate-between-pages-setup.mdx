### [Descargar código inicial (Opcional)](#descargar-código-inicial-opcional)

Si NO continúas desde la lección anterior, puedes descargar, instalar y ejecutar el código inicial para esta lección a continuación. Esto configura un directorio `nextjs-blog` para que sea idéntico al resultado de la lección anterior.

Nuevamente, esto NO es necesario si acabas de terminar la lección anterior.

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/navigate-between-pages-starter"
```

Luego sigue las instrucciones de la salida del comando (ejecuta `cd` en el directorio e inicia el servidor de desarrollo).
