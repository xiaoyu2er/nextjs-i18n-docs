Antes de implementar, vamos a subir nuestra aplicación Next.js a [GitHub](https://github.com/vercel/next.js) si aún no lo has hecho. Esto facilitará la implementación.

*   En tu cuenta personal de GitHub, crea un nuevo repositorio llamado `nextjs-blog`.
*   El repositorio puede ser público o privado. **No** es necesario inicializarlo con un README u otros archivos.
*   Si necesitas ayuda para configurar tu repositorio, consulta [esta guía en GitHub](https://help.github.com/en/github/getting-started-with-github/create-a-repo).

Luego:

*   Si no has inicializado el repositorio git localmente para tu aplicación Next.js, hazlo ahora.
*   Sube la aplicación Next.js a tu repositorio de GitHub.

Para subir a GitHub, puedes ejecutar los siguientes comandos (reemplaza `<username>` con tu nombre de usuario de GitHub):

```
git remote add origin https://github.com/<username>/nextjs-blog.git
git push -u origin main
```

Una vez que tu repositorio de GitHub esté listo, continúa con la siguiente página.
