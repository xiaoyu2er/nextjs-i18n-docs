Soporte experimental para [enlaces con tipado estático (statically typed links)](/docs/app/building-your-application/configuring/typescript#statically-typed-links). Esta característica requiere el uso del App Router así como TypeScript en tu proyecto.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
