Puedes configurar el nivel de registro y si se muestra la URL completa en la consola cuando se ejecuta Next.js en modo desarrollo.

Actualmente, `logging` solo se aplica a las solicitudes de datos que utilizan la API `fetch`. Aún no se aplica a otros registros dentro de Next.js.

```js filename="next.config.js"
module.exports = {
  logging: {
    fetches: {
      fullUrl: true,
    },
  },
}
```
