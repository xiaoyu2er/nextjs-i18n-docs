Un archivo **loading** puede crear estados de carga instantáneos basados en [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming).

Por defecto, este archivo es un [Componente del Servidor (Server Component)](/docs/app/building-your-application/rendering/server-components), pero también puede usarse como un Componente del Cliente (Client Component) mediante la directiva `"use client"`.

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // O un componente esqueleto de carga personalizado
  return <p>Cargando...</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // O un componente esqueleto de carga personalizado
  return <p>Cargando...</p>
}
```

Los componentes de interfaz de usuario (UI) de carga no aceptan ningún parámetro.

> **Dato útil**
>
> - Al diseñar la interfaz de carga, puede resultarle útil utilizar las [React Developer Tools](https://react.dev/learn/react-developer-tools) para alternar manualmente los límites de Suspense.
