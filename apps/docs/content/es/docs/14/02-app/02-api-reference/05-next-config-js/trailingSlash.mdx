Por defecto, Next.js redireccionará las URLs con barras diagonales finales a su versión sin la barra diagonal. Por ejemplo, `/about/` será redirigido a `/about`. Puedes configurar este comportamiento para que actúe de manera opuesta, donde las URLs sin barras diagonales finales sean redirigidas a sus contrapartes con la barra diagonal.

Abre `next.config.js` y agrega la configuración `trailingSlash`:

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

Con esta opción habilitada, URLs como `/about` serán redirigidas a `/about/`.

Cuando se usa junto con la configuración [`output: "export"`](/docs/app/building-your-application/deploying/static-exports), la página `/about` generará `/about/index.html` (en lugar del valor predeterminado `/about.html`).
