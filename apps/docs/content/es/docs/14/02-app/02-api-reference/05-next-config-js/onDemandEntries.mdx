Next.js expone algunas opciones que te permiten controlar cómo el servidor descartará o mantendrá en memoria las páginas construidas durante el desarrollo.

Para cambiar los valores predeterminados, abre `next.config.js` y añade la configuración `onDemandEntries`:

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // período (en ms) durante el cual el servidor mantendrá las páginas en el búfer
    maxInactiveAge: 25 * 1000,
    // número de páginas que se mantendrán simultáneamente sin ser descartadas
    pagesBufferLength: 2,
  },
}
```
