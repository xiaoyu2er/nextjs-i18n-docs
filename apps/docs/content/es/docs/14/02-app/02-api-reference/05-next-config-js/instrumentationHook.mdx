La opción experimental `instrumentationHook` le permite configurar instrumentación mediante el [archivo `instrumentation`](/docs/app/api-reference/file-conventions/instrumentation) en su aplicación Next.js.

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
