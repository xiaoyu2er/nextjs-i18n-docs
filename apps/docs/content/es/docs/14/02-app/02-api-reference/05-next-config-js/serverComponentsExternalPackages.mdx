Las dependencias utilizadas dentro de [Componentes del Servidor (Server Components)](/docs/app/building-your-application/rendering/server-components) y [Manejadores de Ruta (Route Handlers)](/docs/app/building-your-application/routing/route-handlers) serán empaquetadas automáticamente por Next.js.

Si una dependencia está utilizando características específicas de Node.js, puedes optar por excluir dependencias específicas del empaquetado de Componentes del Servidor y usar el `require` nativo de Node.js.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    serverComponentsExternalPackages: ['@acme/ui'],
  },
}

module.exports = nextConfig
```
