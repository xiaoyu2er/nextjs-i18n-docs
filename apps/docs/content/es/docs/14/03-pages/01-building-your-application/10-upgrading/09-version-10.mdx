title: Versión 10
description: Actualice su aplicación Next.js de la versión 9 a la versión 10.
---

No hubo cambios importantes entre las versiones 9 y 10.

Para actualizar a la versión 10, ejecute el siguiente comando:

```bash filename="Terminal"
npm i next@10
```

```bash filename="Terminal"
yarn add next@10
```

```bash filename="Terminal"
pnpm up next@10
```

```bash filename="Terminal"
bun add next@10
```

> **Nota importante:** Si está utilizando TypeScript, asegúrese de actualizar también `@types/react` y `@types/react-dom` a sus versiones correspondientes.
