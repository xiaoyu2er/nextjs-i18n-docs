### `src`

La ruta del archivo de fuente como string o un array de objetos (con tipo `Array<{path: string, weight?: string, style?: string}>`) relativa al directorio donde se llama a la funci√≥n de carga de fuente.

Usado en `next/font/local`

- Requerido

Ejemplos:

- `src:'./fonts/my-font.woff2'` donde `my-font.woff2` se coloca en un directorio llamado `fonts` dentro del directorio `app`
- `src:[{path: './inter/Inter-Thin.ttf', weight: '100',},{path: './inter/Inter-Regular.ttf',weight: '400',},{path: './inter/Inter-Bold-Italic.ttf', weight: '700',style: 'italic',},]`
- si la funci√≥n de carga de fuente se llama en `app/page.tsx` usando `src:'../styles/fonts/my-font.ttf'`, entonces `my-font.ttf` se coloca en `styles/fonts` en la ra√≠z del proyecto

### `weight`

El [`peso`](https://fonts.google.com/knowledge/glossary/weight) de la fuente con las siguientes posibilidades:

- Un string con los valores posibles de los pesos disponibles para la fuente espec√≠fica o un rango de valores si es una fuente [variable](https://fonts.google.com/variablefonts)
- Un array de valores de peso si la fuente no es una [fuente variable de Google](https://fonts.google.com/variablefonts). Aplica solo para `next/font/google`.

Usado en `next/font/google` y `next/font/local`

- Requerido si la fuente que se usa **no** es [variable](https://fonts.google.com/variablefonts)

Ejemplos:

- `weight: '400'`: Un string para un solo valor de peso - para la fuente [`Inter`](https://fonts.google.com/specimen/Inter?query=inter), los valores posibles son `'100'`, `'200'`, `'300'`, `'400'`, `'500'`, `'600'`, `'700'`, `'800'`, `'900'` o `'variable'` donde `'variable'` es el predeterminado)
- `weight: '100 900'`: Un string para el rango entre `100` y `900` para una fuente variable
- `weight: ['100','400','900']`: Un array de 3 valores posibles para una fuente no variable

### `style`

El [`estilo`](https://developer.mozilla.org/docs/Web/CSS/font-style) de la fuente con las siguientes posibilidades:

- Un string con [valor](https://developer.mozilla.org/docs/Web/CSS/font-style#values) predeterminado de `'normal'`
- Un array de valores de estilo si la fuente no es una [fuente variable de Google](https://fonts.google.com/variablefonts). Aplica solo para `next/font/google`.

Usado en `next/font/google` y `next/font/local`

- Opcional

Ejemplos:

- `style: 'italic'`: Un string - puede ser `normal` o `italic` para `next/font/google`
- `style: 'oblique'`: Un string - puede tomar cualquier valor para `next/font/local` pero se espera que provenga de [estilos de fuente est√°ndar](https://developer.mozilla.org/docs/Web/CSS/font-style)
- `style: ['italic','normal']`: Un array de 2 valores para `next/font/google` - los valores son `normal` e `italic`

### `subsets`

Los [`subconjuntos`](https://fonts.google.com/knowledge/glossary/subsetting) de la fuente definidos por un array de valores string con los nombres de cada subconjunto que deseas [precargar](/docs/app/api-reference/components/font#specifying-a-subset). Las fuentes especificadas mediante `subsets` tendr√°n una etiqueta de precarga de enlace inyectada en el head cuando la opci√≥n [`preload`](#preload) sea true, que es el valor predeterminado.

Usado en `next/font/google`

- Opcional

Ejemplos:

- `subsets: ['latin']`: Un array con el subconjunto `latin`

Puedes encontrar una lista de todos los subconjuntos en la p√°gina de Google Fonts para tu fuente.

### `axes`

Algunas fuentes variables tienen `ejes` adicionales que se pueden incluir. Por defecto, solo se incluye el peso de la fuente para mantener el tama√±o del archivo. Los valores posibles de `axes` dependen de la fuente espec√≠fica.

Usado en `next/font/google`

- Opcional

Ejemplos:

- `axes: ['slnt']`: Un array con el valor `slnt` para la fuente variable `Inter` que tiene `slnt` como `axes` adicional como se muestra [aqu√≠](https://fonts.google.com/variablefonts?vfquery=inter#font-families). Puedes encontrar los valores posibles de `axes` para tu fuente usando el filtro en la [p√°gina de fuentes variables de Google](https://fonts.google.com/variablefonts#font-families) y buscando ejes distintos a `wght`

### `display`

El [`display`](https://developer.mozilla.org/docs/Web/CSS/@font-face/font-display) de la fuente con posibles valores string de `'auto'`, `'block'`, `'swap'`, `'fallback'` o `'optional'` con valor predeterminado de `'swap'`.

Usado en `next/font/google` y `next/font/local`

- Opcional

Ejemplos:

- `display: 'optional'`: Un string asignado al valor `optional`

### `preload`

Un valor booleano que especifica si la fuente debe [precargarse](/docs/app/api-reference/components/font#preloading) o no. El valor predeterminado es `true`.

Usado en `next/font/google` y `next/font/local`

- Opcional

Ejemplos:

- `preload: false`

### `fallback`

La fuente de respaldo que se usar√° si no se puede cargar la fuente. Un array de strings de fuentes de respaldo sin valor predeterminado.

- Opcional

Usado en `next/font/google` y `next/font/local`

Ejemplos:

- `fallback: ['system-ui', 'arial']`: Un array que establece las fuentes de respaldo en `system-ui` o `arial`

### `adjustFontFallback`

- Para `next/font/google`: Un valor booleano que establece si se debe usar una fuente de respaldo autom√°tica para reducir el [Cambio Acumulativo de Dise√±o (CLS)](https://web.dev/cls/). El valor predeterminado es `true`.
- Para `next/font/local`: Un string o valor booleano `false` que establece si se debe usar una fuente de respaldo autom√°tica para reducir el [Cambio Acumulativo de Dise√±o (CLS)](https://web.dev/cls/). Los valores posibles son `'Arial'`, `'Times New Roman'` o `false`. El valor predeterminado es `'Arial'`.

Usado en `next/font/google` y `next/font/local`

- Opcional

Ejemplos:

- `adjustFontFallback: false`: para `next/font/google`
- `adjustFontFallback: 'Times New Roman'`: para `next/font/local`

### `variable`

Un valor string para definir el nombre de la variable CSS que se usar√° si el estilo se aplica con el [m√©todo de variables CSS](#css-variables).

Usado en `next/font/google` y `next/font/local`

- Opcional

Ejemplos:

- `variable: '--my-font'`: Se declara la variable CSS `--my-font`

### `declarations`

Un array de pares clave-valor de [descriptores](https://developer.mozilla.org/docs/Web/CSS/@font-face#descriptors) de fuentes que definen m√°s el `@font-face` generado.

Usado en `next/font/local`

- Opcional

Ejemplos:

- `declarations: [{ prop: 'ascent-override', value: '90%' }]`
