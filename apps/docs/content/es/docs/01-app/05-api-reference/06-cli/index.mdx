Next.js incluye **dos** herramientas de interfaz de línea de comandos (CLI):

- **`create-next-app`**: Crea rápidamente una nueva aplicación Next.js usando la plantilla predeterminada o un [ejemplo](https://github.com/vercel/next.js/tree/canary/examples) de un repositorio público de GitHub.
- **`next`**: Ejecuta el servidor de desarrollo de Next.js, construye tu aplicación y más.
