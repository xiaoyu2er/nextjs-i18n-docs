Agrega o genera un archivo `manifest.(json|webmanifest)` que cumpla con la [Especificación de Manifest Web](https://developer.mozilla.org/docs/Web/Manifest) en el directorio **raíz** de `app` para proporcionar información sobre tu aplicación web al navegador.

## Archivo Manifest estático

```json filename="app/manifest.json | app/manifest.webmanifest"
{
  "name": "My Next.js Application",
  "short_name": "Next.js App",
  "description": "An application built with Next.js",
  "start_url": "/"
  // ...
}
```

## Generar un archivo Manifest

Agrega un archivo `manifest.js` o `manifest.ts` que retorne un [objeto `Manifest`](#manifest-object).

> Bueno saber: `manifest.js` es un Manejador de Rutas especial que se almacena en caché por defecto a menos que use una [API Dinámica](/docs/app/deep-dive/caching#dynamic-apis) u opción de [configuración dinámica](/docs/app/deep-dive/caching#segment-config-options).

```ts filename="app/manifest.ts" switcher
import type { MetadataRoute } from 'next'

export default function manifest(): MetadataRoute.Manifest {
  return {
    name: 'Next.js App',
    short_name: 'Next.js App',
    description: 'Next.js App',
    start_url: '/',
    display: 'standalone',
    background_color: '#fff',
    theme_color: '#fff',
    icons: [
      {
        src: '/favicon.ico',
        sizes: 'any',
        type: 'image/x-icon',
      },
    ],
  }
}
```

```js filename="app/manifest.js" switcher
export default function manifest() {
  return {
    name: 'Next.js App',
    short_name: 'Next.js App',
    description: 'Next.js App',
    start_url: '/',
    display: 'standalone',
    background_color: '#fff',

    icons: [
      {
        src: '/favicon.ico',
        sizes: 'any',
        type: 'image/x-icon',
      },
    ],
  }
}
```

### Objeto Manifest

El objeto manifest contiene una extensa lista de opciones que pueden actualizarse debido a nuevos estándares web. Para información sobre todas las opciones actuales, consulta el tipo `MetadataRoute.Manifest` en tu editor de código si usas [TypeScript](https://nextjs.org/docs/app/api-reference/config/typescript#ide-plugin) o revisa la documentación de [MDN](https://developer.mozilla.org/docs/Web/Manifest).
