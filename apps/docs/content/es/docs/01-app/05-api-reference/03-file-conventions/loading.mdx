Un archivo **loading** puede crear estados de carga instantáneos basados en [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming).

Por defecto, este archivo es un [Componente de Servidor (Server Component)](/docs/app/getting-started/server-and-client-components), pero también puede usarse como Componente de Cliente mediante la directiva `"use client"`.

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // O un componente esqueleto de carga personalizado
  return <p>Cargando...</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // O un componente esqueleto de carga personalizado
  return <p>Cargando...</p>
}
```

Los componentes de interfaz de usuario de carga no aceptan ningún parámetro.

> **Es bueno saber**:
>
> - Al diseñar interfaces de carga, puede resultar útil utilizar las [Herramientas de Desarrollo de React](https://react.dev/learn/react-developer-tools) para alternar manualmente los límites de Suspense.
