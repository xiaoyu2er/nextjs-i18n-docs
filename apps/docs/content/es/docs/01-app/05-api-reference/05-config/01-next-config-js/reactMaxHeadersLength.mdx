title: reactMaxHeadersLength
description: Longitud máxima de los encabezados emitidos por React que se añaden a la respuesta.
---

Durante el renderizado estático, React puede emitir encabezados que pueden agregarse a la respuesta. Estos pueden utilizarse para mejorar el rendimiento permitiendo que el navegador precargue recursos como fuentes, scripts y hojas de estilo. El valor predeterminado es `6000`, pero puedes sobrescribir este valor configurando la opción `reactMaxHeadersLength` en `next.config.js`:

```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **Es bueno saberlo**: Esta opción solo está disponible en App Router.

Dependiendo del tipo de proxy entre el navegador y el servidor, los encabezados pueden truncarse. Por ejemplo, si estás utilizando un proxy inverso que no admite encabezados largos, deberías establecer un valor más bajo para asegurarte de que los encabezados no se trunquen.
