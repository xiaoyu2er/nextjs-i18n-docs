Las dependencias utilizadas dentro de [Componentes del Servidor](/docs/app/getting-started/server-and-client-components) y [Manejadores de Ruta](/docs/app/building-your-application/routing/route-handlers) serán automáticamente empaquetadas por Next.js.

Si una dependencia utiliza características específicas de Node.js, puedes optar por excluir dependencias específicas del empaquetado de Componentes del Servidor y usar el `require` nativo de Node.js.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
