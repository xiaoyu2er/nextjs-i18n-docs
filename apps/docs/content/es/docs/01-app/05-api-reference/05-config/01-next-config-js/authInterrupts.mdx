La opción de configuración `authInterrupts` le permite utilizar las APIs [`forbidden`](/docs/app/api-reference/functions/forbidden) y [`unauthorized`](/docs/app/api-reference/functions/unauthorized) en su aplicación. Si bien estas funciones son experimentales, debe habilitar la opción `authInterrupts` en su archivo `next.config.js` para utilizarlas:

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    authInterrupts: true,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
module.exports = {
  experimental: {
    authInterrupts: true,
  },
}
```
