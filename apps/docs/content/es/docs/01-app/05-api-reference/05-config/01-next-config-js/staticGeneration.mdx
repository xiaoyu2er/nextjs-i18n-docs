Las opciones `staticGeneration*` le permiten configurar el proceso de Generación Estática (Static Generation) para casos de uso avanzados.

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
const nextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

## Opciones de Configuración

Las siguientes opciones están disponibles:

- `staticGenerationRetryCount`: Número de intentos para regenerar una página fallida antes de marcar el build como fallido.
- `staticGenerationMaxConcurrency`: Número máximo de páginas a procesar por worker.
- `staticGenerationMinPagesPerWorker`: Número mínimo de páginas a procesar antes de iniciar un nuevo worker.
