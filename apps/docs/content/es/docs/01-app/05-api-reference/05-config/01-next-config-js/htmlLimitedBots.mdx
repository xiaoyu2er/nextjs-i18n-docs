La configuración `htmlLimitedBots` permite especificar una lista de agentes de usuario que deben recibir metadatos de bloqueo en lugar de [metadatos de transmisión (streaming metadata)](/docs/app/api-reference/functions/generate-metadata#streaming-metadata).

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const config: NextConfig = {
  htmlLimitedBots: /MySpecialBot|MyAnotherSpecialBot|SimpleCrawler/,
}

export default config
```

```js filename="next.config.js" switcher
module.exports = {
  htmlLimitedBots: /MySpecialBot|MyAnotherSpecialBot|SimpleCrawler/,
}
```

## Lista predeterminada

Next.js incluye [una lista predeterminada de bots con HTML limitado](https://github.com/vercel/next.js/blob/canary/packages/next/src/shared/lib/router/utils/html-bots.ts).

Al especificar una configuración `htmlLimitedBots`, se sobrescribirá la lista predeterminada de Next.js, lo que le permitirá tener control total sobre qué agentes de usuario deben optar por este comportamiento. Sin embargo, este es un comportamiento avanzado y la configuración predeterminada debería ser suficiente para la mayoría de los casos.
