Por defecto, Next.js agregará el encabezado `x-powered-by`. Para desactivarlo, abre `next.config.js` y deshabilita la configuración `poweredByHeader`:

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
