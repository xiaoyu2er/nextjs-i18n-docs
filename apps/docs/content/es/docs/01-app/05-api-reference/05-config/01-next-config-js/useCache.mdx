La bandera `useCache` es una característica experimental en Next.js que permite utilizar la [directiva `use cache`](/docs/app/api-reference/directives/use-cache) independientemente de [`dynamicIO`](/docs/app/api-reference/config/next-config-js/dynamicIO). Cuando está habilitada, puedes usar `use cache` en tu aplicación incluso si `dynamicIO` está desactivado.

## Uso

Para habilitar la bandera `useCache`, configúrala como `true` en la sección `experimental` de tu archivo `next.config.ts`:

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

Cuando `useCache` está habilitado, puedes utilizar las siguientes funciones y configuraciones de caché:

- La [directiva `use cache`](/docs/app/api-reference/directives/use-cache)
- La función [`cacheLife`](/docs/app/api-reference/config/next-config-js/cacheLife) con `use cache`
- La función [`cacheTag`](/docs/app/api-reference/functions/cacheTag)
