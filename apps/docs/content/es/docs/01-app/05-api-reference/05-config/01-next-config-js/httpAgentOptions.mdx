En versiones de Node.js anteriores a la 18, Next.js aplica automáticamente un polyfill para `fetch()` con [undici](/docs/architecture/supported-browsers#polyfills) y habilita [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive) por defecto.

Para deshabilitar HTTP Keep-Alive en todas las llamadas `fetch()` del lado del servidor, abra `next.config.js` y agregue la configuración `httpAgentOptions`:

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
