Next.js expone algunas opciones que le permiten controlar cómo el servidor eliminará o mantendrá en memoria las páginas construidas en desarrollo.

Para cambiar los valores predeterminados, abra `next.config.js` y agregue la configuración `onDemandEntries`:

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // período (en ms) durante el cual el servidor mantendrá las páginas en el búfer
    maxInactiveAge: 25 * 1000,
    // número de páginas que deben mantenerse simultáneamente sin ser eliminadas
    pagesBufferLength: 2,
  },
}
```
