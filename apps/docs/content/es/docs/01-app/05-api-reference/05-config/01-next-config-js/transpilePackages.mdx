Next.js puede transpilar y empaquetar automáticamente dependencias de paquetes locales (como monorepos) o dependencias externas (`node_modules`). Esto reemplaza el paquete `next-transpile-modules`.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['nombre-del-paquete'],
}

module.exports = nextConfig
```
