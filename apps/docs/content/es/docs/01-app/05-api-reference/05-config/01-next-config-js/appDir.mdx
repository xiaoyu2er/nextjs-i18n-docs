> **Es bueno saberlo**: Esta opción **ya no es necesaria** a partir de Next.js 13.4. El Enrutador de Aplicación (App Router) ahora es estable.

El Enrutador de Aplicación ([directorio `app`](/docs/app/building-your-application/routing)) habilita soporte para [diseños (layouts)](/docs/app/api-reference/file-conventions/layout), [Componentes del Servidor (Server Components)](/docs/app/getting-started/server-and-client-components), [transmisión en flujo (streaming)](/docs/app/building-your-application/routing/loading-ui-and-streaming) y [obtención de datos colocalizada (colocated data fetching)](/docs/app/building-your-application/data-fetching).

El uso del directorio `app` habilitará automáticamente el [Modo Estricto de React](https://react.dev/reference/react/StrictMode). Aprende cómo [adoptar `app` incrementalmente](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app).
