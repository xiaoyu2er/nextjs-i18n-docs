Utilice la opción `crossOrigin` para agregar un [atributo `crossOrigin`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) en todas las etiquetas `<script>` generadas por el componente <AppOnly>[`next/script`](/docs/app/guides/scripts)</AppOnly> <PagesOnly>[`next/script`](/docs/pages/guides/scripts) y [`next/head`](/docs/pages/api-reference/components/head)</PagesOnly>, y definir cómo se deben manejar las solicitudes de origen cruzado (cross-origin).

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## Opciones

- `'anonymous'`: Agrega el atributo [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous).
- `'use-credentials'`: Agrega [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials).
