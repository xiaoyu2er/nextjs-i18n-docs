> Estás viendo la documentación del Enrutador de Páginas (Pages Router). Consulta la documentación del [Enrutador de Aplicación (App Router)](/docs/app) para conocer las últimas funciones.

El **Enrutador de Páginas (Pages Router)** utiliza un sistema de enrutamiento basado en archivos intuitivo para mapear cada archivo a una ruta.

Antes de Next.js 13, el Enrutador de Páginas (Pages Router) era la forma principal de crear rutas en Next.js. Todavía es compatible con versiones más recientes de Next.js, pero recomendamos migrar al nuevo [Enrutador de Aplicación (App Router)](/docs/app) para aprovechar las últimas funciones de React.
