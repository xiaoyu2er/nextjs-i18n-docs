Excluir dependencias específicas del empaquetado automático de la opción [`bundlePagesRouterDependencies`](/docs/pages/api-reference/config/next-config-js/bundlePagesRouterDependencies).

Estas páginas utilizarán entonces el `require` nativo de Node.js para resolver la dependencia.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
