title: Cómo actualizar a la versión 10
nav_title: Versión 10
description: Actualiza tu aplicación Next.js desde la versión 9 a la versión 10.
---

No hubo cambios importantes entre las versiones 9 y 10.

Para actualizar a la versión 10, ejecuta el siguiente comando:

```bash filename="Terminal"
npm i next@10
```

```bash filename="Terminal"
yarn add next@10
```

```bash filename="Terminal"
pnpm up next@10
```

```bash filename="Terminal"
bun add next@10
```

> **Nota importante:** Si estás usando TypeScript, asegúrate de actualizar también `@types/react` y `@types/react-dom` a sus versiones correspondientes.
