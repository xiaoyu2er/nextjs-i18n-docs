title: Obtención de Datos
description: Next.js permite obtener datos de múltiples formas, con prerenderizado, renderizado del lado del servidor (SSR) o generación de sitios estáticos, y regeneración estática incremental. Aprende a gestionar los datos de tu aplicación en Next.js.
---

La obtención de datos en Next.js te permite renderizar tu contenido de diferentes maneras, dependiendo del caso de uso de tu aplicación. Esto incluye prerrenderizado con **Renderizado del Lado del Servidor (SSR)** o **Generación Estática**, y actualización o creación de contenido en tiempo de ejecución con **Regeneración Estática Incremental**.
