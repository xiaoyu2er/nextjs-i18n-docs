Antes de Next.js 13, el Enrutador de Páginas (Pages Router) era la forma principal de crear rutas en Next.js. Utilizaba un enrutador basado en sistema de archivos intuitivo para mapear cada archivo a una ruta. El Enrutador de Páginas (Pages Router) sigue siendo compatible en versiones más recientes de Next.js, pero recomendamos migrar al nuevo [Enrutador de Aplicación (App Router)](/docs/app) para aprovechar las últimas características de React.

Utilice esta sección de la documentación para aplicaciones existentes que usen el Enrutador de Páginas (Pages Router).
