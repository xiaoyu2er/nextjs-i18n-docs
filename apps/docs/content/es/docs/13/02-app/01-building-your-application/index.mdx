Next.js proporciona los bloques fundamentales para crear aplicaciones web full-stack flexibles. Las guías en **Construyendo tu aplicación** explican cómo usar estas características y cómo personalizar el comportamiento de tu aplicación.

Las secciones y páginas están organizadas secuencialmente, desde lo básico hasta lo avanzado, para que puedas seguirlas paso a paso al construir tu aplicación con Next.js. Sin embargo, puedes leerlas en cualquier orden o saltar directamente a las páginas que apliquen a tu caso de uso.

<AppOnly>

Si eres nuevo en Next.js, recomendamos comenzar con las secciones de [Enrutamiento (Routing)](/docs/app/building-your-application/routing), [Renderizado (Rendering)](/docs/app/building-your-application/rendering), [Obtención de datos (Data Fetching)](/docs/app/building-your-application/data-fetching) y [Estilos (Styling)](/docs/app/building-your-application/styling), ya que introducen los conceptos fundamentales de Next.js y la web para ayudarte a comenzar. Luego, puedes profundizar en otras secciones como [Optimización (Optimizing)](/docs/app/building-your-application/optimizing) y [Configuración (Configuring)](/docs/app/building-your-application/configuring). Finalmente, cuando estés listo, revisa las secciones de [Despliegue (Deploying)](/docs/app/building-your-application/deploying) y [Actualización (Upgrading)](/docs/app/building-your-application/upgrading).

</AppOnly>
