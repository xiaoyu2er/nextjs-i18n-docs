`useParams` es un hook para **Componentes de Cliente** que permite leer los [parámetros dinámicos](/docs/app/building-your-application/routing/dynamic-routes) de una ruta rellenados por la URL actual.

```tsx filename="app/example-client-component.tsx" switcher
'use client'

import { useParams } from 'next/navigation'

export default function ExampleClientComponent() {
  const params = useParams()

  // Ruta -> /shop/[tag]/[item]
  // URL -> /shop/shoes/nike-air-max-97
  // `params` -> { tag: 'shoes', item: 'nike-air-max-97' }
  console.log(params)

  return <></>
}
```

```jsx filename="app/example-client-component.js" switcher
'use client'

import { useParams } from 'next/navigation'

export default function ExampleClientComponent() {
  const params = useParams()

  // Ruta -> /shop/[tag]/[item]
  // URL -> /shop/shoes/nike-air-max-97
  // `params` -> { tag: 'shoes', item: 'nike-air-max-97' }
  console.log(params)

  return <></>
}
```

## Parámetros

```tsx
const params = useParams()
```

`useParams` no recibe ningún parámetro.

## Retorno

`useParams` retorna un objeto que contiene los [parámetros dinámicos](/docs/app/building-your-application/routing/dynamic-routes) rellenados de la ruta actual.

- Cada propiedad en el objeto es un segmento dinámico activo.
- El nombre de la propiedad corresponde al nombre del segmento, y el valor de la propiedad es con qué se ha rellenado el segmento.
- El valor de la propiedad será un `string` o un array de `string` dependiendo del [tipo de segmento dinámico](/docs/app/building-your-application/routing/dynamic-routes).
- Si la ruta no contiene parámetros dinámicos, `useParams` retorna un objeto vacío.
- Si se usa en `pages`, `useParams` retornará `null`.

Por ejemplo:

| Ruta                           | URL         | `useParams()`             |
| ------------------------------- | ----------- | ------------------------- |
| `app/shop/page.js`              | `/shop`     | `null`                    |
| `app/shop/[slug]/page.js`       | `/shop/1`   | `{ slug: '1' }`           |
| `app/shop/[tag]/[item]/page.js` | `/shop/1/2` | `{ tag: '1', item: '2' }` |
| `app/shop/[...slug]/page.js`    | `/shop/1/2` | `{ slug: ['1', '2'] }`    |
