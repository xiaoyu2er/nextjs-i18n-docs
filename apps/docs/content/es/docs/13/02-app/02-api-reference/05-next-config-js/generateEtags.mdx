Next.js genera [etags](https://en.wikipedia.org/wiki/HTTP_ETag) para cada página por defecto. Es posible que desees deshabilitar la generación de etags para páginas HTML dependiendo de tu estrategia de caché.

Abre `next.config.js` y deshabilita la opción `generateEtags`:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
