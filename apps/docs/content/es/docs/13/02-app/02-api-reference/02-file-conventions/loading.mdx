Un archivo **loading** puede crear estados de carga instantáneos basados en [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming).

Por defecto, este archivo es un [Componente de Servidor (Server Component)](/docs/app/building-your-application/rendering/server-components) - pero también puede usarse como Componente de Cliente (Client Component) mediante la directiva `"use client"`.

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // O un componente de skeleton loading personalizado
  return <p>'Loading...'</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // O un componente de skeleton loading personalizado
  return <p>'Loading...'</p>
}
```

Los componentes de interfaz de usuario de carga no aceptan ningún parámetro.

> **Dato útil**
>
> - Al diseñar interfaces de carga, puede resultarte útil usar las [Herramientas de Desarrollo de React](https://react.dev/learn/react-developer-tools) para alternar manualmente los límites de Suspense.
