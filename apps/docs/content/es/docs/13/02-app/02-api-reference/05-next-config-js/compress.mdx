Next.js proporciona compresión [**gzip**](https://tools.ietf.org/html/rfc6713#section-3) para comprimir el contenido renderizado y archivos estáticos. En general, querrás habilitar la compresión en un proxy HTTP como [nginx](https://www.nginx.com/), para descargar la carga del proceso de `Node.js`.

Para desactivar la **compresión**, abre `next.config.js` y deshabilita la configuración `compress`:

```js filename="next.config.js"
module.exports = {
  compress: false,
}
```
