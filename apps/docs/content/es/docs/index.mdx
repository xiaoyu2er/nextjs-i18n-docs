¡Bienvenido a la documentación de Next.js!

## ¿Qué es Next.js?

Next.js es un framework de React para construir aplicaciones web full-stack. Utilizas Componentes de React para construir interfaces de usuario, y Next.js para características adicionales y optimizaciones.

También configura automáticamente herramientas de bajo nivel como bundlers y compiladores. Así puedes enfocarte en construir tu producto y lanzarlo rápidamente.

Ya seas un desarrollador individual o parte de un equipo más grande, Next.js puede ayudarte a construir aplicaciones React interactivas, dinámicas y rápidas.

## Cómo usar la documentación

La documentación está organizada en 4 secciones:

- [Primeros pasos](/docs/app/getting-started): Tutoriales paso a paso para ayudarte a crear una nueva aplicación y aprender las características principales de Next.js.
- [Guías](/docs/app/guides): Tutoriales sobre casos de uso específicos, elige lo que sea relevante para ti.
- [Análisis profundo](/docs/app/deep-dive): Explicaciones detalladas sobre cómo funciona Next.js.
- [Referencia API](/docs/app/api-reference): Referencia técnica detallada de cada característica.

Usa la barra lateral para navegar por las secciones, o busca (`Ctrl+K` o `Cmd+K`) para encontrar rápidamente una página.

## App Router y Pages Router

Next.js tiene dos routers diferentes:

- **App Router**: El router más nuevo que soporta nuevas características de React como Server Components.
- **Pages Router**: El router original, todavía soportado y siendo mejorado.

En la parte superior de la barra lateral, notarás un menú desplegable que te permite cambiar entre la documentación del [App Router](/docs/app) y el [Pages Router](/docs/pages).

## Conocimientos previos

Nuestra documentación asume cierta familiaridad con el desarrollo web. Antes de comenzar, será útil si estás cómodo con:

- HTML
- CSS
- JavaScript
- React

Si eres nuevo en React o necesitas un repaso, recomendamos comenzar con nuestro [curso Fundamentos de React](/learn/react-foundations), y el [curso Fundamentos de Next.js](/learn/dashboard-app) donde construirás una aplicación mientras aprendes.

## Accesibilidad

Para la mejor experiencia al usar un lector de pantalla, recomendamos usar Firefox con NVDA, o Safari con VoiceOver.
