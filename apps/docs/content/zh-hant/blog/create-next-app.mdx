我們很高興在此介紹全新的 Create Next App。

只需一個指令，Create Next App 就能為您建立一個基於 Next.js 的現代化 React 應用程式。

開始使用只需執行以下指令：

```bash filename="Terminal"
npx create-next-app
```

Create Next App 已從底層全面重建，以提供最佳的開發者體驗：

*   **互動式體驗**：現在執行不帶參數的 `npx create-next-app` 會啟動互動式介面，引導您完成專案設定。
*   **零依賴**：初始化專案現在只需 **1 秒**。Create Next App 零依賴且安裝大小僅 **604 kB**。在優化前，舊版本大小為 **5.38 MB**，減少了超過 **4.7 MB**！
*   **離線支援**：Create Next App 會自動檢測您是否處於離線狀態，並使用本地套件快取來初始化專案。
*   **新的預設專案模板**：Create Next App 使用專為現代 Next.js 應用程式設計的新模板。由於 Create Next App 現在與 Next.js 本身同步維護，此模板將始終保持與最新版 Next.js 同步！
*   **範例支援**：Create Next App 可以使用 [Next.js 範例集](https://github.com/vercel/next.js/tree/canary/examples)中的範例來初始化您的應用程式（例如 `npx create-next-app --example api-routes`）。
*   **經過測試**：此套件是 Next.js monorepo 的一部分，並使用與 Next.js 相同的整合測試套件進行測試，確保每個版本都能如預期運作。

Create Next App 先前是由[社群維護](https://open.segment.com/create-next-app/)的專案，但我們認為管理 Next.js 的第一印象非常重要，特別是當我們在 [Next.js 範例集](https://github.com/vercel/next.js/tree/canary/examples)中推薦它時。

我們與 [Segment](https://segment.com/) 合作轉移了套件的所有權，非常感謝他們先前的管理，特別是 [Fouad Matin](https://twitter.com/fouadmatin)。
