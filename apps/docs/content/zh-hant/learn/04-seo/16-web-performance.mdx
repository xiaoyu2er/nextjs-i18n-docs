[網頁指標 (Web Vitals)](https://web.dev/vitals/) 是由 Google 發起的計畫，旨在提供統一的指引與指標來衡量使用者在網頁上的體驗。

[核心網頁指標 (Core Web Vitals)](https://developers.google.com/search/blog/2020/11/timing-for-page-experience) 是網頁指標的子集，目前包含三個衡量載入速度、互動性與視覺穩定性的指標。這些指標分別是：[最大內容繪製 (Largest Contentful Paint, LCP)](/learn/seo/lcp)、[首次輸入延遲 (First Input Delay, FID)](/learn/seo/fid) 和 [累積版面配置位移 (Cumulative Layout Shift, CLS)](/learn/seo/cls)。

在這三個指標中取得良好分數，將為您的使用者創造更流暢的網站體驗。

若網站在核心網頁指標的各項評分表現不佳，將會影響其搜尋引擎排名，因為 Google 已開始將[核心網頁指標作為搜尋演算法的排名因素](https://developers.google.com/search/blog/2020/11/timing-for-page-experience)。不佳的指標表現可能對您的網站流量與業務造成影響。

在本課程中，您將學習：

* 核心網頁指標的簡要背景
* 核心網頁指標對 SEO 和使用者體驗 (UX) 的影響，以及對您網站的衝擊
* 為何您應在開發過程中重視核心網頁指標，以及如何測量這些指標
* 如何使用 Next.js 改善您的核心網頁指標並監控變化
