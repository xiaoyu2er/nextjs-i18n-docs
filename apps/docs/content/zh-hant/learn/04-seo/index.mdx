[什麼是 SEO？](#what-is-seo)
----------------------------

SEO 代表 **搜尋引擎最佳化 (Search Engine Optimization)**。SEO 的目標是制定能提升網站在搜尋引擎結果中排名的策略。排名越高，網站獲得的自然流量就越多，最終將為您帶來更多業務！

### [本課程將學到什麼](#what-youll-learn-in-this-course)

在本課程中，我們將討論：

*   搜尋系統與搜尋引擎機器人（如 Googlebot）
*   SEO 策略對網站的影響
*   如何整合最佳 SEO 實踐
*   使用 Next.js 實現爬取 (Crawling)、索引 (Indexing)、渲染 (Rendering) 和排名 (Ranking)
*   網路效能主題，包含核心網頁指標 (Core Web Vitals)
