讓我們看看如何利用 Next.js 的功能來提升範例應用的核心網頁指標 (Core Web Vitals)。

在本課程中，您將學習：

*   什麼是 Lighthouse 以及如何使用它。
*   如何使用 `next/image` 自動優化圖片。
*   如何動態載入函式庫和元件以減少初始 JS 打包體積。
*   如何預先連線 (preconnect) 第三方腳本。
*   Next.js 如何預設優化網頁字型載入。
*   如何優化任何第三方腳本的載入。
