在本課程中，我們將探討不同的指標、[核心 Web Vitals](https://web.dev/vitals/) 對 SEO 的影響，以及它們對使用者體驗的重要性。

測量核心 Web Vitals 時有三種不同的評級：

**「良好」**、**「需要改進」** 和 **「不佳」**。這些評級會根據測量的項目有所不同：

![核心 Web Vitals](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/seo/vitals-light.png)

您可以透過兩種方式來看待核心 Web Vitals：

*   **嘗試在每個指標上獲得最高分數**。追求完美很棒，但在具有許多依賴項的大型網站上可能會很棘手。
*   **與行業競爭對手進行基準比較**。您不是在 Google 搜尋中與每個完美優化的網站競爭，而是與其他針對目標關鍵字排名的網站競爭。
