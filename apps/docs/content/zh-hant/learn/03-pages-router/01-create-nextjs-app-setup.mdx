首先，請確保您的開發環境已準備就緒。

*   如果尚未安裝 **Node.js**，請[從此處安裝](https://nodejs.org/en/)。您需要 Node.js 版本 **18** 或更高版本。
*   本教學將使用您自己的文字編輯器和終端機應用程式。

> 如果您使用 Windows 系統，建議[下載 Git for Windows](https://gitforwindows.org/) 並使用內附的 Git Bash，它支援本教學中的 UNIX 專用指令。[Windows Subsystem for Linux (WSL)](https://docs.microsoft.com/en-us/windows/wsl/install-win10) 是另一個選擇。

### [建立 Next.js 應用程式](#create-a-nextjs-app)

要建立 Next.js 應用程式，請開啟終端機，`cd` 進入您想建立應用程式的目錄，然後執行以下指令：

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/learn-starter"
```

> 底層使用的是名為 [`create-next-app`](/docs/api-reference/create-next-app) 的工具，它會為您初始化 Next.js 應用程式。透過 `--example` 標記使用了[此範本](https://github.com/vercel/next-learn/tree/main/basics/learn-starter)。
> 
> 如果無法運作，請參考[此頁面](https://github.com/vercel/next-learn/blob/main/basics/errors/install.md)。

### [執行開發伺服器](#run-the-development-server)

現在您已有一個名為 `nextjs-blog` 的新目錄。請 `cd` 進入該目錄：

```
cd nextjs-blog
```

接著執行以下指令：

```
npm run dev
```

這會在連接埠 **3000** 上啟動您的 Next.js 應用程式的「開發伺服器 (development server)」（稍後會詳細說明）。

讓我們檢查是否運作正常。從瀏覽器開啟 [http://localhost:3000](http://localhost:3000)。
