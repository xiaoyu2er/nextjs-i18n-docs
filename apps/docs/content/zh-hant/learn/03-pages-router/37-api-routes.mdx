Next.js 支援 [API 路由](/docs/api-routes/introduction)，讓您可以輕鬆建立作為 Node.js 無伺服器函式 (serverless function) 的 API 端點。雖然我們的部落格應用程式不需要使用它，但在本課程中我們將簡要討論如何使用它。

### [本課程將學習的內容](#what-youll-learn-in-this-lesson)

在本課程中，您將學習：

*   如何建立 API 路由。
*   關於 [API 路由](/docs/api-routes/introduction) 的一些實用資訊。
