我們已經在首頁填充了部落格資料，但尚未建立個別的部落格頁面（這是[預期結果](https://next-learn-starter.vercel.app/)）。我們希望這些頁面的 URL 能根據部落格資料動態產生，這意味著需要使用[動態路由](/docs/routing/dynamic-routes)。

### [本課程將學習的內容](#what-youll-learn-in-this-lesson)

在本課程中，您將學習：

* 如何使用 [`getStaticPaths`](/docs/basic-features/data-fetching#getstaticpaths-static-generation) 搭配[動態路由](/docs/routing/dynamic-routes)來靜態產生頁面。
* 如何撰寫 [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) 來取得每篇部落格文章的資料。
* 如何使用 [`remark`](https://github.com/remarkjs/remark) 渲染 markdown 內容。
* 如何美化日期字串的顯示格式。
* 如何連結至使用[動態路由](/docs/routing/dynamic-routes)的頁面。
* 關於[動態路由](/docs/routing/dynamic-routes)的一些實用資訊。
