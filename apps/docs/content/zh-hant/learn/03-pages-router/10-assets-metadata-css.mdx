我們新增的第二個頁面目前沒有樣式。讓我們添加一些 CSS 來美化這個頁面。

Next.js 內建支援 [CSS](/docs/basic-features/built-in-css-support) 和 [Sass](/docs/basic-features/built-in-css-support#sass-support)。本課程中我們將使用 CSS。

本課程還會介紹 Next.js 如何處理靜態資源（例如圖片）和頁面元資料（例如 `<title>` 標籤）。

### [本課程將學習的內容](#what-youll-learn-in-this-lesson)

在本課程中，您將學習：

* 如何在 Next.js 中添加[靜態檔案](/docs/basic-features/static-file-serving)（圖片等）。
* 如何自訂每個頁面 `<head>` 中的內容。
* 如何使用 [CSS Modules](/docs/basic-features/built-in-css-support#adding-component-level-css) 建立可重複使用的 React 元件樣式。
* 如何在 [`pages/_app.js`](/docs/advanced-features/custom-app) 中[添加全域 CSS](/docs/basic-features/built-in-css-support#adding-a-global-stylesheet)。
* 一些在 Next.js 中實用的樣式技巧。

### [先決條件](#prerequisites)

* 基礎 CSS 知識。本課程將介紹如何在 Next.js 應用中添加 CSS，但不會涵蓋 CSS 基礎知識。

> 如需 Next.js 樣式的詳細文件，請參閱 [CSS 文件](/docs/basic-features/built-in-css-support)。
