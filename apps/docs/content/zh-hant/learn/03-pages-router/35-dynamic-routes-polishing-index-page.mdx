接下來，讓我們更新首頁 (`pages/index.js`)。我們需要使用 [`Link`](/docs/api-reference/next/link) 元件為每篇文章頁面添加連結。

打開 `pages/index.js` 並在檔案頂部添加以下導入語句，用於引入 [`Link`](/docs/api-reference/next/link) 和 `Date`：

```
import Link from 'next/link';
import Date from '../components/date';
```

然後，在同一個檔案的 `Home` 元件底部附近，將 `li` 標籤替換為以下內容：

```
<li className={utilStyles.listItem} key={id}>
  <Link href={`/posts/${id}`}>{title}</Link>
  <br />
  <small className={utilStyles.lightText}>
    <Date dateString={date} />
  </small>
</li>
```

如果你訪問 [http://localhost:3000](http://localhost:3000)，頁面現在會顯示每篇文章的連結：

![連結](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/dynamic-routes/links.png)
