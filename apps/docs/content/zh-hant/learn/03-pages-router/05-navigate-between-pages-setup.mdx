### [下載起始程式碼 (選用)](#下載起始程式碼-選用)

如果您不是從上一課繼續學習，可以下載、安裝並執行本課的起始程式碼。這會建立一個 `nextjs-blog` 目錄，其內容與上一課的結果完全相同。

再次強調，如果您剛完成上一課，則無需執行此步驟。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/navigate-between-pages-starter"
```

然後按照命令輸出的指示操作（進入目錄並啟動開發伺服器）。
