**如果您是從上一課繼續學習，**可以跳過本頁。

### [下載起始程式碼（選用）](#下載起始程式碼-optional)

如果您不是從上一課繼續學習，可以下載、安裝並執行本課的起始程式碼。這將建立一個 `nextjs-blog` 目錄，其內容與上一課的結果完全相同。

再次強調，如果您剛完成上一課，則無需執行此步驟。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/basics-final"
```

接著請依照命令列輸出的指示操作（`cd` 進入目錄並啟動開發伺服器）。

您還需要更新以下檔案：

*   將 `public/images/profile.jpg` 替換為您的照片（建議尺寸：寬高各 400px）。
*   將 `components/layout.js` 中的 `const name = '[您的名字]'` 替換為您的名字。
*   將 `pages/index.js` 中的 `<p>[您的自我介紹]</p>` 替換為您的自我介紹。
