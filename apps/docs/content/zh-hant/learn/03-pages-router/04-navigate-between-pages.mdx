目前我們建立的 Next.js 應用程式只有一個頁面。網站和網頁應用程式通常會有多個不同的頁面。

讓我們來探索如何為應用程式新增更多頁面。

[本課程將學習的內容](#what-youll-learn-in-this-lesson)
--------------------------------------------------------------------

在本課程中，您將：

*   使用整合的[檔案系統路由 (file system routing)](/docs/routing/introduction) 建立新頁面。
*   學習如何使用 [`Link`](https://nextjs.org/docs/api-reference/next/link) 元件來啟用頁面間的客戶端導航 (client-side navigation)。
*   了解內建的程式碼分割 (code splitting) 和預取 (prefetching) 支援功能。

> 如需 Next.js 路由的詳細文件，請參閱[路由文件](/docs/routing/introduction)。
