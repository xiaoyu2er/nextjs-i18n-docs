我們想要建立一個部落格（這是[預期成果](https://next-learn-starter.vercel.app/)），但目前尚未加入任何部落格內容。在本課程中，我們將學習如何將外部部落格資料獲取到應用程式中。我們會將部落格內容儲存在檔案系統中，但即使內容儲存在其他地方（例如資料庫或[無頭內容管理系統 (Headless CMS)](https://en.wikipedia.org/wiki/Headless_content_management_system)），此方法同樣適用。

### [本課程將學習的內容](#what-youll-learn-in-this-lesson)

在本課程中，您將學習：

* Next.js 的[預渲染 (pre-rendering)](/docs/basic-features/pages#pre-rendering) 功能。
* 兩種預渲染形式：[靜態生成 (Static Generation)](/docs/basic-features/pages#static-generation-recommended) 與[伺服器端渲染 (Server-side Rendering)](/docs/basic-features/pages#server-side-rendering)。
* 使用資料與不使用資料的靜態生成：[含資料](/docs/basic-features/pages#static-generation-with-data) 和 [不含資料](/docs/basic-features/pages#static-generation-without-data)。
* [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) 以及如何使用它將外部部落格資料導入首頁。
* 關於 [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) 的一些實用資訊。
