讓我們嘗試編輯起始頁面。

*   確保 Next.js 開發伺服器仍在運行。
*   使用文字編輯器開啟 `pages/index.js`。
*   找到 `<h1>` 標籤下方顯示 **"Welcome to"** 的文字，將其更改為 **"Learn"**。
*   儲存檔案。

當您儲存檔案後，瀏覽器會自動更新頁面顯示新文字：

![學習 Next.js](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/create-nextjs-app/learn-nextjs.png)

Next.js 開發伺服器已啟用 [快速刷新 (Fast Refresh)](/docs/basic-features/fast-refresh)。當您變更檔案時，Next.js 會幾乎立即在瀏覽器中自動套用變更，無需手動刷新！這將幫助您快速迭代應用程式。
