**若您延續前一課程進度**，可跳過此頁面。

### [下載起始程式碼 (選用)](#下載起始程式碼-選用)

若您並非延續前一課程進度，可下載、安裝並執行下方提供的本課程起始程式碼。此操作將建立一個 `nextjs-blog` 目錄，其內容與前一課程的成果完全相同。

再次強調，若您剛完成前一課程，此步驟並非必要。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/data-fetching-starter"
```

接著請依照指令輸出中的指示操作（切換至該目錄並啟動開發伺服器）。

您還需更新以下檔案：

*   將 `public/images/profile.jpg` 替換為您的個人照片（建議尺寸：寬高各 400 像素）。
*   將 `components/layout.js` 中的 `const name = '[您的姓名]'` 替換為您的姓名。
*   將 `pages/index.js` 中的 `<p>[您的自我介紹]</p>` 替換為您的自我介紹內容。
