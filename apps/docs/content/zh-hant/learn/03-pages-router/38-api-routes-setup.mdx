**若您延續前一個課程進行**，可以跳過此頁面。

### [下載起始程式碼 (選用)](#下載起始程式碼-選用)

若您並非延續前一個課程，您可以下載、安裝並執行本課程的起始程式碼。這會建立一個 `nextjs-blog` 目錄，其內容與前一個課程的結果完全相同。

再次提醒，若您剛完成前一個課程，此步驟並非必要。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/api-routes-starter"
```

接著請依照指令輸出的指示操作（進入目錄並啟動開發伺服器）。

您還需要更新以下檔案：

*   將 `public/images/profile.jpg` 替換為您的照片（建議尺寸：寬高 400px）。
*   將 `components/layout.js` 中的 `const name = '[您的名字]'` 替換為您的名字。
*   將 `pages/index.js` 中的 `<p>[您的自我介紹]</p>` 替換為您的自我介紹。
