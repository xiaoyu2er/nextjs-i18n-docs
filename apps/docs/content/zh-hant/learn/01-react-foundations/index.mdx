要有效學習 Next.js，熟悉 JavaScript、React 及相關網頁開發概念會有所幫助。但 JavaScript 和 React 是龐大的主題，您如何知道自己何時準備好開始使用 Next.js？

歡迎來到 React 基礎知識課程！這個適合初學者、以範例為導向的課程，將引導您掌握 Next.js 所需的先備知識。您將逐步建立一個簡單專案：從 JavaScript 應用程式開始，然後遷移至 React 和 Next.js。

每個章節都建立在前一章的基礎上，因此您可以根據現有知識選擇從哪裡開始。

### [先備知識](#prerequisite-knowledge)

本教學假設您具備 HTML、CSS 和 JavaScript 知識，但不需 React 基礎。如果您已熟悉 React，可以直接跳至 [從 React 到 Next.js](/learn/react-foundations/from-react-to-nextjs) 章節或學習 [儀表板應用程式](/learn/dashboard-app) 課程。

### [系統需求](#system-requirements)

開始本課程前，請確保已安裝以下環境：

*   Node.js 20.12.0 或更新版本
*   作業系統：macOS、Windows（包含 WSL）或 Linux
*   VSCode 或其他您選擇的文字編輯器
