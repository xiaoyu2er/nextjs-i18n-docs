Next.js 每週下載量超過 500 萬次，擁有來自全球龐大且活躍的開發者社群。以下是您可以參與社群的方式：

## 貢獻專案

您可以透過以下幾種方式為 Next.js 的發展做出貢獻：

- [文件](/docs/community/contribution-guide)：建議改進內容或撰寫新章節，幫助使用者理解如何使用 Next.js。
- [範例專案](https://github.com/vercel/next.js/tree/canary/contributing/examples)：透過建立新範例或改進現有範例，協助開發者整合 Next.js 與其他工具和服務。
- [程式碼庫](https://github.com/vercel/next.js/tree/canary/contributing/core)：深入了解底層架構，參與錯誤修復並建議新功能。

## 討論交流

如果您對 Next.js 有疑問，或想幫助他人，隨時歡迎加入討論：

- [GitHub 討論區](https://github.com/vercel/next.js/discussions)
- [Discord](https://discord.com/invite/bUG2bvbtHy)
- [Reddit](https://www.reddit.com/r/nextjs)

## 社群媒體

在 [Twitter](https://twitter.com/nextjs) 追蹤 Next.js 獲取最新消息，並訂閱 [Vercel YouTube 頻道](https://www.youtube.com/@VercelHQ)觀看 Next.js 相關影片。

## 行為準則

我們致力於建立包容友善的社群環境。因此，我們要求所有成員遵守 [行為準則](https://github.com/vercel/next.js/blob/canary/CODE_OF_CONDUCT.md)。該文件概述了我們對參與者行為的期望，請閱讀並協助我們維護安全尊重的環境。
