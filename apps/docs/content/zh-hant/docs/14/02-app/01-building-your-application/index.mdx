Next.js 提供了構建靈活、全端網頁應用程式的基本模組。**構建您的應用程式**指南將說明如何使用這些功能，以及如何自訂應用程式的行為。

各節內容與頁面按從基礎到進階的順序排列，讓您在構建 Next.js 應用程式時能逐步跟隨。不過，您也可以按任何順序閱讀，或直接跳至符合您使用情境的頁面。

<AppOnly>

如果您是 Next.js 的新手，我們建議從[路由 (Routing)](/docs/app/building-your-application/routing)、[渲染 (Rendering)](/docs/app/building-your-application/rendering)、[資料獲取 (Data Fetching)](/docs/app/building-your-application/data-fetching) 和[樣式 (Styling)](/docs/app/building-your-application/styling) 章節開始，這些內容介紹了 Next.js 與網頁開發的基本概念，能幫助您快速入門。接著，您可以深入探討其他章節，例如[優化 (Optimizing)](/docs/app/building-your-application/optimizing) 和[配置 (Configuring)](/docs/app/building-your-application/configuring)。最後，當您準備好時，可以查看[部署 (Deploying)](/docs/app/building-your-application/deploying) 和[升級 (Upgrading)](/docs/app/building-your-application/upgrading) 章節。
