本節文件涵蓋了**元數據檔案約定**。您可以通過在路由區段中添加特殊的元數據檔案來定義基於檔案的元數據。

每種檔案約定都可以使用靜態檔案（例如 `opengraph-image.jpg`）或使用代碼生成檔案的動態變體（例如 `opengraph-image.js`）來定義。

一旦檔案被定義，Next.js 將自動提供該檔案（在生產環境中會添加哈希值以進行緩存），並使用正確的元數據（例如資源的 URL、檔案類型和圖像大小）更新相關的 head 元素。
