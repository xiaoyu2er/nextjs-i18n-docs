Next.js 提供 **兩種** 命令列介面 (CLI) 工具：

- **`create-next-app`**：使用預設範本或從公開 Github 儲存庫中的 [範例](https://github.com/vercel/next.js/tree/canary/examples) 快速建立新的 Next.js 應用程式。
- **`next`**：執行 Next.js 開發伺服器、建置您的應用程式等更多功能。
