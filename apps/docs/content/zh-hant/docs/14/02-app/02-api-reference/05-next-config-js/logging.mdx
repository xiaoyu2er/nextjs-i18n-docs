您可以設定日誌記錄等級以及是否在開發模式下執行 Next.js 時，將完整 URL 記錄到控制台。

目前，`logging` 僅適用於使用 `fetch` API 進行的資料獲取。它還不適用於 Next.js 內的其他日誌記錄。

```js filename="next.config.js"
module.exports = {
  logging: {
    fetches: {
      fullUrl: true,
    },
  },
}
```
