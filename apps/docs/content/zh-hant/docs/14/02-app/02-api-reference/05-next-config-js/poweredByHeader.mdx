Next.js 預設會加入 `x-powered-by` 標頭。若要選擇停用此功能，請開啟 `next.config.js` 並停用 `poweredByHeader` 設定：

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
