實驗性的 `instrumentationHook` 選項允許您透過 [`instrumentation` 檔案](/docs/app/api-reference/file-conventions/instrumentation)在 Next.js 應用程式中設定儀器化功能。

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
