使用 `crossOrigin` 選項可在 <AppOnly>[`next/script`](/docs/app/building-your-application/optimizing/scripts) 元件</AppOnly> <PagesOnly>[`next/script`](/docs/pages/building-your-application/optimizing/scripts) 和 [`next/head`](/docs/pages/api-reference/components/head) 元件</PagesOnly> 生成的所有 `<script>` 標籤中添加 [`crossOrigin` 屬性](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin)，並定義應如何處理跨來源請求。

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## 選項

- `'anonymous'`: 添加 [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous) 屬性。
- `'use-credentials'`: 添加 [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials) 屬性。
