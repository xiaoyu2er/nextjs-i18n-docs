[靜態類型連結](/docs/app/building-your-application/configuring/typescript#statically-typed-links) 的實驗性支援功能。此功能需在專案中使用 App Router 及 TypeScript。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
