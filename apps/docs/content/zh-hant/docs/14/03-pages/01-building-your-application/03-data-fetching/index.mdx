description: Next.js 允許您透過預渲染、伺服器端渲染或靜態網站生成，以及增量靜態再生等多種方式獲取資料。了解如何在 Next.js 中管理應用程式資料。
---

Next.js 中的資料獲取功能讓您可以根據應用程式的使用情境，以不同方式渲染內容。這些方式包括使用**伺服器渲染 (Server-side Rendering)** 或**靜態生成 (Static Generation)** 進行預渲染，以及透過**增量靜態再生 (Incremental Static Regeneration)** 在運行時更新或建立內容。
