在 Next.js 13 之前，頁面路由 (Pages Router) 是 Next.js 中建立路由的主要方式。它使用直觀的檔案系統路由，將每個檔案映射到一個路由。較新版本的 Next.js 仍支援頁面路由 (Pages Router)，但我們建議遷移至新的 [應用路由 (App Router)](/docs/app) 以利用 React 的最新功能。

若您的現有應用程式使用頁面路由 (Pages Router)，請參考本節文件。
