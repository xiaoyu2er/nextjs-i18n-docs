Next.js 提供了一些選項，讓您可以控制開發伺服器如何釋放或保留記憶體中已構建的頁面。

要修改預設值，請開啟 `next.config.js` 並新增 `onDemandEntries` 配置：

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // 伺服器將頁面保留在緩衝區的時間週期（單位：毫秒）
    maxInactiveAge: 25 * 1000,
    // 應同時保留而不被釋放的頁面數量
    pagesBufferLength: 2,
  },
}
```
