> **須知**：自 Next.js 13.4 起**不再**需要此選項。App Router 現已穩定。

App Router ([`app` 目錄](/docs/app/building-your-application/routing)) 支援[佈局 (layouts)](/docs/app/building-your-application/routing/pages-and-layouts)、[伺服器元件 (Server Components)](/docs/app/building-your-application/rendering/server-components)、[串流 (streaming)](/docs/app/building-your-application/routing/loading-ui-and-streaming) 以及[同位置資料獲取 (colocated data fetching)](/docs/app/building-your-application/data-fetching)。

使用 `app` 目錄將自動啟用 [React 嚴格模式 (React Strict Mode)](https://react.dev/reference/react/StrictMode)。了解如何[逐步採用 `app`](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app)。
