**loading** 檔案可以建立基於 [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming) 的即時載入狀態。

預設情況下，此檔案是一個 [伺服器元件 (Server Component)](/docs/app/building-your-application/rendering/server-components) - 但也可以透過 `"use client"` 指令作為用戶端元件 (Client Component) 使用。

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // 或自訂的載入骨架元件
  return <p>'Loading...'</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // 或自訂的載入骨架元件
  return <p>'Loading...'</p>
}
```

載入 UI 元件不接受任何參數。

> **小知識**
>
> - 在設計載入 UI 時，您可能會發現使用 [React 開發者工具 (React Developer Tools)](https://react.dev/learn/react-developer-tools) 手動切換 Suspense 邊界很有幫助。
