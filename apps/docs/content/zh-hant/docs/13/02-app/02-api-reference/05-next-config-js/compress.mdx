Next.js 提供 [**gzip**](https://tools.ietf.org/html/rfc6713#section-3) 壓縮功能來壓縮渲染內容和靜態檔案。通常您會希望在 HTTP 代理伺服器 (如 [nginx](https://www.nginx.com/)) 上啟用壓縮，以減輕 `Node.js` 程序的負載。

若要停用**壓縮**功能，請開啟 `next.config.js` 並停用 `compress` 設定：

```js filename="next.config.js"
module.exports = {
  compress: false,
}
```
