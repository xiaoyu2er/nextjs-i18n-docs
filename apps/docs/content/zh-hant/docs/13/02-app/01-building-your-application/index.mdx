Next.js 提供了構建靈活的全端網頁應用程式所需的基礎模組。**構建你的應用程式** 章節中的指南將解釋如何使用這些功能，以及如何自訂應用程式的行為。

這些章節和頁面按照從基礎到進階的順序組織，讓你在構建 Next.js 應用程式時可以逐步跟隨。不過，你也可以按任意順序閱讀，或直接跳轉到符合你使用情境的頁面。

如果你是 Next.js 的新手，我們建議從 [路由 (Routing)](/docs/app/building-your-application/routing)、[渲染 (Rendering)](/docs/app/building-your-application/rendering)、[資料獲取 (Data Fetching)](/docs/app/building-your-application/data-fetching) 和 [樣式 (Styling)](/docs/app/building-your-application/styling) 章節開始，這些章節介紹了 Next.js 和網頁開發的基礎概念，幫助你入門。接著，你可以深入瞭解其他章節，例如 [優化 (Optimizing)](/docs/app/building-your-application/optimizing) 和 [配置 (Configuring)](/docs/app/building-your-application/configuring)。最後，當你準備好時，可以查看 [部署 (Deploying)](/docs/app/building-your-application/deploying) 和 [升級 (Upgrading)](/docs/app/building-your-application/upgrading) 章節。
