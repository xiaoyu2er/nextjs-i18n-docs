這個**入門指南**章節將幫助您建立第一個 Next.js 應用程式，並學習每個專案都會用到的核心功能。

## 預備知識

我們的文件假設您對網頁開發有一定了解。在開始之前，建議您熟悉以下技術：

- HTML
- CSS
- JavaScript
- React

如果您是 React 新手或需要複習，我們建議先學習 [React 基礎課程](/learn/react-foundations)，以及 [Next.js 基礎課程](/learn/dashboard-app)，這個課程會讓您在學習過程中實際建構一個應用程式。
