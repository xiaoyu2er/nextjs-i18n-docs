## 最新版本

要更新至 Next.js 的最新版本，你可以使用 `upgrade` 程式碼修改工具 (codemod)：

```bash filename="終端機"
npx @next/codemod@canary upgrade latest
```

如果你偏好手動升級，請安裝最新版本的 Next.js 和 React：

```bash filename="終端機"
npm i next@latest react@latest react-dom@latest eslint-config-next@latest
```

## Canary 版本

要更新至最新的 canary 版本，請確保你已使用 Next.js 的最新版本且一切運作正常。接著執行以下指令：

```bash filename="終端機"
npm i next@canary
```
