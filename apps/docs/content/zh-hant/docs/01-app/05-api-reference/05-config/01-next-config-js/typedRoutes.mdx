實驗性支援 [靜態類型連結 (statically typed links)](/docs/app/api-reference/config/typescript#statically-typed-links)。此功能需在專案中使用 App Router 及 TypeScript。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
