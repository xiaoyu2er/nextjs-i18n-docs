在[伺服器元件 (Server Components)](/docs/app/getting-started/server-and-client-components) 和[路由處理器 (Route Handlers)](/docs/app/building-your-application/routing/route-handlers) 中使用的依賴項會自動由 Next.js 進行打包。

如果某個依賴項使用了 Node.js 的特定功能，您可以選擇將特定依賴項從伺服器元件打包中排除，並使用原生 Node.js 的 `require`。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
