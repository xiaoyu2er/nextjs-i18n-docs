title: reactMaxHeadersLength
description: 關於 React 發出並添加到回應中的標頭最大長度設定。
---

在靜態渲染期間，React 可以發出能被添加到回應中的標頭。這些標頭可用於提升效能，例如讓瀏覽器預先載入字型、腳本和樣式表等資源。預設值為 `6000`，但您可以透過在 `next.config.js` 中配置 `reactMaxHeadersLength` 選項來覆寫此值：

```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **須知**：此選項僅在 App Router 中可用。

根據瀏覽器與伺服器之間使用的代理類型，標頭可能會被截斷。例如，如果您使用的反向代理不支援過長的標頭，則應設定較小的值以確保標頭不會被截斷。
