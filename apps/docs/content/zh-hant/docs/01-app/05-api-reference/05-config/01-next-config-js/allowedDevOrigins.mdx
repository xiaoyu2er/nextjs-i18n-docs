Next.js 在開發期間不會自動阻擋跨來源請求，但在未來的重大版本中預設將會阻擋，以防止在開發模式下未經授權存取內部資源/端點。

若要設定 Next.js 應用程式允許來自伺服器初始化主機名稱 (預設為 `localhost`) 以外的來源請求，您可以使用 `allowedDevOrigins` 設定選項。

`allowedDevOrigins` 允許您設定在開發模式下可使用的額外來源。例如，若要使用 `local-origin.dev` 而非僅限 `localhost`，請開啟 `next.config.js` 並新增 `allowedDevOrigins` 設定：

```js filename="next.config.js"
module.exports = {
  allowedDevOrigins: ['local-origin.dev', '*.local-origin.dev'],
}
```
