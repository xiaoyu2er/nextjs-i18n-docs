`useCache` 標誌是 Next.js 中的一項實驗性功能，它允許獨立於 [`dynamicIO`](/docs/app/api-reference/config/next-config-js/dynamicIO) 使用 [`use cache` 指令](/docs/app/api-reference/directives/use-cache)。啟用後，即使關閉了 `dynamicIO`，您仍可在應用程式中使用 `use cache`。

## 使用方法

要啟用 `useCache` 標誌，請在 `next.config.ts` 檔案的 `experimental` 區段中將其設為 `true`：

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

啟用 `useCache` 後，您可以使用以下快取函數和配置：

- [`use cache` 指令](/docs/app/api-reference/directives/use-cache)
- 搭配 `use cache` 使用的 [`cacheLife` 函數](/docs/app/api-reference/config/next-config-js/cacheLife)
- [`cacheTag` 函數](/docs/app/api-reference/functions/cacheTag)
