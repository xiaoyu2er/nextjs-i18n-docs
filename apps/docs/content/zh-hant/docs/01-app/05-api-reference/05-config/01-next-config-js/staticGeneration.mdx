`staticGeneration*` 選項允許你針對進階使用情境配置靜態生成 (Static Generation) 流程。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
const nextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

## 配置選項

以下選項可供使用：

- `staticGenerationRetryCount`: 在構建失敗前，重試失敗頁面生成的次數。
- `staticGenerationMaxConcurrency`: 每個工作執行緒 (worker) 最多可處理的頁面數量。
- `staticGenerationMinPagesPerWorker`: 啟動新工作執行緒 (worker) 前需處理的最少頁面數量。
