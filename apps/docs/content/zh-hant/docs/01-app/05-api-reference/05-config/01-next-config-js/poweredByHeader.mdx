Next.js 預設會加入 `x-powered-by` 標頭。若要停用此功能，請開啟 `next.config.js` 並將 `poweredByHeader` 設定設為 false：

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
