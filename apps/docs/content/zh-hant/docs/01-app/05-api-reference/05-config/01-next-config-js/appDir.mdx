> **須知**：自 Next.js 13.4 起**不再**需要此選項。App Router 現已穩定。

App Router ([`app` 目錄](/docs/app/building-your-application/routing)) 支援[佈局 (layouts)](/docs/app/api-reference/file-conventions/layout)、[伺服器元件 (Server Components)](/docs/app/getting-started/server-and-client-components)、[串流 (streaming)](/docs/app/building-your-application/routing/loading-ui-and-streaming) 以及[同位置資料獲取 (colocated data fetching)](/docs/app/building-your-application/data-fetching)。

使用 `app` 目錄將自動啟用 [React 嚴格模式 (React Strict Mode)](https://react.dev/reference/react/StrictMode)。了解如何[逐步採用 `app`](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app)。
