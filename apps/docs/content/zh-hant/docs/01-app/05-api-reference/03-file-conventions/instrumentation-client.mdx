`instrumentation-client.js|ts` 檔案可讓您新增監控與分析程式碼，這些程式碼會在應用程式的前端程式碼開始執行前運作。這對於設定效能追蹤、錯誤監控或任何其他用戶端可觀測性工具非常有用。

若要使用此功能，請將檔案放置在應用程式的**根目錄**或 `src` 資料夾內。

## 使用方式

與[伺服器端檢測](/docs/app/guides/instrumentation)不同，您不需要匯出任何特定函式。您可以直接在檔案中撰寫監控程式碼：

```ts filename="instrumentation-client.ts" switcher
// 設定效能監控
performance.mark('app-init')

// 初始化分析工具
console.log('Analytics initialized')

// 設定錯誤追蹤
window.addEventListener('error', (event) => {
  // 傳送至您的錯誤追蹤服務
  reportError(event.error)
})
```

