[Tailwind CSS](https://tailwindcss.com/) 是一個以工具優先的 CSS 框架，完全兼容 Next.js。本指南將逐步介紹如何在您的 Next.js 應用程式中安裝 Tailwind CSS。

## 安裝 Tailwind

安裝必要的 Tailwind CSS 套件：

```bash filename="終端機"
npm install -D tailwindcss @tailwindcss/postcss postcss
```

> **小提示**：如果您使用 `create-next-app` CLI 創建專案，Next.js 會詢問您是否要安裝 Tailwind 並自動配置專案。

## 配置 Tailwind

在專案根目錄創建一個 `postcss.config.mjs` 文件，並將 `@tailwindcss/postcss` 插件添加到您的 PostCSS 配置中：

```js filename="postcss.config.mjs" highlight={4}
/** @type {import('tailwindcss').Config} */
export default {
  plugins: {
    '@tailwindcss/postcss': {},
  },
}
```

從 [Tailwind v4](https://tailwindcss.com/blog/tailwindcss-v4) 開始，預設情況下不需要任何配置。如果您確實需要配置 Tailwind，可以參考 [官方文件](https://tailwindcss.com/blog/tailwindcss-v4#css-first-configuration) 來配置全域 CSS 文件。

如果您有現有的 Tailwind v3 專案，也可以使用 [升級 CLI](https://tailwindcss.com/docs/upgrade-guide) 和 [指南](https://tailwindcss.com/docs/upgrade-guide)。

## 導入樣式

將 [Tailwind CSS 指令](https://tailwindcss.com/docs/functions-and-directives#directives) 添加到您的應用程式中的 [全域樣式表](/docs/app/getting-started/css#global-css)，Tailwind 將使用這些指令來注入其生成的樣式，例如：

```css filename="app/globals.css"
@import 'tailwindcss';
```

在 [根佈局](/docs/app/building-your-application/routing/layouts-and-templates#root-layout-required) (`app/layout.tsx`) 中導入 `globals.css` 樣式表，以將樣式應用到應用程式中的每個路由。

```tsx filename="app/layout.tsx" switcher
import type { Metadata } from 'next'

// 這些樣式將應用到應用程式中的每個路由
import './globals.css'

export const metadata: Metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
}

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

```jsx filename="app/layout.js" switcher
// 這些樣式將應用到應用程式中的每個路由
import './globals.css'

export const metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
}

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

## 導入樣式

將 [Tailwind CSS 指令](https://tailwindcss.com/docs/functions-and-directives#directives) 添加到您的應用程式中的 [全域樣式表](/docs/app/getting-started/css#global-css)，Tailwind 將使用這些指令來注入其生成的樣式，例如：

```css filename="styles/globals.css"
@import 'tailwindcss';
```

在 [自訂應用程式文件](/docs/pages/building-your-application/routing/custom-app) (`pages/_app.js`) 中導入 `globals.css` 樣式表，以將樣式應用到應用程式中的每個路由。

```tsx filename="pages/_app.tsx" switcher
// 這些樣式將應用到應用程式中的每個路由
import '@/styles/globals.css'
import type { AppProps } from 'next/app'

export default function App({ Component, pageProps }: AppProps) {
  return <Component {...pageProps} />
}
```

```jsx filename="pages/_app.js" switcher
// 這些樣式將應用到應用程式中的每個路由
import '@/styles/globals.css'

export default function App({ Component, pageProps }) {
  return <Component {...pageProps} />
}
```
