## 從 13 升級至 14

要更新至 Next.js 版本 14，請使用您偏好的套件管理器執行以下指令：

```bash filename="終端機"
npm i next@next-14 react@18 react-dom@18 && npm i eslint-config-next@next-14 -D
```

```bash filename="終端機"
yarn add next@next-14 react@18 react-dom@18 && yarn add eslint-config-next@next-14 -D
```

```bash filename="終端機"
pnpm i next@next-14 react@18 react-dom@18 && pnpm i eslint-config-next@next-14 -D
```

```bash filename="終端機"
bun add next@next-14 react@18 react-dom@18 && bun add eslint-config-next@next-14 -D
```

> **須知事項：** 如果您使用 TypeScript，請確保同時將 `@types/react` 和 `@types/react-dom` 升級至最新版本。

### v14 版本摘要

- 最低 Node.js 版本要求已從 16.14 提升至 18.17，因為 16.x 已結束生命週期。
- `next export` 指令已被移除，改用 `output: 'export'` 設定。詳情請參閱[文件](https://nextjs.org/docs/app/guides/static-exports)。
- `next/server` 中的 `ImageResponse` 導入已更名為 `next/og`。[提供 codemod 工具](/docs/app/guides/upgrading/codemods#next-og-import)可安全自動地替換您的導入語句。
- `@next/font` 套件已完全移除，改用內建的 `next/font`。[提供 codemod 工具](/docs/app/guides/upgrading/codemods#built-in-next-font)可安全自動地替換您的導入語句。
- `next-swc` 的 WASM 目標支援已被移除。
