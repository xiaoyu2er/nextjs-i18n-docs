---

[JSON-LD](https://json-ld.org/) 是一種結構化資料格式，可被搜尋引擎和 AI 用來幫助它們理解頁面內容之外的結構。例如，您可以用它來描述人物、事件、組織、電影、書籍、食譜等各種實體類型。

我們目前對於 JSON-LD 的建議是將結構化資料以 `<script>` 標籤的形式渲染在您的 `layout.js` 或 `page.js` 元件中。例如：

```tsx filename="app/products/[id]/page.tsx" switcher
export default async function Page({ params }) {
  const { id } = await params
  const product = await getProduct(id)

  const jsonLd = {
    '@context': 'https://schema.org',
    '@type': 'Product',
    name: product.name,
    image: product.image,
    description: product.description,
  }

  return (
    <section>
      {/* Add JSON-LD to your page */}
      <script
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(jsonLd) }}
      />
      {/* ... */}
    </section>
  )
}
```


您可以使用 Google 的 [Rich Results Test](https://search.google.com/test/rich-results) 或通用的 [Schema Markup Validator](https://validator.schema.org/) 來驗證和測試您的結構化資料。

您可以使用社群套件如 [`schema-dts`](https://www.npmjs.com/package/schema-dts) 來為 JSON-LD 加上 TypeScript 型別：

```tsx
import { Product, WithContext } from 'schema-dts'

const jsonLd: WithContext<Product> = {
  '@context': 'https://schema.org',
  '@type': 'Product',
  name: 'Next.js Sticker',
  image: 'https://nextjs.org/imgs/sticker.png',
  description: 'Dynamic at the speed of static.',
}
```
