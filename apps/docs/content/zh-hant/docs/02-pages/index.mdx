> 您正在查看的是頁面路由器 (Pages Router) 的文件。如需最新功能，請參閱 [應用路由器 (App Router)](/docs/app) 文件。

**頁面路由器 (Pages Router)** 採用直觀的檔案系統路由器，將每個檔案對應至一個路由。

在 Next.js 13 之前，頁面路由器 (Pages Router) 是 Next.js 中建立路由的主要方式。新版本的 Next.js 仍支援此功能，但我們建議遷移至新的 [應用路由器 (App Router)](/docs/app) 以利用 React 的最新功能。
