為 Pages Router 應用程式啟用伺服器端自動依賴項打包功能。此功能與 App Router 中的自動依賴項打包行為一致。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  bundlePagesRouterDependencies: true,
}

module.exports = nextConfig
```

可以使用 [`serverExternalPackages`](/docs/pages/api-reference/config/next-config-js/serverExternalPackages) 選項明確排除特定套件不被打包。

| ---------- | --------------------------------------------------------------------------------------------------------- |
| `v15.0.0` | 從實驗性功能轉為穩定功能。名稱從 `bundlePagesExternals` 更改為 `bundlePagesRouterDependencies` |
