Bevor wir über [Datenabruf](/docs/basic-features/data-fetching) sprechen, lassen Sie uns eines der wichtigsten Konzepte in Next.js besprechen: [**Pre-Rendering**](/docs/basic-features/pages#pre-rendering).

Standardmäßig führt Next.js für jede Seite ein Pre-Rendering durch. Das bedeutet, dass Next.js _HTML für jede Seite im Voraus generiert_, anstatt alles clientseitig mit JavaScript zu erledigen. Pre-Rendering kann zu besserer Leistung und verbessertem [SEO](https://de.wikipedia.org/wiki/Suchmaschinenoptimierung) führen.

Jedes generierte HTML ist mit dem minimal notwendigen JavaScript-Code für diese Seite verknüpft. Wenn eine Seite vom Browser geladen wird, wird ihr JavaScript-Code ausgeführt und macht die Seite vollständig interaktiv. (Dieser Prozess wird **Hydration** genannt.)

### [Überprüfen, ob Pre-Rendering stattfindet](#check-that-pre-rendering-is-happening)

Sie können überprüfen, ob Pre-Rendering stattfindet, indem Sie folgende Schritte ausführen:

1.  Deaktivieren Sie JavaScript in Ihrem Browser. ([Hier erfahren Sie, wie das in Chrome geht](https://developer.chrome.com/docs/devtools/javascript/disable/)).
2.  [Versuchen Sie, auf diese Seite zuzugreifen](https://next-learn-starter.vercel.app/) (das Endergebnis dieses Tutorials).

Sie sollten sehen, dass Ihre App ohne JavaScript gerendert wird. Das liegt daran, dass Next.js die App in statisches HTML pre-rendered, sodass Sie die Benutzeroberfläche der App sehen können, ohne JavaScript ausführen zu müssen.

> **Hinweis**: Sie können die obigen Schritte auch auf `localhost` ausprobieren, aber CSS wird nicht geladen, wenn Sie JavaScript deaktivieren.

Wenn Ihre App eine reine React.js-App (ohne Next.js) ist, gibt es kein [Pre-Rendering](/docs/basic-features/pages#pre-rendering), sodass Sie die App nicht sehen können, wenn Sie JavaScript deaktivieren. Beispiel:

*   Aktivieren Sie JavaScript in Ihrem Browser und [rufen Sie diese Seite auf](https://create-react-template.vercel.app/). Dies ist eine reine React.js-App, die mit [Create React App](https://create-react-app.dev/) erstellt wurde.
*   Deaktivieren Sie nun JavaScript und rufen Sie [dieselbe Seite](https://create-react-template.vercel.app/) erneut auf.
*   Sie werden die App nicht mehr sehen – stattdessen wird die Meldung "Sie müssen JavaScript aktivieren, um diese App auszuführen" angezeigt. Das liegt daran, dass die App nicht in statisches HTML pre-rendered wurde.
