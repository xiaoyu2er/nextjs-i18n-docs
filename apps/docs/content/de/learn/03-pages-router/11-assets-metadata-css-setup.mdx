### [Starter-Code herunterladen (optional)](#download-starter-code-optional)

Falls Sie NICHT von der vorherigen Lektion fortfahren, können Sie den Starter-Code für diese Lektion unten herunterladen, installieren und ausführen. Dies richtet ein `nextjs-blog`-Verzeichnis so ein, dass es identisch mit dem Ergebnis der vorherigen Lektion ist.

Nochmals: Dies ist NICHT notwendig, wenn Sie gerade die vorherige Lektion abgeschlossen haben.

```
npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/assets-metadata-css-starter"
```

Folgen Sie dann den Anweisungen in der Befehlsausgabe (wechseln Sie mit `cd` in das Verzeichnis und starten Sie den Entwicklungsserver).
