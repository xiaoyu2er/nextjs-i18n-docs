**Falls Sie von der vorherigen Lektion fortfahren,** können Sie diese Seite überspringen.

### [Starter-Code herunterladen (optional)](#download-starter-code-optional)

Falls Sie NICHT von der vorherigen Lektion fortfahren, können Sie den Starter-Code für diese Lektion unten herunterladen, installieren und ausführen. Damit wird ein `nextjs-blog`-Verzeichnis eingerichtet, das identisch mit dem Ergebnis der vorherigen Lektion ist.

Nochmals: Dies ist NICHT notwendig, wenn Sie gerade die vorherige Lektion abgeschlossen haben.

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/navigate-between-pages-starter"
```

Folgen Sie dann den Anweisungen in der Kommandozeilenausgabe (wechseln Sie mit `cd` in das Verzeichnis und starten Sie den Entwicklungsserver).
