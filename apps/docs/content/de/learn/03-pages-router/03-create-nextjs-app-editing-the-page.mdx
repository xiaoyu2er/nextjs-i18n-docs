Lassen Sie uns versuchen, die Startseite zu bearbeiten.

*   Stellen Sie sicher, dass der Next.js-Entwicklungsserver noch läuft.
*   Öffnen Sie `pages/index.js` mit Ihrem Texteditor.
*   Suchen Sie den Text **"Welcome to"** unter dem `<h1>`-Tag und ändern Sie ihn in **"Learn"**.
*   Speichern Sie die Datei.

Sobald Sie die Datei speichern, aktualisiert der Browser die Seite automatisch mit dem neuen Text:

![Learn Next.js](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/create-nextjs-app/learn-nextjs.png)

Der Next.js-Entwicklungsserver hat [Fast Refresh](/docs/basic-features/fast-refresh) aktiviert. Wenn Sie Änderungen an Dateien vornehmen, wendet Next.js diese Änderungen fast sofort im Browser an. Kein Neuladen erforderlich! Dies hilft Ihnen, schnell an Ihrer App zu arbeiten.
