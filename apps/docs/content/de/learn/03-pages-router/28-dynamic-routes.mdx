Wir haben die Indexseite mit Blog-Daten gefüllt, aber wir haben noch keine einzelnen Blog-Seiten erstellt (hier ist das [gewünschte Ergebnis](https://next-learn-starter.vercel.app/)). Wir möchten, dass die URL für diese Seiten von den Blog-Daten abhängt, was bedeutet, dass wir [dynamische Routen](/docs/routing/dynamic-routes) verwenden müssen.

### [Was Sie in dieser Lektion lernen werden](#what-youll-learn-in-this-lesson)

In dieser Lektion lernen Sie:

*   Wie Sie Seiten mit [dynamischen Routen](/docs/routing/dynamic-routes) statisch generieren können, indem Sie [`getStaticPaths`](/docs/basic-features/data-fetching#getstaticpaths-static-generation) verwenden.
*   Wie Sie [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) schreiben, um die Daten für jeden Blog-Beitrag abzurufen.
*   Wie Sie Markdown mit [`remark`](https://github.com/remarkjs/remark) rendern.
*   Wie Sie Datumszeichenfolgen formatieren.
*   Wie Sie auf eine Seite mit [dynamischen Routen](/docs/routing/dynamic-routes) verlinken.
*   Einige nützliche Informationen zu [dynamischen Routen](/docs/routing/dynamic-routes).
