Als Nächstes aktualisieren wir unsere Index-Seite (`pages/index.js`). Wir müssen Links zu jeder Beitragsseite hinzufügen, indem wir die [`Link`](/docs/api-reference/next/link)-Komponente verwenden.

Öffnen Sie `pages/index.js` und fügen Sie oben in der Datei folgende Imports für [`Link`](/docs/api-reference/next/link) und `Date` hinzu:

```
import Link from 'next/link';
import Date from '../components/date';
```

Ersetzen Sie dann weiter unten in der `Home`-Komponente derselben Datei das `li`-Tag durch folgenden Code:

```
<li className={utilStyles.listItem} key={id}>
  <Link href={`/posts/${id}`}>{title}</Link>
  <br />
  <small className={utilStyles.lightText}>
    <Date dateString={date} />
  </small>
</li>
```

Wenn Sie nun zu [http://localhost:3000](http://localhost:3000) navigieren, enthält die Seite Links zu jedem Artikel:

![Links](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/dynamic-routes/links.png)

> Falls etwas nicht funktioniert, überprüfen Sie, ob Ihr Code [so aussieht](https://github.com/vercel/next-learn/blob/main/basics/api-routes-starter/pages/posts/%5Bid%5D.js).

Das war's! Bevor wir diese Lektion abschließen, werfen wir auf der nächsten Seite einen Blick auf einige Tipps zu [dynamischen Routen](/docs/routing/dynamic-routes).
