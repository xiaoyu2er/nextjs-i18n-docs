### [Starter-Code herunterladen (optional)](#download-starter-code-optional)

Falls Sie NICHT von der vorherigen Lektion kommen, können Sie den Starter-Code für diese Lektion unten herunterladen, installieren und ausführen. Dadurch wird ein `nextjs-blog`-Verzeichnis erstellt, das identisch mit dem Ergebnis der vorherigen Lektion ist.

Nochmals: Dies ist NICHT notwendig, wenn Sie gerade die vorherige Lektion abgeschlossen haben.

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/api-routes-starter"
```

Folgen Sie dann den Anweisungen in der Befehlsausgabe (wechseln Sie mit `cd` in das Verzeichnis und starten Sie den Entwicklungsserver).

Sie sollten außerdem folgende Dateien aktualisieren:

*   `public/images/profile.jpg` mit Ihrem Foto (Empfohlen: 400px Breite/Höhe).
*   `const name = '[Ihr Name]'` in `components/layout.js` mit Ihrem Namen.
*   `<p>[Ihre Selbstvorstellung]</p>` in `pages/index.js` mit Ihrer Selbstvorstellung.
