Um Next.js effektiv zu lernen, ist es hilfreich, mit JavaScript, React und verwandten Webentwicklungskonzepten vertraut zu sein. Aber JavaScript und React sind umfangreiche Themen. Woher wissen Sie, wann Sie bereit sind, mit Next.js zu beginnen?

Willkommen zum Kurs "React Grundlagen"! Dieser einsteigerfreundliche, beispielorientierte Kurs führt Sie durch die erforderlichen Vorkenntnisse für Next.js. Sie werden Schritt für Schritt ein einfaches Projekt aufbauen; beginnend mit einer JavaScript-Anwendung, die dann zu React und schließlich zu Next.js migriert wird.

Jeder Abschnitt baut auf dem vorherigen auf, sodass Sie je nach Ihrem Vorwissen entscheiden können, wo Sie beginnen möchten.

### [Voraussetzungen](#prerequisite-knowledge)

Dieses Tutorial setzt Kenntnisse in HTML, CSS und JavaScript voraus, aber keine React-Kenntnisse. Wenn Sie bereits mit React vertraut sind, können Sie direkt zum Kapitel [Von React zu Next.js](/learn/react-foundations/from-react-to-nextjs) springen oder den Kurs [Dashboard App](/learn/dashboard-app) belegen.

### [Systemanforderungen](#system-requirements)

Bevor Sie mit diesem Kurs beginnen, stellen Sie sicher, dass Folgendes installiert ist:

*   Node.js 20.12.0 oder höher.
*   Betriebssysteme: macOS, Windows (einschließlich WSL) oder Linux.
*   VSCode oder einen anderen Texteditor Ihrer Wahl.
