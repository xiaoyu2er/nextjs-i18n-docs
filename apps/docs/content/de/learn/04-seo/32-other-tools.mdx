Sie können die Erfassung von Felddaten in den folgenden Tools finden:

*   [PageSpeed Insights](https://developers.google.com/speed/pagespeed/insights/): Googles Tool zur Messung der Seitenleistung.
*   [Chrome User Experience Report](https://developers.google.com/web/tools/chrome-user-experience-report): Open-Source-Datensatz für Felddaten.
*   [Search Console](https://support.google.com/webmasters/answer/9205520): Google Search, Core Web Vitals-Bericht.

Falls Sie Labordaten benötigen, bietet Google ebenfalls mehrere Mess-Tools an:

*   [Lighthouse](https://developers.google.com/web/tools/lighthouse?hl=en): Googles Open-Source-Tool zur automatisierten Verbesserung der Webseitenqualität.
*   [Chrome DevTools](https://developer.chrome.com/docs/devtools/): Sammlung von Webentwickler-Tools, direkt in den Google Chrome-Browser integriert.

Beachten Sie, dass Sie für beide Tools die [Total Blocking Time (TBT)](https://web.dev/tbt/) als Alternative zur [First Input Delay (FID)](https://web.dev/fid/) verwenden müssen, da FID nur durch Felddaten gemessen werden kann.
