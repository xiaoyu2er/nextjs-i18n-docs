[Was ist SEO?](#what-is-seo)
----------------------------

SEO steht für **Search Engine Optimization** (Suchmaschinenoptimierung). Das Ziel von SEO ist es, eine Strategie zu entwickeln, die Ihre Position in den Suchergebnissen verbessert. Je höher das Ranking, desto mehr organischen Traffic erhält Ihre Website, was letztendlich zu mehr Geschäft führt!

### [Was Sie in diesem Kurs lernen werden](#what-youll-learn-in-this-course)

In diesem Kurs behandeln wir:

*   Suchsysteme und Suchmaschinen-Robots wie Googlebot.
*   Die Auswirkungen einer SEO-Strategie auf Ihre Website.
*   Wie Sie bewährte SEO-Praktiken umsetzen.
*   Crawling, Indexierung, Rendering und Ranking mit Next.js.
*   Web-Performance-Themen einschließlich Core Web Vitals.
