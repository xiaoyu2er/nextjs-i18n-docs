LCP zeigt an, wie lange es dauert, bis das größte Element für Nutzer geladen und sichtbar ist.

Die Metrik **Largest Contentful Paint (LCP)** betrachtet die Ladeleistung Ihrer Seite. LCP misst die Zeit, die benötigt wird, um das größte Element auf der Seite innerhalb des Viewports sichtbar zu machen. Dabei kann es sich um einen großen Textblock, ein Video oder ein Bild handeln, das den primären Bereich der Seite einnimmt.

**Hinweis:** Dies ist nicht das [First Contentful Paint (FCP)](https://web.dev/fcp/), das die Zeit vom Beginn des Seitenladens bis zur Darstellung des ersten Elements auf dem Bildschirm misst.

Während das DOM gerendert wird, kann sich das größte Element auf der Seite ändern. Das Largest Contentful Paint hört erst auf zu zählen, wenn das größte Bild oder Element auf dem Bildschirm zu sehen ist.
