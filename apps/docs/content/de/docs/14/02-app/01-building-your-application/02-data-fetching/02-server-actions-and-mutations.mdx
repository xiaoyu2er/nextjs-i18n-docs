## Verhalten

- Server Actions k√∂nnen mit dem `action`-Attribut in einem [`<form>`-Element](#formulare) aufgerufen werden:

- Server Actions erben die [Laufzeitumgebung](/docs/app/building-your-application/rendering/edge-and-nodejs-runtimes) von der Seite oder dem Layout, in dem sie verwendet werden.
- Server Actions erben die [Route Segment Config](/docs/app/api-reference/file-conventions/route-segment-config) von der Seite oder dem Layout, in dem sie verwendet werden, einschlie√ülich Feldern wie `maxDuration`.
