Eine **Page (Seite)** ist eine Benutzeroberfläche, die für eine bestimmte Route einzigartig ist.

```tsx filename="app/blog/[slug]/page.tsx" switcher
export default function Page({
  params,
  searchParams,
}: {
  params: { slug: string }
  searchParams: { [key: string]: string | string[] | undefined }
}) {
  return <h1>My Page</h1>
}
```

```jsx filename="app/blog/[slug]/page.js" switcher
export default function Page({ params, searchParams }) {
  return <h1>My Page</h1>
}
```

## Props

### `params` (optional)

Ein Objekt, das die [dynamischen Routenparameter](/docs/app/building-your-application/routing/dynamic-routes) vom Root-Segment bis zu dieser Seite enthält. Zum Beispiel:

| Beispiel                           | URL         | `params`                       |
| ---------------------------------- | ----------- | ------------------------------ |
| `app/shop/[slug]/page.js`          | `/shop/1`   | `{ slug: '1' }`                |
| `app/shop/[category]/[item]/page.js` | `/shop/1/2` | `{ category: '1', item: '2' }` |
| `app/shop/[...slug]/page.js`       | `/shop/1/2` | `{ slug: ['1', '2'] }`         |

### `searchParams` (optional)

Ein Objekt, das die [Suchparameter](https://developer.mozilla.org/docs/Learn/Common_questions/What_is_a_URL#parameters) der aktuellen URL enthält. Zum Beispiel:

| URL             | `searchParams`       |
| --------------- | -------------------- |
| `/shop?a=1`     | `{ a: '1' }`         |
| `/shop?a=1&b=2` | `{ a: '1', b: '2' }` |
| `/shop?a=1&a=2` | `{ a: ['1', '2'] }`  |

> **Gut zu wissen**:
>
> - `searchParams` ist eine **[dynamische API](/docs/app/building-your-application/rendering/server-components#server-rendering-strategies#dynamic-functions)**, deren Werte nicht im Voraus bekannt sein können. Die Verwendung führt dazu, dass die Seite zur Laufzeit **[dynamisches Rendering (dynamic rendering)](/docs/app/building-your-application/rendering/server-components#dynamic-rendering)** verwendet.
> - `searchParams` gibt ein einfaches JavaScript-Objekt zurück und keine `URLSearchParams`-Instanz.
