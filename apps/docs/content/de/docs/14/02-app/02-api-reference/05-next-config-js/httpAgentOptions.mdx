In Node.js-Versionen vor 18 polyfillt Next.js automatisch `fetch()` mit [undici](/docs/architecture/supported-browsers#polyfills) und aktiviert standardmäßig [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive).

Um HTTP Keep-Alive für alle `fetch()`-Aufrufe auf der Serverseite zu deaktivieren, öffnen Sie `next.config.js` und fügen Sie die `httpAgentOptions`-Konfiguration hinzu:

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
