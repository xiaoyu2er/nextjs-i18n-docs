Next.js generiert standardmäßig [Etags](https://en.wikipedia.org/wiki/HTTP_ETag) für jede Seite. Je nach Cache-Strategie können Sie die Etag-Generierung für HTML-Seiten deaktivieren.

Öffnen Sie `next.config.js` und deaktivieren Sie die `generateEtags`-Option:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
