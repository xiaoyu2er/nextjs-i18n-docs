Eine **loading**-Datei kann sofortige Ladezustände erstellen, die auf [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming) basieren.

Standardmäßig ist diese Datei eine [Server-Komponente (Server Component)](/docs/app/building-your-application/rendering/server-components) - kann aber auch als Client-Komponente durch die `"use client"`-Direktive verwendet werden.

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // Oder eine benutzerdefinierte Lade-Skeleton-Komponente
  return <p>Loading...</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // Oder eine benutzerdefinierte Lade-Skeleton-Komponente
  return <p>Loading...</p>
}
```

Lade-UI-Komponenten akzeptieren keine Parameter.

> **Gut zu wissen**
>
> - Beim Entwurf von Lade-UI kann es hilfreich sein, die [React Developer Tools](https://react.dev/learn/react-developer-tools) zu verwenden, um Suspense-Grenzen manuell umzuschalten.
