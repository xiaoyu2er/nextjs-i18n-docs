> **Wissenswert**: Diese Option wird **ab** Next.js 13.4 **nicht mehr** benötigt. Der App-Router ist nun stabil.

Der App-Router ([`app`-Verzeichnis](/docs/app/building-your-application/routing)) ermöglicht Unterstützung für [Layouts](/docs/app/building-your-application/routing/pages-and-layouts), [Server-Komponenten (Server Components)](/docs/app/building-your-application/rendering/server-components), [Streaming](/docs/app/building-your-application/routing/loading-ui-and-streaming) und [kollokierte Datenabfrage (colocated data fetching)](/docs/app/building-your-application/data-fetching).

Die Verwendung des `app`-Verzeichnisses aktiviert automatisch den [React Strict Mode](https://react.dev/reference/react/StrictMode). Erfahren Sie, wie Sie [`app` schrittweise einführen](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app) können.
