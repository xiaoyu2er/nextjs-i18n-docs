Das `useCache`-Flag ist eine experimentelle Funktion in Next.js, die es ermöglicht, die [`use cache`-Direktive](/docs/app/api-reference/directives/use-cache) unabhängig von [`dynamicIO`](/docs/app/api-reference/config/next-config-js/dynamicIO) zu verwenden. Wenn aktiviert, können Sie `use cache` in Ihrer Anwendung nutzen, selbst wenn `dynamicIO` deaktiviert ist.

## Verwendung

Um das `useCache`-Flag zu aktivieren, setzen Sie es in der `experimental`-Sektion Ihrer `next.config.ts`-Datei auf `true`:

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

Wenn `useCache` aktiviert ist, können Sie die folgenden Cache-Funktionen und Konfigurationen verwenden:

- Die [`use cache`-Direktive](/docs/app/api-reference/directives/use-cache)
- Die [`cacheLife`-Funktion](/docs/app/api-reference/config/next-config-js/cacheLife) mit `use cache`
- Die [`cacheTag`-Funktion](/docs/app/api-reference/functions/cacheTag)
