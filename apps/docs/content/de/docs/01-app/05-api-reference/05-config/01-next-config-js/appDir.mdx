> **Wichtig zu wissen**: Diese Option wird **ab** Next.js 13.4 **nicht mehr** benötigt. Der App-Router ist jetzt stabil.

Der App-Router ([`app`-Verzeichnis](/docs/app/building-your-application/routing)) ermöglicht Unterstützung für [Layouts](/docs/app/api-reference/file-conventions/layout), [Server-Komponenten (Server Components)](/docs/app/getting-started/server-and-client-components), [Streaming](/docs/app/api-reference/file-conventions/loading) und [lokalisiertes Datenabrufen (colocated data fetching)](/docs/app/building-your-application/data-fetching).

Die Verwendung des `app`-Verzeichnisses aktiviert automatisch den [React Strict Mode](https://react.dev/reference/react/StrictMode). Erfahren Sie, wie Sie [`app` schrittweise einführen](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app) können.
