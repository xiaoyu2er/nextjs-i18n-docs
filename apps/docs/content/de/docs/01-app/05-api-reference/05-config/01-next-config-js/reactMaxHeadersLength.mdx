title: reactMaxHeadersLength
description: Die maximale Länge der Header, die von React ausgegeben und zur Antwort hinzugefügt werden.
---

Beim statischen Rendering (Static Rendering) kann React Header ausgeben, die zur Antwort hinzugefügt werden können. Diese können die Performance verbessern, indem sie dem Browser ermöglichen, Ressourcen wie Schriftarten, Skripte und Stylesheets vorab zu laden. Der Standardwert ist `6000`, aber Sie können diesen Wert überschreiben, indem Sie die Option `reactMaxHeadersLength` in `next.config.js` konfigurieren:

```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **Wissenswert**: Diese Option ist nur im App Router verfügbar.

Abhängig vom Typ des Proxys zwischen Browser und Server können die Header abgeschnitten werden. Wenn Sie beispielsweise einen Reverse Proxy verwenden, der keine langen Header unterstützt, sollten Sie einen niedrigeren Wert festlegen, um sicherzustellen, dass die Header nicht abgeschnitten werden.
