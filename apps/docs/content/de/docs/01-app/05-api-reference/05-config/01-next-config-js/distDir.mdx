Sie können einen Namen für ein benutzerdefiniertes Build-Verzeichnis angeben, das anstelle von `.next` verwendet werden soll.

Öffnen Sie `next.config.js` und fügen Sie die `distDir`-Konfiguration hinzu:

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

Wenn Sie nun `next build` ausführen, verwendet Next.js `build` anstelle des Standardordners `.next`.

> `distDir` **sollte nicht** Ihr Projektverzeichnis verlassen. Beispielsweise ist `../build` ein **ungültiges** Verzeichnis.
