Standardmäßig fügt Next.js den `x-powered-by`-Header hinzu. Um dies zu deaktivieren, öffnen Sie `next.config.js` und setzen Sie die Konfiguration `poweredByHeader` auf `false`:

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
