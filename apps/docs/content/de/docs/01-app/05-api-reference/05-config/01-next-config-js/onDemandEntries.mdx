Next.js bietet einige Optionen, mit denen Sie steuern können, wie der Server im Entwicklungsmodus erstellte Seiten verwaltet oder im Speicher behält.

Um die Standardeinstellungen zu ändern, öffnen Sie `next.config.js` und fügen Sie die `onDemandEntries`-Konfiguration hinzu:

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // Zeitraum (in ms), in dem der Server Seiten im Puffer behält
    maxInactiveAge: 25 * 1000,
    // Anzahl der Seiten, die gleichzeitig ohne Entsorgung behalten werden sollen
    pagesBufferLength: 2,
  },
}
```
