Abhängigkeiten, die in [Server Components](/docs/app/getting-started/server-and-client-components) und [Route Handlers](/docs/app/building-your-application/routing/route-handlers) verwendet werden, werden automatisch von Next.js gebündelt.

Falls eine Abhängigkeit Node.js-spezifische Funktionen nutzt, können Sie bestimmte Abhängigkeiten vom Bundling der Server Components ausschließen und das native Node.js `require` verwenden.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
