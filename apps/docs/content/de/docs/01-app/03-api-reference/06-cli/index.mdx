Next.js bietet **zwei** Command Line Interface (CLI)-Tools:

- **`create-next-app`**: Erstellt schnell eine neue Next.js-Anwendung mit der Standardvorlage oder einem [Beispiel](https://github.com/vercel/next.js/tree/canary/examples) aus einem öffentlichen GitHub-Repository.
- **`next`**: Startet den Next.js-Entwicklungsserver, baut die Anwendung und mehr.
