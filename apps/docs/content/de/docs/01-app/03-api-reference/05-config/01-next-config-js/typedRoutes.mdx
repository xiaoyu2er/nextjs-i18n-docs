Experimentelle Unterstützung für [statisch typisierte Links](/docs/app/api-reference/config/typescript#statically-typed-links). Diese Funktion erfordert die Verwendung des App Routers sowie TypeScript in Ihrem Projekt.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
