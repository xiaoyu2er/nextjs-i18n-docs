Next.js generiert standardmäßig [ETags](https://en.wikipedia.org/wiki/HTTP_ETag) für jede Seite. Je nach Ihrer Cache-Strategie möchten Sie möglicherweise die ETag-Generierung für HTML-Seiten deaktivieren.

Öffnen Sie `next.config.js` und deaktivieren Sie die Option `generateEtags`:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
