---

Vor Next.js 13 war der Pages Router die primäre Methode zum Erstellen von Routen in Next.js. Er nutzte einen intuitiven Dateisystem-Router, um jede Datei einer Route zuzuordnen. Der Pages Router wird in neueren Versionen von Next.js weiterhin unterstützt, aber wir empfehlen die Migration zum neuen [App Router](/docs/app), um die neuesten React-Funktionen nutzen zu können.

Nutzen Sie diesen Abschnitt der Dokumentation für bestehende Anwendungen, die den Pages Router verwenden.
