Next.js bietet [**Gzip**](https://tools.ietf.org/html/rfc6713#section-3)-Komprimierung, um gerenderte Inhalte und statische Dateien zu komprimieren. Im Allgemeinen sollten Sie die Komprimierung auf einem HTTP-Proxy wie [nginx](https://www.nginx.com/) aktivieren, um die Last vom `Node.js`-Prozess zu entlasten.

Um die **Komprimierung** zu deaktivieren, öffnen Sie `next.config.js` und deaktivieren Sie die `compress`-Konfiguration:

```js filename="next.config.js"
module.exports = {
  compress: false,
}
```
