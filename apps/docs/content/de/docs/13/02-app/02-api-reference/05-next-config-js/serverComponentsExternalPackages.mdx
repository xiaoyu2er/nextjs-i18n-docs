Abhängigkeiten, die in [Server Components](/docs/app/building-your-application/rendering/server-components) und [Route Handlers](/docs/app/building-your-application/routing/route-handlers) verwendet werden, werden automatisch von Next.js gebündelt.

Falls eine Abhängigkeit Node.js-spezifische Funktionen nutzt, können Sie bestimmte Abhängigkeiten vom Server Components-Bundling ausschließen und stattdessen die native Node.js-`require`-Funktion verwenden.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    serverComponentsExternalPackages: ['@acme/ui'],
  },
}

module.exports = nextConfig
```
