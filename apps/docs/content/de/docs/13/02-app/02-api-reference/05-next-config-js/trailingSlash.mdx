Standardmäßig leitet Next.js URLs mit nachgestelltem Schrägstrich zu ihren Entsprechungen ohne Schrägstrich um. Zum Beispiel wird `/about/` zu `/about` weitergeleitet. Sie können dieses Verhalten so konfigurieren, dass es umgekehrt funktioniert, wobei URLs ohne nachgestellten Schrägstrich zu ihren Entsprechungen mit Schrägstrich weitergeleitet werden.

Öffnen Sie `next.config.js` und fügen Sie die `trailingSlash`-Konfiguration hinzu:

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

Mit dieser Option werden URLs wie `/about` zu `/about/` weitergeleitet.
