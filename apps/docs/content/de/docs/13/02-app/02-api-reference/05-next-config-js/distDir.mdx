Sie können einen Namen für ein benutzerdefiniertes Build-Verzeichnis festlegen, das anstelle von `.next` verwendet wird.

Öffnen Sie `next.config.js` und fügen Sie die `distDir`-Konfiguration hinzu:

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

Wenn Sie nun `next build` ausführen, verwendet Next.js `build` anstelle des standardmäßigen `.next`-Ordners.

> `distDir` **sollte** Ihr Projektverzeichnis **nicht verlassen**. Beispielsweise ist `../build` ein **ungültiges** Verzeichnis.
