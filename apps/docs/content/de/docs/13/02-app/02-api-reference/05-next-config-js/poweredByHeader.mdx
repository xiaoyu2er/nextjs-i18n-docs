Standardmäßig fügt Next.js den Header `x-powered-by` hinzu. Um dies zu deaktivieren, öffnen Sie `next.config.js` und setzen die Konfiguration `poweredByHeader` auf false:

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
