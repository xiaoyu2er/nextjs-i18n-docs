> Sie sehen die Dokumentation für den Pages Router. Die Dokumentation für den neueren [App Router](/docs/app) mit den aktuellsten Funktionen finden Sie hier.

Der **Pages Router** verwendet einen intuitiven Dateisystem-Router, um jede Datei einer Route zuzuordnen.

Vor Next.js 13 war der Pages Router die primäre Methode zum Erstellen von Routen in Next.js. Er wird in neueren Versionen von Next.js weiterhin unterstützt, aber wir empfehlen die Migration zum neuen [App Router](/docs/app), um die neuesten React-Funktionen nutzen zu können.
