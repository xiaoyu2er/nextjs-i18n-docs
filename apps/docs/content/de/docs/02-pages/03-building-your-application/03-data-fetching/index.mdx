---

Die Datenabfrage in Next.js ermöglicht es Ihnen, Ihre Inhalte auf unterschiedliche Weise darzustellen, abhängig vom Anwendungsfall. Dazu gehören Pre-Rendering mit **Server-seitigem Rendering (SSR)** oder **Statischer Generierung (Static Generation)**, sowie das Aktualisieren oder Erstellen von Inhalten zur Laufzeit mit **Inkrementeller Statischer Regenerierung (ISR)**.
