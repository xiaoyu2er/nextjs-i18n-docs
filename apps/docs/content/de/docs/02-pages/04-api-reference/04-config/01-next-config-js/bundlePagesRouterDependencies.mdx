description: Aktiviert das automatische Bündeln von Abhängigkeiten für den Pages Router
---

Aktiviert das automatische Bündeln von serverseitigen Abhängigkeiten für Anwendungen mit Pages Router. Entspricht dem automatischen Bündeln von Abhängigkeiten im App Router.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  bundlePagesRouterDependencies: true,
}

module.exports = nextConfig
```

Bestimmte Pakete können explizit vom Bündeln ausgeschlossen werden, indem die Option [`serverExternalPackages`](/docs/pages/api-reference/config/next-config-js/serverExternalPackages) verwendet wird.

## Versionsverlauf

| Version   | Änderungen                                                                                               |
| --------- | ------------------------------------------------------------------------------------------------------- |
| `v15.0.0` | Wechsel von experimentell zu stabil. Umbenennung von `bundlePagesExternals` zu `bundlePagesRouterDependencies` |
