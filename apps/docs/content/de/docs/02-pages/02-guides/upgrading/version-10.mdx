Es gab keine Breaking Changes zwischen den Versionen 9 und 10.

Um auf Version 10 zu aktualisieren, fÃ¼hren Sie folgenden Befehl aus:

```bash filename="Terminal"
npm i next@10
```

```bash filename="Terminal"
yarn add next@10
```

```bash filename="Terminal"
pnpm up next@10
```

```bash filename="Terminal"
bun add next@10
```

> **Wissenswert:** Wenn Sie TypeScript verwenden, stellen Sie sicher, dass Sie auch `@types/react` und `@types/react-dom` auf die entsprechenden Versionen aktualisieren.
