Willkommen in der Next.js-Dokumentation!

## Was ist Next.js?

Next.js ist ein React-Framework für die Entwicklung von Full-Stack-Webanwendungen. Sie verwenden React-Komponenten, um Benutzeroberflächen zu erstellen, und Next.js für zusätzliche Funktionen und Optimierungen.

Es konfiguriert auch Low-Level-Tools wie Bundler und Compiler automatisch. So können Sie sich stattdessen auf den Aufbau Ihres Produkts und schnelle Veröffentlichungen konzentrieren.

Egal, ob Sie Einzelentwickler oder Teil eines größeren Teams sind – Next.js hilft Ihnen dabei, interaktive, dynamische und schnelle React-Anwendungen zu erstellen.

## App-Router und Pages-Router

Next.js bietet zwei verschiedene Router:

- **App-Router**: Der neuere Router, der neue React-Funktionen wie Server-Komponenten unterstützt.
- **Pages-Router**: Der ursprüngliche Router, der weiterhin unterstützt und verbessert wird.

Oberhalb der Seitenleiste finden Sie ein Dropdown-Menü, mit dem Sie zwischen der [App-Router](/docs/app)- und der [Pages-Router](/docs/pages)-Dokumentation wechseln können.

## Voraussetzungen

Unsere Dokumentation setzt Grundkenntnisse in der Webentwicklung voraus. Bevor Sie beginnen, sollten Sie sich mit Folgendem auskennen:

- HTML
- CSS
- JavaScript
- React

Wenn Sie neu in React sind oder Ihr Wissen auffrischen möchten, empfehlen wir unseren [React Foundations-Kurs](/learn/react-foundations) und den [Next.js Foundations-Kurs](/learn/dashboard-app), bei dem Sie während des Lernens eine Anwendung erstellen.
