الصفحة الثانية التي أضفناها حالياً لا تحتوي على تنسيقات. دعونا نضيف بعض CSS لتنسيق الصفحة.

يحتوي Next.js على دعم مدمج لـ [CSS](/docs/basic-features/built-in-css-support) و [Sass](/docs/basic-features/built-in-css-support#sass-support). في هذه الدورة، سنستخدم CSS.

سنتحدث أيضاً في هذا الدرس عن كيفية تعامل Next.js مع الأصول الثابتة مثل الصور وبيانات الصفحة الوصفية مثل وسم `<title>`.

### [ما الذي ستتعلمه في هذا الدرس](#what-youll-learn-in-this-lesson)

في هذا الدرس، ستتعلم:

* كيفية إضافة [ملفات ثابتة](/docs/basic-features/static-file-serving) (صور، إلخ) إلى Next.js.
* كيفية تخصيص المحتوى داخل `<head>` لكل صفحة.
* كيفية إنشاء مكون React قابل لإعادة الاستخدام باستخدام [وحدات CSS](/docs/basic-features/built-in-css-support#adding-component-level-css).
* كيفية [إضافة CSS عام](/docs/basic-features/built-in-css-support#adding-a-global-stylesheet) في ملف [`pages/_app.js`](/docs/advanced-features/custom-app).
* بعض النصائح المفاهيمية حول التنسيق في Next.js.

### [المتطلبات الأساسية](#prerequisites)

* معرفة أساسية بـ CSS. ستناقش هذه الدورة كيفية إضافة CSS في تطبيق Next.js، لكنها لن تغطي أساسيات CSS.

> إذا كنت تبحث عن وثائق مفصلة حول التنسيق في Next.js، يمكنك الاطلاع على [وثائق CSS](/docs/basic-features/built-in-css-support).
