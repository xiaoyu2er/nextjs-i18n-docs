**إذا كنت تتابع من الدرس السابق،** يمكنك تخطي هذه الصفحة.

### [تحميل كود البداية (اختياري)](#download-starter-code-optional)

إذا كنت لا تتابع من الدرس السابق، يمكنك تحميل وتثبيت وتشغيل كود البداية لهذا الدرس أدناه. هذا ينشئ مجلد `nextjs-blog` بحيث يكون مطابقًا لنتيجة الدرس السابق.

مرة أخرى، هذا ليس ضروريًا إذا كنت قد أنهيت للتو الدرس السابق.

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/api-routes-starter"
```

ثم اتبع التعليمات من ناتج الأمر (انتقل إلى المجلد `cd` وابدأ خادم التطوير).

يجب أيضًا تحديث الملفات التالية:

* `public/images/profile.jpg` بصورتك الشخصية (مستحسن: عرض/ارتفاع 400 بكسل).
* `const name = '[اسمك]'` في `components/layout.js` باسمك.
* `<p>[مقدمة عن نفسك]</p>` في `pages/index.js` بمقدمة عن نفسك.
