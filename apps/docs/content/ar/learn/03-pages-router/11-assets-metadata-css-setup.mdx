**إذا كنت تتابع من الدرس السابق،** يمكنك تخطي هذه الصفحة.

### [تحميل كود البداية (اختياري)](#download-starter-code-optional)

إذا كنت لا تتابع من الدرس السابق، يمكنك تنزيل وتثبيت وتشغيل كود البداية لهذا الدرس أدناه. هذا ينشئ دليل `nextjs-blog` بحيث يكون مطابقًا لنتيجة الدرس السابق.

مرة أخرى، هذا ليس ضروريًا إذا كنت قد أنهيت للتو الدرس السابق.

```
npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/assets-metadata-css-starter"
```

ثم اتبع التعليمات من ناتج الأمر (انتقل إلى الدليل `cd` وابدأ خادم التطوير).
