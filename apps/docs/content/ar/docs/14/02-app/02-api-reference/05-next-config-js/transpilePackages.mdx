يمكن لـ Next.js تحويل وحزم التبعيات تلقائيًا من الحزم المحلية (مثل المستودعات الأحادية) أو من التبعيات الخارجية (`node_modules`). هذا يحل محل حزمة `next-transpile-modules`.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['@acme/ui', 'lodash-es'],
}

module.exports = nextConfig
```
