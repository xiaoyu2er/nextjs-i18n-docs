يمكنك تكوين مستوى تسجيل البيانات وما إذا كان سيتم تسجيل عنوان URL الكامل في وحدة التحكم عند تشغيل Next.js في وضع التطوير.

حاليًا، `logging` ينطبق فقط على جلب البيانات باستخدام واجهة `fetch` API. ولا ينطبق بعد على السجلات الأخرى داخل Next.js.

```js filename="next.config.js"
module.exports = {
  logging: {
    fetches: {
      fullUrl: true,
    },
  },
}
```
