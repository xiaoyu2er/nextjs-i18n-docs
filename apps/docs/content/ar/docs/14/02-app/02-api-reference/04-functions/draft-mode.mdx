تتيح لك وظيفة `draftMode` اكتشاف [وضع المسودة (Draft Mode)](/docs/app/building-your-application/configuring/draft-mode) داخل [مكون الخادم (Server Component)](/docs/app/building-your-application/rendering/server-components).

```jsx filename="app/page.js"
import { draftMode } from 'next/headers'

export default function Page() {
  const { isEnabled } = draftMode()
  return (
    <main>
      <h1>My Blog Post</h1>
      <p>Draft Mode is currently {isEnabled ? 'Enabled' : 'Disabled'}</p>
    </main>
  )
}
```
