---

{/* المحتوى في هذا المستند مشترك بين موجه التطبيق وموجه الصفحات. يمكنك استخدام مكون `<PagesOnly>المحتوى</PagesOnly>` لإضافة محتوى خاص بموجه الصفحات. أي محتوى مشترك لا يجب أن يكون مغلفًا في مكون. */}

افتراضيًا، سيضيف Next.js رأس `x-powered-by`. لإلغاء تفعيله، افتح ملف `next.config.js` وعطل إعداد `poweredByHeader`:

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
