يمكن لملف **loading** إنشاء حالات تحميل فورية مبنية على [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming).

بشكل افتراضي، هذا الملف هو [مكون من جانب الخادم (Server Component)](/docs/app/building-your-application/rendering/server-components) - ولكن يمكن أيضًا استخدامه كمكون عميل (Client Component) من خلال التوجيه `"use client"`.

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // أو مكون هيكل تحميل مخصص
  return <p>Loading...</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // أو مكون هيكل تحميل مخصص
  return <p>Loading...</p>
}
```

لا تقبل مكونات واجهة مستخدم التحميل أي معاملات.

> **معلومة مفيدة**
>
> - أثناء تصميم واجهة مستخدم التحميل، قد تجد أنه من المفيد استخدام [أداة مطوري React (React Developer Tools)](https://react.dev/learn/react-developer-tools) لتبديل حدود Suspense يدويًا.
