يسمح لك جلب البيانات في Next.js بعرض محتواك بطرق مختلفة، اعتمادًا على حالة استخدام تطبيقك. تشمل هذه الطرق التصيير المسبق باستخدام **التصيير من جانب الخادم (SSR)** أو **التوليد الثابت (Static Generation)**، وتحديث أو إنشاء المحتوى أثناء التشغيل باستخدام **التجديد الثابت التدريجي (ISR)**.
