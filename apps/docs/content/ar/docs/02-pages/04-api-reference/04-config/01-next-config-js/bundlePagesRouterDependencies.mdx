تمكين تجميع التبعيات التلقائي من جانب الخادم لتطبيقات موجه صفحات التطبيق (Pages Router). يتوافق مع تجميع التبعيات التلقائي في موجه التطبيق (App Router).

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  bundlePagesRouterDependencies: true,
}

module.exports = nextConfig
```

يمكنك استثناء حزم معينة من التجميع بشكل صريح باستخدام خيار [`serverExternalPackages`](/docs/pages/api-reference/config/next-config-js/serverExternalPackages).

## سجل الإصدارات

| الإصدار   | التغييرات                                                                                                   |
| --------- | --------------------------------------------------------------------------------------------------------- |
| `v15.0.0` | تم نقله من التجريبي إلى المستقر. تم تغيير الاسم من `bundlePagesExternals` إلى `bundlePagesRouterDependencies` |
