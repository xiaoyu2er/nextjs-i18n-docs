[Turbopack](https://turbo.build/pack) (بيتا) هو حزمة تدريجية مُحسنة لجافاسكريبت وTypeScript، مكتوبة بلغة Rust ومدمجة في Next.js.

## الاستخدام

يمكن استخدام Turbopack في Next.js سواء في دليل `pages` أو `app` لتطوير محلي أسرع. لتفعيل Turbopack، استخدم العلم `--turbo` عند تشغيل خادم التطوير الخاص بـ Next.js.

```json filename="package.json" highlight={3}
{
  "scripts": {
    "dev": "next dev --turbo",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
```

## الميزات المدعومة

لمعرفة المزيد عن الميزات المدعومة حاليًا في Turbopack، راجع [الوثائق](https://turbo.build/pack/docs/features).

## الميزات غير المدعومة

حاليًا، يدعم Turbopack فقط `next dev` ولا يدعم `next build`. نحن نعمل حاليًا على دعم عمليات البناء مع اقترابنا من تحقيق الاستقرار.
