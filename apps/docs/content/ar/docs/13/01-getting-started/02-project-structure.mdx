توفر هذه الصفحة نظرة عامة على هيكل الملفات والمجلدات في مشروع Next.js. وهي تغطي الملفات والمجلدات الرئيسية، وملفات التكوين، واتفاقيات التوجيه داخل دليل `app` و `pages`.

## المجلدات الرئيسية

|                                                                          |                                    |
| ------------------------------------------------------------------------ | ---------------------------------- |
| [`app`](/docs/app/building-your-application/routing)                     | موجه التطبيق (App Router)          |
| [`pages`](/docs/pages/building-your-application/routing)                 | موجه الصفحات (Pages Router)        |
| [`public`](/docs/app/building-your-application/optimizing/static-assets) | الموارد الثابتة التي يتم تقديمها   |
| [`src`](/docs/app/building-your-application/configuring/src-directory)   | مجلد مصدر التطبيق الاختياري        |
