دعم تجريبي لـ[الروابط ذات النوع الثابت (statically typed links)](/docs/app/building-your-application/configuring/typescript#statically-typed-links). تتطلب هذه الميزة استخدام موجه التطبيق (App Router) بالإضافة إلى TypeScript في مشروعك.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
