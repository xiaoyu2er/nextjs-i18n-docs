> **من الجيد معرفته**: منذ Next.js 13.4، الوضع الصارم (Strict Mode) مفعل افتراضيًا مع موجه `app`، لذا الإعدادات أعلاه ضرورية فقط لموجه `pages`. يمكنك تعطيل الوضع الصارم عن طريق تعيين `reactStrictMode: false`.

> **مقترح**: ننصح بشدة بتفعيل الوضع الصارم في تطبيق Next.js الخاص بك لتحضير تطبيقك لمستقبل React.

[الوضع الصارم (Strict Mode)](https://react.dev/reference/react/StrictMode) في React هو ميزة خاصة بوضع التطوير فقط لتسليط الضوء على المشاكل المحتملة في التطبيق. يساعد في تحديد دورات الحياة غير الآمنة، واستخدام واجهات برمجة التطبيقات القديمة، وعدد من الميزات الأخرى.

بيئة تشغيل Next.js متوافقة مع الوضع الصارم. لتفعيل الوضع الصارم، قم بتكوين الخيار التالي في ملف `next.config.js` الخاص بك:

```js filename="next.config.js"
module.exports = {
  reactStrictMode: true,
}
```

إذا كنت أو فريقك غير مستعدين لاستخدام الوضع الصارم في تطبيقك بالكامل، لا بأس! يمكنك الهجرة تدريجيًا على أساس صفحة بصفحة باستخدام `<React.StrictMode>`.
