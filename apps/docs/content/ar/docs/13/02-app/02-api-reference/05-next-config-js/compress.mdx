يوفر Next.js ضغط [**gzip**](https://tools.ietf.org/html/rfc6713#section-3) لضغط المحتوى المعروض والملفات الثابتة. بشكل عام، سترغب في تمكين الضغط على وكيل HTTP مثل [nginx](https://www.nginx.com/)، لتخفيف الحمل عن عملية `Node.js`.

لتعطيل **الضغط**، افتح ملف `next.config.js` وعطل إعداد `compress`:

```js filename="next.config.js"
module.exports = {
  compress: false,
}
```
