يقوم Next.js بتوليد [علامات etag](https://en.wikipedia.org/wiki/HTTP_ETag) افتراضيًا لكل صفحة. قد ترغب في تعطيل توليد etag لصفحات HTML اعتمادًا على إستراتيجية التخزين المؤقت الخاصة بك.

افتح ملف `next.config.js` وعطّل خيار `generateEtags`:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
