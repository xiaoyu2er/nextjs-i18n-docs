خرائط المصدر (source maps) مفعلة افتراضيًا أثناء التطوير. أثناء عمليات بناء الإنتاج، يتم تعطيلها لمنع تسريب المصدر على العميل، إلا إذا قمت بالاشتراك صراحة باستخدام علامة التكوين.

يوفر Next.js علامة تكوين يمكنك استخدامها لتمكين إنشاء خرائط المصدر للمتصفح أثناء بناء الإنتاج:

```js filename="next.config.js"
module.exports = {
  productionBrowserSourceMaps: true,
}
```

عند تفعيل خيار `productionBrowserSourceMaps`، سيتم إخراج خرائط المصدر في نفس دليل ملفات JavaScript. سيقوم Next.js تلقائيًا بخدمة هذه الملفات عند الطلب.

- إضافة خرائط المصدر قد تزيد من وقت `next build`
- تزيد من استخدام الذاكرة أثناء `next build`
