أضف أو أنشئ ملف `sitemap.xml` يتوافق مع [تنسيق Sitemaps XML](https://www.sitemaps.org/protocol.html) في **الجذر** من دليل `app` لمساعدة زواحف محركات البحث على فحص موقعك بكفاءة أكبر.

## ملف `sitemap.xml` الثابت

```xml filename="app/sitemap.xml"
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://acme.com</loc>
    <lastmod>2023-04-06T15:02:24.021Z</lastmod>
    <changefreq>yearly</changefreq>
    <priority>1</priority>
  </url>
  <url>
    <loc>https://acme.com/about</loc>
    <lastmod>2023-04-06T15:02:24.021Z</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  <url>
    <loc>https://acme.com/blog</loc>
    <lastmod>2023-04-06T15:02:24.021Z</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.5</priority>
  </url>
</urlset>
```

## إنشاء خريطة الموقع

أضف ملف `sitemap.js` أو `sitemap.ts` الذي يُرجع [`Sitemap`](#sitemap-return-type).

```ts filename="app/sitemap.ts" switcher
import { MetadataRoute } from 'next'

export default function sitemap(): MetadataRoute.Sitemap {
  return [
    {
      url: 'https://acme.com',
      lastModified: new Date(),
      changeFrequency: 'yearly',
      priority: 1,
    },
    {
      url: 'https://acme.com/about',
      lastModified: new Date(),
      changeFrequency: 'monthly',
      priority: 0.8,
    },
    {
      url: 'https://acme.com/blog',
      lastModified: new Date(),
      changeFrequency: 'weekly',
      priority: 0.5,
    },
  ]
}
```

### نوع الإرجاع لخريطة الموقع

```tsx
type Sitemap = Array<{
  url: string
  lastModified?: string | Date
  changeFrequency?:
    | 'always'
    | 'hourly'
    | 'daily'
    | 'weekly'
    | 'monthly'
    | 'yearly'
    | 'never'
  priority?: number
}>
```

> **معلومة مفيدة**
>
> - في المستقبل، سندعم خرائط مواقع متعددة وفهارس خرائط المواقع.
