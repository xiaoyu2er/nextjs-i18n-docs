هذا الملحق حاليًا في مرحلة تجريبية. يرجى استخدام [نقاش GitHub هذا](https://github.com/vercel/next.js/discussions/77800) لتقديم ملاحظاتك حول أي مشكلات تواجهها.

تعلم المزيد من [وثائق Rspack](https://rspack.rs/guide/tech/next) وجرب [هذا المثال](https://github.com/vercel/next.js/tree/canary/examples/with-rspack).
