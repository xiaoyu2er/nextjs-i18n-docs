يأتي Next.js مع **أداتين** لواجهة سطر الأوامر (CLI):

- **`create-next-app`**: إنشاء تطبيق Next.js جديد بسرعة باستخدام القالب الافتراضي أو [مثال](https://github.com/vercel/next.js/tree/canary/examples) من مستودع GitHub العام.
- **`next`**: تشغيل خادم التطوير الخاص بـ Next.js، بناء التطبيق، والمزيد.
