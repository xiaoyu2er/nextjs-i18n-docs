يمكن لملف **تحميل** إنشاء حالات تحميل فورية مبنية على [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming).

بشكل افتراضي، هذا الملف هو [مكون خادم (Server Component)](/docs/app/getting-started/server-and-client-components) - ولكن يمكن أيضًا استخدامه كمكون عميل (Client Component) من خلال التوجيه `"use client"`.

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // Or a custom loading skeleton component
  return <p>Loading...</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // Or a custom loading skeleton component
  return <p>Loading...</p>
}
```

لا تقبل مكونات واجهة المستخدم للتحميل أي معاملات.

> **من الجيد معرفة**:
>
> - أثناء تصميم واجهة مستخدم التحميل، قد تجد أنه من المفيد استخدام [أدوات مطوري React](https://react.dev/learn/react-developer-tools) لتبديل حدود Suspense يدويًا.
