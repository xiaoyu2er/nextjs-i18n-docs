{/* يتم مشاركة محتوى هذا المستند بين موجه التطبيق (app router) وموجه الصفحات (pages router). يمكنك استخدام المكون `<PagesOnly>المحتوى</PagesOnly>` لإضافة محتوى خاص بموجه الصفحات. أي محتوى مشترك لا يجب أن يكون مغلفًا داخل مكون. */}

يوفر Next.js بعض الخيارات التي تمنحك تحكمًا في كيفية تعامل الخادم مع التخلص من الصفحات المبنية أو الاحتفاظ بها في الذاكرة أثناء التطوير.

لتغيير الإعدادات الافتراضية، افتح ملف `next.config.js` وأضف تكوين `onDemandEntries`:

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // المدة (بالمللي ثانية) التي سيحتفظ فيها الخادم بالصفحات في المخزن المؤقت
    maxInactiveAge: 25 * 1000,
    // عدد الصفحات التي يجب الاحتفاظ بها في نفس الوقت دون التخلص منها
    pagesBufferLength: 2,
  },
}
```
