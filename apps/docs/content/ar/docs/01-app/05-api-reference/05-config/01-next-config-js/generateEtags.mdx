سيقوم Next.js بإنشاء [علامات etags](https://en.wikipedia.org/wiki/HTTP_ETag) لكل صفحة افتراضيًا. قد ترغب في تعطيل إنشاء علامات etags لصفحات HTML اعتمادًا على استراتيجية التخزين المؤقت الخاصة بك.

افتح ملف `next.config.js` وعطل خيار `generateEtags`:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
