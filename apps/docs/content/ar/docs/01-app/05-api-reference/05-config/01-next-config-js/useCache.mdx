علامة `useCache` هي ميزة تجريبية في Next.js تتيح استخدام [توجيه use cache](/docs/app/api-reference/directives/use-cache) بشكل مستقل عن [`dynamicIO`](/docs/app/api-reference/config/next-config-js/dynamicIO). عند التمكين، يمكنك استخدام `use cache` في تطبيقك حتى إذا كان `dynamicIO` معطلاً.

## الاستخدام

لتمكين علامة `useCache`، قم بتعيينها إلى `true` في قسم `experimental` في ملف `next.config.ts` الخاص بك:

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

عند تمكين `useCache`، يمكنك استخدام دوال وضوابط التخزين المؤقت التالية:

- [توجيه use cache](/docs/app/api-reference/directives/use-cache)
- [دالة cacheLife](/docs/app/api-reference/config/next-config-js/cacheLife) مع `use cache`
- [دالة cacheTag](/docs/app/api-reference/functions/cacheTag)
