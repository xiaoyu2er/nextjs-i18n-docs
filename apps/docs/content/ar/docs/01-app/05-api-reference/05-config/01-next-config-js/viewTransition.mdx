`viewTransition` هو علامة تجريبية تمكن واجهة برمجة التطبيقات الجديدة [View Transitions API](https://developer.mozilla.org/en-US/docs/Web/API/View_Transition_API) في React. تتيح لك هذه الواجهة الاستفادة من واجهة متصفح View Transitions الأصلية لإنشاء انتقالات سلسة بين حالات واجهة المستخدم.

لتمكين هذه الميزة، تحتاج إلى تعيين خاصية `viewTransition` إلى `true` في ملف `next.config.js`.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    viewTransition: true,
  },
}

module.exports = nextConfig
```

> ملاحظة هامة: هذه الميزة ليست من تطوير أو صيانة فريق Next.js - إنها واجهة برمجة تطبيقات تجريبية من فريق React. لا تزال في **مراحل مبكرة** و **غير موصى بها للاستخدام في الإنتاج**. لا يزال التنفيذ قيد التطوير، وقد يتغير سلوكه في إصدارات React المستقبلية.
> يتطلب تمكين هذه الميزة فهمًا للطبيعة التجريبية للواجهة. لفهم سلوكها بالكامل، راجع [طلب سحب React](https://github.com/facebook/react/pull/31975) والمناقشات المرتبطة به.

## الاستخدام

بعد التمكين، يمكنك استيراد مكون `ViewTransition` من React في تطبيقك:

```jsx
import { unstable_ViewTransition as ViewTransition } from 'react'
```

ومع ذلك، فإن الوثائق والأمثلة محدودة حاليًا، وسوف تحتاج إلى الرجوع مباشرة إلى كود المصدر والمناقشات الخاصة بـ React لفهم كيفية عمل هذا.
