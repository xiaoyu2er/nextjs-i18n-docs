استخدم خيار `crossOrigin` لإضافة [سمة `crossOrigin`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) في جميع علامات `<script>` التي يتم إنشاؤها بواسطة مكون <AppOnly>[`next/script`](/docs/app/guides/scripts)</AppOnly> <PagesOnly>[`next/script`](/docs/pages/guides/scripts) و [`next/head`](/docs/pages/api-reference/components/head)</PagesOnly>، وذلك لتحديد كيفية معالجة طلبات cross-origin.

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## الخيارات

- `'anonymous'`: يضيف سمة [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous).
- `'use-credentials'`: يضيف سمة [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials).
