> **معلومة مفيدة**: هذا الخيار **لم يعد** مطلوبًا بدءًا من Next.js 13.4. حيث أصبح موجه التطبيق (App Router) مستقرًا الآن.

يمكّن موجه التطبيق ([دليل `app`](/docs/app/building-your-application/routing)) دعم [التخطيطات (layouts)](/docs/app/api-reference/file-conventions/layout)، [مكونات الخادم (Server Components)](/docs/app/getting-started/server-and-client-components)، [البث (streaming)](/docs/app/building-your-application/routing/loading-ui-and-streaming)، و[جلب البيانات المترافق (colocated data fetching)](/docs/app/building-your-application/data-fetching).

سيؤدي استخدام دليل `app` إلى تمكين [وضع React الصارم (React Strict Mode)](https://react.dev/reference/react/StrictMode) تلقائيًا. تعلم كيفية [تبني دليل `app` تدريجيًا](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app).
