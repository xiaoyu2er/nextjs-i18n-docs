خرائط المصدر (source maps) مفعلة بشكل افتراضي أثناء التطوير. أثناء عمليات بناء الإنتاج، يتم تعطيلها لمنع تسريب المصدر إلى العميل، إلا إذا قمت بالاشتراك صراحةً باستخدام علامة التكوين.

يوفر Next.js علامة تكوين يمكنك استخدامها لتمكين إنشاء خرائط المصدر في المتصفح أثناء عملية بناء الإنتاج:

```js filename="next.config.js"
module.exports = {
  productionBrowserSourceMaps: true,
}
```

عند تفعيل خيار `productionBrowserSourceMaps`، سيتم إخراج خرائط المصدر في نفس الدليل الذي يحتوي على ملفات JavaScript. سيقوم Next.js تلقائيًا بتقديم هذه الملفات عند الطلب.

- إضافة خرائط المصدر قد تزيد من وقت `next build`
- تزيد من استخدام الذاكرة أثناء `next build`
