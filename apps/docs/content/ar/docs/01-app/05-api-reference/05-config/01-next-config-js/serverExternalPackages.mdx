سيتم تجميع التبعيات المستخدمة داخل [مكونات الخادم (Server Components)](/docs/app/getting-started/server-and-client-components) و [معالجات المسار (Route Handlers)](/docs/app/building-your-application/routing/route-handlers) تلقائيًا بواسطة Next.js.

إذا كانت التبعية تستخدم ميزات محددة في Node.js، يمكنك اختيار استثناء تبعيات محددة من تجميع مكونات الخادم واستخدام الوظيفة الأصلية `require` في Node.js.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
