يمكنك تحديد اسم لاستخدامه كدليل بناء مخصص بدلاً من `.next`.

افتح ملف `next.config.js` وأضف إعداد `distDir`:

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

الآن عند تشغيل أمر `next build`، سيستخدم Next.js دليل `build` بدلاً من المجلد الافتراضي `.next`.

> **يجب ألا** يخرج `distDir` من دليل مشروعك. على سبيل المثال، `../build` يعتبر دليلًا **غير صالح**.
