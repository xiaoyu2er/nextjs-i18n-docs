في إصدارات Node.js السابقة للإصدار 18، يقوم Next.js تلقائيًا بملء `fetch()` باستخدام [undici](/docs/architecture/supported-browsers#polyfills) وتمكين [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive) افتراضيًا.

لتعطيل HTTP Keep-Alive لجميع استدعاءات `fetch()` على جانب الخادم، افتح ملف `next.config.js` وأضف إعداد `httpAgentOptions`:

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
