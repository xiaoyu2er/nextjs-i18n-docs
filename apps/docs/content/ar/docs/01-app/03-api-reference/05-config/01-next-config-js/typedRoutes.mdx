دعم تجريبي لـ[الروابط ذات الكتابة الثابتة](/docs/app/api-reference/config/typescript#statically-typed-links). تتطلب هذه الميزة استخدام موجه التطبيق (App Router) بالإضافة إلى TypeScript في مشروعك.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
