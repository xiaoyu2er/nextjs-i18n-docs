> 現在表示しているのは Pages Router のドキュメントです。最新機能については [App Router](/docs/app) のドキュメントを参照してください。

**Pages Router** は直感的なファイルシステムベースのルーターを使用して、各ファイルをルートにマッピングします。

Next.js 13 より前では、Pages Router が Next.js でルートを作成する主要な方法でした。新しいバージョンの Next.js でも引き続きサポートされていますが、React の最新機能を活用するために新しい [App Router](/docs/app) への移行を推奨しています。
