Pages Router アプリケーション向けにサーバーサイド依存関係の自動バンドリングを有効化します。App Router の自動依存関係バンドリングと同様の動作です。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  bundlePagesRouterDependencies: true,
}

module.exports = nextConfig
```

特定のパッケージをバンドル対象から除外するには、[`serverExternalPackages`](/docs/pages/api-reference/config/next-config-js/serverExternalPackages) オプションを使用してください。
