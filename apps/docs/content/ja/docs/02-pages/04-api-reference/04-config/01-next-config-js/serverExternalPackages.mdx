[`bundlePagesRouterDependencies`](/docs/pages/api-reference/config/next-config-js/bundlePagesRouterDependencies) オプションによる自動バンドリングから特定の依存関係を除外します。

これらのページでは、ネイティブのNode.js `require` を使用して依存関係が解決されます。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
