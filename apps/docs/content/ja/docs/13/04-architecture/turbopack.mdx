[Turbopack](https://turbo.build/pack) (ベータ版) は JavaScript と TypeScript に最適化されたインクリメンタルバンドラーで、Rust で記述され Next.js に組み込まれています。

## 使用方法

Turbopack は Next.js の `pages` と `app` ディレクトリで使用でき、ローカル開発を高速化します。Turbopack を有効にするには、Next.js 開発サーバーを実行する際に `--turbo` フラグを使用します。

```json filename="package.json" highlight={3}
{
  "scripts": {
    "dev": "next dev --turbo",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
```

## サポートされている機能

Turbopack で現在サポートされている機能の詳細については、[ドキュメント](https://turbo.build/pack/docs/features)を参照してください。

## サポートされていない機能

Turbopack は現在 `next dev` のみをサポートしており、`next build` はサポートしていません。安定版に向けてビルドのサポートに取り組んでいます。
