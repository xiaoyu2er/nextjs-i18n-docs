[静的に型付けされたリンク](/docs/app/building-your-application/configuring/typescript#statically-typed-links)の実験的サポートです。この機能を使用するには、プロジェクトでApp RouterとTypeScriptを使用している必要があります。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
