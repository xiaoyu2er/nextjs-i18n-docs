<AppOnly>

コードを編集し、Next.jsがアプリケーションをコンパイルしている間、ページの右下隅にコンパイルインジケーターが表示されます。

> **豆知識**: このインジケーターは開発モードでのみ表示され、本番モードでアプリをビルド・実行する際には表示されません。

場合によっては、このインジケーターがページ上で不適切な位置に表示されることがあります（例：チャットランチャーと重なる場合など）。位置を変更するには、`next.config.js`を開き、`devIndicators`オブジェクト内の`buildActivityPosition`を`bottom-right`（デフォルト）、`bottom-left`、`top-right`、`top-left`のいずれかに設定します：

```js filename="next.config.js"
module.exports = {
  devIndicators: {
    buildActivityPosition: 'bottom-right',
  },
}
```

このインジケーターが不要な場合、`next.config.js`を開き、`devIndicators`オブジェクト内の`buildActivity`設定を無効にすることで削除できます：

```js filename="next.config.js"
module.exports = {
  devIndicators: {
    buildActivity: false,
  },
}
```

</AppOnly>

<PagesOnly>

> **豆知識**: このインジケーターはNext.jsバージョン10.0.1で削除されました。最新バージョンのNext.jsにアップグレードすることを推奨します。

ページが[自動静的最適化](/docs/pages/building-your-application/rendering/automatic-static-optimization)の条件を満たす場合、そのことを示すインジケーターが表示されます。

自動静的最適化は非常に有益であり、開発時にすぐに条件を満たしているかどうかが分かるため、これは便利です。

Electronアプリケーションを開発している場合など、このインジケーターが不要な場合は、`next.config.js`を開き、`devIndicators`内の`autoPrerender`設定を無効にすることで削除できます：

```js filename="next.config.js"
module.exports = {
  devIndicators: {
    autoPrerender: false,
  },
}
```

</PagesOnly>
