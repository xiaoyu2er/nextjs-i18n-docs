このドキュメントセクションでは、**メタデータファイル規約**について説明します。ルートセグメントに特別なメタデータファイルを追加することで、ファイルベースのメタデータを定義できます。

各ファイル規約は、静的ファイル（例: `opengraph-image.jpg`）またはコードを使用してファイルを生成する動的バリアント（例: `opengraph-image.js`）で定義できます。

ファイルが定義されると、Next.js は自動的にファイルを提供し（本番環境ではキャッシュ用にハッシュ付きで）、アセットの URL、ファイルタイプ、画像サイズなどの適切なメタデータで関連する head 要素を更新します。
