Next.js はレンダリングされたコンテンツと静的ファイルの圧縮に [**gzip**](https://tools.ietf.org/html/rfc6713#section-3) 圧縮を提供します。一般的には、`Node.js` プロセスからの負荷を軽減するために [nginx](https://www.nginx.com/) などの HTTP プロキシで圧縮を有効にすることをお勧めします。

**圧縮**を無効にするには、`next.config.js` を開き、`compress` 設定を無効にします:

```js filename="next.config.js"
module.exports = {
  compress: false,
}
```
