Next.js 13以前では、ページルーター (Pages Router) がNext.jsでルートを作成する主要な方法でした。各ファイルをルートにマッピングする直感的なファイルシステムベースのルーターを使用していました。ページルーター (Pages Router) は新しいバージョンのNext.jsでも引き続きサポートされていますが、Reactの最新機能を活用するためには新しい[App Router](/docs/app)への移行をお勧めします。

このドキュメントセクションは、ページルーター (Pages Router) を使用している既存のアプリケーション向けです。
