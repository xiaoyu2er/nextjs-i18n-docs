## バージョン13から14へのアップグレード

Next.jsバージョン14にアップデートするには、お使いのパッケージマネージャーで以下のコマンドを実行してください：

```bash filename="ターミナル"
npm i next@next-14 react@18 react-dom@18 && npm i eslint-config-next@next-14 -D
```

```bash filename="ターミナル"
yarn add next@next-14 react@18 react-dom@18 && yarn add eslint-config-next@next-14 -D
```

```bash filename="ターミナル"
pnpm i next@next-14 react@18 react-dom@18 && pnpm i eslint-config-next@next-14 -D
```

```bash filename="ターミナル"
bun add next@next-14 react@18 react-dom@18 && bun add eslint-config-next@next-14 -D
```

> **知っておくと便利：** TypeScriptを使用している場合、`@types/react`と`@types/react-dom`も最新バージョンにアップグレードしてください。

### v14の主な変更点

- Node.jsの最低バージョン要件が16.14から18.17に引き上げられました（16.xはサポート終了のため）
- `next export`コマンドが削除され、`output: 'export'`設定に置き換えられました。詳細は[ドキュメント](https://nextjs.org/docs/app/guides/static-exports)をご覧ください。
- `ImageResponse`のインポート元が`next/server`から`next/og`に変更されました。安全にインポートを自動変更する[コードモッド](/docs/app/guides/upgrading/codemods#next-og-import)が利用可能です。
- `@next/font`パッケージが完全に削除され、組み込みの`next/font`に置き換えられました。安全にインポートを自動変更する[コードモッド](/docs/app/guides/upgrading/codemods#built-in-next-font)が利用可能です。
- `next-swc`のWASMターゲットが削除されました。
