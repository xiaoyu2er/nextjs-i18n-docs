[Tailwind CSS](https://tailwindcss.com/)は、Next.jsと完全に互換性のあるユーティリティファーストのCSSフレームワークです。このガイドでは、Next.jsアプリケーションにTailwind CSSをインストールする方法を説明します。

## Tailwindのインストール

必要なTailwind CSSパッケージをインストールします:

```bash filename="Terminal"
npm install -D tailwindcss @tailwindcss/postcss postcss
```

> **補足**: `create-next-app` CLIを使用してプロジェクトを作成する場合、Next.jsはTailwindのインストールとプロジェクトの自動設定を提案します。

## Tailwindの設定

プロジェクトのルートに`postcss.config.mjs`ファイルを作成し、PostCSS設定に`@tailwindcss/postcss`プラグインを追加します:

```js filename="postcss.config.mjs" highlight={4}
/** @type {import('tailwindcss').Config} */
export default {
  plugins: {
    '@tailwindcss/postcss': {},
  },
}
```

[Tailwind v4](https://tailwindcss.com/blog/tailwindcss-v4)以降、デフォルトでは設定は不要です。設定が必要な場合は、グローバルCSSファイルの設定について[公式ドキュメント](https://tailwindcss.com/blog/tailwindcss-v4#css-first-configuration)を参照してください。

既存のTailwind v3プロジェクトがある場合は、[アップグレードCLI](https://tailwindcss.com/docs/upgrade-guide)と[ガイド](https://tailwindcss.com/docs/upgrade-guide)が利用可能です。

## スタイルのインポート

Tailwindが生成するスタイルを注入するための[Tailwind CSSディレクティブ](https://tailwindcss.com/docs/functions-and-directives#directives)を、アプリケーションの[グローバルスタイルシート](/docs/app/getting-started/css#global-css)に追加します。例:

```css filename="app/globals.css"
@import 'tailwindcss';
```

[ルートレイアウト](/docs/app/api-reference/file-conventions/layout#root-layout)（`app/layout.tsx`）内で`globals.css`スタイルシートをインポートし、アプリケーションのすべてのルートにスタイルを適用します。

```tsx filename="app/layout.tsx" switcher
import type { Metadata } from 'next'

// これらのスタイルはアプリケーションのすべてのルートに適用されます
import './globals.css'

export const metadata: Metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
}

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

```jsx filename="app/layout.js" switcher
// これらのスタイルはアプリケーションのすべてのルートに適用されます
import './globals.css'

export const metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
}

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

## スタイルのインポート

Tailwindが生成するスタイルを注入するための[Tailwind CSSディレクティブ](https://tailwindcss.com/docs/functions-and-directives#directives)を、アプリケーションの[グローバルスタイルシート](/docs/app/getting-started/css#global-css)に追加します。例:

```css filename="styles/globals.css"
@import 'tailwindcss';
```

[カスタムAppファイル](/docs/pages/building-your-application/routing/custom-app)（`pages/_app.js`）内で`globals.css`スタイルシートをインポートし、アプリケーションのすべてのルートにスタイルを適用します。

```tsx filename="pages/_app.tsx" switcher
// これらのスタイルはアプリケーションのすべてのルートに適用されます
import '@/styles/globals.css'
import type { AppProps } from 'next/app'

export default function App({ Component, pageProps }: AppProps) {
  return <Component {...pageProps} />
}
```

```jsx filename="pages/_app.js" switcher
// これらのスタイルはアプリケーションのすべてのルートに適用されます
import '@/styles/globals.css'

export default function App({ Component, pageProps }) {
  return <Component {...pageProps} />
}
```
