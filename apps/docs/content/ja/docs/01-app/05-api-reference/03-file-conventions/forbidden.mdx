**forbidden** ファイルは、認証中に [`forbidden`](/docs/app/api-reference/functions/forbidden) 関数が呼び出された際に表示する UI をレンダリングするために使用されます。UI をカスタマイズできることに加えて、Next.js は `403` ステータスコードを返します。

```tsx filename="app/forbidden.tsx" switcher
import Link from 'next/link'

export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>
      <p>You are not authorized to access this resource.</p>
      <Link href="/">Return Home</Link>
    </div>
  )
}
```

```jsx filename="app/forbidden.jsx" switcher

export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>

```

## リファレンス

### Props

`forbidden.js` コンポーネントは props を受け取りません。
