**loading** ファイルを使用すると、[Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming) を基盤とした即時のローディング状態を作成できます。

デフォルトでは、このファイルは[サーバーコンポーネント (Server Component)](/docs/app/getting-started/server-and-client-components) ですが、`"use client"` ディレクティブを使用してクライアントコンポーネントとしても利用可能です。

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // またはカスタムのローディングスケルトンコンポーネント
  return <p>Loading...</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // またはカスタムのローディングスケルトンコンポーネント
  return <p>Loading...</p>
}
```

ローディング UI コンポーネントはパラメータを受け取りません。

> **知っておくと便利**:
>
> - ローディング UI を設計する際には、[React Developer Tools](https://react.dev/learn/react-developer-tools) を使用して Suspense バウンダリを手動で切り替えると便利な場合があります。
