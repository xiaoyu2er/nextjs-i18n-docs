`instrumentation-client.js|ts`ファイルを使用すると、アプリケーションのフロントエンドコードが実行される前に、監視や分析コードを追加できます。これはパフォーマンス追跡、エラーモニタリング、その他のクライアントサイド可観測性ツールのセットアップに役立ちます。

利用するには、このファイルをアプリケーションの**ルート**ディレクトリか、`src`フォルダ内に配置してください。

## 使用方法

[サーバーサイド計装](/docs/app/guides/instrumentation)とは異なり、特定の関数をエクスポートする必要はありません。監視コードを直接ファイルに記述できます:

```ts filename="instrumentation-client.ts" switcher
// パフォーマンス監視をセットアップ
performance.mark('app-init')

// アナリティクスを初期化
console.log('Analytics initialized')

// エラートラッキングをセットアップ
window.addEventListener('error', (event) => {
  // エラートラッキングサービスに送信
  reportError(event.error)
})
```
