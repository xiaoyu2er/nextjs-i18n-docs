Next.jsは、ローカルパッケージ（モノレポなど）や外部依存関係（`node_modules`）から自動的にトランスパイルしバンドルすることができます。これは`next-transpile-modules`パッケージの代わりとなる機能です。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['package-name'],
}

module.exports = nextConfig
```
