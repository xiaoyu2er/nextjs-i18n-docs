Next.js はデフォルトで全てのページに対して [etag](https://en.wikipedia.org/wiki/HTTP_ETag) を生成します。キャッシュ戦略によっては、HTML ページに対する etag 生成を無効化したい場合があるかもしれません。

`next.config.js` を開き、`generateEtags` オプションを無効にします:

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
