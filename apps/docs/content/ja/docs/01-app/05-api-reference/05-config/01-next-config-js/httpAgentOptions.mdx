Node.js バージョン 18 より前では、Next.js は自動的に `fetch()` を [undici](/docs/architecture/supported-browsers#polyfills) でポリフィルし、デフォルトで [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive) を有効にします。

サーバーサイドのすべての `fetch()` 呼び出しに対して HTTP Keep-Alive を無効にするには、`next.config.js` を開き、`httpAgentOptions` 設定を追加します:

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
