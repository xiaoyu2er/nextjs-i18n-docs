Next.jsは、プロジェクトにTypeScriptエラーが存在する場合、**本番ビルド** (`next build`) を失敗させます。

アプリケーションにエラーがある場合でも、Next.jsに危険を承知で本番コードを生成させたい場合は、組み込みの型チェックステップを無効にすることができます。

無効にする場合は、ビルドまたはデプロイプロセスの一部として型チェックを実行していることを確認してください。そうしないと非常に危険な場合があります。

`next.config.js` を開き、`typescript` 設定で `ignoreBuildErrors` オプションを有効にしてください：

```js filename="next.config.js"
module.exports = {
  typescript: {
    // !! 警告 !!
    // プロジェクトに型エラーがあっても、危険を承知で
    // 本番ビルドを成功させる
    // !! 警告 !!
    ignoreBuildErrors: true,
  },
}
```
