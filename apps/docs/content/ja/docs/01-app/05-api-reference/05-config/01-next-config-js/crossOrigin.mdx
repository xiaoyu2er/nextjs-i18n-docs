`crossOrigin` オプションを使用すると、<AppOnly>[`next/script`](/docs/app/guides/scripts) コンポーネント</AppOnly> <PagesOnly>[`next/script`](/docs/pages/guides/scripts) および [`next/head`](/docs/pages/api-reference/components/head) コンポーネント</PagesOnly>によって生成されるすべての `<script>` タグに [`crossOrigin` 属性](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin)を追加し、クロスオリジンリクエストの処理方法を定義できます。

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## オプション

- `'anonymous'`: [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous) 属性を追加します。
- `'use-credentials'`: [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials) 属性を追加します。
