`useCache` フラグは Next.js の実験的機能で、[`use cache` ディレクティブ](/docs/app/api-reference/directives/use-cache) を [`dynamicIO`](/docs/app/api-reference/config/next-config-js/dynamicIO) とは独立して使用できるようにします。有効にすると、`dynamicIO` がオフになっている場合でもアプリケーション内で `use cache` を使用できます。

## 使用方法

`useCache` フラグを有効にするには、`next.config.ts` ファイルの `experimental` セクションで `true` に設定します：

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

`useCache` が有効になると、以下のキャッシュ関連の機能と設定が使用可能になります：

- [`use cache` ディレクティブ](/docs/app/api-reference/directives/use-cache)
- `use cache` と組み合わせた [`cacheLife` 関数](/docs/app/api-reference/config/next-config-js/cacheLife)
- [`cacheTag` 関数](/docs/app/api-reference/functions/cacheTag)
