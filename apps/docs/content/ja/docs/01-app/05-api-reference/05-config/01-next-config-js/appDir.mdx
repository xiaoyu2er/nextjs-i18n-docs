> **参考**: このオプションはNext.js 13.4以降**不要**になりました。App Routerは現在安定版です。

App Router（[`app`ディレクトリ](/docs/app/building-your-application/routing)）は、[レイアウト](/docs/app/api-reference/file-conventions/layout)、[サーバーコンポーネント (Server Components)](/docs/app/getting-started/server-and-client-components)、[ストリーミング](/docs/app/api-reference/file-conventions/loading)、[データ取得の同配置 (colocated data fetching)](/docs/app/building-your-application/data-fetching)のサポートを有効にします。

`app`ディレクトリを使用すると、自動的に[React Strict Mode](https://react.dev/reference/react/StrictMode)が有効になります。[`app`ディレクトリへの段階的な移行方法](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app)について学びましょう。
