> **ご存知ですか？**: Next.js 13.4以降、このオプションは**不要**になりました。App Routerは現在安定版です。

App Router（[`app`ディレクトリ](/docs/app/building-your-application/routing)）は、以下の機能のサポートを有効にします：
- [レイアウト](/docs/app/api-reference/file-conventions/layout)
- [サーバーコンポーネント (Server Components)](/docs/app/getting-started/server-and-client-components)
- [ストリーミング](/docs/app/building-your-application/routing/loading-ui-and-streaming)
- [データ取得の同配置 (colocated data fetching)](/docs/app/building-your-application/data-fetching)

`app`ディレクトリを使用すると、自動的に[React Strict Mode](https://react.dev/reference/react/StrictMode)が有効になります。[`app`への段階的な移行方法](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app)について学びましょう。
