Next.js には **2つ** のコマンドラインインターフェース (CLI) ツールが付属しています：

- **`create-next-app`**: デフォルトテンプレートまたは公開 GitHub リポジトリの[例](https://github.com/vercel/next.js/tree/canary/examples)を使用して、新しい Next.js アプリケーションを素早く作成します。
- **`next`**: Next.js 開発サーバーの実行、アプリケーションのビルドなどを行います。
