Next.js ドキュメントへようこそ！

この「**はじめよう**」セクションでは、最初の Next.js アプリを作成し、すべてのプロジェクトで使用するコア機能を学ぶことができます。

## 前提知識

当ドキュメントでは、Web 開発にある程度慣れていることを前提としています。始める前に、以下の技術に慣れていると役立ちます:

- HTML
- CSS
- JavaScript
- React

React に初めて触れる方や復習が必要な方は、まず [React Foundations コース](/learn/react-foundations) から始めることをお勧めします。また、学習しながらアプリケーションを構築する [Next.js Foundations コース](/learn/dashboard-app) もご利用いただけます。
