週間500万ダウンロードを超える Next.js は、世界中に大規模で活発な開発者コミュニティを持っています。以下はコミュニティに参加する方法です：

## コントリビューション

Next.js の開発に貢献する方法はいくつかあります：

- [ドキュメント](/docs/community/contribution-guide): Next.js の使用方法を理解するための改善提案や新しいセクションの執筆
- [サンプルコード](https://github.com/vercel/next.js/tree/canary/contributing/examples): 新しいサンプルの作成や既存サンプルの改善を通じて、他のツールやサービスとの統合を支援
- [コードベース](https://github.com/vercel/next.js/tree/canary/contributing/core): 基盤となるアーキテクチャについて学び、バグ修正や新機能提案に貢献

## ディスカッション

Next.js に関する質問や他の開発者を助けたい場合、以下の場で議論に参加できます：

- [GitHub Discussions](https://github.com/vercel/next.js/discussions)
- [Discord](https://discord.com/invite/bUG2bvbtHy)
- [Reddit](https://www.reddit.com/r/nextjs)

## ソーシャルメディア

最新情報は [Twitter](https://twitter.com/nextjs) でフォローし、Next.js 関連動画は [Vercel YouTube チャンネル](https://www.youtube.com/@VercelHQ) を購読してください。

## 行動規範

私たちは包括的で歓迎的なコミュニティの構築を信じています。すべてのメンバーに [行動規範](https://github.com/vercel/next.js/blob/canary/CODE_OF_CONDUCT.md) の遵守をお願いしています。この文書は参加者に期待される行動を規定しています。安全で敬意ある環境を維持するため、ぜひお読みください。
