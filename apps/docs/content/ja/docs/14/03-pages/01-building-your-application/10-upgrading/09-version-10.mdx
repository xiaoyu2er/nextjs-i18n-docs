---

バージョン9からバージョン10の間には破壊的変更はありません。

バージョン10にアップグレードするには、以下のコマンドを実行してください:

```bash filename="Terminal"
npm i next@10
```

```bash filename="Terminal"
yarn add next@10
```

```bash filename="Terminal"
pnpm up next@10
```

```bash filename="Terminal"
bun add next@10
```

> **補足:** TypeScriptを使用している場合、`@types/react` と `@types/react-dom` も対応するバージョンにアップグレードしてください。
