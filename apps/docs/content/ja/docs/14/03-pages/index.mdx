Next.js 13より前では、ページルーター (Pages Router) がNext.jsでルートを作成する主要な方法でした。各ファイルをルートにマッピングする直感的なファイルシステムベースのルーターを使用していました。ページルーター (Pages Router) は新しいバージョンのNext.jsでも引き続きサポートされていますが、Reactの最新機能を活用するために新しい[App Router](/docs/app)への移行を推奨しています。

ページルーター (Pages Router) を使用している既存のアプリケーションについては、このドキュメントセクションを参照してください。
