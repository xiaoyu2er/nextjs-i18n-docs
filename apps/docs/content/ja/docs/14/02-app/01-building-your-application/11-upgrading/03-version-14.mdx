## バージョン13から14へのアップグレード

Next.jsバージョン14に更新するには、使用しているパッケージマネージャーで次のコマンドを実行してください：

```bash filename="Terminal"
npm i next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash filename="Terminal"
yarn add next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash filename="Terminal"
pnpm up next react react-dom eslint-config-next --latest
```

```bash filename="Terminal"
bun add next@latest react@latest react-dom@latest eslint-config-next@latest
```

> **豆知識:** TypeScriptを使用している場合、`@types/react`と`@types/react-dom`も最新バージョンにアップグレードしてください。

### v14の主な変更点

- Node.jsの最低バージョン要件が16.14から18.17に引き上げられました（16.xはサポート終了のため）
- `next export`コマンドが削除され、代わりに`output: 'export'`設定を使用するようになりました。詳細は[ドキュメント](https://nextjs.org/docs/app/building-your-application/deploying/static-exports)をご覧ください。
- `ImageResponse`のインポート元が`next/server`から`next/og`に変更されました。インポートを安全かつ自動的にリネームする[コードモッド](/docs/app/building-your-application/upgrading/codemods#next-og-import)が利用可能です。
- `@next/font`パッケージが完全に削除され、組み込みの`next/font`を使用するようになりました。インポートを安全かつ自動的にリネームする[コードモッド](/docs/app/building-your-application/upgrading/codemods#built-in-next-font)が利用可能です。
- `next-swc`のWASMターゲットが削除されました。
