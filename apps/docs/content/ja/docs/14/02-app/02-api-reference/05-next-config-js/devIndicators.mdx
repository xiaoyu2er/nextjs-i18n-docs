<AppOnly>

コードを編集し、Next.jsがアプリケーションをコンパイルしている間、ページの右下隅にコンパイルインジケーターが表示されます。

> **豆知識**: このインジケーターは開発モードでのみ表示され、本番モードでアプリをビルド・実行する際には表示されません。

場合によっては、このインジケーターがページ上で不適切な位置に表示されることがあります（例えばチャットランチャーと競合する場合など）。位置を変更するには、`next.config.js`を開き、`devIndicators`オブジェクト内の`buildActivityPosition`を`bottom-right`（デフォルト）、`bottom-left`、`top-right`、または`top-left`に設定します：

```js filename="next.config.js"
module.exports = {
  devIndicators: {
    buildActivityPosition: 'bottom-right',
  },
}
```

このインジケーターが不要な場合もあります。非表示にするには、`next.config.js`を開き、`devIndicators`オブジェクト内の`buildActivity`設定を無効にします：

```js filename="next.config.js"
module.exports = {
  devIndicators: {
    buildActivity: false,
  },
}
```

</AppOnly>

<PagesOnly>

> **豆知識**: このインジケーターはNext.jsバージョン10.0.1で削除されました。最新バージョンのNext.jsにアップグレードすることをお勧めします。

ページが[自動静的最適化](/docs/pages/building-your-application/rendering/automatic-static-optimization)の条件を満たしている場合、そのことを示すインジケーターが表示されます。

自動静的最適化は非常に有益な機能であり、開発中に条件を満たしているかどうかを即座に知ることができるため、これは便利です。

Electronアプリケーションを開発している場合など、このインジケーターが不要な場合があります。非表示にするには、`next.config.js`を開き、`devIndicators`内の`autoPrerender`設定を無効にします：

```js filename="next.config.js"
module.exports = {
  devIndicators: {
    autoPrerender: false,
  },
}
```

</PagesOnly>
