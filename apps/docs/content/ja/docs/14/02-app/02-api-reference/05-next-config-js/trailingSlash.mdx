デフォルトでは、Next.jsは末尾スラッシュ付きのURLをスラッシュなしのURLにリダイレクトします。例えば`/about/`は`/about`にリダイレクトされます。この動作を逆に設定することも可能で、末尾スラッシュなしのURLをスラッシュ付きのURLにリダイレクトするようにできます。

`next.config.js`を開き、`trailingSlash`設定を追加します：

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

このオプションを設定すると、`/about`のようなURLは`/about/`にリダイレクトされます。

[`output: "export"`](/docs/app/building-your-application/deploying/static-exports)設定と併用する場合、`/about`ページは`/about/index.html`を出力します（デフォルトの`/about.html`ではなく）。
