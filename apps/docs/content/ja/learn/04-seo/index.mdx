[SEOとは？](#what-is-seo)
----------------------------

SEOは**Search Engine Optimization（検索エンジン最適化）**の略称です。SEOの目的は、検索エンジン結果ページ（SERP）でのランキングを向上させる戦略を構築することです。ランキングが高ければ高いほど、サイトへの自然流入（オーガニックトラフィック）が増え、最終的にはビジネスの成長につながります！

### [このコースで学べること](#what-youll-learn-in-this-course)

このコースでは以下のトピックについて解説します：

*   検索システムとGooglebotなどの検索エンジンロボット
*   SEO戦略がサイトに与える影響
*   SEOのベストプラクティスの実装方法
*   Next.jsにおけるクローリング、インデックス登録、レンダリング、ランキング
*   コアウェブバイタルを含むウェブパフォーマンスのトピック
