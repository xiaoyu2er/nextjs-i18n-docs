このレッスンでは、さまざまなメトリクスについて説明し、[Core Web Vitals](https://web.dev/vitals/) が SEO に与える影響と、ユーザーエクスペリエンスにおける重要性について解説します。

Core Web Vitals を測定する際には、3つの異なる評価値があります：

**「良好」**、**「要改善」**、**「不良」**。これらの値は測定対象の指標によって異なります：

![Core Web Vitals](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/seo/vitals-light.png)

Core Web Vitals には2つの異なるアプローチ方法があります：

*   **各メトリクスで可能な限り高いスコアを目指す**。完璧を目指すことは素晴らしいですが、多くの依存関係を持つ大規模なウェブサイトでは難しい場合があります。
*   **業界の競合他社と比較する**。Google 検索で完全に最適化されたすべてのウェブサイトと競争しているわけではなく、ターゲットキーワードでランキングしている他社と比較します。
