[Next.js Speed Insights](https://nextjs.org/analytics) を使用すると、Core Web Vitals を活用してページのパフォーマンスを分析・測定できます。

![Next.js Speed Insights](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/seo/analytics.png)

[Vercel デプロイ](https://vercel.com/docs/concepts/speed-insights)では、設定不要で[リアルユーザー体験スコア (Real Experience Score)](https://vercel.com/docs/concepts/speed-insights#metrics)の収集を開始できます。
