スターターページを編集してみましょう。

*   Next.js 開発サーバーがまだ実行されていることを確認してください。
*   テキストエディタで `pages/index.js` を開きます。
*   `<h1>` タグの下にある **"Welcome to"** というテキストを見つけ、**"Learn"** に変更します。
*   ファイルを保存します。

ファイルを保存するとすぐに、ブラウザが自動的に新しいテキストでページを更新します:

![Learn Next.js](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/create-nextjs-app/learn-nextjs.png)

Next.js 開発サーバーには [Fast Refresh](/docs/basic-features/fast-refresh) が有効になっています。ファイルに変更を加えると、Next.js はほぼ瞬時にブラウザに変更を適用します。ページの再読み込みは不要です！これにより、アプリの迅速な反復開発が可能になります。
