**前のレッスンから続けている場合**、このページはスキップできます。

### [スターターコードのダウンロード（オプション）](#download-starter-code-optional)

前のレッスンから続けていない場合、以下のスターターコードをダウンロード、インストール、実行できます。これにより、`nextjs-blog`ディレクトリが前のレッスンの結果と同一の状態でセットアップされます。

繰り返しますが、前のレッスンを終えたばかりの場合はこの作業は不要です。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/data-fetching-starter"
```

その後、コマンド出力の指示に従ってください（ディレクトリに`cd`して開発サーバーを起動します）。

以下のファイルも更新する必要があります:

*   `public/images/profile.jpg` を自分の写真に置き換え（推奨サイズ: 400px幅/高さ）
*   `components/layout.js`内の `const name = '[Your Name]'` を自分の名前に変更
*   `pages/index.js`内の `<p>[Your Self Introduction]</p>` を自己紹介文に置き換え
