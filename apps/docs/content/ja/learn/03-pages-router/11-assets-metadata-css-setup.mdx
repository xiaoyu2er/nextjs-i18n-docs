**前のレッスンから続けている場合**、このページはスキップできます。

### [スターターコードのダウンロード（オプション）](#download-starter-code-optional)

前のレッスンから続けていない場合は、以下のスターターコードをダウンロード、インストール、実行できます。これにより、`nextjs-blog`ディレクトリが前のレッスンの結果と同一の状態でセットアップされます。

繰り返しますが、前のレッスンを終えたばかりの場合はこの作業は不要です。

```
npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/assets-metadata-css-starter"
```

その後、コマンド出力の指示に従ってください（ディレクトリに`cd`して開発サーバーを起動します）。
