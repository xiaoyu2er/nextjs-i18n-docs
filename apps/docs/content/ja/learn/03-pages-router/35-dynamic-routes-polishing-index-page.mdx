次に、インデックスページ（`pages/index.js`）を更新しましょう。[`Link`](/docs/api-reference/next/link) コンポーネントを使用して、各投稿ページへのリンクを追加する必要があります。

`pages/index.js` を開き、ファイルの先頭に [`Link`](/docs/api-reference/next/link) と `Date` のインポートを追加します：

```
import Link from 'next/link';
import Date from '../components/date';
```

そして、同じファイル内の `Home` コンポーネントの下部付近で、`li` タグを以下のように置き換えます：

```
<li className={utilStyles.listItem} key={id}>
  <Link href={`/posts/${id}`}>{title}</Link>
  <br />
  <small className={utilStyles.lightText}>
    <Date dateString={date} />
  </small>
</li>
```

[http://localhost:3000](http://localhost:3000) にアクセスすると、各記事へのリンクが表示されるようになります：

![リンク](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/dynamic-routes/links.png)

> 動作しない場合は、[こちらのコード](https://github.com/vercel/next-learn/blob/main/basics/api-routes-starter/pages/posts/%5Bid%5D.js)と比較してください。

以上です！このレッスンを終える前に、次のページで[ダイナミックルーティング](/docs/routing/dynamic-routes)に関するヒントをいくつか紹介します。
