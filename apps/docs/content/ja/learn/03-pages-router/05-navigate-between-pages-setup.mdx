**前のレッスンから続けている場合**、このページはスキップできます。

### [スターターコードのダウンロード（オプション）](#download-starter-code-optional)

前のレッスンから続けていない場合、以下のスターターコードをダウンロード、インストール、実行できます。これにより、前のレッスンの結果と同一の`nextjs-blog`ディレクトリがセットアップされます。

繰り返しますが、前のレッスンを終えたばかりの場合はこの作業は不要です。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/navigate-between-pages-starter"
```

その後、コマンド出力の指示に従ってください（ディレクトリに`cd`して開発サーバーを起動します）。
