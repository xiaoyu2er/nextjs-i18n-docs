**前回のレッスンから続けている場合**、このページはスキップできます。

### [スターターコードのダウンロード（オプション）](#download-starter-code-optional)

前回のレッスンから続けていない場合は、以下のスターターコードをダウンロード、インストール、実行できます。これにより、`nextjs-blog`ディレクトリが前回のレッスンの結果と同一の状態でセットアップされます。

繰り返しますが、前回のレッスンを終えたばかりの場合はこの作業は不要です。

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/api-routes-starter"
```

その後、コマンド出力の指示に従ってください（ディレクトリに`cd`して開発サーバーを起動します）。

以下のファイルも更新する必要があります：

*   `public/images/profile.jpg` を自分の写真に（推奨：幅/高さ400px）
*   `components/layout.js`内の`const name = '[Your Name]'`を自分の名前に
*   `pages/index.js`内の`<p>[Your Self Introduction]</p>`を自己紹介文に
