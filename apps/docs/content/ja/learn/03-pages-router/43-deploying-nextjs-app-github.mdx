デプロイする前に、まだ行っていない場合はNext.jsアプリを[GitHub](https://github.com/vercel/next.js)にプッシュしましょう。これによりデプロイが簡単になります。

*   GitHubアカウントで`nextjs-blog`という名前の新しいリポジトリを作成してください。
*   リポジトリは公開（パブリック）でも非公開（プライベート）でも構いません。READMEやその他のファイルで初期化する必要は**ありません**。
*   リポジトリの設定にヘルプが必要な場合は、[GitHubのこのガイド](https://help.github.com/en/github/getting-started-with-github/create-a-repo)を参照してください。

次に:

*   ローカルのNext.jsアプリでgitリポジトリを初期化していない場合は、今すぐ行ってください。
*   Next.jsアプリをGitHubリポジトリにプッシュします。

GitHubにプッシュするには、以下のコマンドを実行します（`<username>`はGitHubのユーザー名に置き換えてください）:

```
git remote add origin https://github.com/<username>/nextjs-blog.git
git push -u origin main
```

GitHubリポジトリの準備が整ったら、次のページに進んでください。
