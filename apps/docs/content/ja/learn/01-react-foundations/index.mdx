Next.jsを効果的に学ぶには、JavaScript、React、および関連するウェブ開発の概念に慣れていることが役立ちます。しかし、JavaScriptとReactは広大なトピックです。いつNext.jsを使い始める準備ができているのか、どうやって判断すればよいでしょうか？

React基礎コースへようこそ！この初心者向けの実例中心のコースでは、Next.jsの前提知識を段階的にガイドします。簡単なプロジェクトをステップバイステップで構築していきます。まずはJavaScriptアプリケーションから始め、それをReactとNext.jsに移行していきます。

各セクションは前のセクションの内容を基に構築されているので、既に知っている内容に応じて開始地点を選ぶことができます。

### [前提知識](#prerequisite-knowledge)

このチュートリアルでは、HTML、CSS、JavaScriptの知識を前提とし、Reactの知識は必要ありません。Reactに既に慣れている場合は、[ReactからNext.jsへ](/learn/react-foundations/from-react-to-nextjs)の章に進むか、[ダッシュボードアプリ](/learn/dashboard-app)コースを受講してください。

### [システム要件](#system-requirements)

このコースを開始する前に、以下がインストールされていることを確認してください:

*   Node.js 20.12.0以降がインストールされていること
*   対応OS: macOS、Windows（WSL含む）、またはLinux
*   VSCodeまたはお好みのテキストエディタ
