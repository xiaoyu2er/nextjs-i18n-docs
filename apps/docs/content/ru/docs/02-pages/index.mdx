**Роутер страниц (Pages Router)** использует интуитивно понятный файловый роутер, который сопоставляет каждый файл с маршрутом.

До версии Next.js 13 роутер страниц был основным способом создания маршрутов в Next.js. Он по-прежнему поддерживается в новых версиях Next.js, но мы рекомендуем перейти на новый [роутер приложения (App Router)](/docs/app), чтобы использовать последние функции React.
