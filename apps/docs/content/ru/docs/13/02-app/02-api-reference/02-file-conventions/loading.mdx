Файл **loading** позволяет создавать мгновенные состояния загрузки на основе [Suspense](/docs/app/building-your-application/routing/loading-ui-and-streaming).

По умолчанию этот файл является [Серверным Компонентом (Server Component)](/docs/app/building-your-application/rendering/server-components), но также может использоваться как Клиентский Компонент с помощью директивы `"use client"`.

```tsx filename="app/feed/loading.tsx" switcher
export default function Loading() {
  // Или кастомный компонент-скелетон загрузки
  return <p>'Loading...'</p>
}
```

```jsx filename="app/feed/loading.js" switcher
export default function Loading() {
  // Или кастомный компонент-скелетон загрузки
  return <p>'Loading...'</p>
}
```

Компоненты UI загрузки не принимают никаких параметров.

> **Полезно знать**
>
> - При разработке UI загрузки может быть полезно использовать [React Developer Tools](https://react.dev/learn/react-developer-tools) для ручного переключения границ Suspense.
