Next.js прерывает **продакшен-сборку** (`next build`), если в вашем проекте присутствуют ошибки TypeScript.

Если вы хотите, чтобы Next.js опасно выполнял сборку продакшен-кода даже при наличии ошибок в приложении, вы можете отключить встроенную проверку типов.

При отключении убедитесь, что проверка типов выполняется как часть процесса сборки или деплоя, иначе это может быть очень опасно.

Откройте `next.config.js` и включите опцию `ignoreBuildErrors` в конфигурации `typescript`:

```js filename="next.config.js"
module.exports = {
  typescript: {
    // !! ВНИМАНИЕ !!
    // Опасное разрешение на успешное завершение продакшен-сборки
    // даже при наличии ошибок типов в проекте.
    // !! ВНИМАНИЕ !!
    ignoreBuildErrors: true,
  },
}
```
