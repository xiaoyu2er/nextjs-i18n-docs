Next.js предоставляет сжатие [**gzip**](https://tools.ietf.org/html/rfc6713#section-3) для сжатия рендеренного контента и статических файлов. Обычно рекомендуется включать сжатие на HTTP-прокси, таком как [nginx](https://www.nginx.com/), чтобы разгрузить процесс `Node.js`.

Чтобы отключить **сжатие**, откройте `next.config.js` и отключите настройку `compress`:

```js filename="next.config.js"
module.exports = {
  compress: false,
}
```
