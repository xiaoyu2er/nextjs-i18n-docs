По умолчанию Next.js перенаправляет URL-адреса с завершающим слешем на их аналоги без слеша. Например, `/about/` будет перенаправлен на `/about`. Вы можете настроить обратное поведение, когда URL-адреса без завершающего слеша перенаправляются на их аналоги со слешем.

Откройте `next.config.js` и добавьте конфигурацию `trailingSlash`:

```js filename="next.config.js"
module.exports = {
  trailingSlash: true,
}
```

При включённой этой опции URL-адреса типа `/about` будут перенаправляться на `/about/`.
