> **Важно знать**: Начиная с Next.js 13.4 эта опция **больше не требуется**. Маршрутизатор приложений (App Router) теперь стабилен.

Маршрутизатор приложений ([директория `app`](/docs/app/building-your-application/routing)) обеспечивает поддержку [макетов](/docs/app/building-your-application/routing/pages-and-layouts), [серверных компонентов (Server Components)](/docs/app/building-your-application/rendering/server-components), [потоковой передачи (streaming)](/docs/app/building-your-application/routing/loading-ui-and-streaming) и [совмещённого получения данных (colocated data fetching)](/docs/app/building-your-application/data-fetching).

Использование директории `app` автоматически включает [строгий режим React (React Strict Mode)](https://react.dev/reference/react/StrictMode). Узнайте, как [постепенно внедрить `app`](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app).
