> **Полезно знать**: Начиная с Next.js 13.4, строгий режим (Strict Mode) включён по умолчанию для `app` маршрутизатора, поэтому приведённая конфигурация нужна только для `pages`. Вы всё ещё можете отключить строгий режим, установив `reactStrictMode: false`.

> **Рекомендация**: Мы настоятельно рекомендуем включить строгий режим в вашем приложении Next.js, чтобы лучше подготовить его к будущему React.

[Строгий режим React](https://react.dev/reference/react/StrictMode) — это функция только для режима разработки, которая помогает выявлять потенциальные проблемы в приложении. Он помогает обнаруживать небезопасные методы жизненного цикла, использование устаревших API и ряд других особенностей.

Среда выполнения Next.js совместима со строгим режимом. Чтобы включить строгий режим, настройте следующую опцию в вашем `next.config.js`:

```js filename="next.config.js"
module.exports = {
  reactStrictMode: true,
}
```

Если вы или ваша команда не готовы использовать строгий режим во всём приложении — это нормально! Вы можете постепенно внедрять его на отдельных страницах, используя `<React.StrictMode>`.
