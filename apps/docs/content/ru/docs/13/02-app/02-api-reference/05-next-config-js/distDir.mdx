Вы можете указать имя пользовательской директории сборки вместо `.next`.

Откройте файл `next.config.js` и добавьте конфигурацию `distDir`:

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

Теперь при запуске `next build` Next.js будет использовать папку `build` вместо стандартной `.next`.

> `distDir` **не должен** указывать за пределы директории вашего проекта. Например, `../build` является **недопустимым** путём.
