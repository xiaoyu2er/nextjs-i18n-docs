Экспериментальная поддержка [статически типизированных ссылок](/docs/app/building-your-application/configuring/typescript#statically-typed-links). Для использования этой функции требуется App Router и TypeScript в вашем проекте.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
