Next.js включает **два** инструмента командной строки (CLI):

- **`create-next-app`**: Быстро создает новое приложение Next.js, используя шаблон по умолчанию или [пример](https://github.com/vercel/next.js/tree/canary/examples) из публичного репозитория Github.
- **`next`**: Запускает сервер разработки Next.js, собирает приложение и выполняет другие задачи.
