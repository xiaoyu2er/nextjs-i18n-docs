Этот раздел документации посвящён **соглашениям по файлам метаданных**. Метаданные на основе файлов можно определить, добавив специальные файлы метаданных в сегменты маршрутов.

Каждое соглашение по файлам может быть определено либо с помощью статического файла (например, `opengraph-image.jpg`), либо с использованием динамического варианта, который генерирует файл с помощью кода (например, `opengraph-image.js`).

После определения файла Next.js автоматически обслуживает его (с хешами в production-сборке для кэширования) и обновляет соответствующие элементы head с правильными метаданными, такими как URL ресурса, тип файла и размер изображения.
