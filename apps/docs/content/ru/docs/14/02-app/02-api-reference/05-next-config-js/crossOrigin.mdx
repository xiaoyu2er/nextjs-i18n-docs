Используйте опцию `crossOrigin` для добавления атрибута [`crossOrigin`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin) во все теги `<script>`, генерируемые компонентом <AppOnly>[`next/script`](/docs/app/building-your-application/optimizing/scripts)</AppOnly> <PagesOnly>[`next/script`](/docs/pages/building-your-application/optimizing/scripts) и [`next/head`](/docs/pages/api-reference/components/head)</PagesOnly>, чтобы определить, как должны обрабатываться кросс-доменные запросы.

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## Опции

- `'anonymous'`: Добавляет атрибут [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous).
- `'use-credentials'`: Добавляет атрибут [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials).
