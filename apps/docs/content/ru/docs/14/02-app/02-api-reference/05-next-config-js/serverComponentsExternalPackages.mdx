---

Зависимости, используемые внутри [серверных компонентов (Server Components)](/docs/app/building-your-application/rendering/server-components) и [обработчиков маршрутов (Route Handlers)](/docs/app/building-your-application/routing/route-handlers), автоматически включаются в бандл Next.js.

Если зависимость использует специфичные для Node.js функции, вы можете исключить определенные зависимости из бандлинга серверных компонентов и использовать нативный Node.js `require`.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    serverComponentsExternalPackages: ['@acme/ui'],
  },
}

module.exports = nextConfig
```
