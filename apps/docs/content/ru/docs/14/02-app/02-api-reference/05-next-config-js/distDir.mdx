Вы можете указать имя пользовательской директории сборки, которая будет использоваться вместо `.next`.

Откройте файл `next.config.js` и добавьте конфигурацию `distDir`:

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

Теперь при запуске `next build` Next.js будет использовать папку `build` вместо стандартной `.next`.

> Директория `distDir` **не должна** выходить за пределы вашего проекта. Например, путь `../build` является **недопустимым**.
