## Обновление с версии 13 до 14

Для обновления до Next.js версии 14 выполните следующую команду, используя предпочитаемый менеджер пакетов:

```bash filename="Терминал"
npm i next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash filename="Терминал"
yarn add next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash filename="Терминал"
pnpm up next react react-dom eslint-config-next --latest
```

```bash filename="Терминал"
bun add next@latest react@latest react-dom@latest eslint-config-next@latest
```

> **Полезно знать:** Если вы используете TypeScript, убедитесь, что также обновили `@types/react` и `@types/react-dom` до их последних версий.

### Краткое описание версии 14

- Минимальная версия Node.js увеличена с 16.14 до 18.17, так как версия 16.x достигла конца жизненного цикла.
- Команда `next export` удалена в пользу конфигурации `output: 'export'`. Подробнее см. в [документации](https://nextjs.org/docs/app/building-your-application/deploying/static-exports).
- Импорт `ImageResponse` из `next/server` переименован в `next/og`. Доступен [кодмод](/docs/app/building-your-application/upgrading/codemods#next-og-import) для безопасного и автоматического переименования импортов.
- Пакет `@next/font` полностью удалён в пользу встроенного `next/font`. Доступен [кодмод](/docs/app/building-your-application/upgrading/codemods#built-in-next-font) для безопасного и автоматического переименования импортов.
- Целевая платформа WASM для `next-swc` удалена.
