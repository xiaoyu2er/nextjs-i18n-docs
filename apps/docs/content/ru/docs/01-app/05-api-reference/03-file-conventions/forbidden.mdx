Файл **forbidden** используется для отображения интерфейса при вызове функции [`forbidden`](/docs/app/api-reference/functions/forbidden) во время аутентификации. Помимо возможности кастомизации интерфейса, Next.js возвращает статус-код `403`.

```tsx filename="app/forbidden.tsx" switcher
import Link from 'next/link'

export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>
      <p>You are not authorized to access this resource.</p>
      <Link href="/">Return Home</Link>
    </div>
  )
}
```

```jsx filename="app/forbidden.jsx" switcher
import Link from 'next/link'

export default function Forbidden() {
  return (
    <div>
      <h2>Forbidden</h2>
      <p>You are not authorized to access this resource.</p>
      <Link href="/">Return Home</Link>
    </div>
  )
}
```

## Справочник

### Пропсы

Компоненты `forbidden.js` не принимают пропсов.
