Файл **unauthorized** используется для отображения интерфейса, когда вызывается функция [`unauthorized`](/docs/app/api-reference/functions/unauthorized) во время аутентификации. Помимо возможности кастомизировать интерфейс, Next.js возвращает статус-код `401`.

```tsx filename="app/unauthorized.tsx" switcher
import Login from '@/app/components/Login'

export default function Unauthorized() {
  return (
    <main>
      <h1>401 - Неавторизован</h1>
      <p>Пожалуйста, войдите для доступа к этой странице.</p>
      <Login />
    </main>
  )
}
```

## Справочник

### Пропсы

Компоненты `unauthorized.js` не принимают пропсов.

## Примеры

### Отображение интерфейса входа для неавторизованных пользователей

Вы можете использовать функцию [`unauthorized`](/docs/app/api-reference/functions/unauthorized) для отображения файла `unauthorized.js` с интерфейсом входа.

```tsx filename="app/dashboard/page.tsx" switcher
import { verifySession } from '@/app/lib/dal'
import { unauthorized } from 'next/navigation'

export default async function DashboardPage() {
  const session = await verifySession()

  if (!session) {
    unauthorized()
  }

  return <div>Dashboard</div>
}
```
