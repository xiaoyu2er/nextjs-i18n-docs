В версиях Node.js до 18 Next.js автоматически полифиллит `fetch()` с помощью [undici](/docs/architecture/supported-browsers#polyfills) и по умолчанию включает [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive).

Чтобы отключить HTTP Keep-Alive для всех вызовов `fetch()` на стороне сервера, откройте `next.config.js` и добавьте конфигурацию `httpAgentOptions`:

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
