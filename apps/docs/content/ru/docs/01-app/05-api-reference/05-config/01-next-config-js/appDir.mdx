> **Важно знать**: Начиная с Next.js 13.4 эта опция **больше не требуется**. Маршрутизатор приложения (App Router) теперь стабилен.

Маршрутизатор приложения ([директория `app`](/docs/app/building-your-application/routing)) обеспечивает поддержку [макетов](/docs/app/api-reference/file-conventions/layout), [серверных компонентов (Server Components)](/docs/app/getting-started/server-and-client-components), [потоковой передачи (streaming)](/docs/app/api-reference/file-conventions/loading) и [совмещённого получения данных (colocated data fetching)](/docs/app/building-your-application/data-fetching).

Использование директории `app` автоматически включает [строгий режим React (React Strict Mode)](https://react.dev/reference/react/StrictMode). Узнайте, как [постепенно внедрить директорию `app`](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app).
