Next.js предоставляет некоторые опции, которые позволяют контролировать, как сервер будет освобождать или удерживать в памяти собранные страницы в режиме разработки.

Чтобы изменить значения по умолчанию, откройте файл `next.config.js` и добавьте конфигурацию `onDemandEntries`:

```js filename="next.config.js"
module.exports = {
  onDemandEntries: {
    // период (в мс), в течение которого сервер будет хранить страницы в буфере
    maxInactiveAge: 25 * 1000,
    // количество страниц, которые должны одновременно храниться без освобождения
    pagesBufferLength: 2,
  },
}
```
