Опции `staticGeneration*` позволяют настроить процесс статической генерации (Static Generation) для сложных случаев использования.

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
const nextConfig = {
  experimental: {
    staticGenerationRetryCount: 1,
    staticGenerationMaxConcurrency: 8,
    staticGenerationMinPagesPerWorker: 25,
  },
}

export default nextConfig
```

## Настройки конфигурации

Доступны следующие опции:

- `staticGenerationRetryCount`: Количество попыток повторной генерации страницы при сбое до завершения сборки с ошибкой.
- `staticGenerationMaxConcurrency`: Максимальное количество страниц, обрабатываемых одним воркером.
- `staticGenerationMinPagesPerWorker`: Минимальное количество страниц для обработки перед запуском нового воркера.
