description: Исключение определенных зависимостей из бандлинга серверных компонентов и использование нативного Node.js `require`.
---

Зависимости, используемые внутри [серверных компонентов](/docs/app/getting-started/server-and-client-components) и [обработчиков маршрутов](/docs/app/building-your-application/routing/route-handlers), автоматически включаются в бандл Next.js.

Если зависимость использует специфичные для Node.js функции, вы можете исключить определенные зависимости из бандлинга серверных компонентов и использовать нативный Node.js `require`.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
