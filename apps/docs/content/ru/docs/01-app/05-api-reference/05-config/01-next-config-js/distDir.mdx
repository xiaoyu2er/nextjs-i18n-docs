Вы можете указать имя пользовательской директории сборки, которая будет использоваться вместо `.next`.

Откройте `next.config.js` и добавьте конфигурацию `distDir`:

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

Теперь при запуске `next build` Next.js будет использовать директорию `build` вместо стандартной `.next`.

> `distDir` **не должна** выходить за пределы вашего проекта. Например, `../build` является **недопустимой** директорией.
