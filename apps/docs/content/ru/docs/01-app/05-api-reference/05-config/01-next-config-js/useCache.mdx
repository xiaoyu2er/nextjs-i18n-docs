Флаг `useCache` — это экспериментальная функция в Next.js, которая позволяет использовать [директиву `use cache`](/docs/app/api-reference/directives/use-cache) независимо от [`dynamicIO`](/docs/app/api-reference/config/next-config-js/dynamicIO). При включении этой функции вы можете использовать `use cache` в своем приложении, даже если `dynamicIO` отключен.

## Использование

Чтобы включить флаг `useCache`, установите его в значение `true` в разделе `experimental` вашего файла `next.config.ts`:

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

Когда `useCache` включен, вы можете использовать следующие функции и настройки кэширования:

- [Директиву `use cache`](/docs/app/api-reference/directives/use-cache)
- Функцию [`cacheLife`](/docs/app/api-reference/config/next-config-js/cacheLife) с `use cache`
- Функцию [`cacheTag`](/docs/app/api-reference/functions/cacheTag)
