Next.js включает **два** инструмента командной строки (CLI):

- **`create-next-app`**: Быстро создает новое приложение Next.js с использованием шаблона по умолчанию или [примера](https://github.com/vercel/next.js/tree/canary/examples) из публичного репозитория GitHub.
- **`next`**: Запускает сервер разработки Next.js, собирает приложение и выполняет другие задачи.
