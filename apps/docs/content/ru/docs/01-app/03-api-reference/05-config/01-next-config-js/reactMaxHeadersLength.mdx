title: reactMaxHeadersLength
description: Максимальная длина заголовков, которые генерируются React и добавляются в ответ.
---

При статическом рендеринге React может генерировать заголовки, которые можно добавить в ответ. Это позволяет улучшить производительность, давая браузеру возможность предварительно загружать ресурсы, такие как шрифты, скрипты и таблицы стилей. Значение по умолчанию — `6000`, но его можно переопределить, настроив параметр `reactMaxHeadersLength` в файле `next.config.js`:

```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **Полезно знать**: Эта опция доступна только в App Router.

В зависимости от типа прокси между браузером и сервером заголовки могут быть обрезаны. Например, если вы используете обратный прокси, который не поддерживает длинные заголовки, следует установить меньшее значение, чтобы избежать их обрезки.
