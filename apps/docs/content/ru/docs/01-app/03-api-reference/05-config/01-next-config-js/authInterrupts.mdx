Опция конфигурации `authInterrupts` позволяет использовать API [`forbidden`](/docs/app/api-reference/functions/forbidden) и [`unauthorized`](/docs/app/api-reference/functions/unauthorized) в вашем приложении. Поскольку эти функции являются экспериментальными, для их использования необходимо включить опцию `authInterrupts` в файле `next.config.js`:

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    authInterrupts: true,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
module.exports = {
  experimental: {
    authInterrupts: true,
  },
}
```
