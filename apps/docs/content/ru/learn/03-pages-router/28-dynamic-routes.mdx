Мы заполнили индексную страницу данными блога, но пока не создали отдельные страницы для каждой записи (вот [желаемый результат](https://next-learn-starter.vercel.app/)). Мы хотим, чтобы URL этих страниц зависел от данных блога, что означает необходимость использования [динамических маршрутов](/docs/routing/dynamic-routes).

### [Что вы узнаете в этом уроке](#what-youll-learn-in-this-lesson)

В этом уроке вы узнаете:

* Как статически генерировать страницы с [динамическими маршрутами](/docs/routing/dynamic-routes) с помощью [`getStaticPaths`](/docs/basic-features/data-fetching#getstaticpaths-static-generation).
* Как писать [`getStaticProps`](/docs/basic-features/data-fetching#getstaticprops-static-generation) для получения данных каждой записи блога.
* Как отображать markdown с помощью [`remark`](https://github.com/remarkjs/remark).
* Как красиво форматировать строки с датами.
* Как ссылаться на страницы с [динамическими маршрутами](/docs/routing/dynamic-routes).
* Полезную информацию о [динамических маршрутах](/docs/routing/dynamic-routes).
