Добавленная нами вторая страница пока не имеет стилей. Давайте добавим CSS для оформления страницы.

Next.js имеет встроенную поддержку [CSS](/docs/basic-features/built-in-css-support) и [Sass](/docs/basic-features/built-in-css-support#sass-support). В этом курсе мы будем использовать CSS.

В этом уроке также будет рассмотрено, как Next.js работает со статическими ресурсами (например, изображениями) и метаданными страниц, такими как тег `<title>`.

### [Что вы изучите в этом уроке](#what-youll-learn-in-this-lesson)

В этом уроке вы узнаете:

* Как добавлять [статические файлы](/docs/basic-features/static-file-serving) (изображения и другие) в Next.js.
* Как настраивать содержимое `<head>` для каждой страницы.
* Как создавать переиспользуемые React-компоненты со стилями с помощью [CSS Modules](/docs/basic-features/built-in-css-support#adding-component-level-css).
* Как [добавлять глобальные CSS-стили](/docs/basic-features/built-in-css-support#adding-a-global-stylesheet) в [`pages/_app.js`](/docs/advanced-features/custom-app).
* Полезные советы по стилизации в Next.js.

### [Необходимые знания](#prerequisites)

* Базовые знания CSS. Этот курс покажет, как добавлять CSS в Next.js-приложение, но не будет охватывать основы CSS.

> Если вам нужна подробная документация по стилизации в Next.js, ознакомьтесь с [документацией по CSS](/docs/basic-features/built-in-css-support).
