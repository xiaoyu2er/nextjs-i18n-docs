**Если вы продолжаете с предыдущего урока,** эту страницу можно пропустить.

### [Загрузка стартового кода (опционально)](#download-starter-code-optional)

Если вы НЕ продолжаете с предыдущего урока, вы можете скачать, установить и запустить стартовый код для этого урока ниже. Это создаст директорию `nextjs-blog`, идентичную результату предыдущего урока.

Повторяем, это НЕ требуется, если вы только что завершили предыдущий урок.

```
npx create-next-app@latest nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/main/basics/basics-final"
```

Затем следуйте инструкциям из вывода команды (перейдите в директорию с помощью `cd` и запустите сервер разработки).

Также рекомендуется обновить следующие файлы:

*   `public/images/profile.jpg` - замените на ваше фото (рекомендуемый размер: 400px в ширину/высоту).
*   `const name = '[Ваше имя]'` в `components/layout.js` - укажите ваше имя.
*   `<p>[Ваше самопредставление]</p>` в `pages/index.js` - добавьте информацию о себе.
