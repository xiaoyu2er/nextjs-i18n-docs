Прежде чем развернуть приложение, давайте загрузим наше Next.js-приложение в [GitHub](https://github.com/vercel/next.js), если вы ещё этого не сделали. Это упростит процесс развертывания.

* В своём личном аккаунте GitHub создайте новый репозиторий с именем `nextjs-blog`.
* Репозиторий может быть публичным или приватным. **Не** нужно инициализировать его с файлами README или другими.
* Если вам нужна помощь в настройке репозитория, ознакомьтесь с [этим руководством на GitHub](https://help.github.com/en/github/getting-started-with-github/create-a-repo).

Затем:

* Если вы ещё не инициализировали git-репозиторий локально для вашего Next.js-приложения, сделайте это сейчас.
* Загрузите Next.js-приложение в ваш GitHub-репозиторий.

Для загрузки в GitHub выполните следующие команды (замените `<username>` на ваше имя пользователя GitHub):

```
git remote add origin https://github.com/<username>/nextjs-blog.git
git push -u origin main
```

После того как ваш GitHub-репозиторий будет готов, переходите к следующей странице.
