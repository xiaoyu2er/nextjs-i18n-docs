Пока что наше приложение Next.js содержит только одну страницу. Однако веб-сайты и приложения обычно состоят из множества различных страниц.

Давайте рассмотрим, как добавить больше страниц в наше приложение.

[Что вы узнаете в этом уроке](#что-вы-узнаете-в-этом-уроке)
-----------------------------------------------------------

В этом уроке вы:

* Создадите новую страницу, используя встроенную [файловую систему маршрутизации](/docs/routing/introduction).
* Узнаете, как использовать компонент [`Link`](https://nextjs.org/docs/api-reference/next/link) для клиентской навигации между страницами.
* Познакомитесь с встроенной поддержкой разделения кода (code splitting) и предварительной загрузки (prefetching).

> Если вам нужна подробная документация по маршрутизации в Next.js, ознакомьтесь с [документацией по маршрутизации](/docs/routing/introduction).
