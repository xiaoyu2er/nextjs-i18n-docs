[Next.js Speed Insights](https://nextjs.org/analytics) позволяет анализировать и измерять производительность страниц с использованием Core Web Vitals.

![Next.js Speed Insights](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/seo/analytics.png)

Вы можете начать собирать данные о [Real Experience Score](https://vercel.com/docs/concepts/speed-insights#metrics) без дополнительной настройки при [развертывании на Vercel](https://vercel.com/docs/concepts/speed-insights).
