Основная цель поисковой системы Google — предоставлять пользователям наилучшие результаты с учётом локализации и опечаток. В каждом случае Google стремится обеспечить пользователям быстрый и плавный результат с отличным взаимодействием с сайтом. [Скорость загрузки веб-страниц](https://developers.google.com/web/updates/2018/07/search-ads-speed) на мобильных устройствах является фактором ранжирования с 2018 года. Однако до недавнего времени не было ясно, какие именно метрики производительности использует алгоритм Google Search для ранжирования.

Это изменилось в июне 2021 года, когда Google предоставил [набор конкретных метрик и диапазонов](https://developers.google.com/search/blog/2021/04/more-details-page-experience) для анализа и оптимизации производительности.

### [Вес показателей в Lighthouse (v6)](#lighthouse-v6-weights-for-vitals)

Три метрики не обязательно имеют одинаковую значимость. В [Lighthouse](https://developers.google.com/web/tools/lighthouse) каждому из Core Web Vitals присвоен разный вес:

*   **Largest Contentful Paint (LCP)**: 25%
*   **Total Blocking Time (TBT)**: 25%
*   **First Contentful Paint (FCP)**: 15%
*   **Speed Index (SI)**: 15%
*   **Time to Interactive (TTI)**: 15%
*   **Cumulative Layout Shift (CLS)**: 5%

Это похоже на [First Input Delay (FID)](/learn/seo/fid).

**Примечание:** Влияние на ранжирование в Google будет одинаковым для всех страниц, попадающих в зелёную зону по всем Core Web Vitals, независимо от их индивидуальных показателей.

### [Влияние на пользовательский опыт](#ux-impact)

Большинство обсуждений Core Web Vitals в первую очередь сосредоточено на SEO.

Хотя верно, что Core Web Vitals — это инициатива, направленная на измерение и улучшение взаимодействия с страницей и поискового ранжирования, в конечном итоге выигрывают от этих изменений пользователи.

Core Web Vitals помогают стремиться к наилучшему взаимодействию с страницей. Согласно [исследованию Amazon за 2012 год](https://www.fastcompany.com/1825005/how-one-second-could-cost-amazon-16-billion-sales), одна дополнительная секунда загрузки может потенциально стоить компании 1,6 миллиарда долларов. Такие исследования демонстрируют важность отличного взаимодействия с страницей и быстрой работы сайта, чего и помогают достичь Core Web Vitals.
