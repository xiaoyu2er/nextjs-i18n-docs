В этом уроке мы рассмотрим различные метрики, влияние [Core Web Vitals](https://web.dev/vitals/) на SEO и их важность для пользовательского опыта.

При измерении Core Web Vitals существует три различных значения:

**"Хорошо"**, **"Требует улучшений"** и **"Плохо"**. Эти значения различаются в зависимости от измеряемого показателя:

![Core Web Vitals](https://h8DxKfmAPhn8O0p3.public.blob.vercel-storage.com/learn/pages-router/seo/vitals-light.png)

Существует два подхода к работе с Core Web Vitals:

*   **Стремиться к максимально возможному баллу по каждой метрике**. Стремление к совершенству — это прекрасно, но может быть сложно реализовать на крупных сайтах со множеством зависимостей.
*   **Сравнивать свои показатели с конкурентами в вашей отрасли**. Вы конкурируете не со всеми идеально оптимизированными сайтами в поиске Google, а с теми, кто ранжируется по вашим целевым ключевым словам.
