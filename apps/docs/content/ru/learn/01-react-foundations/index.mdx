Для эффективного изучения Next.js полезно быть знакомым с JavaScript, React и связанными концепциями веб-разработки. Но JavaScript и React — это обширные темы. Как понять, что вы готовы начать использовать Next.js?

Добро пожаловать на курс «Основы React»! Этот дружелюбный к новичкам курс с примерами проведёт вас через необходимые знания для работы с Next.js. Вы будете шаг за шагом создавать простой проект: начиная с JavaScript-приложения, затем перейдёте к React и Next.js.

Каждый раздел основывается на предыдущем, поэтому вы можете выбрать, с чего начать, в зависимости от того, что уже знаете.

### [Необходимые знания](#prerequisite-knowledge)

Это руководство предполагает знание HTML, CSS, JavaScript и отсутствие опыта работы с React. Если вы уже знакомы с React, можете перейти к главе [От React к Next.js](/learn/react-foundations/from-react-to-nextjs) или пройти курс [Приложение Dashboard](/learn/dashboard-app).

### [Системные требования](#system-requirements)

Перед началом курса убедитесь, что у вас установлено:

*   Node.js версии 20.12.0 или новее.
*   Операционная система: macOS, Windows (включая WSL) или Linux.
*   VSCode или другой текстовый редактор на ваш выбор.

### [Присоединяйтесь к обсуждению](#join-the-conversation)
