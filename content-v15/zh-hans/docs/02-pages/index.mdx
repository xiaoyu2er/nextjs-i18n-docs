**页面路由器** 使用直观的文件系统路由将每个文件映射到一个路由。

{/* md5:1db9c0588aa01958c3a39202e4f31b60 */}
在 Next.js 13 之前，页面路由器是 Next.js 中创建路由的主要方式。它在较新版本的 Next.js 中仍然受支持，但我们建议迁移到新的 [应用路由器](/docs/app) 以利用 React 的最新功能。
