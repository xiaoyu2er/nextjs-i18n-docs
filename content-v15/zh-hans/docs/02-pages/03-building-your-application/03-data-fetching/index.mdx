---
title: 数据获取 (Data Fetching)
description: Next.js 允许您以多种方式获取数据，包括预渲染 (pre-rendering)、服务端渲染 (server-side rendering)、静态站点生成 (static-site generation) 或增量静态再生 (incremental static regeneration)。了解如何在 Next.js 中管理您的应用程序数据。
---

Next.js 中的数据获取允许您根据应用程序的使用情况以不同方式渲染内容。这包括使用 **服务端渲染 (Server-side Rendering)** 或 **静态生成 (Static Generation)** 进行预渲染 (pre-rendering)，以及使用 **增量静态再生 (Incremental Static Regeneration)** 在运行时更新或创建内容。
