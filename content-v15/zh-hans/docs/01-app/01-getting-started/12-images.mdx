{/* md5:c0bfe49a98c758d003b5e9d4dca23911 */}
Next.js [`<Image>`](/docs/app/api-reference/components/image) 组件扩展了 HTML `<img>` 元素，以提供：

{/* md5:c15ec5e5cacb101065f1c04a49a20b38 */}
- **尺寸优化：** 使用 WebP 等现代图像格式，自动为每台设备提供尺寸合适的图像。
- **视觉稳定性：** 在图像加载时自动防止[布局偏移 (layout shift)](https://web.dev/articles/cls)。
- **更快的页面加载：** 仅当图像进入视口时才使用原生浏览器延迟加载进行加载，并可选地提供模糊占位符。
- **资源灵活性：** 按需调整图像大小，即使是存储在远程服务器上的图像。

{/* md5:da8ea2a39a3f794ccdc7b1303b576295 */}
要开始使用 `<Image>`，请从 `next/image` 导入它并在你的组件中渲染它。

```tsx filename="app/page.tsx" switcher
import Image from 'next/image'

export default function Page() {
  return <Image src="" alt="" />
}
```

```jsx filename="app/page.js" switcher
import Image from 'next/image'

export default function Page() {
  return <Image src="" alt="" />
}
```

{/* md5:a29da34f52a2a2f78437efbb9b5bd58a */}
`src` 属性可以是[本地](#local-images)或[远程](#remote-images)图像。

{/* md5:3b603a867dc5f5c8bde0cf11c27ea62d */}

## 本地图像

{/* md5:bda2df642487f11bc65a55ea94229132 */}
你可以将静态文件（如图像和字体）存储在根目录下名为 [`public`](/docs/app/api-reference/file-conventions/public-folder) 的文件夹中。`public` 文件夹中的文件随后可以通过你的代码从基 URL (`/`) 开始引用。

{/* md5:0340bb3d673cb8d49e021daa7dd71da2 */}

```tsx filename="app/page.tsx" switcher
import Image from 'next/image'

export default function Page() {
  return (
    <Image
      src="/profile.png"
      alt="Picture of the author"
      width={500}
      height={500}
    />
  )
}
```

```jsx filename="app/page.js" switcher
import Image from 'next/image'

export default function Page() {
  return (
    <Image
      src="/profile.png"
      alt="Picture of the author"
      width={500}
      height={500}
    />
  )
}
```

{/* md5:b53822df23434ff69c569e082f3a204c */}
如果图像是静态导入的，Next.js 将自动确定其固有的 [`width`](/docs/app/api-reference/components/image#width-and-height) 和 [`height`](/docs/app/api-reference/components/image#width-and-height)。这些值用于确定图像比例，并在图像加载时防止[累积布局偏移 (Cumulative Layout Shift)](https://web.dev/articles/cls)。

```tsx filename="app/page.tsx" switcher
import Image from 'next/image'
import ProfileImage from './profile.png'

export default function Page() {
  return (
    <Image
      src={ProfileImage}
      alt="Picture of the author"
      // width={500} automatically provided
      // height={500} automatically provided
      // blurDataURL="data:..." automatically provided
      // placeholder="blur" // Optional blur-up while loading
    />
  )
}
```

```jsx filename="app/page.js" switcher
import Image from 'next/image'
import ProfileImage from './profile.png'

export default function Page() {
  return (
    <Image
      src={ProfileImage}
      alt="Picture of the author"
      // width={500} automatically provided
      // height={500} automatically provided
      // blurDataURL="data:..." automatically provided
      // placeholder="blur" // Optional blur-up while loading
    />
  )
}
```

## 远程图像

{/* md5:bbbe6037fc70a164dcad5fb5800f983b */}
要使用远程图像，你可以为 `src` 属性提供一个 URL 字符串。

```tsx filename="app/page.tsx" switcher
import Image from 'next/image'

export default function Page() {
  return (
    <Image
      src="https://s3.amazonaws.com/my-bucket/profile.png"
      alt="Picture of the author"
      width={500}
      height={500}
    />
  )
}
```

```jsx filename="app/page.js" switcher
import Image from 'next/image'

export default function Page() {
  return (
    <Image
      src="https://s3.amazonaws.com/my-bucket/profile.png"
      alt="Picture of the author"
      width={500}
      height={500}
    />
  )
}
```

{/* md5:099d889b9ec729c0c15d8dbf9d81b657 */}
由于 Next.js 在构建过程中无法访问远程文件，你需要手动提供 [`width`](/docs/app/api-reference/components/image#width-and-height)、[`height`](/docs/app/api-reference/components/image#width-and-height) 和可选的 [`blurDataURL`](/docs/app/api-reference/components/image#blurdataurl) 属性。`width` 和 `height` 用于推断图像的正确宽高比，并避免图像加载时的布局偏移。或者，你可以使用 [`fill` 属性](/docs/app/api-reference/components/image#fill) 使图像填满其父元素的大小。

{/* md5:79dfa0ed4c41ea9d1aec94be9adf2929 */}
为了安全地允许来自远程服务器的图像，你需要在 [`next.config.js`](/docs/app/api-reference/config/next-config-js) 中定义一个支持的 URL 模式列表。请尽可能具体以防止恶意使用。例如，以下配置将只允许来自特定 AWS S3 存储桶的图像：

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const config: NextConfig = {
  images: {
    remotePatterns: [
      {
        protocol: 'https',
        hostname: 's3.amazonaws.com',
        port: '',
        pathname: '/my-bucket/**',
        search: '',
      },
    ],
  },
}

export default config
```

```js filename="next.config.js" switcher
module.exports = {
  images: {
    remotePatterns: [
      {
        protocol: 'https',
        hostname: 's3.amazonaws.com',
        port: '',
        pathname: '/my-bucket/**',
        search: '',
      },
    ],
  },
}
```
