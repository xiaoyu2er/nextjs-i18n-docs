## 最新版本

{/* md5:8f09994841ebebe969f700b75fafbb86 */}
要更新到 Next.js 的最新版本，你可以使用 `upgrade` codemod：

```bash filename="Terminal"
npx @next/codemod@latest upgrade latest
```

{/* md5:10043443a7ea49a4e5fd5f427a113d1d */}
如果您更喜欢手动升级，请安装最新的 Next.js 和 React 版本：

```bash package="pnpm"
pnpm i next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash package="npm"
npm i next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash package="yarn"
yarn add next@latest react@latest react-dom@latest eslint-config-next@latest
```

```bash package="bun"
bun add next@latest react@latest react-dom@latest eslint-config-next@latest
```

{/* md5:1407aab52cedef2dcc1a88c222176e90 */}
## Canary 版本

{/* md5:e97960bc6a978b8c431a8fada1a95368 */}
要更新到最新的 canary 版本，请确保您使用的是最新版本的 Next.js，并且一切运行正常。然后，运行以下命令：

```bash filename="Terminal"
npm i next@canary
```
