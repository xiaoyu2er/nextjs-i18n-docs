在 18 之前的 Node.js 版本中，Next.js 会自动使用 [undici](/docs/architecture/supported-browsers#polyfills) 对 `fetch()` 进行 polyfill 支持，并默认启用 [HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive)。

若要在服务器端禁用所有 `fetch()` 调用的 HTTP Keep-Alive，请打开 `next.config.js` 并添加 `httpAgentOptions` 配置：

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
