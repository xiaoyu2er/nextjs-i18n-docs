源映射 (Source Maps) 在开发期间默认启用。在生产构建期间，除非您明确使用配置标志选择加入，否则它们会被禁用，以防止您的源代码泄露到客户端。

Next.js 提供了一个配置标志，您可在生产构建期间使用它来启用浏览器源映射生成：

```js filename="next.config.js"
module.exports = {
  productionBrowserSourceMaps: true,
}
```

启用 `productionBrowserSourceMaps` 选项后，源映射将输出在与 JavaScript 文件相同的目录中。Next.js 在请求这些文件时将自动提供它们。

- 添加源映射可能会增加 `next build` 时间
- 增加 `next build` 期间的内存使用量
