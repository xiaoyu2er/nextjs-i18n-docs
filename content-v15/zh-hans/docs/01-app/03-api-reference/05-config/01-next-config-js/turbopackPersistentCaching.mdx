## 使用方法

Turbopack 持久化缓存 (Persistent Caching) 可让 Turbopack 减少在 `next dev` 或 `next build` 命令间的工作。启用后，Turbopack 将在构建间将数据保存到 `.next` 文件夹并恢复，这可以大大加快后续构建和开发会话的速度。

> **警告:** 持久化缓存 (Persistent Caching) 仍在开发中，尚未稳定。采用该功能的用户可能会遇到一些稳定性问题。

> **小贴士**: 请注意，虽然 `next dev` 和 `next build` 可以共享缓存数据，但由于配置和环境变量不同，大多数缓存条目是特定于命令的。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    turbopackPersistentCaching: true,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    turbopackPersistentCaching: true,
  },
}

module.exports = nextConfig
```
