Next.js 默认会为每个页面生成 [ETag](https://en.wikipedia.org/wiki/HTTP_ETag)。根据您的缓存策略，您可能需要为 HTML 页面禁用 ETag 生成。

打开 next.config.js 并禁用 generateEtags 选项：

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
