您可以指定一个名称，用作自定义构建目录以替代 `.next`。

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

现在，如果您运行 `next build`， Next.js 将使用 `build` 而不是默认的 `.next` 文件夹。

> `distDir` **不应** 超出您的项目目录。 例如， `../build` 是一个 **无效** 目录。
