---
title: serverExternalPackages
description: 将特定依赖项排除在服务器组件打包之外，使用原生 Node.js `require`。
---

在 [服务器组件 (Server Components)](/docs/app/getting-started/server-and-client-components) 和 [路由处理程序 (Route Handlers)](/docs/app/api-reference/file-conventions/route) 中使用的依赖项将由 Next.js 自动打包。

如果某个依赖项使用了 Node.js 特有的功能，您可以选择将其排除在服务器组件打包之外，并使用原生 Node.js `require`。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
