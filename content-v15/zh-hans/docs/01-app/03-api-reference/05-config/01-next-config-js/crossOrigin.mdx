---

{/* 本文档的内容在应用路由和页面路由之间共享。您可以使用 `<PagesOnly>Content</PagesOnly>` 组件来添加仅适用于页面路由的内容。任何共享内容不应包裹在组件中。 */}

使用 `crossOrigin` 选项可为所有 <AppOnly>\[`next/script`\](/docs/app/guides/scripts) 组件</AppOnly> <PagesOnly>\[`next/script`\](/docs/pages/guides/scripts) 和 [`next/head`](/docs/pages/api-reference/components/head) 组件</PagesOnly> 生成的 `<script>` 标签添加 [`crossOrigin` 属性](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin)，并定义如何处理跨域请求。

```js filename="next.config.js"
module.exports = {
  crossOrigin: 'anonymous',
}
```

## 选项

- `'anonymous'`: 添加 [`crossOrigin="anonymous"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#anonymous) 属性。
- `'use-credentials'`: 添加 [`crossOrigin="use-credentials"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes/crossorigin#use-credentials)。
