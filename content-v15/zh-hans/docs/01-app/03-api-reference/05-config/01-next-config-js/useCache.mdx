`useCache` 标志是 Next.js 中的一项实验性功能，它允许 [`use cache` 指令](/docs/app/api-reference/directives/use-cache) 独立于 [`cacheComponents`](/docs/app/api-reference/config/next-config-js/cacheComponents) 使用。启用后，即使 `cacheComponents` 已关闭，您也可以在应用程序中使用 `use cache`。

## 使用方法

要启用 `useCache` 标志，请在您的 `next.config.ts` 文件的 `experimental` 部分将其设置为 `true`：

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

启用 `useCache` 后，您可以使用以下缓存函数和配置：

- [`use cache` 指令](/docs/app/api-reference/directives/use-cache)
- 与 `use cache` 配合使用的 [`cacheLife` 函数](/docs/app/api-reference/config/next-config-js/cacheLife)
- [`cacheTag` 函数](/docs/app/api-reference/functions/cacheTag)
