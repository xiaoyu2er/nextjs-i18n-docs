---
title: reactMaxHeadersLength
description: React 发出并添加到响应中的标头的最大长度。
---

在静态渲染期间，React 可以发出可添加到响应中的标头。这些可以用于通过允许浏览器预加载诸如字体、脚本和样式表之类的资源来提高性能。默认值为 `6000`，但您可以通过在 `next.config.js` 中配置 `reactMaxHeadersLength` 选项来覆盖此值：
```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```


```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **要知道**：此选项仅在 App Router 中可用。


根据浏览器和服务器之间的代理类型，标头可能会被截断。例如，如果您使用的是不支持长标头的反向代理，则应该设置较低的值以确保标头不会被截断。
