---
title: reactMaxHeadersLength
description: React 生成并添加到响应中的响应头 (headers) 的最大长度。
---

在静态渲染 (static rendering) 期间，React 可以生成会添加到响应中的响应头 (headers)。这些响应头可用于提升性能，允许浏览器预加载资源，例如字体、脚本和样式表。默认值为 `6000`，但您可以通过在 `next.config.js` 中配置 `reactMaxHeadersLength` 选项来覆盖此值：

```js filename="next.config.js"
module.exports = {
  reactMaxHeadersLength: 1000,
}
```

> **值得了解**：此选项仅在 App Router 中可用。

根据浏览器和服务器之间的代理 (proxy) 类型，响应头 (headers) 可能会被截断。例如，如果您使用的是不支持长响应头 (headers) 的反向代理 (reverse proxy)，则应设置一个较低的值，以确保响应头不会被截断。
