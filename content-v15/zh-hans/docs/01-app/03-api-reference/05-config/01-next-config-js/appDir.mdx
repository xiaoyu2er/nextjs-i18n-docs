title: 应用目录 (appDir)
description: 启用 App Router 以使用布局、流式传输等功能。
version: 旧版
---

> **值得了解**：自 Next.js 13.4 起，该选项不再需要。App Router 现已稳定。

App Router ([`app` 目录](/docs/app)) 支持 [布局](/docs/app/api-reference/file-conventions/layout)、[Server 组件 (Server Components)](/docs/app/getting-started/server-and-client-components)、[流式传输 (streaming)](/docs/app/api-reference/file-conventions/loading) 和 [就地数据获取 (colocated data fetching)](/docs/app/getting-started/fetching-data)。

使用 `app` 目录将自动启用 [React Strict Mode (React 严格模式)](https://react.dev/reference/react/StrictMode)。了解如何通过 [渐进式采用 `app`](/docs/app/guides/migrating/app-router-migration#migrating-from-pages-to-app)。
