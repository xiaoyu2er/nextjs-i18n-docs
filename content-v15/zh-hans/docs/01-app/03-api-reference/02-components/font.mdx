{/* md5:a8ca6ec161ba0dfad520fa7f3627c0f2 */}
[`next/font`](/docs/app/api-reference/components/font) 会自动优化你的字体（包括自定义字体），并移除外部网络请求，以提升隐私性和性能。

{/* md5:ffd86d10afe3de4fabf67509601c2261 */}
它包括针对任何字体文件的**内置自动自托管**。这意味着你可以以零[布局偏移](https://web.dev/articles/cls)的方式最佳地加载网页字体。

{/* md5:ab12f4fb0003e2a6d6bf7287d1e4f079 */}
你也可以方便地使用所有 [Google Fonts](https://fonts.google.com/)。CSS 和字体文件会在构建时下载，并与你的其他静态资源一起自托管。**浏览器不会向 Google 发送任何请求。**

<AppOnly>

```tsx filename="app/layout.tsx" switcher
import { Inter } from 'next/font/google'

// If loading a variable font, you don't need to specify the font weight
const inter = Inter({
  subsets: ['latin'],
  display: 'swap',
})

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en" className={inter.className}>
      <body>{children}</body>
    </html>
  )
}
```

```jsx filename="app/layout.js" switcher
import { Inter } from 'next/font/google'

// If loading a variable font, you don't need to specify the font weight
const inter = Inter({
  subsets: ['latin'],
  display: 'swap',
})

export default function RootLayout({ children }) {
  return (
    <html lang="en" className={inter.className}>
      <body>{children}</body>
    </html>
  )
}
```

</AppOnly>

### `src`

{/* md5:381fa2a1daaf8d3208baa76457f0c90e */}
字体文件的路径，可以是字符串或对象数组（类型为 `Array<{path: string, weight?: string, style?: string}>`），路径相对于调用字体加载器函数的目录。

{/* md5:ae6ba5d5bdf6f30d70c0737cdaee3f18 */}
用于 `next/font/local`

{/* md5:97fc66470937743603cd906ae180a541 */}
- 必需

{/* md5:fd05d7f6bfc90e815087880e8e915b43 */}
示例:

{/* md5:c0381729e1dba5b02f3f2a72969c6949 */}
- `src:'./fonts/my-font.woff2'`，其中 `my-font.woff2` 放置在 `app` 目录内名为 `fonts` 的目录中
- `src:[{path: './inter/Inter-Thin.ttf', weight: '100',},{path: './inter/Inter-Regular.ttf',weight: '400',},{path: './inter/Inter-Bold-Italic.ttf', weight: '700',style: 'italic',},]`
- 如果字体加载器函数在 `app/page.tsx` 中使用 `src:'../styles/fonts/my-font.ttf'` 调用，那么 `my-font.ttf` 放置在项目根目录下的 `styles/fonts` 中

{/* md5:2414104d9a8193cef618b258a2f16c82 */}
### `weight`

{/* md5:cb0bea5343d7ea1a518222447427acaa */}
字体 [`weight`](https://fonts.google.com/knowledge/glossary/weight)，有以下几种可能：

{/* md5:7fbe1de21421d763db569c04a4d00ef9 */}
- 一个字符串，其值可以是特定字体可用的权重值，或者如果是[可变](https://fonts.google.com/variablefonts)字体，则可以是权重值范围。
- 如果字体不是[可变 Google 字体](https://fonts.google.com/variablefonts)，则可以是权重值数组。仅适用于 `next/font/google`。

{/* md5:010cd0e3cbb781ef61d0e9c381c9ea25 */}
用于 `next/font/google` 和 `next/font/local`

{/* md5:5fbdad89def623b42988c8d5707eafcb */}
- 如果使用的字体**不是**[可变](https://fonts.google.com/variablefonts)字体，则必需。

{/* md5:fd05d7f6bfc90e815087880e8e915b43 */}
示例:

{/* md5:0aecf66a3062c9f8fd7f3fb7b52afeb0 */}
- `weight: '400'`：单个权重值的字符串 - 对于字体 [`Inter`](https://fonts.google.com/specimen/Inter?query=inter)，可能的值有 `'100'`、`'200'`、`'300'`、`'400'`、`'500'`、`'600'`、`'700'`、`'800'`、`'900'` 或 `'variable'`（其中 `'variable'` 是默认值）
- `weight: '100 900'`：可变字体在 `100` 到 `900` 之间的范围字符串
- `weight: ['100','400','900']`：非可变字体的 3 个可能值的数组

{/* md5:d048a7bb78fa2611a32fdd04257cc49f */}
### `style`

{/* md5:cfd06207082284445d406c10d08435e0 */}
字体 [`style`](https://developer.mozilla.org/docs/Web/CSS/font-style)，有以下几种可能：

{/* md5:5bba1baf9e2abe3152a32b231efa61f2 */}
- 一个字符串[值](https://developer.mozilla.org/docs/Web/CSS/font-style#values)，默认值为 `'normal'`
- 如果字体不是[可变 Google 字体](https://fonts.google.com/variablefonts)，则可以是样式值数组。仅适用于 `next/font/google`。

{/* md5:010cd0e3cbb781ef61d0e9c381c9ea25 */}
用于 `next/font/google` 和 `next/font/local`

{/* md5:f0a247afcc6e28306a96dda0dba12a4f */}
- 可选

{/* md5:fd05d7f6bfc90e815087880e8e915b43 */}
示例:

{/* md5:c21d85106f643d02dc581c4aef3f095a */}
- `style: 'italic'`：一个字符串 - 对于 `next/font/google`，可以是 `normal` 或 `italic`
- `style: 'oblique'`：一个字符串 - 对于 `next/font/local`，它可以是任何值，但预期应来自[标准字体样式](https://developer.mozilla.org/docs/Web/CSS/font-style)
- `style: ['italic','normal']`：`next/font/google` 的 2 个值的数组 - 值来自 `normal` 和 `italic`

{/* md5:b8cfb76700f533bd3340f12d1b1f5941 */}
### `subsets`

{/* md5:90ccd86a4fdf14501399cec337ad53fe */}
字体 [`subsets`](https://fonts.google.com/knowledge/glossary/subsetting)，由字符串值数组定义，每个字符串是你希望[预加载](/docs/app/api-reference/components/font#specifying-a-subset)的子集名称。当 [`preload`](#preload) 选项为 true（默认为 true）时，通过 `subsets` 指定的字体将在头部注入一个链接预加载标签。

{/* md5:12061c96b2c424f1b5fa9e50fa51bd82 */}
用于 `next/font/google`

{/* md5:f0a247afcc6e28306a96dda0dba12a4f */}
- 可选

{/* md5:fd05d7f6bfc90e815087880e8e915b43 */}
示例:

{/* md5:272207eb48bb963466b9ccee7a8061bd */}
- `subsets: ['latin']`：一个包含子集 `latin` 的数组

{/* md5:c0a04644d4247756c567230ab1c45d00 */}
你可以在 Google Fonts 页面上找到你的字体的所有子集列表。

{/* md5:2487b138b3298caed312dea54dbe722a */}
### `axes`

{/* md5:88388525f86b5a0fa3b31fd0e5c2ed24 */}
某些可变字体有额外的 `axes` 可以包含。默认情况下，为了减小文件大小，只包含字体粗细。`axes` 的可能值取决于具体的字体。

{/* md5:12061c96b2c424f1b5fa9e50fa51bd82 */}
用于 `next/font/google`

{/* md5:f0a247afcc6e28306a96dda0dba12a4f */}
- 可选

{/* md5:fd05d7f6bfc90e815087880e8e915b43 */}
示例:

{/* md5:adc2398e3c44ceacb34320ddf04568be */}

### 在 <head> 中应用字体

{/* md5:19cff4d533002b628a55d03ba813a119 */}
你也可以在没有包装器和 `className` 的情况下使用字体，通过将其注入到 `<head>` 中，如下所示：

```jsx filename="pages/_app.js"
import { Inter } from 'next/font/google'

const inter = Inter({ subsets: ['latin'] })

export default function MyApp({ Component, pageProps }) {
  return (
    <>
      <style jsx global>{`
        html {
          font-family: ${inter.style.fontFamily};
        }
      `}</style>
      <Component {...pageProps} />
    </>
  )
}
```

### 指定子集

{/* md5:53b3c82b9f590a70e6656df326c44039 */}
Google 字体会自动[子集化](https://fonts.google.com/knowledge/glossary/subsetting)。这减少了字体文件的大小并提高了性能。你需要定义你想要预加载哪些子集。如果在 [`preload`](/docs/app/api-reference/components/font#preload) 为 `true` 时未指定任何子集，将会导致警告。

{/* md5:1649f28f3b25bfcb3b9adb06d2af7e5f */}
这可以通过将其添加到函数调用来完成：

<AppOnly>

```tsx filename="app/layout.tsx" switcher
const inter = Inter({ subsets: ['latin'] })
```

### 使用字体定义文件

{/* md5:ccb0dd569359ab6bc3df217ca9c94bac */}
每次调用 `localFont` 或 Google 字体函数时，该字体都将在你的应用程序中作为一个实例托管。因此，如果你需要在多个地方使用相同的字体，你应该在一个地方加载它，并在需要的地方导入相关的字体对象。这是使用字体定义文件完成的。

{/* md5:94739fa37afe0273769120db36456b4d */}

### 预加载

<AppOnly>

{/* md5:2663bcb16e154414e5a4c44389755e03 */}
当字体函数在你网站的页面上被调用时，它不是全局可用的，也不会在所有路由上预加载。相反，字体仅根据使用它的文件类型在相关路由上预加载：

{/* md5:49c35a234c2b121d64698f39e109a5ce */}
- 如果是[唯一页面](/docs/app/api-reference/file-conventions/page)，则在其唯一路由上预加载。
- 如果是[布局](/docs/app/api-reference/file-conventions/layout)，则在该布局包装的所有路由上预加载。
- 如果是[根布局](/docs/app/api-reference/file-conventions/layout#root-layout)，则在所有路由上预加载。

</AppOnly>

{/* md5:965a9992a06c435b99ee69ee0261121e */}

{/* md5:4d8989cd7d6b6e2ac0f590483bdd1c41 */}
