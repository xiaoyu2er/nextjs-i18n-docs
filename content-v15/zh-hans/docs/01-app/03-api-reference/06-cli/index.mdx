Next.js 附带**两个**命令行界面 (CLI) 工具：

{/* md5:02b348f1c400a23e5af67ddb9472f55d */}
- **`create-next-app`**: 使用默认模板或来自公共 GitHub 仓库的 [示例](https://github.com/vercel/next.js/tree/canary/examples) 快速创建一个新的 Next.js 应用程序。
- **`next`**: 运行 Next.js 开发服务器、构建您的应用程序等等。
