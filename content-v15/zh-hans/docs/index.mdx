## Next.js 是什么？

Next.js 是一个 React 框架，用于构建全栈 Web 应用程序。您可以使用 React 组件构建用户界面，并使用 Next.js 获得额外功能和优化。

它还自动配置底层工具，如打包器和编译器。您可以转而专注于构建您的产品并快速交付。

无论您是独立开发者还是大型团队的一员，Next.js 都能帮助您构建交互式、动态且快速的 React 应用程序。

## App Router 和 Pages Router

Next.js 有两个不同的路由器：

- **App Router**：较新的路由器，支持新的 React 功能，如服务端组件。
- **Pages Router**：原始路由器，仍受支持并不断改进。

在侧边栏顶部，您会看到一个下拉菜单，允许您在 [App Router](/docs/app) 和 [Pages Router](/docs/pages) 文档之间切换。

### React 版本处理

App Router 和 Pages Router 处理 React 版本的方式不同：

- **App Router**：内置使用 [React canary 版本](https://react.dev/blog/2023/05/03/react-canaries)，其中包含所有稳定的 React 19 更改，以及在新 React 发布前在框架中验证的较新功能。
- **Pages Router**：使用项目 `package.json` 中安装的 React 版本。

这种方法确保新 React 功能在 App Router 中可靠运行，同时保持现有 Pages Router 应用程序的向后兼容性。
