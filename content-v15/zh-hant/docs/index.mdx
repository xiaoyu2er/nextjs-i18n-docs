## 什麼是 Next.js？

Next.js 是一個用於建構全端 Web 應用程式的 React 框架。您可以使用 React 元件來建構使用者介面，並使用 Next.js 進行額外的功能和最佳化。它還會自動配置較底層的工具，例如捆綁器 (bundlers) 和編譯器 (compilers)。您反而可以專注於建構您的產品並快速發布。無論您是個人開發者還是大型團隊的一員，Next.js 都能幫助您建構互動性、動態性和快速的 React 應用程式。

## App Router 和 Pages Router

Next.js 擁有兩種不同的路由器：

- **App Router**：較新的路由器，支援新 React 功能，例如伺服器組件 (Server Components)。
- **Pages Router**：原始路由器，仍獲支援並持續改善中。

在側邊欄頂部，您會注意到一個下拉選單，允許您在 [App Router](/docs/app) 和 [Pages Router](/docs/pages) 文件之間切換。

### React 版本處理

App Router 和 Pages Router 處理 React 版本的方式不同：

- **App Router**：使用內建的 [React Canary Releases](https://react.dev/blog/2023/05/03/react-canaries)，包含所有穩定的 React 19 變更，以及在新 React 版本發布之前，在框架中驗證的新功能。

- **Pages Router**：使用專案 `package.json` 中安裝的 React 版本。

此方法確保新的 React 功能在 App Router 中能可靠運作，同時維持現有 Pages Router 應用程式的向後相容性。
