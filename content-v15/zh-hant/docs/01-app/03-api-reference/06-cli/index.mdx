Next.js 內建**兩個**命令行介面 (CLI) 工具：

- **`create-next-app`**: 使用預設範本或來自公共 GitHub 儲存庫的 [範例](https://github.com/vercel/next.js/tree/canary/examples) 快速建立新的 Next.js 應用程式。
- **`next`**: 運行 Next.js 開發伺服器、建構您的應用程式，以及其他功能。
