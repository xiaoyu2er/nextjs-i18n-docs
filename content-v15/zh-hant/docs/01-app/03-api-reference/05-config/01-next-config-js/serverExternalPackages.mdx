---
title: 伺服器外部套件 (serverExternalPackages)
description: 停用特定相依套件從伺服器組件 (Server Components) 的打包，並使用原生的 Node.js `require`。

在 [伺服器組件 (Server Components)](/docs/app/getting-started/server-and-client-components) 和 [路由處理程序 (Route Handlers)](/docs/app/api-reference/file-conventions/route) 中使用的相依套件將由 Next.js 自動打包。

如果相依套件使用了 Node.js 特定功能，你可以選擇將特定相依套件排除於伺服器組件 (Server Components) 打包之外，並使用原生的 Node.js `require`。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  serverExternalPackages: ['@acme/ui'],
}

module.exports = nextConfig
```
