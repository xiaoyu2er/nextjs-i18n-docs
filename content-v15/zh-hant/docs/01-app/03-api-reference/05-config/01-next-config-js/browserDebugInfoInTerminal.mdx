---
title: 瀏覽器除錯資訊終端 (browserDebugInfoInTerminal)
description: 在開發過程中將瀏覽器控制台日誌和錯誤轉發到您的終端。
version: 實驗性 (experimental)
---

`experimental.browserDebugInfoInTerminal` 選項會將源自瀏覽器的控制台輸出與執行時錯誤轉發至開發伺服器終端。

此選項預設為停用。啟用後僅在開發模式中運作。

## 使用方法

啟用轉發：

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    browserDebugInfoInTerminal: true,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    browserDebugInfoInTerminal: true,
  },
}

module.exports = nextConfig
```

### 序列化限制

深度嵌套的物件/陣列使用 **合理的預設值** 進行截斷。您可以調整這些限制：

- **depthLimit**：(選用) 限制嵌套物件/陣列的字串化深度。預設：5
- **edgeLimit**：(選用) 每個物件或陣列包含的屬性或元素的最大數量。預設：100

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    browserDebugInfoInTerminal: {
      depthLimit: 5,
      edgeLimit: 100,
    },
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    browserDebugInfoInTerminal: {
      depthLimit: 5,
      edgeLimit: 100,
    },
  },
}

module.exports = nextConfig
```

### 來源位置

啟用此功能時，預設包含來源位置資訊。

```tsx filename="app/page.tsx" highlight={8}
'use client'

export default function Home() {
  return (
    <button
      type="button"
      onClick={() => {
        console.log('Hello World')
      }}
    >
      Click me
    </button>
  )
}
```

按下按鈕會將此訊息列印至終端。

```bash filename="Terminal"
[browser] Hello World (app/page.tsx:8:17)
```

若要抑制此訊息，請設定 `showSourceLocation: false`。

- **showSourceLocation**：包含可用的來源位置資訊。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    browserDebugInfoInTerminal: {
      showSourceLocation: false,
    },
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    browserDebugInfoInTerminal: {
      showSourceLocation: false,
    },
  },
}

module.exports = nextConfig
```
