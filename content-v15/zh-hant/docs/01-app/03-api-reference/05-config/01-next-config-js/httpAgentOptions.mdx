在 Node.js 18 之前的版本中，Next.js 會自動使用 [undici](/docs/architecture/supported-browsers#polyfills) 為 `fetch()` 進行填充 (polyfill)，並預設啟用 HTTP 保持連線 (HTTP Keep-Alive)。

若要在伺服器端停用所有 `fetch()` 調用的 HTTP 保持連線 (HTTP Keep-Alive)，請開啟 `next.config.js` 並新增 `httpAgentOptions` 配置:

```js filename="next.config.js"
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
