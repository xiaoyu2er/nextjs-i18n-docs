---
title: 類型化路由 (typedRoutes)
description: 啟用靜態類型化鏈接 (statically typed links) 支援。
---

> **注意**: 此選項已標記為穩定版本，因此您應使用 `typedRoutes` 而非 `experimental.typedRoutes`。

支持 [靜態類型化鏈接 (statically typed links)](/docs/app/api-reference/config/typescript#statically-typed-links)。此功能要求您的項目中使用 TypeScript。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  typedRoutes: true,
}

module.exports = nextConfig
```
