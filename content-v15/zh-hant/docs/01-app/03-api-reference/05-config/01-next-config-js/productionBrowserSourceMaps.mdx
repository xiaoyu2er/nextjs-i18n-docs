源碼映射 (Source Maps) 在開發期間預設啟用。在生產構建期間，停用它們可防止原始碼洩漏至客戶端，除非您特別透過配置標誌啟用。

Next.js 提供了一個配置標誌，您可在生產構建期間使用它以啟用瀏覽器源碼映射生成：

```js filename="next.config.js"
module.exports = {
  productionBrowserSourceMaps: true,
}
```

當 `productionBrowserSourceMaps` 選項啟用時，源碼映射 (Source Maps) 將與 JavaScript 檔案在同一目錄中輸出。當請求時，Next.js 將自動提供這些檔案。

- 添加源碼映射可能會增加 `next build` 的時間
- 在 `next build` 期間增加記憶體使用量
