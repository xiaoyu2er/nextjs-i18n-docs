## 使用方法

Turbopack 持久緩存 (Persistent Caching) 能減少 `next dev` 或 `next build` 命令之間的工作量。啟用後，Turbopack 會在構建之間將數據保存並恢復到 `.next` 資料夾，這可以顯著加快後續構建和開發會話的速度。

> **警告**：持久緩存 (Persistent Caching) 仍在開發中，尚未穩定。採用此功能的用戶應預先了解可能會遇到穩定性問題。

> **小提示**：注意，雖然 `next dev` 和 `next build` 可以共享緩存數據，但由於配置和環境變量不同，大多數緩存條目是專用於命令的。

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    turbopackPersistentCaching: true,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    turbopackPersistentCaching: true,
  },
}

module.exports = nextConfig
```
