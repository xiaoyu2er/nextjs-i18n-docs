``viewTransition`` 是一個實驗性標記，可在 React 中啟用新的 [View Transitions API](https://developer.mozilla.org/en-US/docs/Web/API/View_Transition_API)。此 API 允許您利用原生的 View Transitions 瀏覽器 API 來在 UI 狀態之間建立無縫過渡。

要啟用此功能，您需要在 `next.config.js` 文件中將 `viewTransition` 屬性設置為 `true`。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    viewTransition: true,
  },
}

module.exports = nextConfig
```

> 重要提示：此功能並非由 Next.js 團隊開發或維護——它是來自 React 團隊的實驗性 API。它仍處於 **早期階段**，**不建議用於生產環境**。該實現仍在迭代中，其行為可能在未來的 React 版本中發生變化。
> 啟用此功能需要了解該 API 的實驗性質。為了完全理解其行為，請參閱 [React 拉取請求 (Pull Request)](https://github.com/facebook/react/pull/31975) 和相關討論。

## 使用方法

啟用後，您可以在應用中從 React 導入 `ViewTransition` 組件：

```jsx
import { unstable_ViewTransition as ViewTransition } from 'react'
```

然而，文檔和示例目前有限，您將需要直接參考 React 的源代碼和討論來了解其工作原理。
