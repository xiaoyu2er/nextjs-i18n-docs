Next.js 預設會為每個頁面生成 [ETag](https://en.wikipedia.org/wiki/HTTP_ETag)。根據您的快取策略，您可能想要停用 HTML 頁面的 ETag 生成。

開啟 `next.config.js` 並停用 `generateEtags` 選項：

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```

```js filename="next.config.js"
module.exports = {
  generateEtags: false,
}
```
