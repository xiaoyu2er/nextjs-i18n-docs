---
title: 伺服器標頭 (poweredByHeader)
description: Next.js 預設會新增 `x-powered-by` 標頭。在此處了解如何停用。
---

{/* 此文件內容在應用程式路由器和頁面路由器之間共享。您可以使用 `<PagesOnly>內容</PagesOnly>` 元件來新增專屬於頁面路由器的內容。任何共享內容都不應包裹在元件中。 */}

預設情況下，Next.js 會新增 `x-powered-by` 標頭。若要停用，請開啟 `next.config.js` 並停用 `poweredByHeader` 配置：

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
