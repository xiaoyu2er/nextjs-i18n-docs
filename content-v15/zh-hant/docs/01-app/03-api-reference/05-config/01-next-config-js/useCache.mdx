 `useCache` 標誌是 Next.js 中的一項實驗性功能，它允許 [使用快取 (use cache) 指令](/docs/app/api-reference/directives/use-cache) 獨立於 [`cacheComponents`](/docs/app/api-reference/config/next-config-js/cacheComponents) 使用。啟用後，即使 `cacheComponents` 未開啟，您也可以在應用程式中使用 `use cache`。

## 使用方法

## 使用方法

若要啟用 `useCache` 標誌，請在您的 `next.config.ts` 檔案的 `experimental` 部分將其設為 `true`：

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

```ts filename="next.config.ts"
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    useCache: true,
  },
}

export default nextConfig
```

啟用 `useCache` 後，您可使用以下快取相關函式與設定：

- [使用快取 (use cache) 指令](/docs/app/api-reference/directives/use-cache)
- [`cacheLife` 函式](/docs/app/api-reference/config/next-config-js/cacheLife) 搭配 `use cache` 使用
- [`cacheTag` 函式](/docs/app/api-reference/functions/cacheTag)
