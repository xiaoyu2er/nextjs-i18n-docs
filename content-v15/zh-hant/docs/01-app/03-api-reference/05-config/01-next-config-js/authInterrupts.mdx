`authInterrupts` 配置選項允許您在應用程式中使用 [`forbidden`](/docs/app/api-reference/functions/forbidden) 和 [`unauthorized`](/docs/app/api-reference/functions/unauthorized) API。雖然這些函式是實驗性的，但您必須在 `next.config.js` 檔案中啟用 `authInterrupts` 選項才能使用它們：

```ts filename="next.config.ts" switcher
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  experimental: {
    authInterrupts: true,
  },
}

export default nextConfig
```

```js filename="next.config.js" switcher
module.exports = {
  experimental: {
    authInterrupts: true,
  },
}
```
