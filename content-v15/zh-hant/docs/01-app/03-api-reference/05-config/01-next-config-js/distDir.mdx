您可以指定一個名稱，用作自訂構建目錄以取代 `.next`。

開啟 `next.config.js` 並新增 `distDir` 設定：

```js filename="next.config.js"
module.exports = {
  distDir: 'build',
}
```

現在如果您執行 `next build`，Next.js 將使用 `build` 取代預設的 `.next` 資料夾。

> `distDir` **不應** 離開您的專案目錄。例如，`../build` 是**無效**的目錄。
