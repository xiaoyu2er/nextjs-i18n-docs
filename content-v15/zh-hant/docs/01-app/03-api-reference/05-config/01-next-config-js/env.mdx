若要將環境變數 (environment variables) 添加到 JavaScript bundle 中，請開啟 `next.config.js` 並添加 `env` 配置：

```js filename="next.config.js"
module.exports = {
  env: {
    customKey: 'my-value',
  },
}
```

現在你可以在程式碼中訪問 `process.env.customKey`。例如：

```jsx
function Page() {
  return <h1>The value of customKey is: {process.env.customKey}</h1>
}

export default Page
```

Next.js 會在建構時將 `process.env.customKey` 替換為 `'my-value'`。由於 Webpack [DefinePlugin](https://webpack.js.org/plugins/define-plugin/) 的性質，嘗試解構 `process.env` 變數無法運作。

例如，以下程式碼列：

```jsx
return <h1>The value of customKey is: {process.env.customKey}</h1>
```

將會變成：

```jsx
return <h1>The value of customKey is: {'my-value'}</h1>
```
