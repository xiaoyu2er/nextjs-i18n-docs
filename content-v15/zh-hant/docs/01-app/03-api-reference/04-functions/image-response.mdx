title: ImageResponse 影像回應

ImageResponse 建構子允許您使用 JSX 和 CSS 生成動態影像。這對於生成社交媒體影像（例如 Open Graph 影像、Twitter 卡片等）非常有用。

## 參考

### 參數

ImageResponse 可用的參數如下：

```jsx
import { ImageResponse } from 'next/og'

new ImageResponse(
  element: ReactElement,
  options: {
    width?: number = 1200
    height?: number = 630
    emoji?: 'twemoji' | 'blobmoji' | 'noto' | 'openmoji' = 'twemoji',
    fonts?: {
      name: string,
      data: ArrayBuffer,
      weight: number,
      style: 'normal' | 'italic'
    }[]
    debug?: boolean = false

    // Options that will be passed to the HTTP response
    status?: number = 200
    statusText?: string
    headers?: Record<string, string>
  },
)
```

> [Vercel OG Playground](https://og-playground.vercel.app/) 中提供範例。

### 支援的 HTML 和 CSS 功能

`ImageResponse` 支援常見的 CSS 屬性，包括 flexbox 和絕對定位、自訂字體、文字換行、置中以及嵌套影像。

請參閱 [Satori 的文件](https://github.com/vercel/satori#css) 以獲取支援的 HTML 和 CSS 功能列表。

## 行為

- `ImageResponse` 使用 [@vercel/og](https://vercel.com/docs/concepts/functions/edge-functions/og-image-generation)、[Satori](https://github.com/vercel/satori) 和 Resvg 將 HTML 和 CSS 轉換為 PNG。
- 僅支援 flexbox 和部分 CSS 屬性。高級佈局（例如 `display: grid`）無法運作。
- 最大捆包大小 (Bundle Size) 為 `500KB`。捆包大小 (Bundle Size) 包含您的 JSX、CSS、字體、影像及其他所有資源。如果您超過限制，請考慮減少任何資源的大小或在執行時獲取。
- 僅支援 `ttf`、`otf` 和 `woff` 字體格式。為了最大化字體解析速度，建議使用 `ttf` 或 `otf` 而不是 `woff`。

## 範例

### 路由處理器

`ImageResponse` 可用於路由處理器 (Route Handlers) 來在請求時間動態生成影像。

```js filename="app/api/route.js"
import { ImageResponse } from 'next/og'

export async function GET() {
  try {
    return new ImageResponse(
      (
        <div
          style={{
            height: '100%',
            width: '100%',
            display: 'flex',
            flexDirection: 'column',
            alignItems: 'center',
            justifyContent: 'center',
            backgroundColor: 'white',
            padding: '40px',
          }}
        >
          <div
            style={{
              fontSize: 60,
              fontWeight: 'bold',
              color: 'black',
              textAlign: 'center',
            }}
          >
            Welcome to My Site
          </div>
          <div
            style={{
              fontSize: 30,
              color: '#666',
              marginTop: '20px',
            }}
          >
            Generated with Next.js ImageResponse
          </div>
        </div>
      ),
      {
        width: 1200,
        height: 630,
      }
    )
  } catch (e) {
    console.log(`${e.message}`)
    return new Response(`Failed to generate the image`, {
      status: 500,
    })
  }
}
```

### 基於檔案的元數據 (File-based Metadata)

您可以使用 `ImageResponse` 於 [`opengraph-image.tsx`](/docs/app/api-reference/file-conventions/metadata/opengraph-image) 檔案來在編譯時或在請求時間動態生成 Open Graph 影像。

```tsx filename="app/opengraph-image.tsx"
import { ImageResponse } from 'next/og'

// Image metadata
export const alt = 'My site'
export const size = {
  width: 1200,
  height: 630,
}

export const contentType = 'image/png'

// Image generation
export default async function Image() {
  return new ImageResponse(
    (
      // ImageResponse JSX element
      <div
        style={{
          fontSize: 128,
          background: 'white',
          width: '100%',
          height: '100%',
          display: 'flex',
          alignItems: 'center',
          justifyContent: 'center',
        }}
      >
        My site
      </div>
    ),
    // ImageResponse options
    {
      // For convenience, we can re-use the exported opengraph-image
      // size config to also set the ImageResponse's width and height.
      ...size,
    }
  )
}
```

### 自訂字體 (Custom fonts)

您可以透過在選項中提供 `fonts` 陣列來在 `ImageResponse` 中使用自訂字體。

```tsx filename="app/opengraph-image.tsx"
import { ImageResponse } from 'next/og'
import { readFile } from 'node:fs/promises'
import { join } from 'node:path'

// Image metadata
export const alt = 'My site'
export const size = {
  width: 1200,
  height: 630,
}

export const contentType = 'image/png'

// Image generation
export default async function Image() {
  // Font loading, process.cwd() is Next.js project directory
  const interSemiBold = await readFile(
    join(process.cwd(), 'assets/Inter-SemiBold.ttf')
  )

  return new ImageResponse(
    (
      // ...
    ),
    // ImageResponse options
    {
      // For convenience, we can re-use the exported opengraph-image
      // size config to also set the ImageResponse's width and height.
      ...size,
      fonts: [
        {
          name: 'Inter',
          data: interSemiBold,
          style: 'normal',
          weight: 400,
        },
      ],
    }
  )
}
```
