NextRequest 使用額外的便利性方法擴展了 [Web Request API](https://developer.mozilla.org/docs/Web/API/Request)。

## `cookies`

讀取或變更請求的 [`Set-Cookie`](https://developer.mozilla.org/docs/Web/HTTP/Headers/Set-Cookie) 標頭。

### `set(name, value)`

提供名稱，在請求中設定具有該值的 Cookie。

```ts
// Given incoming request /home
// Set a cookie to hide the banner
// request will have a `Set-Cookie:show-banner=false;path=/home` header
request.cookies.set('show-banner', 'false')
```

### `get(name)`

提供 Cookie 名稱，返回該 Cookie 的值。如果未找到 Cookie，則返回 `undefined`。如果找到多個 Cookie，則返回第一個。

```ts
// Given incoming request /home
// { name: 'show-banner', value: 'false', Path: '/home' }
request.cookies.get('show-banner')
```

### `getAll()`

提供 Cookie 名稱，返回該 Cookie 的值。如果未提供名稱，則返回請求中的所有 Cookie。

```ts
// Given incoming request /home
// [
//   { name: 'experiments', value: 'new-pricing-page', Path: '/home' },
//   { name: 'experiments', value: 'winter-launch', Path: '/home' },
// ]
request.cookies.getAll('experiments')
// Alternatively, get all cookies for the request
request.cookies.getAll()
```

### `delete(name)`

提供 Cookie 名稱，從請求中刪除該 Cookie。

```ts
// Returns true for deleted, false is nothing is deleted
request.cookies.delete('experiments')
```

### `has(name)`

提供 Cookie 名稱，如果請求中存在該 Cookie 則返回 `true`。

```ts
// Returns true if cookie exists, false if it does not
request.cookies.has('experiments')
```

### `clear()`

從請求中移除 `Set-Cookie` 標頭。

```ts
request.cookies.clear()
```

## `nextUrl`

使用額外的便利性方法擴展原生的 [`URL`](https://developer.mozilla.org/docs/Web/API/URL) API，包括 Next.js 特有的屬性。

```ts
// Given a request to /home, pathname is /home
request.nextUrl.pathname
// Given a request to /home?name=lee, searchParams is { 'name': 'lee' }
request.nextUrl.searchParams
```

下列選項可用：

<PagesOnly>

| 屬性          | 類型                      | 說明                                                                                                                            |
| ----------------- | ------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `basePath`        | `string`                  | 的 [基礎路徑](/docs/pages/api-reference/config/next-config-js/basePath) 的 URL。                                                  |
| `buildId`         | `string` \| `undefined`   | Next.js 應用的建構識別符。可以 [自訂](/docs/pages/api-reference/config/next-config-js/generateBuildId)。 |
| `defaultLocale`   | `string` \| `undefined`   | [國際化](/docs/pages/guides/internationalization) 預設的語言區域。                                                |
| `domainLocale`    |                           |                                                                                                                                        |
| - `defaultLocale` | `string`                  | 網域內的預設語言區域。                                                                                                    |
| - `domain`        | `string`                  | 與特定語言區域相關的網域。                                                                                          |
| - `http`          | `boolean` \| `undefined`  | 指示網域是否使用 HTTP。                                                                                                 |
| `locales`         | `string[]` \| `undefined` | 可用語言區域的陣列。                                                                                                         |
| `locale`          | `string` \| `undefined`   | 目前活躍的語言區域。                                                                                                           |
| `url`             | `URL`                     | URL 物件。                                                                                                                        |

</PagesOnly>

<AppOnly>

| 屬性       | 類型                    | 說明                                                                                                                          |
| -------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| `basePath`     | `string`                | 的 [基礎路徑](/docs/app/api-reference/config/next-config-js/basePath) 的 URL。                                                  |
| `buildId`      | `string` \| `undefined` | Next.js 應用的建構識別符。可以 [自訂](/docs/app/api-reference/config/next-config-js/generateBuildId)。 |
| `pathname`     | `string`                | 的 URL 路徑名稱。                                                                                                             |
| `searchParams` | `Object`                | 的 URL 搜尋參數。                                                                                                    |

> **注意：** Pages Router 的國際化屬性無法在 App Router 中使用。瞭解更多有關 [App Router 的國際化](/docs/app/guides/internationalization)。

</AppOnly>
