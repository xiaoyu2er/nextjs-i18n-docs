title: 標頭 (headers)

`headers` 是一個 **非同步 (async)** 函式，允許您從 [伺服器組件 (Server Component)](/docs/app/getting-started/server-and-client-components) **讀取** HTTP 請求標頭。

```tsx filename="app/page.tsx" switcher
import { headers } from 'next/headers'

export default async function Page() {
  const headersList = await headers()
  const userAgent = headersList.get('user-agent')
}
```

```jsx filename="app/page.js" switcher
import { headers } from 'next/headers'

export default async function Page() {
  const headersList = await headers()
  const userAgent = headersList.get('user-agent')
}
```

## 參考

### 參數

`headers` 不接收任何參數。

### 回傳值

`headers` 回傳一個 **唯讀** 的 [Web Headers](https://developer.mozilla.org/docs/Web/API/Headers) 物件。

- [`Headers.entries()`](https://developer.mozilla.org/docs/Web/API/Headers/entries): 回傳一個 [`迭代器 (iterator)`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Iteration_protocols)，讓您可以遍歷此物件中包含的所有鍵值對。
- [`Headers.forEach()`](https://developer.mozilla.org/docs/Web/API/Headers/forEach): 針對此 `Headers` 物件中的每一個鍵值對執行提供的函式。
- [`Headers.get()`](https://developer.mozilla.org/docs/Web/API/Headers/get): 回傳一個 [`字串 (String)`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/String) 序列，表示 `Headers` 物件中具有指定名稱的標頭的所有值。
- [`Headers.has()`](https://developer.mozilla.org/docs/Web/API/Headers/has): 回傳一個布林值，表示 `Headers` 物件是否包含某個標頭。
- [`Headers.keys()`](https://developer.mozilla.org/docs/Web/API/Headers/keys): 回傳 [`迭代器 (iterator)`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Iteration_protocols)，讓您遍歷此物件中包含的鍵值對的所有鍵。
- [`Headers.values()`](https://developer.mozilla.org/docs/Web/API/Headers/values): 回傳 [`迭代器 (iterator)`](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Iteration_protocols)，讓您遍歷此物件中包含的鍵值對的所有值。

## 小知識

- `headers` 是一個 **非同步 (async)** 函式，回傳 Promise。您必須使用 `async/await` 或 React 的 [`use`](https://react.dev/reference/react/use) 函式。
  - 在 `v14` 及更早版本中，`headers` 是一個同步函式。為了協助反向相容性，您仍可在 Next.js 15 中同步存取它，但此行為未來將被棄用。
- 由於 `headers` 為唯讀，您無法 `set` 或 `delete` 要求的出站標頭。
- `headers` 是一個 [動態 API (Dynamic API)](/docs/app/getting-started/partial-prerendering#dynamic-rendering#dynamic-apis)，其回傳值無法事先預知。使用它會將路由選擇 **[動態渲染 (dynamic rendering)](/docs/app/getting-started/partial-prerendering#dynamic-rendering)**。

## 範例

### 使用 Authorization 標頭

```jsx filename="app/page.js"
import { headers } from 'next/headers'

export default async function Page() {
  const authorization = (await headers()).get('authorization')
  const res = await fetch('...', {
    headers: { authorization }, // Forward the authorization header
  })
  const user = await res.json()

  return <h1>{user.name}</h1>
}
```
