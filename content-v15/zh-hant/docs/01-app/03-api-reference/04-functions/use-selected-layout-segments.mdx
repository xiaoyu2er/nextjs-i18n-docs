`useSelectedLayoutSegments` 是一個 **Client Component (客戶端組件)** 鉤子，可讓您讀取它被調用的 **佈局 (Layout)** 下方的活動路由段。

它適用於在父層 **佈局 (Layout)** 中建立需要知晓活動子段（例如 **麵包屑 (breadcrumbs)**）的 UI。

```tsx filename="app/example-client-component.tsx" switcher
'use client'

import { useSelectedLayoutSegments } from 'next/navigation'

export default function ExampleClientComponent() {
  const segments = useSelectedLayoutSegments()

  return (
    <ul>
      {segments.map((segment, index) => (
        <li key={index}>{segment}</li>
      ))}
    </ul>
  )
}
```

```jsx filename="app/example-client-component.js" switcher
'use client'

import { useSelectedLayoutSegments } from 'next/navigation'

export default function ExampleClientComponent() {
  const segments = useSelectedLayoutSegments()

  return (
    <ul>
      {segments.map((segment, index) => (
        <li key={index}>{segment}</li>
      ))}
    </ul>
  )
}
```

> **小知識**：
>
> -   由於 `useSelectedLayoutSegments` 是一個 [Client Component (客戶端組件)](/docs/app/getting-started/server-and-client-components) 鉤子，且 **佈局 (Layouts)** 預設為 [Server Components (伺服器組件)](/docs/app/getting-started/server-and-client-components)，因此 `useSelectedLayoutSegments` 通常透過導入到佈局中的 Client Component (客戶端組件) 來呼叫。
> -   返回的路由段 (Route Segments) 包含 [Route Groups (路由群組)](/docs/app/api-reference/file-conventions/route-groups)，您可能不希望在 UI 中包含它們。您可以使用 [`filter`](<https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/filter>) 陣列方法來移除以括號開頭的項目。

## 參數

```tsx
const segments = useSelectedLayoutSegments(parallelRoutesKey?: string)
```

`useSelectedLayoutSegments` _optionally_ 接受一個 [`parallelRoutesKey`](/docs/app/api-reference/file-conventions/parallel-routes#with-useselectedlayoutsegments)，這讓您能夠讀取該槽位內的活動路由段。

## 返回值

`useSelectedLayoutSegments` 傳回一個包含活動段的字串陣列，這些段位於該鉤子被呼叫的佈局下方一層。如果沒有一個則傳回空陣列。

例如，對於下方給出的佈局和 URL，所傳回的路由段將為：

| 佈局 (Layout)           | 已訪問的 URL          | 返回的路由段 (Route Segments) |
| ------------------------- | --------------------- | --------------------------- |
| `app/layout.js`           | `/`                   | `[]`                        |
| `app/layout.js`           | `/dashboard`          | `['dashboard']`             |
| `app/layout.js`           | `/dashboard/settings` | `['dashboard', 'settings']` |
| `app/dashboard/layout.js` | `/dashboard`          | `[]`                        |
| `app/dashboard/layout.js` | `/dashboard/settings` | `['settings']`              |
