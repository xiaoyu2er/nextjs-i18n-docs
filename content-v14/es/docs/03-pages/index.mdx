Antes de Next.js 13, el enrutador de páginas (Pages Router) era la forma principal de crear rutas en Next.js. Utilizaba un sistema de enrutamiento basado en archivos intuitivo para mapear cada archivo a una ruta. El enrutador de páginas (Pages Router) sigue siendo compatible en versiones más recientes de Next.js, pero recomendamos migrar al nuevo [Enrutador de aplicación (App Router)](/docs/app) para aprovechar las últimas características de React.

Utilice esta sección de la documentación para aplicaciones existentes que usen el enrutador de páginas (Pages Router).
