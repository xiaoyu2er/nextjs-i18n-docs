---
title: draftMode
description: draftMode 関数の API リファレンス。
---

`draftMode` 関数を使用すると、[サーバーコンポーネント (Server Component)](/docs/app/building-your-application/rendering/server-components) 内で [ドラフトモード (Draft Mode)](/docs/app/building-your-application/configuring/draft-mode) を検出できます。

```jsx filename="app/page.js"
import { draftMode } from 'next/headers'

export default function Page() {
  const { isEnabled } = draftMode()
  return (
    <main>
      <h1>My Blog Post</h1>
      <p>ドラフトモードは現在 {isEnabled ? '有効' : '無効'}</p>
    </main>
  )
}
```
