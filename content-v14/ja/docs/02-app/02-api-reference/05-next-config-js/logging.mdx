Next.jsを開発モードで実行する際に、ログレベルと完全なURLをコンソールに出力するかどうかを設定できます。

現在、`logging`は`fetch` APIを使用したデータフェッチにのみ適用されます。Next.js内部の他のログにはまだ適用されていません。

```js filename="next.config.js"
module.exports = {
  logging: {
    fetches: {
      fullUrl: true,
    },
  },
}
```
