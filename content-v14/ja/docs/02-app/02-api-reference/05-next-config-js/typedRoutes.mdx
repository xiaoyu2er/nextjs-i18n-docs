---
title: typedRoutes（実験的機能）
nav_title: typedRoutes
description: 静的に型付けされたリンクの実験的サポートを有効化します。
---

[静的に型付けされたリンク](/docs/app/building-your-application/configuring/typescript#statically-typed-links)の実験的サポートです。この機能を使用するには、App RouterとTypeScriptをプロジェクトで使用している必要があります。

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
