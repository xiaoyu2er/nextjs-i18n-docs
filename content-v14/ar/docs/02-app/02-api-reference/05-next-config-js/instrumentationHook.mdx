يسمح لك الخيار التجريبي `instrumentationHook` بإعداد أدوات القياس عبر [ملف `instrumentation`](/docs/app/api-reference/file-conventions/instrumentation) في تطبيق Next.js الخاص بك.

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
