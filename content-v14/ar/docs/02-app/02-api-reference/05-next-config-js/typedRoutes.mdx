---

دعم تجريبي للـ[روابط ذات أنواع ثابتة](/docs/app/building-your-application/configuring/typescript#statically-typed-links). هذه الميزة تتطلب استخدام موجه التطبيق (App Router) بالإضافة إلى TypeScript في مشروعك.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
