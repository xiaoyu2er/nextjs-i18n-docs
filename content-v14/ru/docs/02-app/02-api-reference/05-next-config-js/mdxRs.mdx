---
title: mdxRs
description: Использование нового компилятора Rust для компиляции MDX-файлов в App Router.
---

Для использования с `@next/mdx`. Компиляция MDX-файлов с помощью нового компилятора Rust.

```js filename="next.config.js"
const withMDX = require('@next/mdx')()

/** @type {import('next').NextConfig} */
const nextConfig = {
  pageExtensions: ['ts', 'tsx', 'mdx'],
  experimental: {
    mdxRs: true,
  },
}

module.exports = withMDX(nextConfig)
```
