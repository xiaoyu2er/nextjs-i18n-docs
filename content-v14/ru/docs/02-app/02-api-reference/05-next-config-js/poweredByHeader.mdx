По умолчанию Next.js добавляет заголовок `x-powered-by`. Чтобы отключить его, откройте файл `next.config.js` и отключите настройку `poweredByHeader`:

```js filename="next.config.js"
module.exports = {
  poweredByHeader: false,
}
```
