Next.js может автоматически транспилировать и собирать зависимости из локальных пакетов (например, монорепозиториев) или внешних зависимостей (`node_modules`). Эта функциональность заменяет пакет `next-transpile-modules`.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['@acme/ui', 'lodash-es'],
}

module.exports = nextConfig
```
