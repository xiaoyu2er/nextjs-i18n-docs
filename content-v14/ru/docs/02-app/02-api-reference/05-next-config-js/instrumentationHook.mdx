Экспериментальная опция `instrumentationHook` позволяет настроить инструментирование через файл [`instrumentation`](/docs/app/api-reference/file-conventions/instrumentation) в вашем Next.js приложении.

```js filename="next.config.js"
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
