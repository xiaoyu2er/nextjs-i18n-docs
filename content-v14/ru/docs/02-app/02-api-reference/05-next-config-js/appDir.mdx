---

> **Полезно знать**: Начиная с Next.js 13.4 эта опция **больше не требуется**. Маршрутизатор приложений (App Router) теперь стабилен.

Маршрутизатор приложений ([директория `app`](/docs/app/building-your-application/routing)) обеспечивает поддержку:
- [Макетов (layouts)](/docs/app/building-your-application/routing/pages-and-layouts)
- [Серверных компонентов (Server Components)](/docs/app/building-your-application/rendering/server-components)
- [Потоковой передачи (streaming)](/docs/app/building-your-application/routing/loading-ui-and-streaming)
- [Совмещенного получения данных (colocated data fetching)](/docs/app/building-your-application/data-fetching)

Использование директории `app` автоматически включает [Строгий режим React (React Strict Mode)](https://react.dev/reference/react/StrictMode). Узнайте, как [постепенно внедрить директорию `app`](/docs/app/building-your-application/upgrading/app-router-migration#migrating-from-pages-to-app).
