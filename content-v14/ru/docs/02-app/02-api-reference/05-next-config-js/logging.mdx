Вы можете настроить уровень логирования и отображение полного URL в консоли при работе Next.js в режиме разработки.

В настоящее время `logging` применяется только к запросам данных с использованием API `fetch`. Оно пока не применяется к другим логам внутри Next.js.

```js filename="next.config.js"
module.exports = {
  logging: {
    fetches: {
      fullUrl: true,
    },
  },
}
```
