---
title: ImageResponse
description: Справочник API для конструктора ImageResponse.
---

Конструктор `ImageResponse` позволяет генерировать динамические изображения с использованием JSX и CSS. Это полезно для создания изображений для социальных сетей, таких как Open Graph изображения, Twitter-карточки и других.

Доступные параметры для `ImageResponse`:

```jsx
import { ImageResponse } from 'next/og'

new ImageResponse(
  element: ReactElement,
  options: {
    width?: number = 1200
    height?: number = 630
    emoji?: 'twemoji' | 'blobmoji' | 'noto' | 'openmoji' = 'twemoji',
    fonts?: {
      name: string,
      data: ArrayBuffer,
      weight: number,
      style: 'normal' | 'italic'
    }[]
    debug?: boolean = false

    // Параметры, которые будут переданы в HTTP-ответ
    status?: number = 200
    statusText?: string
    headers?: Record<string, string>
  },
)
```

## Поддерживаемые CSS-свойства

Ознакомьтесь с [документацией Satori](https://github.com/vercel/satori#css) для получения списка поддерживаемых HTML- и CSS-функций.
