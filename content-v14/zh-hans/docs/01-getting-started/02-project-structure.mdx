点击文件和文件夹名称以了解更多关于每个约定的信息。

## app 路由约定

以下文件约定用于在 `app` 路由中定义路由和处理元数据。

### 路由文件

|                                                                                 |                     |                              |
| ------------------------------------------------------------------------------- | ------------------- | ---------------------------- |
| [`layout`](/docs/app/api-reference/file-conventions/layout)                     | .js .jsx .tsx | 布局 (layout) |
| [`page`](/docs/app/api-reference/file-conventions/page)                         | .js .jsx .tsx | 页面 (page) |
| [`loading`](/docs/app/api-reference/file-conventions/loading)                   | .js .jsx .tsx | 加载 UI (loading UI) |
| [`not-found`](/docs/app/api-reference/file-conventions/not-found)               | .js .jsx .tsx | 未找到 UI (not-found UI) |
| [`error`](/docs/app/api-reference/file-conventions/error)                       | .js .jsx .tsx | 错误 UI (error UI) |
| [`global-error`](/docs/app/api-reference/file-conventions/error#global-errorjs) | .js .jsx .tsx | 全局错误 UI (global-error UI) |
| [`route`](/docs/app/api-reference/file-conventions/route)                       | .js .ts | API 端点 (API endpoint) |
| [`template`](/docs/app/api-reference/file-conventions/template)                 | .js .jsx .tsx | 重新渲染布局 (template) |
| [`default`](/docs/app/api-reference/file-conventions/default)                   | .js .jsx .tsx | 并行路由后备页面 (Parallel route fallback page) |
