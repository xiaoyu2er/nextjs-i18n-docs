Next.js 提供了创建灵活的全栈 Web 应用程序的构建块。**Building Your Application** 中的指南解释了如何使用这些功能以及如何自定义应用程序的行为。

这些章节和页面按从基础到高级的顺序排列，因此您可以在构建 Next.js 应用程序时逐步跟进。然而，您可以按任何顺序阅读它们，或直接跳转到适用于您用例的页面。

<AppOnly>

如果您是 Next.js 的新手，我们建议从以下章节开始学习：[Routing](/docs/app/building-your-application/routing)、[Rendering](/docs/app/building-your-application/rendering)、[Data Fetching](/docs/app/building-your-application/data-fetching) 和 [Styling](/docs/app/building-your-application/styling)，因为它们介绍了有助于您入门的基础 Next.js 和 Web 概念。然后，您可以深入学习其他章节，例如 [Optimizing](/docs/app/building-your-application/optimizing) 和 [Configuring](/docs/app/building-your-application/configuring)。最后，当您准备好时，请查看 [Deploying](/docs/app/building-your-application/deploying) 和 [Upgrading](/docs/app/building-your-application/upgrading) 章节。

</AppOnly>
